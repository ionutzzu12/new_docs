You can add one or more users to connect to the FTP server by navigating to **System&#x20;**>**&#x20;Device Management > System Services** from the *FTP Service* section.&#x20;

**To add an FTP user:**

1. Navigate to **System&#x20;***>&#x20;***Device Management**.
2. Click the **System Services** tab.  &#x20;
3. From the *FTP Service* section, click **Add User**.
   The *Add User* dialog box appears.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/wHXYkC_kVZ1ZdK-ztqQKL_add-ftp-user.png)
4. Enter a username and click **Add User**.&#x20;
5. Click the **Copy&#x20;**&#x69;con next to the password and save it to a text file. Save the password in a safe place. You cannot use the new user account without this password.
6. Click **Close**.
