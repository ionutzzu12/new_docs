To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: GE Fanuc VersaMax Microax Micro, IC200UDR064
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/uybBXWF-ml7i-6dgALThi_ge-fanuc-snp-serial.png" size="80" width="513" height="285" position="flex-start" caption="GE Fanuc SNP Serial reference" showCaption="true" indent="2"></Image>

2. Launch Proficy Machine Edition v.80 software and then select the COM port.
3. Navigate to **Target > Go Online with Target1**.
