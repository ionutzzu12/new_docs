Breeding codes are used to describe the different types of breeding events. Whenever a breeding event is entered, the user assigns a breeding code to it. You can view all the current breeding codes from the Breeding Codes page. You can also update and add new breeding codes from this page, provided you have the right permissions.

You can find this feature under the :Link[DairyComp/MyDC menu]{label="DairyComp/MyDC menu" overridedLabel="DairyComp/MyDC menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} > :Link[Settings]{label="Settings" overridedLabel="Settings" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} option on the \{\{Buttons-UI.FeaturesSecondary}} tab.

Use the sortable **Code** column to reorder the items in the list.

<ExpandableHeading>
  ## Update Breeding Codes

  To update the description of an existing breeding code, click on the applicable code in the list. A side panel appears in which you can make your changes.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/4PvATGUBx3WBjxPB2_GF0_dc-breeding-codes-side-panel.png" size="64" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/dc-breeding-codes-side-panel.png" githubPath="Content/Resources/Images/Platform/dairycomp/dc-breeding-codes-side-panel.png" position="flex-start" showCaption="false"></Image>

  Enter the new description for the breeding code in the field provided, then click **Save**. Note that only the description can be changed; you cannot change the code letter itself.
</ExpandableHeading>

<ExpandableHeading>
  ## Add Breeding Codes

  Click the **Add New Breeding Code** button at the top of the page to open a window for creating a new breeding code.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/-NCK41U1Q-zavVCOKkK2M_dc-breeding-codes-add.png" size="45" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/dc-breeding-codes-add.png" githubPath="Content/Resources/Images/Platform/dairycomp/dc-breeding-codes-add.png" position="flex-start" showCaption="false"></Image>

  1. Enter a new value in the **Code** field.This value is mandatory and must be unique (i.e., not currently in use). Also, this value must be a single letter.:::hint\{type="info"}
     **NOTE:&#x20;**&#x41;s stated in the window, it is recommended that you not use the letters P, O, E, or R as codes, since these letters may appear in bred event as breeding results.
     \:::
  2. In the **Description** field, enter a description of the new breeding code.This field is optional.
  3. Click **Save**.Your new breeding code now appears in the list.
</ExpandableHeading>

<ExpandableHeading>
  ## Delete Breeding Codes

  1. Select the applicable breeding code from the list to open its side panel.
  2. In the side panel, click **Delete**.
  3. In the confirmation window that appears, click **Yes**.A message appears at the bottom of the page indicating that the breeding code has been deleted.
</ExpandableHeading>
