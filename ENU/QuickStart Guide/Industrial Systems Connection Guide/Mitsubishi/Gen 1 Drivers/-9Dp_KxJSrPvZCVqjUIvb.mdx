To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: Mitsubishi FX5U-32M
   - **Communication I/F**: RS485. Refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/U58spZmOnhdKtSHmuUYF__mitsubishi-fx5u-gx-reference.png" size="80" width="874" height="848" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch the GX Work3-V1.035M software, and then click **Online > Read from PLC**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/IAfpmxbpAyighIJKb6qqs_mitsubishi-fx5u-gx-online-read482x338.png" size="80" width="482" height="338" position="flex-start" caption="Read from PLC option" showCaption="true" indent="2"></Image>

3. Select series.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/i370u2PB5LmGW37CBt_ZN_mitsubishi-fx5u-serial-series.png" size="80" width="745" height="401" position="flex-start" caption="Series selection drop-down list" showCaption="true" indent="2"></Image>

4. Select connection destination.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/2TkGj96zd91C12dOJU4_2_mitsubishi-fx5u-serial-connection.png" size="80" width="825" height="467" position="flex-start" caption="Select Connection Destination dialog box" showCaption="true" indent="2"></Image>
