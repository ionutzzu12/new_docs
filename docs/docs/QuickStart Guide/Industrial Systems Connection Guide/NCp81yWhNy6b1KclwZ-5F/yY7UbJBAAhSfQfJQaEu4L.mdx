To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

Refer to the following parameters.&#x20;

**Online Simulator**: YES

**Broadcast command**: NO

**Extend address mode**: NO

**To configure the device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: Hitachi EH-D20DT
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/3dPbbaAimnqhvA039NkU0_hitachiserialreference.png" size="80" width="370" height="461" position="flex-start" caption="Hitachi Serial reference" showCaption="true" indent="2"></Image>

2. Use PLC software on your computer to configure your device for Ethernet access.&#x20;
