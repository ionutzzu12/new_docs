The **Disposal Codes** page allows you to view all available disposal codes for your \{\{masterVAS-Terminology.d-airy}}. Whenever a disposal event is entered, a disposal code is assigned to it.\{\{masterVAS-Products.DCU-users-short}} can also edit the code descriptions and add new disposal codes if they have the required permissions.

<hint type="warning">
  **IMPORTANT:&#x20;**&#x55;se caution when changing disposal codes, as some are commonly used in the \{\{masterVAS-Terminology.d-airy}} industry and linked to processing centers that computerize \{\{masterVAS-Terminology.d-airy}} data.
</hint>

<ExpandableHeading>
  ## Add Disposal Codes

  1. Click the **Add New Disposal Code** button at the top of the page to open a window for creating a new disposal code.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/kzVhUc2fH4TiMbg9X9y_r_dc-disposal-add.png" alt caption size="41"}
  2. Enter a new value in the **Code** field.This value is mandatory and must be unique (i.e., not currently in use). Also, this value must be a number.
  3. In the **Description** field, enter a description of the new disposal code.This field is optional.
  4. Click **Save** to add your new disposal code to the list.
</ExpandableHeading>

<ExpandableHeading>
  ## Edit Disposal Codes

  Only the description can be changed; you cannot change the code number. To edit the description of an existing disposal code, click on the applicable code in the list. Use the side panel to make changes to the code description, and then click **Save**.
