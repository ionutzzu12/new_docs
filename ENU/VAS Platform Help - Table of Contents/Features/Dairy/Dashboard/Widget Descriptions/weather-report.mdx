  ### How do I add this widget to my Dashboard or Overview Page?

  To add this widget to your \{\{Buttons-UI.Dashboard}} or \{\{masterVAS-Terminology.O-verview-}} page, check the box for it under the **General** category in the [Page Customization](../customize-dashboards.mdx) options.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/sFMJnllZdVhvHV3cfIpFD_general-widgets.png" size="22" isUploading="false" initialPath="../../../../Resources/Images/Platform/dashboard/general-widgets.png" githubPath="Content/Resources/Images/Platform/dashboard/general-widgets.png" position="flex-start" showCaption="false"></Image>
</ExpandableHeading>

The \{\{Widgets.WeatherReport}} widget shown below allows you to check the forecast at the \{\{masterVAS-Terminology.d-airy}} location right from within \{\{masterVAS-Products.Pulse-only}}. The widget uses the \{\{masterVAS-Terminology.d-airy}} address that is defined on the [Settings (Dairy)](../../settings-dairy.mdx) page.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/YAOLRSV0rxhmfcyIrEtn8_weather-widget-full.png" size="56" isUploading="false" initialPath="../../../../Resources/Images/Platform/dashboard/weather-widget-full.png" githubPath="Content/Resources/Images/Platform/dashboard/weather-widget-full.png" position="flex-start" showCaption="false"></Image>

When customizing this widget on your Dashboard or \{\{masterVAS-Terminology.O-verview-}} page, note that the full page width setting displays the additional **Current Conditions** information shown the example above.
