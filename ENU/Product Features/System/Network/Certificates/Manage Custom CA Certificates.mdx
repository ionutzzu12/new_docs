You can manage custom CA certificates by navigating to **System&#x20;**> **Network&#x20;**> **Certificates**.&#x20;

Click the **Action menu&#x20;**&#x66;or a certificate to manage it.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/j5HLBccwdEKsI-NXYR8Mv_image.png "Action menu for a custom certificate")

- **Remove Certificate**
- **Show Details**: Review the following details.&#x20;
  - **Issuer**: The entity issuing the certificate
  - **Not After**: The end date for the certificate
  - **Not Before**: The begin date for the certificate
  - **Subject**: The entity for which the certificate is issued
  - **Signature Algorithm**: The algorithm type used
  - **Trust Chain**: Verification that the sender and all intermediate certificates are trustworthy
