Puede acceder y enviar \{\{FRS\_VARS.Request\_Offering-Solicitar-oferta}} a través de ofertas externas haciendo clic en una URL de un sitio web externo. Puede ver y solicitar servicios, realizar un seguimiento de su solicitud utilizando ID de usuario y de oferta en la URL, y elegir cómo se muestra la solicitud, ya sea en una ventana nueva, dentro de \{\{FRS\_VARS.ProductName-Neurons-para-ITSM}} o dentro del **Catálogo de servicios**.

Para utilizar las ofertas externas creadas por su administrador, siga estos pasos:

1. Inicie sesión en \{\{FRS\_VARS.ProductName-Neurons-para-ITSM}}.
2. Desde el espacio de trabajo **Catálogo de servicios**, busque el registro de la oferta externa utilizando la barra de búsqueda.

<Paragraph indent="1">
  Se muestran los registros de ofertas externas.
</Paragraph>

3. Haga clic en una tarjeta de oferta externa.

<Paragraph indent="1">
  Un cuadro de diálogo le preguntará si desea continuar o permanecer en la página.
</Paragraph>

4. Si selecciona **Continuar**, la URL se abrirá en una nueva pestaña con su ID de usuario y su ID de oferta incluidos.
