Before you can start using your task lists, use Task Builder to generate a custom \{\{task list}}.

To build a task list from a web browser, select \{\{Buttons-UI.FeaturesMain}} > \{\{Buttons-UI.FeaturesSecondary}} > \{\{Buttons-UI.FeaturesTertiary}}.

Follow the guidelines below to select your task list settings:

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/rdPUgatZo6-YidWFuX8ZT_build-task-list-mydc.png" size="56" isUploading="false" initialPath="../../../../Resources/Images/Platform/tasks/build-task-list-mydc.png" githubPath="Content/Resources/Images/Platform/tasks/build-task-list-mydc.png" position="flex-start" showCaption="false"></Image>

1. Task Type: Select **Vet** or **ToDo**.
2. Task List Name: Enter a name for your list.
3. Task Name: If you created a ToDo task list, enter a name for the task itself. If you created a Vet task list, the system automatically sets the **Task Name** to `VETC Reason for vet check`.
4. Required: Check the box if the task is required.
5. Select Events allowed (ToDo only): If you want to make event selection faster when you create a ToDo task list, you can limit the events that can be added to an animal upon task completion. Check the **Select Events allowed** box and select the events from the drop-down list.
6. Click **Next**.

Follow the guidelines below to build your task list:

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/aaZOUh250OBX-okU3PyQh_build-task-list-mydc-2.png" size="55" isUploading="false" initialPath="../../../../Resources/Images/Platform/tasks/build-task-list-mydc-2.png" githubPath="Content/Resources/Images/Platform/tasks/build-task-list-mydc-2.png" position="flex-start" showCaption="false"></Image>

For example, searching for "\{\{masterVAS-Terminology.s-ire}}" finds all associated items in your \{\{masterVAS-Terminology.d-airy}}: \{\{LSIR}}, MGSIR, \{\{PSIRC}}, \{\{SIR1}}, etc. Reorder the columns by dragging and dropping the item names

    or click the **x** next to the item name to delete it.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Task List Output: To add items you want to see in the list, drag and drop items from the items list on the left to the **Task List Output** field on the right.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Filters: To set filters, drag and drop items from the items list into the **Filters** area, and then choose an **Operator** from the drop-down and set a **Value** for the operator.
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Sorting: To customize how items are sorted on your report, drag and drop items from the items list into the **Sorting** area, and then choose the sort order:
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    - A - Z: Ascending sort (low to high, earliest to latest)
    - Z - A: Descending sort (high to low, latest to earliest)
      Click **Save and Create** to build your list.
  </WorkflowBlockItem>
</WorkflowBlock>
