## PULSE Update - April 2025

ENHANCEMENT\{\{masterVAS-Products.DCUnlimited-short}} ONLY\{\{masterVAS-Integrations.Cowfile}} sharing for select Integration \{\{Partners.PartnersUIoption}}! \{\{General.CompanyName-short}} now offers \{\{General.DCUnlimited}}\{\{masterVAS-Terminology.d-airies}} the option to quickly and easily share their most recent \{\{masterVAS-Products.DC305-short}}\{\{masterVAS-Terminology.c-owfile}} backup with select \{\{Partners.PartnerName}}s using the [Integrations Gallery](../Features/partners/integrations-gallery.mdx) in \{\{masterVAS-Products.Pulse-only}}. This automatically provides access to your \{\{masterVAS-Terminology.h-erd}} data to simplify file sharing when needed — no need to download or upload your \{\{masterVAS-Terminology.c-owfiles}} manually! If at any point you decide to stop sharing your latest \{\{masterVAS-Terminology.c-owfile}}, you can turn off the connection with a single click. You always have full control over who can access your \{\{masterVAS-Terminology.h-erd}} data!

The following partners now support \{\{masterVAS-Terminology.c-owfile}} sharing:

- [AgriTech Analytics for DC305](<../../../VAS Platform Help - Table of Contents/Features/Partners/Herd Analytics/Cowfile Backup Sharing/agritechanalytics-info.mdx>)
- [Minnesota DHIA](<../../../VAS Platform Help - Table of Contents/Features/Partners/Herd Analytics/Cowfile Backup Sharing/minndhia-info.mdx>)
- [Select Sires](<../../../VAS Platform Help - Table of Contents/Features/Partners/Herd Analytics/Cowfile Backup Sharing/selectsires-info.mdx>)

To get started, connect the integration in the [Integrations Gallery](../Features/partners/integrations-gallery.mdx) in \{\{masterVAS-Products.Pulse-only}}. Then see [Activate Cowfile Backup Sharing](<../../ENU/VAS Platform Help - Table of Contents/Features/partners/herd-analytics/activate-cowfile-backup-sharing.mdx>) to learn more about connecting and using the \{\{masterVAS-Products.DC305-short}}\{\{masterVAS-Integrations.Cowfile}} feature.
