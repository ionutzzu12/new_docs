You can create Flow Groups by navigating to **Analytics&#x20;**>**&#x20;Instances**. Groups serve as containers for analytics flows. A group can contain a single flow or multiple flows, typically related to each other in some way.

<hint type="info">
  **Note**: An empty group cannot be deleted from the list of available groups. The only way to delete empty groups is to do a system reboot.
</hint>

**To create a flow group:**

1. Navigate to **Analytics&#x20;***>&#x20;***Instances**.&#x20;
   The *Instances&#x20;*&#x70;ane appears.&#x20;
2. Click the **Create group** icon.&#x20;
   The *Create Group&#x20;*&#x64;ialog box appears.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/v11fNBV-76QUHCq_2FUqC_creategroupicon.png" size="80" width="867" height="458" caption="Create group icon" position="center" showCaption="true"></Image>
3. Enter a group name and click **Create**.&#x20;

You can then select the appropriate flow group from the drop-down list.&#x20;
