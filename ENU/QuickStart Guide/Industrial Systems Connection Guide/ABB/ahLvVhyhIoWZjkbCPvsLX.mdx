- Step 1: Set up and configure the PLC device
- Step 2: Set up the PLC device in  DeviceHub

Find the list of registers in the :Link[TotalFlow XSeries Remote Controller User's Manual]{href="https://library.e.abb.com/public/f767e1b1766dfa26c1257b1a005b0e6b/XRC%20Manual%20AA.pdf" newTab="true" hasDisabledNofollow="false"}.
