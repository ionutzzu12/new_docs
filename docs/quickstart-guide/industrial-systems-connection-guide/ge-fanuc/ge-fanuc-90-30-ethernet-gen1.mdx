- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, use Otherwise, use PLC software on your computer to configure the device for Ethernet access.&#x20;

Refer to the following device details:

- Device: GE Fanuc 90-30 IC693CPU363
- Device type (driver): GE 90-30 Series (Ethernet)
- I/F: Ethernet
