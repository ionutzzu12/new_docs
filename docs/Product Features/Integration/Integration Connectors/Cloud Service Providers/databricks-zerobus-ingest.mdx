</hint>

Databricks integration enables Edge System to connect with Databricks via Zerobus Ingest. For more information, see the :Link[Databricks Zerobus Ingest Integration Guide]{label="Databricks Zerobus Ingest Integration Guide" spaceId="0c3K9AkyXsnPI4j8vUaa5" docId="igrvvLyJdHguwAVX9zlpQ" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.

## Supported Communication

- Outbound (Write from Edge to Databricks): Yes
- Inbound (Read from Databricks to Edge): No
