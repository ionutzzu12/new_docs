You can add a processor to an existing Analytics flow by navigating to **Analytics&#x20;**>**&#x20;Instances Table**.&#x20;

# Add an Input or Output Processor from the Instances Table

**To add a processor from the Instances Table:&#x20;**

1. Navigate to **Analytics&#x20;***>&#x20;***Instances Table**.&#x20;
2. Click the **Add +** icon for an existing respective input or output processor.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/1QoUDAA8rTFx7gVta6NMo_addprocessorinstancestable.png" size="80" width="1547" height="405" caption="Add processor icon" position="center" showCaption="true"></Image>
3. From the add processor menu, select one of the following.&#x20;
   - **New processor**: Select **Events&#x20;**&#x6F;r **Values**, and then select the appropriate input or output processor. See [Input Processors](<../../../docs/Product Features/4B5Sgl13RYerzVbfE9zCC/input-processors.mdx>) and :Link[Output Processors]{label="Output Processors" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="OWy76JTGh_Nglz3ySUF2O" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.&#x20;
     - Events (individual data source): The receiving processor reacts to each individual data value immediately.&#x20;
     - Values (combined data sources): The function processor waits for values from all of the combined data sources before reacting.&#x20;
   - **Existing processor in group**: Select an existing input or output processor and then select **Events&#x20;**&#x6F;r **Values**.&#x20;
     - Events (individual data source): The receiving processor reacts to each individual data value immediately.&#x20;
     - Values (combined data sources): The function processor waits for values from all of the combined data sources before reacting.&#x20;

After saving the update, the new processor is connected to the selected input or output processor.&#x20;

# Add a Function Processor from the Instances Table

You can connect multiple functions from existing input and output processors by navigating to **Analytics > Instances Table**.&#x20;

Connecting from an input processor in an analytics flow places the selected flow's function processor in the *Outputs&#x20;*&#x63;olumn of the selected flow. Connecting from an output processor in a flow places the selected flow's function processor in the *Inputs&#x20;*&#x63;olumn of the selected flow.

When you view the flows in the *Instances&#x20;*&#x70;ane, the two functions are connected.

**To connect a function processor to a flow:**

1. Navigate to **Analytics&#x20;***>&#x20;***Instances Table**.&#x20;
2. Click the **Add +** icon for an existing respective input or output processor.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/PUmwy_NBxhCxKNY0zlbf8_addprocessorinstancestable.png" size="80" width="1547" height="405" caption="Add processor icon" position="center" showCaption="true"></Image>
3. From the add processor menu, select one of the following.&#x20;
   - **New processor**: Select **Events&#x20;**&#x6F;r **Values**, and then select the appropriate function processor. See [Input Processors](<../../../docs/Product Features/4B5Sgl13RYerzVbfE9zCC/input-processors.mdx>) and :Link[Output Processors]{label="Output Processors" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="OWy76JTGh_Nglz3ySUF2O" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.&#x20;
     - Events (individual data source): The receiving processor reacts to each individual data value immediately.&#x20;
     - Values (combined data sources): The function processor waits for values from all of the combined data sources before reacting.&#x20;
   - **Existing processor in group**: Select an existing function processor and then select **Events&#x20;**&#x6F;r **Values**.&#x20;
     - Events (individual data source): The receiving processor reacts to each individual data value immediately.&#x20;
     - Values (combined data sources): The function processor waits for values from all of the combined data sources before reacting.&#x20;

After saving the update, the function is added to either the inputs or outputs column to the flow.&#x20;
