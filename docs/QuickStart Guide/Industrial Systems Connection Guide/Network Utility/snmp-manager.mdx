To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the Device

Find your SNMP agent's configuration and note down its settings.

# Step 2: Set up the Device in Litmus Edge DeviceHub

## Upload required MIB files

<hint type="info">
  See [SNMP Manager Reference](<../../../docs/QuickStart Guide/Industrial Systems Connection Guide/Network Utility/snmp-manager/snmp-manager-reference.mdx>) to review the MIB files which are already included in the driver.
</hint>

1. Enable the Litmus Edge FTP server and add a user. See [Manage FTP and System Services](<../../../docs/docs/Product Features/System/HFXGt4JuZ4d5xfoGvq9tl/IwQ7f8NSzF3VzctmDBgvd/TYxrVCP29FtxbvLZePcxg.mdx>) for more information.
2. Create a folder in the devicehub/ folder and note down the name of the folder.
3. Upload the required MIB files to the folder you created.

If you set **SNMP Version&#x20;**&#x74;o **V1&#x20;**&#x6F;r **V2C**, enter the value for the **Community String&#x20;**&#x70;arameter based on your SNMP agent's settings.

If you set the **SNMP Version&#x20;**&#x74;o **V3**, all the parameters will have values selected based on your SNMP agent's settings. The following parameters will be available:

- Username
- Context Name
- Authentication Protocol
- Authentication Passphrase
- Privacy Protocol
- Privacy Passphrase
