When you [Connect a Device](<Connect a Device.mdx>), you may have the option to use the Asse&#x74;*&#x20;Discovery* tool.&#x20;

# Overview

When you use Asset *Discovery&#x20;*&#x77;hen configuring a device, it searches for devices on the network that are specific to the device protocol and respective driver. After searching for and selecting the device that is discovered on the network, certain parameters are pre-configured that allow the device to be added to Litmus Edge.&#x20;

To search for any device on the network, use the [Device Discovery (Network)](<././Asset Discovery.md>) feature. This feature is not specific to device protocols or devices.&#x20;

# Supported Protocols and Drivers

This feature is only available for the following device protocols:

- BACnet (available for the [BacnetIP](<././././QuickStart Guide/Industrial Systems Connection Guide/BACnet/BacnetIP.md>) driver)
- OPC UA (only available for the [OPC UA Client Advanced](<././././QuickStart Guide/Industrial Systems Connection Guide/OPC UA/OPC UA Client Advanced.md>) driver)
