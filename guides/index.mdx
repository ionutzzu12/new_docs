Archbee CLI lets you preview, validate, and manage documentation locally. Write in Markdown or MDX, choose from 4 templates, and see changes instantly with hot reload.

blabla

## Get Started

```bash
npm install -g @archbee/cli
archbee new my-docs
cd my-docs
archbee dev
```
