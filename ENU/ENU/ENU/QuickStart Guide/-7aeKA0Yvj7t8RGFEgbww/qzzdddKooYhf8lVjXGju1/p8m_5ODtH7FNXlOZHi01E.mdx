To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: LS XBM-DR165
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Z12vA1LDvmBuvrKLc5lcf_ls-xbm-serial-reference.png" size="80" width="398" height="861" position="flex-start" caption="LS XBM Serial reference" showCaption="true" indent="2"></Image>

2. Launch XG5000 v.3.71 software and click **Online > Online Settings**.
3. Set up COM port, and then click **Auto Detect**.
