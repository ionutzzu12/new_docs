\{\{masterVAS-Products.Pulse-only}} provides a default set of \{\{Worklists.HospitalListSingular}} event shortcuts and sound files. In the current \{\{masterVAS-Products.Pulse-only}} release, you can customize up to 7 event shortcuts by using \{\{masterVAS-Products.Pulse-only}}\{\{Buttons-UI.FeaturesSecondary}} page. Hospital list sounds are not currently customizable. See [Tasks & Worklist Settings](<../../../Settings/Health Settings/tasks-and-worklist-settings.mdx>) for instructions on customizing event shortcuts for your \{\{Worklists.HospListCommonUse}}s.

All other \{\{Worklists.HospListCommonUse}}\{\{masterVAS-Terminology.c-onfiguration}} steps are done in \{\{DC-UI.DC-ProdNameShort}}. See [Create Hospital Lists](create-hospital-lists.mdx) to learn more.
