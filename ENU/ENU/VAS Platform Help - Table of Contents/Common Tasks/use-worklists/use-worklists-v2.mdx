<ExpandableHeading>
  ### Required Permissions

  Users must have \{\{masterVAS-Terminology.H-erd-}} \{\{masterVAS-Terminology.R-ead-}} permissions to view \{\{Mobile-UI.ListsTab}} lists, and users must have \{\{masterVAS-Terminology.H-erd-}} \{\{masterVAS-Terminology.W-rite-}} permissions to update \{\{Mobile-UI.ListsTab}} lists. See [Edit Dairy User Permissions](../Features/Config/UserMgmtRoles/EditUserPermissions.htm) if you need help updating user permissions.
</ExpandableHeading>

Worklists include animals that need to be seen by your \{\{masterVAS-Terminology.d-airy}} workers or animal health experts for a variety of reasons, such as performing pregnancy checks or treating animals in a hospital \{\{masterVAS-Terminology.p-en}}. Follow the steps below when checking or treating animals by \{\{Worklists.worklistCommonUse}} type:
