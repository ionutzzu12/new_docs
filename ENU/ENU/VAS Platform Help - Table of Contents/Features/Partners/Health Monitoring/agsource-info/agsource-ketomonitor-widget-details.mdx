## Graph

The graph displays \{\{ketosis}} test data for animals in early lactation, making a distinction between the early \{\{masterVAS-Terminology.f-resh-cow}} period (5 - 11 \{\{DIM}}) and the general \{\{masterVAS-Terminology.f-resh-cow}} period (5 - 20 DIM). The graph displays test results for the previous 6 months, and includes a target ketosis prevalence level line to quickly gauge whether the results are at or near expected levels.

Ketosis is predicted to occur in 5% of \{\{masterVAS-Terminology.f-resh-cow}} animals in their 1st lactation and 15% of animals in later lactations, for an average of 10% for all \{\{masterVAS-Terminology.f-resh-cow}} animals.

### 1st/2nd Lactation

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/5vK_DK_UguGU4AbvfZfuv_ketowidget-fullscreen-1st-2nd-selected.png" size="78" isUploading="false" initialPath="../../Resources/Images/Partners/Keto_widget-fullscreen-1st-2nd-selected.png" githubPath="Content/Resources/Images/Partners/Keto_widget-fullscreen-1st-2nd-selected.png" position="flex-start" showCaption="false"></Image>

1. **References:&#x20;**&#x53;elect to view ketosis test data for All Animals, or by 1st/2nd+ Lactation.
2. **Key:&#x20;**&#x49;dentifies the data shown:\* When All Animals are selected, the graph displays data for all \{\{masterVAS-Terminology.f-resh-cow}} animals (wide bar) and early \{\{masterVAS-Terminology.f-resh-cow}} animals (narrow bar), as well as the 10% target line.
   - When 1st and/or 2nd+ Lactation is selected, the graph displays data for the lactation(s), as well as the 5% (1st lactation) and/or 15% (2nd+ lactation) target lines.
3. **Tooltip:&#x20;**&#x48;over over the graph data to view data details for that month.
4. **Show More Info:&#x20;**&#x53;elect this option to view 12 months of data below the graph in a table format.
5. **Updated:&#x20;**&#x54;imestamp (date and time) of the last data sync from AgSource.

## More Info Table

The table area below the graph provides ketosis test data for the last 12 months.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/1qOjfteQuRug_rC9Ztn3y_ketowidget-fullscreen-more-info-only.png" size="80" isUploading="false" initialPath="../../Resources/Images/Partners/keto_widget-fullscreen-more-info-only.png" githubPath="Content/Resources/Images/Partners/keto_widget-fullscreen-more-info-only.png" position="flex-start" showCaption="false"></Image>

1. **Animals:&#x20;**&#x49;dentifies the animals for which data is shown:\* All Animals: All \{\{masterVAS-Terminology.f-resh-cow}} animals (5-20 DIM)
   - Early \{\{masterVAS-Terminology.F-resh-cow-}} (5-11 DIM): Animals 5-11 DIM 
   - 1st \{\{Lact}}: Animals in their first lactation
   - 2nd+ Lact: Animals in their second (or later) lactation
2. **Updated:&#x20;**&#x54;imestamp (date and time) of the last data sync from AgSource.
3. **Ketosis Data:&#x20;**&#x54;he Predicted Ketosis/Tested Animals data is presented in total number of animals and as a percentage.
4. **Hide Info:&#x20;**&#x53;elect this option to collapse the table.
