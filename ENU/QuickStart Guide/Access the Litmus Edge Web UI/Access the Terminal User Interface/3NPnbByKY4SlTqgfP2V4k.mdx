In some cases, you may need to use the terminal user interface (TUI) as you may have no direct network access to the device. In this case, you can connect locally to the device using a keyboard, mouse, and monitor. This will allow you to perform the required network setup to make the device accessible through the Web UI.

Using the TUI on the gateway device makes it possible to configure Litmus Edge if network access is not possible and you have physical access to the device.

# Connect a Keyboard and Monitor

The Gateway device has back-panel connectors for the monitor and a keyboard. These allow you to access the configuration options of the installed Litmus Edge operating system.

You can:

- Connect a keyboard (for example, a USB connection).
- Connect a monitor (for example, an HDMI, Micro HDMI, or VGA connection).

# Log in to the TUI after Initial Installation

When the system startup is complete, the TUI welcome screen displays the IP addresses and network interface settings. See the section below, *Navigating the TUI*, for tips on using the keyboard to navigate.

The eth0 Ethernet interface is required on first boot.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/PGJBXDQvvt0pS53OSwQu8_le-tui-bootup-v3150.png" size="80" width="872" height="290" caption="Litmus Edge Terminal with IP Address" position="center" showCaption="true"></Image>

**To log in to the TUI after the initial installation:**

1. Press **F2** to enter the setup.
2. When logging in for the first time, you need to select the keyboard layout. The *us* layout is sufficient for most English keyboards. To select another layout, scroll up/down the list using the up/down arrows on your keyboard.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/BNURNY7A1jxyXmWgBmKo__01-r4l7y6hhvvvrocarm40ttuikeyboard.png" size="80" width="765" height="310" caption="Keyboard selection" position="center" showCaption="true"></Image>
3. Press **Enter** to apply the layout and display the login screen. If this is the first time anyone is logging in to this edge device, the credentials are: `admin` / `litmusedge`.&#x20;
4. Enter your credentials. If this is the first time anyone is logging in to this edge device, the credentials are: `admin` / `litmusedge`. Use the **Tab&#x20;**&#x6B;ey to move fields.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/tFb7jGQ7P4jIChnHCCkrr_02-4m3cv6nvxc3h8grzolvotuilogin.png" size="80" width="432" height="201" caption="Login screen" position="center" showCaption="true"></Image>
5. Tab to *Log In* and press **Enter**.

If this is the first time logging in to Litmus Edge, refer to the following *Log in to Litmus Edge for the First Time* section.&#x20;

# Navigating the TUI

- The TUI logs you out if no activity occurs within a few minutes.
- To log in again, press **F2**.

The TUI is limited to keyboard input. Here are some tips for navigating the TUI with the keyboard:

- **F2**: Enter setup to configure settings.
- **F5**: Refresh the screen.
- **Tab**: Move the cursor from one field to the next.
- **Arrows keys**:
  - Move the cursor to select an item in a list, such as the TUI Configuration Options.
  - Use the **Arrow keys** to select an item, tab to **Next** at the bottom of the screen, then press **Enter** to advance to the screen of the selected option.
- **Enter**:
  - Accept the data and move to the next field.
  - Submit the form and display the next screen in the workflow.

## Example of TUI Navigation

When selecting an option:

- Use the **Arrow keys** to move through the list of TUI Configuration Options.
- Use the **Tab&#xA0;**&#x6B;ey to select the item highlighted by the cursor. The cursor moves to the *Next&#x20;*&#x62;utton.
- Press **Enter** when the cursor is on the *Next&#x20;*&#x62;utton to advance to the screen of the selected option.

# Configure Litmus Edge Using the TUI

The TUI has many of the same options that are available through the web browser interface.

Many of these settings need to be set when first deploying a device. Several options can be set later.

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/yvtMpgxLIu6p5TLYEp_4v_image.png)

## Configure Interfaces

Select the interface configuration type and enter the details.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/vtp6y8K5nLJDrgzYV_MzL_loopedgetuiconfigureinterface-1.png" size="80" width="400" height="155" caption="Litmus Edge Terminal Interface Configurations" position="center" showCaption="true"></Image>

**In the following example, you can set a static IP address by providing the following information**:

- **CIDR**: This Classless Inter-Domain Routing (CIDR) notation identifies a block of IP addresses, similar to network subnetting. This example of CIDR notation (`192.168.0.15/24`) means that the first 24 bits of the IP address are used for network routing.
- **Gateway**: The default gateway address on a network can be found using the `ipconfig` command.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/F3s2WX4XK4I2GnRbcmZ-8_tuiinterfaces.png" size="80" width="444" height="264" caption="Litmus Edge Terminal Interface CIDR/Gateway Configuration" position="center" showCaption="true"></Image>

## Configure DNS

Use this dialog to enter space-separated IP addresses of the DNS servers.

- At least one server is required.
- If internet access is enabled, only the first three servers are used.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/93mjWnIAnhjuXen7xLYO-_tuidns.png" size="80" width="750" height="213" caption="Litmus Edge Terminal DNS configuration" position="center" showCaption="true"></Image>

## Configure Docker Subnet

Enter the default subnet for Docker and select **Apply**.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/QinUG4bqQltaHYQwbiJw1_loopedgetuidockersubnet.png" size="80" width="482" height="185" caption="Litmus Edge Terminal Docket Subnet configuration" position="center" showCaption="true"></Image>

## Configure Time

Network Time is the recommended configuration.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/i80K5jrP14fUL2ZdfJtQu_tuitime.png" size="80" width="761" height="216" caption="Litmus Edge Terminal Time Configuration" position="center" showCaption="true"></Image>

The validity period of the activation code is usually several minutes. See :Link[Activate a License]{docId="sXtSfffbJ868Nym8aDvbn" label="Activate a License" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId overridedLabel="Activate a License"} for more information.

## Reboot

**If you decide to not reboot:**

Use the arrow keys to select **Cancel**, then press **Enter**.

The operating system reboots and returns to the initial screen where you can once again press **F2** to log in, then choose TUI Configuration Options.
