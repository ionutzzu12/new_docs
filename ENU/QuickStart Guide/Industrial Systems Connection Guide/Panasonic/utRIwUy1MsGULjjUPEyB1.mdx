To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it as needed.&#x20;
2. **Device**: Panasonic FP Ethernet FP2SH-60K
3. **Communication I/**&#x46;: Ethernet. Refer to the following figure.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/FT5_9hrzag3AGMidO9qZs_panasonic-fp-ethernet.png" size="100" width="817" height="361" caption="Reference image of device" position="center" showCaption="true" indent="2"></Image>

4. Launch Control FPWIN Pro7 v7.2.3.0 software, and then click **Online > Communication parameters**.
5. Set up communication parameters, and then click **Connect**.
