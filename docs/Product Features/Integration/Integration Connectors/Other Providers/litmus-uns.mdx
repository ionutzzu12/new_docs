## Overview

The Litmus UNS integration enables an Edge System to connect to the Litmus MQTT broker.

## Supported Versions

Compatible with MQTT Protocol version 3.1.0 and MQTT Protocol version 3.1.1

## Supported Communication

- Outbound (Publish data from Edge to Litmus MQTT Broker) - Yes
- Inbound (Subscribe data from Litmus MQTT Broker to Edge) - Yes

## Configurations

| **Variable**           | **Detail**                                                                                   |
| ---------------------- | -------------------------------------------------------------------------------------------- |
| Name                   | This field specifies a unique name identifier for the connector.                             |
| Parallel Publish Count | This setting determines the maximum number of messages that can be published simultaneously. |
| MQTT Client ID         | An identifier unique to each client that connects to the MQTT broker.                        |
| UNS Hierarchy          | Defines the organizational structure for which data will be categorized and accessed.        |
| Litmus UNS Token       | A Base64-encoded, one-time token used for secure access to the Litmus UNS endpoint.          |

<hint type="warning">
  **Important:&#x20;**&#x43;ustom topics are not supported within the UNS connectors.
</hint>
