<ExpandableHeading>
  ### Required Permissions

  Only users who are assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions can use this feature. See [Edit Dairy User Permissions](EditUserPermissions.htm) and [Manage Dairy Roles](ManageRoles.htm) for more information.
</ExpandableHeading>

This topic explains how to remove a user from a \{\{masterVAS-Terminology.d-airy}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/qJQmuUbqTNvRqAbtydA5w_user-remove-dairy.png" size="43" isUploading="false" initialPath="../../../../Resources/Images/Platform/users-roles/user-remove-dairy.png" githubPath="Content/Resources/Images/Platform/users-roles/user-remove-dairy.png" position="flex-start" showCaption="false"></Image>

You can find this feature under the :Link[Dairy menu]{label="Dairy menu" overridedLabel="Dairy menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-Dairy" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-Dairy" href="../../../NavPlatform/Menu.htm#Menu-Dairy"}. If you need to remove a user from a company, see :Link[Remove a Company User]{label="Remove a Company User" overridedLabel="Remove a Company User" spaceId docId="ZBxNM9nH4pjlw1ThsUy51" version="v2" docAnchorId="#Remove_a_Company_User" loadingMethod="dynamic" newTab="false" githubPath="../../Company/CompanyUserMgmt.htm#Remove_a_Company_User" href="../../Company/CompanyUserMgmt.htm#Remove_a_Company_User"}.

1. In the **Users** list on the **Users** tab, expand the details of the \{\{masterVAS-Terminology.d-airy}} user that you want to remove.
2. Click the **Remove User** button at the bottom left side of the page.

Once these steps are complete, the user's \{\{masterVAS-Products.Pulse-only}} account will still be active, but the user will no longer be able to view details for the selected \{\{masterVAS-Terminology.d-airy}}. If you want to remove user access to more than one \{\{masterVAS-Terminology.d-airy}}, be sure to :Link[switch to the other dairies]{label="switch to the other dairies" overridedLabel="switch to the other dairies" spaceId docId="efCvqOyrLaeUGS4LRQlc4" version="v2" docAnchorId="#How" loadingMethod="dynamic" newTab="false" githubPath="../../../FAQs/ViewDiffDairy.htm#How" href="../../../FAQs/ViewDiffDairy.htm#How"} and repeat the steps above to remove access to those \{\{masterVAS-Terminology.d-airies}} as well.

<hint type="info">
  **NOTE:&#x20;**&#x41;s a security measure, if only one user in a \{\{masterVAS-Terminology.d-airy}} is assigned \{\{masterVAS-Products.Pulse-only}} \{\{MorphHelp.User\_Management\_UI\_item}} permissions, that user cannot be removed from the \{\{masterVAS-Terminology.d-airy}} and cannot leave the \{\{masterVAS-Terminology.d-airy}}. \{\{MorphHelp.User\_Management\_UI\_item}} permissions must be assigned to at least one other user account if the only user with that permission level wishes to leave the \{\{masterVAS-Terminology.d-airy}}. See [Edit Dairy User Permissions](EditUserPermissions.htm) for more information.
</hint>
