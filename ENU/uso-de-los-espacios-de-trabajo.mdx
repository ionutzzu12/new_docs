Cada espacio de trabajo del portal de Self Service tiene distintas capacidades y se usa para realizar distintas funciones. Para obtener más información sobre cada una de las tareas que puede completar como Usuario de Self Service, seleccione el siguiente enlace:

- Ver, actualizar y cerrar una de las solicitudes de servicios e incidencias enviadas. Para obtener información, consulte [Mis elementos](my-items.htm).
- Envíe una nueva solicitud de servicio o informe de una nueva incidencia. Para obtener información, consulte [Solicitudes de servicio e incidencias](<Uso de los espacios de trabajo/solicitudes-de-servicio-e-incidencias.mdx>).
- Examine los artículos de conocimiento que pueden ayudarle a solucionar un problema. Para obtener información, consulte [Centro de conocimiento](knowledge-center.htm).
- Ver anuncios, Preguntas y respuestas frecuentes y alertas que se publican en el portal. Para obtener información, consulte [Anuncios, Preguntas y respuestas frecuentes y Alertas](<Uso de los espacios de trabajo/anuncios-preguntas-y-respuestas-frecuentes-y-alertas.mdx>).
- Utilice el Panel social para enviar y recibir mensajes de otros usuarios. Aprenda como buscar, filtrar y eliminar mensajes, así como configurar grupos de usuarios con los que compartir mensajes. Para obtener información, consulte [Mensajes de Panel social](social-board-messages.htm).
- Chatear con otros mediante la mensajería instantánea de Chat Zone. Para obtener información, consulte [Chat Zone](chat-zone.htm).
