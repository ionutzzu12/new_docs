You can download the connected OPC UA server CA certificate by navigating to **OPC&#x20;**> **Connections**. This certificate must be installed in the Trusted Root Certification Authorities store to enable trust.

<hint type="warning">
  **Important**: Make sure the server is started before you download the server certificate. See [Manage the OPC UA Server](<../../../docs/Product Features/yrWqhJTW3Gy5_oqParJXv/ONn8_Sd9U6yiBXxIiY1C5/48bjtx80IC-cXk680TnCe.mdx>) to learn more.&#x20;
</hint>

To download the certificate, click **Download server certificate**.

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/o_zUtJ0sPzQpiPkB49G4J_image.png)

After clicking the icon, the `server_cert.der` certificate is downloaded to your hard drive.&#x20;
