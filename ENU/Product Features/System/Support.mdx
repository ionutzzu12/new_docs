The **System&#x20;**>**&#x20;Support** pane is primarily used by Customer Support for troubleshooting purposes.

It includes the following sections.&#x20;

# Components

This section shows the status and version of the underlying software components for Litmus Edge. A green circle by a component's name means that this component is online, and a red circle means that the component is offline.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/GO39_Ck6t0WoVgZU_Gbpg_image.png" size="80" width="1002" height="422" caption="The Components section " position="center" showCaption="true"></Image>

# Support Bundle

The support bundles are primarily used for diagnostic purposes. They should be attached as part of a support ticket when contacting customer support. Support bundles are then reviewed by customer support.

The files in the support bundle are password-protected.

<hint type="info">
  **Note**: You have the option to not include the config files in the support bundle. Select to include config files in the bundle for full diagnosis.&#x20;
</hint>

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/-qWMePMHT5f3UaSKm6AV4_image.png "Support bundle section")

# File Downloads

Allows you to download the DeviceHub Agent file and open-source license bundles.

To download a file, click the **Download&#x20;**&#x69;con.&#x20;

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/DKwCxl0elARuqvtL76znN_image.png" size="80" width="733" height="230" caption="Download icon" position="center" showCaption="true"></Image>

You can view more information on a file by clicking the **Expand&#x20;**&#x69;con.&#x20;
