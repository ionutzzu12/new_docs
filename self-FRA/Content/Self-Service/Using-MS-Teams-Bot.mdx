Vous pouvez utiliser le bot \{\{FRS\_VARS.Company}} Service Manager pour exécuter différentes actions, comme la création d'un incident, la vérification de l'état d'un incident et le chat avec un analyste.

1. Ouvrez le bot \{\{FRS\_VARS.Company}} Service Manager et envoyez un message Ping.

<hint type="info">
  Si le bot n'est pas disponible pour vous, contactez l'administrateur de votre entreprise pour qu'il le configure.
</hint>

2. Sélectionnez l'une des options suivantes, selon vos besoins.
   - **Créer un ticket** - Crée un incident.
   - **Vérifier l'état de mon ticket** - Affiche l'état de tous les incidents ouverts.
   - **Chat avec le support** - Vous êtes mis en relation avec un analyste qui vous aidera pour votre problème/requête.
     - Si un analyste ne répond pas au chat, vous recevez le message « Veuillez patienter, nous vous connectons à un autre analyste ». Vous pouvez attendre la connexion.
     - Si aucun analyste n'est en ligne, le message indique « Tous les analystes sont hors ligne ». Vous pouvez continuer à attendre jusqu'à ce qu'un analyste soit en ligne, ou quitter le chat. Le délai d'attente maximal pour la mise en rapport avec un analyste est de 5 minutes. À l'issue de ce délai, le message suivant apparaît : « Désolé, impossible de vous connecter pour le moment ». Cliquez sur **Oui** pour continuer à utiliser le bot ou sur **Non** pour le quitter.
   - **Quitter la conversation de chat** - Permet de quitter le chat.
   - **Aide** - Affiche une boîte de dialogue qui décrit les options ci-dessus.

<hint type="info">
  - Si vous envoyez un message texte au lieu de choisir l'une des options disponibles, le message suivant s'affiche : « Sélection non valide. Sélectionnez l'une des options disponibles. »
  - Chaque fois que le bot affiche des options, veillez à en sélectionner une. Sinon, le message de sélection non valide s'affiche, et vous êtes invité à sélectionner uniquement l'une des options affichées.
</hint>
