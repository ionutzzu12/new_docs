The Statistics pane is used to manage certificate trust for OPC UA clients. Before a client can connect, the server must be configured to trust that client's certificate.

From here, you can view, trust, or remove client certificates, refresh the client list, and download the server certificate. For more information, see [Manage OPC UA Authentication](<../../../../Product Features/OPC UA Server/Management/dk1QSRWRg4rsZfIN4EtTA.mdx>).&#x20;

If you are using the Docker version of Litmus Edge and having issues connecting to the OPC UA server, see [Issues Connecting to OPC UA Server](../../../Troubleshooting/AD35XvPaCjId2wmw7dl-N/gv8ZRZmsB9jW1F42YujC7.mdx).&#x20;

# Access OPC UA Statistics UI

**To access the OPC UA Statistics pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **OPC&#x20;**>**&#x20;Statistics**.
   The *Statistics&#x20;*&#x70;ane appears.
