You can upgrade an active license offline by navigating to **System&#x20;**>**&#x20;Activation** >**&#x20;License**.&#x20;

# Before You Begin

Before completing the steps below, make sure you do the following:

- Have an Ethernet connection for your first bootup of the Litmus Edge device after installation.
- Have access to a browser that is connected to the internet to complete this activation.
- If your Internet browser is on a separate computer, use a text file to copy the Activation Key and Offline Activation Key so you can transport the information.

# Upgrade a License Offline

**To upgrade a license without an Internet connection:**

1. Navigate to **System&#x20;**\*> \***Activation**.
2. Click the **License&#x20;**&#x74;ab.&#x20;
3. From the *Node-Locked License* section, click the **Upgrade License** icon for the license you need to upgrade. The **Upgrade License** icon only appears for the licenses whose type supports the upgrade operation.
   The *Upgrade License* form appears.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/iY9FwFm3ZWbCJz8IFnxVG_upgrade-license-2.png)
4. Click the **Upgrade License Offline** tab.
   The *Activation Key&#x20;*&#x61;nd *Offline Activation Key* fields appear.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/JrOhOF1NjHUITqXwvrzy0_upgrade-offline-license.png)
5. Paste or enter the license key obtained from Litmus in the **Activation Key** field. Then, click **Generate Request**.
   The *Offline Activation Request* value appears.
6. Click the **Copy&#x20;**&#x69;con for the *Offline Activation Request*.
7. Click **Open License Portal**.&#x20;
   The *Licensing Portal* opens in a new browser tab.&#x20;
8. In the **Activation Request** field paste th&#x65;*&#x20;Offline Activation Request&#x20;*&#x63;opied in step 5.&#x20;
9. Click th&#x65;**&#x20;I'm not a robot** check box. Then, click **Make License**.&#x20;
10. Click the **Copy&#x20;**&#x69;con for the *Activation Result*.&#x20;
11. Return to the browser tab with Litmus Edge and paste the *Activation Result&#x20;*&#x69;n the **Offline Activation Key** field.&#x20;
12. Click **Submit**.&#x20;
    Your license is now upgraded.&#x20;
