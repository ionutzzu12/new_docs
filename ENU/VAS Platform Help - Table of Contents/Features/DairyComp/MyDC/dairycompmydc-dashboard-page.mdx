The \{\{Buttons-UI.FeaturesMain}} Dashboard page allows you to monitor the \{\{masterVAS-Terminology.h-erd}} side of your \{\{masterVAS-Terminology.d-airy}}. You can find this feature under the :Link[DairyComp/MyDC Menu]{label="DairyComp/MyDC Menu" overridedLabel="DairyComp/MyDC Menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}.

The name of the page varies based on your \{\{masterVAS-Products.Pulse-only}} subscription. The page name is "\{\{masterVAS-Products.DC305-simple}} Dashboard" for \{\{masterVAS-Products.DCU-users-short}} and "\{\{masterVAS-Products.MyDC-short}} Dashboard" for \{\{masterVAS-Products.DCP-users-short}}. See [Know Your VAS PULSE Version ](../../GetStarted/KnowYourPulse.htm)for more information.

Like the [Dashboard](../Dashboards/DashboardsOverview.htm), \{\{masterVAS-Terminology.O-verview-}} pages can be customized with any available widget. For instructions on how to add, remove, and rearrange the widgets, see [Customize Dashboards](../../Dairy/Dashboard/customize-dashboards.mdx):
