Litmus Edge is an Industrial Edge Data Platform that allows you to collect, analyze, and act on real-time data at the edge. It offers out-of-the-box support for a wide range of native drivers for industrial devices, control systems, robotics, and other industrial assets.

Litmus Edge collects and normalizes device data and enables data sharing with any edge environment, big data platform, cloud, or enterprise system. Litmus Edge normalizes both raw and processed data so you can visualize and analyze it at the edge, no matter the data source (device type).

You can access Litmus Edge using any of the following:

- Web UI (User Interface)
- Terminal User Interface
- REST APIs

# Security

Litmus Edge is built with security at every step to address the unique challenges of Industrial IoT. Litmus Edge offers:

- **Deployment Security**: Litmus Edge supports offline-first and air-gapped installation.
- **Authentication and Authorization**: Apply enterprise-ready authentication, authorization, and RBAC (Role-Based Access Control) policies. The Litmus Edge user interface is only accessible with a login and password.&#x20;
- **Platform Access Security:** Protected and modern HTTPS web interfaces.&#x20;
- **LDAP/AD authentication**: Litmus Edge LDAP and AD settings allow you to integrate the login system with an external authentication registry.
- **Certificate Management**: Litmus Edge includes a default self-signed certificate to ensure that all communications are secure by default.
- **Encryption**: All communication between a Litmus Edge device and your browser are protected by transport layer encryption, which uses SSL/TLS.
- **Software Updates**: An over-the-air update system delivers updates remotely and ensures their security with thorough verification checks.
- **Token-Based Security**: Litmus Edge provides token-based security for apps authentication.
- **Data Access Security:** Litmus Edge offers a secure and isolated message broker for real-time data subscription.

# Functionality

Litmus Edge allows you to:

- Collect data from any industrial device securely.
- Normalize the collected data.
- Process, analyze, and store data at the edge.
- Run container-based applications on top of your data.
- Perform time-series analysis using ready analytics and KPIs.
- Easily build flows for configuration and analytics.
- Integrate edge systems with cloud or enterprise systems, including Big Data.
