You can configure the following network utilities.&#x20;

- Use [Ping (Gen1)](<../../ENU/QuickStart Guide/-4UjeBrmyWNxfUY_lDonG/network-utility/JYRNYThT10Jt6E60f1pt1.mdx>) to to ping an IP address.
- Use [SNMP Manager](<../../../QuickStart Guide/Industrial Systems Connection Guide/Network Utility/snmp-manager.mdx>) to poll an SNMP agent or start an SNMP trap receiver.&#x20;
- Use [Syslog Server](<../../../QuickStart Guide/Industrial Systems Connection Guide/Network Utility/syslog-server.mdx>) to receive Syslog messages from a Syslog client.
