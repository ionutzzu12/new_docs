The \{\{Worklists.Tasks\&WorklistSettings}} page allows you to:

- Track all \{\{masterVAS-Terminology.p-en}} assignment corrections as \{\{Events.9MOVE}} events on the \{\{masterVAS-Terminology.cowcard}}.
- Configure the event shortcuts that display in the mobile app when processing hospital animals to provide your hospital team the events needed.

You can find this feature under the :Link[DairyComp/MyDC menu]{label="DairyComp/MyDC menu" overridedLabel="DairyComp/MyDC menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} > :Link[Settings]{label="Settings" overridedLabel="Settings" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} option on the \{\{Buttons-UI.FeaturesSecondary}} tab.

  ## Hospital Lists

  Configure Configure the event shortcuts that display in the mobile app when processing hospital animals to provide your hospital team the events needed.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/DpCBLOB4KO84Yf6EIyMW5_settings-tasksandworklists-hospital-lists.png" size="47" isUploading="false" initialPath="../../../../Resources/Images/Platform/mobile-app/worklists/settings-tasks&worklists-hospital-lists.png" githubPath="Content/Resources/Images/Platform/mobile-app/worklists/settings-tasks&worklists-hospital-lists.png" position="flex-start" showCaption="false"></Image>

  Up to 7 events can be added and 7 are selected by default. See :Link[Edit Hospital List Events]{label="Edit Hospital List Events" overridedLabel="Edit Hospital List Events" spaceId docId="635vMF0-dfUW9sY8fzB1y" version="v2" docAnchorId="#edit-hospital-events" loadingMethod="dynamic" newTab="false" href="tasks-and-worklist-settings.mdx"} to learn how to reorder or change which events are selected.

    - \{\{Pneumonia}} (PNEU)

  <ExpandableHeading>
    ### Edit Hospital List Events

    Drag and drop events to reorder them (be sure to \{\{Commands.click}} the dots to the left of the event name to grab that item).

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/0RGl6WI0LBJJL0SLPE21c_hosplistsettings.png" size="43" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/HospListSettings.png" githubPath="Content/Resources/Images/Platform/dairycomp/HospListSettings.png" position="flex-start" showCaption="false"></Image>

    To add new event shortcuts to your hospital lists, and if 7 event shortcuts are already selected, you must first \{\{Commands.click}} the **x** on an event shortcut to delete it and free up space for a new event shortcut. Then, use the drop‑down list to select the new event that your hospital team needs while treating animals.

    After you make your changes, the event shortcuts will be updated in the mobile app during the next \{\{care package}} update. They will then be displayed as shortcuts when processing each animal in the \{\{Worklists.HospListCommonUse}} (see example above). The app automatically provides an **Other** button that allows you to select another event if you do not see a shortcut for the one you need.
  </ExpandableHeading>
