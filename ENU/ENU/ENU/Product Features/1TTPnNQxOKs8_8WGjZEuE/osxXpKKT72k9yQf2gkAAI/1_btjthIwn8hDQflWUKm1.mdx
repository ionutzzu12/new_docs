Zerotouch provisioning allows you to automatically install Litmus Edge configurations, applications, and docker files onto multiple devices.&#x20;

# Zerotouch Installation

See [Zerotouch Installation](<../../../../../QuickStart Guide/Installation and Deployments/QRHOm75sxT_V-s-PP5V7p.mdx>) to learn more about installing Litmus Edge onto devices using Zerotouch.&#x20;

# Access Zerotouch UI

From the Litmus Edge navigation menu, navigate to **System&#x20;**> **Events&#x20;**> **Zerotouch**.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/WvgjcLm6XZBz8qbzfaYez_image.png)

From the *Zerotouch&#x20;*&#x70;ane, you can review applications that were included in a Zerotouch installation, their status, and the reasons for any installation failures.&#x20;
