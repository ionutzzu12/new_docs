<ExpandableHeading>
  ### How do I add this widget to my Dashboard or Overview Page?

  To add this widget to your \{\{Buttons-UI.Dashboard}} or \{\{masterVAS-Terminology.O-verview-}} page, check the box for it under the **Feed** category in the [Page Customization](<../../../../../../VAS Platform Help - Table of Contents/Features/Dairy/Dashboard/customize-dashboards.mdx>) options.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/X1uk9oV82Q3Qj3RXUSJlZ_feed-widgets.png" size="21" isUploading="false" initialPath="../../../../Resources/Images/Platform/dashboard/feed-widgets.png" githubPath="Content/Resources/Images/Platform/dashboard/feed-widgets.png" position="flex-start" showCaption="false"></Image>

  <hint type="warning">
    **IMPORTANT:&#x20;**&#x54;o add **Feed** widgets to your Dashboard, your \{\{masterVAS-Terminology.d-airy}} must have \{\{masterVAS-Products.FeedWatchPlatform}} integration enabled in the \{\{General.ProductName}}.
  </hint>
</ExpandableHeading>

<ExpandableHeading>
  ### What permissions do I need to view this widget?

  To view this widget on a Dashboard or \{\{masterVAS-Terminology.O-verview-}} Page, your user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

The \{\{Widgets.EquipmentStatus}} widget displays the current status of feed mixers that are synchronized into the \{\{General.ProductName}} from \{\{masterVAS-Products.FeedWatchPlatform}}. As shown in the example below, the colors in the equipment header boxes indicate the equipment status:

- A GREEN HEADER BOX indicates that the equipment connection is good.
- A RED HEADER BOX indicates a connection loss between the equipment and \{\{masterVAS-Products.FeedWatchPlatform}}.
- A GRAY HEADER BOX indicates disabled equipment. This type of equipment is only displayed if the **Show Disabled Equipment** box at the top left of the widget is checked.
