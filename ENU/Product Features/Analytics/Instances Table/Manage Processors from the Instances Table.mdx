You have the option of managing processors by navigating to **Analytics&#x20;**>**&#x20;Instances Table**.&#x20;

Refer to the management options below.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/iCQbO5bPVKZcNGa59_CWx_image.png "Management options on the Instances Table")

| **Action**              | **Details**                                                  |
| ----------------------- | ------------------------------------------------------------ |
| \[1]                    | Remove the processor                                         |
| \[2]                    | Update processor connection name                             |
| \[3] **Processor menu** | You have the following options. &#xA;\* **Remove processor** |

- **Invoke processor**: Allows you to schedule a Database Batch processor to start and stop providing input at a specified date and time.&#x20;
- **Reset processor** |
  \| \[4]                    | Update the parameters for the processor                                                                                                                                                                                            |
