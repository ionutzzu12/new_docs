Launching an application to Litmus Edge results in 404 NOT FOUND error.

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/s-9EO78a3UkM1gKu5ZLsY_image.png)

# Possible Causes / Resolutions

      **Possible Cause**
    </td>
    <td>
      **Resolution / Workaround**
    </td>
  </tr>
  <tr>
    <td>
      This Litmus Edge instance was previously activated to a different Litmus Edge Manager. You are trying to launch an application from a catalog that has the same name as a catalog from the previous LEM.
    </td>
    <td>
      On the Litmus Edge instance, delete any catalogs that were synchronized from other Litmus Edge Managers. Then, try launching the application again.
