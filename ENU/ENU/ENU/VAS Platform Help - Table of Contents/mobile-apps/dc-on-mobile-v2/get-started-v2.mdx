The \{\{masterVAS-Products.DConMobile}} app provides the most recent animal data while you are out with the \{\{masterVAS-Terminology.h-erd}} or away from the \{\{masterVAS-Terminology.d-airy}}. When you launch the mobile app, it opens to the \{\{Mobile-UI.HomeTabMobile}} page.

<ExpandableHeading>
  ## I don't see the Home button. How do I find it?

  Follow the steps below if you do not see the \{\{Mobile-UI.Feature-Names}} button at the bottom of your screen.

  1. Tap the **Back** arrow at the top left of the screen until you return to one of the main feature screens.
  2. From there, you'll find the feature buttons at the bottom of the screen.
</ExpandableHeading>

<hint type="warning">
  **IMPORTANT:&#x20;**&#x50;lease be aware that the mobile app displays only the most recent events, tests, and lactations, and only for animals that are alive and on the \{\{masterVAS-Terminology.d-airy}}. For full access to historical animal activity, tests, etc., use the web version of \{\{masterVAS-Products.Pulse-only}}.
</hint>

<ExpandableHeading>
  ## PULSE vs. DC on Mobile - Understanding the Differences

  As mentioned in other topics in the [Get Started](../GetStarted/GettingStarted.htm) section, \{\{masterVAS-Products.Pulse-only}} has two main components for viewing and managing \{\{masterVAS-Terminology.d-airy}} data—the **web version of&#x20;**\{\{masterVAS-Products.Pulse-only}} and the \{\{masterVAS-Products.DConMobile}}**&#x20;app**. While the web version and the mobile app share the same data, what you can do with each differs. See the topics below to learn more:

  <ExpandableHeading>
    ### Use PULSE to monitor your data

    The web version of \{\{masterVAS-Products.Pulse-only}} is available through a browser at [https://platform.vas.com](https://platform.vas.com) from any desktop computer, laptop, or mobile device that has an **active** WiFi or data connection.

    The primary purpose of the web version of \{\{masterVAS-Products.Pulse-only}} is to **monitor, manage, and analyze the status of your&#x20;**\{\{masterVAS-Terminology.d-airy}}. As such, it is typically used by \{\{masterVAS-Terminology.d-airy}} owners and mangers, \{\{masterVAS-Terminology.h-erd}} and feed managers, and \{\{masterVAS-Terminology.d-airy}}\{\{masterVAS-Terminology.c-onsultants}}.

    \{\{masterVAS-Products.Pulse-only}} brings together your most important feed, \{\{masterVAS-Terminology.h-erd}}, parlor, and industry partner data into a single web-based platform, offering the following capabilities from any web browser:

    - Easy access to your data in a single, web-based format
    - Access to data such as animal records, lactation information, and milk production
    - Dashboards so you can graph animal data to provide important insights
    - [Command line](<../../../../VAS Platform Help - Table of Contents/features/command-line.mdx>) functionality to generate many of the same reports you do in \{\{DC-UI.DC-ProductName}} (for \{\{masterVAS-Products.DCUnlimited-short}}\{\{masterVAS-Terminology.d-airies}})
    - \{\{Worklist}} and event status data
  </ExpandableHeading>

  <ExpandableHeading>
    ### Use the mobile app to perform work

    The mobile app is primarily used for work on the \{\{masterVAS-Terminology.d-airy}}. While the mobile app allows you to look up most of the same [CowCard](../../features/manage-your-animals/view-animal-details-on-cowcards.mdx) information that's in the web version of \{\{masterVAS-Products.Pulse-only}}, you can also use the mobile app to scan animals, enter animal events (BRED, MOVE, etc.), process worklists, and enter embryo transfer events.

    Unlike the web version of \{\{masterVAS-Products.Pulse-only}}, the mobile app can be used both online **or offline** and has the ability to work with a paired Bluetooth device to make scanning and working with animals even easier.

    With the functionality defined above, the mobile app is typically used while workers are outside the office or cow-side.
  </ExpandableHeading>

  ## Online & Offline Mode

  When the mobile app has an active WiFi or data connection, you can refresh your data. If you go outside of data range, the app will continue to work in offline \{\{mode}}. Once you are back in data range, you can refresh your data and view the latest updates.

  <hint type="info">
    **NOTE:&#x20;**&#x57;hile the mobile app allows you to continue working in offline mode, some features in the mobile app require you to be online to see data.
  </hint>

  See the topics below to learn more:

  - [Does the mobile app use my cell phone data?](<../../../../../VAS Platform Help - Table of Contents/FAQs/Data Access and Security/does-the-mobile-app-use-my-cell-phone-data.mdx>)

<ExpandableHeading>
  ## Set Up Multiple User Accounts on a Shared Device

  \{\{masterVAS-Terminology.D-airy-}} owners can easily keep a mobile device onsite for \{\{masterVAS-Terminology.d-airy}} workers to use during their shifts. Make sure to set up \{\{masterVAS-Products.Pulse-only}} accounts for all workers who will need to use the mobile device (see [Invite a User to Your Dairy](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/invite-a-user-to-your-dairy.mdx>)), and then allow them each to log in to the \{\{masterVAS-Products.DConMobile}} app on the mobile device. Once each user has logged in and set up their PIN, they can choose their user account from the app start-up screen and enter their PIN to use the app.
</ExpandableHeading>

<ExpandableHeading>
  ## Logging Transactions

  View the \{\{Mobile-UI.TransactionLog}} for a list of transactions you've complete in the app over the last 14 days. Tap on the \{\{Mobile-UI.MoreTab}} tab at the bottom right of the screen, and then tap \{\{Mobile-UI.TransactionLog}} to get started!

  The \{\{Mobile-UI.TransactionLog}} records the following user actions:

  - Adding an animal on the \{\{Mobile-UI.AnimalsTab}} tab
  - Editing items from a CowCard
  - Completing a task on the \{\{Mobile-UI.ListsTab}} tab
  - Completing an event on the \{\{Mobile-UI.EventsTab}} tab
</ExpandableHeading>
