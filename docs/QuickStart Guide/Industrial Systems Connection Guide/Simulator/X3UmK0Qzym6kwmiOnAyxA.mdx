<hint type="success">
  **Tip**:

  - If the **Min\_value** and **Max\_value** tag parameters are set, any generated tag values are overridden.
  - The supported functions for the Formula tag parameter are&#x20;
    - `sin`
    - `cos`
    - `tan`
    - `power`
    - `log`
    - `exp`
    - `rand`
    - `sqrt`
  - We recommend that you use a cyclic function in the formula. The parameter t represents Unix time in milliseconds. When using trigonometric functions such as sine, adjust the period of the function by dividing `t` by 10,000, for example, `sin(t/10000)`.
  - All simulated tags have a default range of **-50** to **50**.
  - Users can override or restrict these defaults using a `CSV` upload.
  - Entering `-` in the *MIN&#x20;*&#x61;nd *MAX* columns prevents the default constraints from being applied.
  - Numeric values in the *MIN&#x20;*&#x61;n&#x64;*&#x20;MAX&#x20;*&#x63;olumns will replace the default settings.
  - The *MIN&#x20;*&#x61;nd *MAX&#x20;*&#x63;onstraints apply to the defined tag formula and do not affect *Boolean&#x20;*&#x74;ags.
</hint>
