You can manage network devices by navigating to **System** >**&#x20;Network&#x20;**> **Wi-Fi**.&#x20;

Litmus Edge Wi-Fi uses WPA for authentication to ensure a secure wireless network.

<hint type="info">
  **Note**: The *Wi-Fi&#x20;*&#x74;ab only appears if the applicable device supports Wi-Fi.
</hint>

# Before You Begin

Make sure you have configured the wlan0 network interface. See [Configure Network Interfaces](fAz7AOfoHQM6AJG1FfS16.mdx) for more information.

# Connect a Device

**To connect a device:&#x20;**

1. Navigate to **System&#x20;**> **Network&#x20;**&#x61;nd click the **Wi-Fi&#x20;**&#x74;ab.&#x20;
2. Click **Networks&#x20;**&#x6F;n the the network device tile, select a network from the drop-down list, and then click **Join**. You can enter the name of the network (SSID) into the **Enter SSID&#xA0;**&#x66;ield to quickly find a network.
   The *Wireless Password* dialog box appears.&#x20;
3. Enter a password for the wireless network in the **Password&#xA0;**&#x66;ield, and then click **Connect**.

The state on the device tile displays CONNECTED.

# Disconnect a Device

To disconnect a device, navigate to th&#x65;*&#x20;Wi-Fi&#x20;*&#x20;tab and click **Disconnect**.&#x20;
