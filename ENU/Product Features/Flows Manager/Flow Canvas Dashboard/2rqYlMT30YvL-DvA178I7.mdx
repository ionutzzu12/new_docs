Once the dashboard elements have been configured, you can review the results in the dashboard.

The URL for the dashboard uses your Litmus Edge IP address.

<hint type="info">
  **Example**: https\://*\<ip address>*/flows/api/ui/#/0
</hint>

**To review dashboard results:**

1. From the Navigation panel, selec&#x74;**&#xA0;Flows Manager.**
2. Click the **Expand&#xA0;**&#x69;con next to a selected Flow.
3. Click the **Menu&#xA0;**&#x69;con, select **View&#x20;**> **Dashboard**, and then drag the **Sidebar&#xA0;**&#x75;p from the bottom of the canvas.
4. From the Dashboard tab, click the **Launch** icon at the upper right of the sidebar dashboard view.
   The dashboard appears in a new tab.
