Review the following information for setting up and configuring the AB CompactLogix (Gen1) driver.&#x20;

# Litmus Edge Compatible Series

| Compatibility parameter        | Compatible items                                                                                                                                                                               |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Driver type**                | Ethernet                                                                                                                                                                                       |
| **Validated devices/series**   | [Rockwell Automation Allen-Bradley CompactLogix](https://www.rockwellautomation.com/en-us/products/hardware/allen-bradley/programmable-controllers/small-controllers/compactlogix-family.html) |
| **Replaced with Gen 2 driver** | [AB CompactLogix Ethernet FreeTag](<././Gen 2 Drivers/AB CompactLogix Ethernet FreeTag.md>)                                                                                                    |

To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1\. Note down the following information and refer to it when needed.&#x20;

**Device**: AB CompactLogix L24ER-QB1B

**Communication I/F**: Ethernet, USB. Refer to the following figure.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/16_zsafVn6V7n8lOUaTLz_ab-compactlogix.png" size="80" width="914" height="662" caption="Reference image of device" position="center" showCaption="true"></Image>

2\. Connect the device to a PC using a USB port.

3\. Launch Rockwell's driver configuration tool, RXLinx classic, and add the SmartGuard USB driver.

4\. Right-click the device and configure the module.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/oh-wD8SV1RJnNg6kSM-cn_ab-compactlogix-parameters.png" size="80" width="894" height="544" caption="Network parameters section" position="center" showCaption="true"></Image>

5\. Add driver: EtherNet/IP device.

6\. Choose adapter: make sure it's the same subnet as the device.

7\. Detect the device.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/h6Dna7P6FeeDE_qwu7vqv_ab-compactlogix-detect.png" size="80" width="1088" height="599" caption="Device detection option" position="center" showCaption="true"></Image>

# Step 2: Set up the PLC Device in Litmus Edge DeviceHub

Configure the following parameters when you [Connect a Device]() with this driver. Update default values to the specific setup of your device.&#x20;

| Parameter           | Value                                        |
| ------------------- | -------------------------------------------- |
| **Type**            | Allen-Bradley                                |
| **Driver**          | AB CompactLogix (Gen1)                       |
| **Network address** | The device address you defined during Step 1 |
| **Port**            | 44818                                        |
| **Slot Number**     | 0                                            |

# Step 3: Configure the List of Registers

When you [Add Tags](#) to the connected device, use the following tag/register mapping table.

<hint type="info">
  **Note**: 

  - The **Browse** icon displayed on the device tile means that this PLC uses Browse Tags. See [Add Tags](#)  for more information.&#x20;
  - Certain device drivers do not have a list of register names in the DeviceHub Add Tag form. These ar&#x65;*&#xA0;Free Tag PLCs*. For Free Tag requirements, see [Troubleshooting - DeviceHub](#).&#x20;
</hint>

<hint type="warning">
  **Caution**: Because this device belongs to a free tag PLC, the case-sensitive register name must be entered exactly as it is defined in the PLC. The device does not display a *Connected&#xA0;*&#x73;tatus until a tag with a correct register name is created.
</hint>

| **Name** | **Value** |
| -------- | --------- |
| R        | BOOL      |
| R        | SINT      |
| R        | INT       |
| R        | BIT(INT)  |
| R        | DINT      |
| R        | REAL      |
| R        | LINT      |
