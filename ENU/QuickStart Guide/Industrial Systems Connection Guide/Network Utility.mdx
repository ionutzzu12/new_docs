You can configure the following network utilities.&#x20;

- Use [Ping (Gen1)](<Network Utility/Ping _Gen1_3_.mdx>) to to ping an IP address.
- Use [SNMP Manager](<Network Utility/SNMP Manager.mdx>) to poll an SNMP agent or start an SNMP trap receiver.&#x20;
- Use [Syslog Server](<Network Utility/Syslog Server.mdx>) to receive Syslog messages from a Syslog client.
