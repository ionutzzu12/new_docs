  ## Add a View from a Template

  To quickly add a \{\{Insights.KPIView-single}} for your \{\{masterVAS-Terminology.d-airy}}, use a prebuilt \{\{Insights.KPIView-single}} from the \{\{Insights.KPIViewTemplates}} tab. <ExpandableHeading indent="2">
  ## Build a Custom View

  Use the following guidelines when building custom \{\{Insights.KPIViews-only}} for use in your \{\{Insights.HerdSights}}\{\{Insights.KPIDashboards-lower-only}}:

  - Click **Customize** to start building a \{\{Insights.KPIView-single}} from an existing template. This option is helpful if you're aiming to build a similar \{\{Insights.KPIView-single}}, but want to change or add more data points or adjust the frequency of the \{\{Insights.KPIView-single}}.
  - Click the **Build Custom&#x20;**\{\{Insights.KPIView-single-upper}} button at the top right of the page to start building a \{\{Insights.KPIView-single}} from scratch.
  - When building your \{\{Insights.KPIViews-only}}, select related data points and \{\{Insights.KPIs}} to measure and compare associated details. For instance, if you need to track weekly replacement females born to heifers, you would only add LACTGRP0 birth data to your \{\{Insights.KPIView-single}}.
  - Create as many \{\{Insights.KPIViews-only}} as you need. Then group your \{\{Insights.KPIViews-only}} in a \{\{Insights.KPIDashboard-lower}} to keep related metrics together.
  - Follow the steps below to create your custom \{\{Insights.KPIView-single}}:1) Build a \{\{Insights.KPIView-single}} from an existing template or build a custom view: \* Click **Customize** to start building a \{\{Insights.KPIView-single}} from an existing template.
    - Click the **Build Custom&#x20;**\{\{Insights.KPIView-single-upper}} button to start building from scratch.

  <Paragraph listStyleType="decimal" listStart="2" indent="2">
    Select whether to track this \{\{Insights.KPI}} on a daily, weekly, or monthly basis and click **Create**.
  </Paragraph>

  <Paragraph listStyleType="decimal" listStart="3" indent="2">
    Use **Search** and the **Categories** filter on the left column to find the specific data points and \{\{Insights.KPIs}} you'd like to track.:::ExpandableHeading\{customAnchorSlug="sm-search"}
  </Paragraph>
