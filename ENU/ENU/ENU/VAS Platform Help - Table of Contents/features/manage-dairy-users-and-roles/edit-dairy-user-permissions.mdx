<ExpandableHeading>
  ### Required Permissions

  Only users who are assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions can use this feature. See Edit Dairy User Permissions and [Manage Dairy Roles](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

This topic explains how to edit the permissions assigned to a \{\{masterVAS-Terminology.d-airy}} user.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wk73PIXxXCkXJQdXoGeOA_user-details-dairy.png" size="43" isUploading="false" initialPath="../../../../Resources/Images/Platform/users-roles/user-details-dairy.png" githubPath="Content/Resources/Images/Platform/users-roles/user-details-dairy.png" position="flex-start" showCaption="false"></Image>

You can find this feature under the :Link[Dairy menu]{label="Dairy menu" overridedLabel="Dairy menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-Dairy" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-Dairy" href="../../../../VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}. If you need to edit the permissions assigned to a company user, see :Link[Edit Company User Permissions]{label="Edit Company User Permissions" overridedLabel="Edit Company User Permissions" spaceId docId="ZBxNM9nH4pjlw1ThsUy51" version="v2" docAnchorId="#Edit_Company_User_Permissions" loadingMethod="dynamic" newTab="false" githubPath="../../Company/CompanyUserMgmt.htm#Edit_Company_User_Permissions" href="../company-view/manage-company-users-and-roles.mdx"}.

1. In the **Users** list on the **Users** tab, expand the details of the \{\{masterVAS-Terminology.d-airy}} user that you want to update.
2. Click the **Edit Roles** button at the bottom right side of the page.
3. In the **Available Roles** list, check the boxes to select additional **Roles** to assign to the user's account.The roles selected control how much or how little a user can do with the data in \{\{masterVAS-Products.Pulse-only}}. Each role checked is displayed in the **Roles Assigned** list. You can click the **X** on the right side of a role to unassign it.
4. \{\{Commands.Click}} the **Save** button at the bottom right side of the page.

See [Manage Dairy Roles](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>)for instructions on how to review and edit the permissions assigned to each \{\{masterVAS-Terminology.d-airy}} role.
