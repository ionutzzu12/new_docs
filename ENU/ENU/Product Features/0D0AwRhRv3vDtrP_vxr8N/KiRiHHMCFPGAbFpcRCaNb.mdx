From the *Device Management* pane, you can do the following:

- Reboot and factory reset the Litmus Edge device. See [Manage Edge Devices](<../../../Product Features/System/Device Management/CY-FRLxvNce4dqT8oHx-T.mdx>).&#x20;
- Upgrade the version of Litmus Edge. See [Software Updates](<../../QuickStart Guide/NyWAQm5OPiDJ27ioRlTW6/MKLEE33uupZUE-L7ZFwRv.mdx>).&#x20;
- Manage the disk space policy.&#x20;
- Manage FTP and SSH services. See [System Services](<../../../Product Features/System/Device Management/VgIBR0tAa0PHaeaWY2I0c.mdx>).&#x20;
- Configure and initiate system backups and restore operations. To prepare for possible hardware failures, Litmus Edge provides a disaster recovery plan to support device replacement. See [Backup/Restore](<../../../Product Features/System/Device Management/m_MfeA_rejY42si5v27V0.mdx>).&#x20;
- Create detailed, granular backup templates and exchange these templates between edge devices as JSON files. See [Backup/Restore](<../../../Product Features/System/Device Management/m_MfeA_rejY42si5v27V0.mdx>).

<hint type="info">
  **Note:** Litmus Edge supports using Trusted Platform Module (TPM) components to encrypt certain Litmus Edge Manager credentials using hardware security. After configuring TPM on your device, LEM credentials become encrypted by Litmus Edge. If you require assistance with enabling TPM, contact Litmus support.
</hint>

Access Device Management UI

From the Litmus Edge navigation panel, navigate to **System&#x20;**>**&#x20;Device Management**.
The *Device Management&#x20;*&#x70;ane appears.&#x20;
