From the *Device Management* pane, you can do the following:

- Reboot and factory reset the Litmus Edge device. See [Manage Edge Devices](<../System/Device Management/_Q9yALYCmzhB-VpRX77mk.mdx>).&#x20;
- Upgrade the version of Litmus Edge. See [Software Updates](<../../QuickStart Guide/Installation and Deployments/3IeYShQnl-tts7ne21_OS.mdx>).&#x20;
- Manage the disk space policy.&#x20;
- Manage FTP and SSH services. See :Link[System Services]{label="System Services" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="5-CeM9XRNWqqh9tyfIspN" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;
- Configure and initiate system backups and restore operations. To prepare for possible hardware failures, Litmus Edge provides a disaster recovery plan to support device replacement. See :Link[Backup/Restore]{label="Backup/Restore" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="BOjcwT6EqoILWd3yWne0_" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;
- Create detailed, granular backup templates and exchange these templates between edge devices as JSON files. See :Link[Backup/Restore]{label="Backup/Restore" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="BOjcwT6EqoILWd3yWne0_" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.

<hint type="info">
  **Note:** Litmus Edge supports using Trusted Platform Module (TPM) components to encrypt certain Litmus Edge Manager credentials using hardware security. After configuring TPM on your device, LEM credentials become encrypted by Litmus Edge. If you require assistance with enabling TPM, contact Litmus support.
</hint>

Access Device Management UI

From the Litmus Edge navigation panel, navigate to **System&#x20;**>**&#x20;Device Management**.
The *Device Management&#x20;*&#x70;ane appears.&#x20;
