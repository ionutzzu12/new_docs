To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: LS Industrial Systems
   - **Communication I/F**: RS232. Refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/D2BMrU-96nvCTBps28UAh_ls-master-k-cpu-direct-serial-reference.png" size="80" width="650" height="511" position="flex-start" caption="LS Master-K CPU Direct Serial reference" showCaption="true" indent="2"></Image>

2. Launch KGL\_WE v3.66 Software and then click **Project > Option > Connection Option**.
3. Set up the COM port.&#x20;
4. Click **Connect**.&#x20;
