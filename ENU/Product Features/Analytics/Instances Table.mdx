An Instance Table is a listed visualization of your current analytics flows, which allows you to quickly add processors to create and connect flows sequentially. Once connected, the processors can be viewed in context from the *Instances&#x20;*&#x70;ane.

<hint type="info">
  **Notes**:

  - You must have at least one existing processor in the *Instances&#xA0;*&#x70;ane to use the processor tables.
  - Properly connected processors and connection definitions display in blue. Processors and connection definitions display green if improperly connected.
</hint>

# Access Instances Table UI

**To access the Instances Table:**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **Analytics&#x20;**>**&#x20;Instances Table**.
   The *Instances Table* pane appears.&#x20;
