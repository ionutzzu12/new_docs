You can configure the following network utilities.&#x20;

- Use [Ping (Gen1)](<../../docs/QuickStart Guide/ihnwJuFWkFifh4xbYZH3n/network-utility/FFbPs1Q7SeD7R39Nn_BNL.mdx>) to to ping an IP address.
- Use [SNMP Manager](<../../../QuickStart Guide/Industrial Systems Connection Guide/Network Utility/snmp-manager.mdx>) to poll an SNMP agent or start an SNMP trap receiver.&#x20;
- Use [Syslog Server](<../../docs/QuickStart Guide/ihnwJuFWkFifh4xbYZH3n/network-utility/syslog-server.mdx>) to receive Syslog messages from a Syslog client.
