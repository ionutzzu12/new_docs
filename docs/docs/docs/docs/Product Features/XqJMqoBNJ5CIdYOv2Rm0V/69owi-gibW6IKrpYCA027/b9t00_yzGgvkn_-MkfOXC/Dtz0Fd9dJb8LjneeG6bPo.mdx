You can create an API credentials by navigating t&#x6F;**&#x20;System&#x20;**>**&#x20;Access Control&#x20;**>**&#x20;Tokens**.&#x20;

**To create API Credentials:**

1. Navigate to **System&#x20;**> **Access Control**.&#x20;
2. Click the **Tokens&#x20;**&#x74;ab.&#x20;
3. From the *API Credentials* section, click the **Add&#x20;**&#x69;con.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/snx-ycj8CnrAxRPFPiIia_create-api-credentials.png)
4. Select the type of API credentials you want to create: **API Key&#x20;**&#x6F;r **OAuth 2.0 Client.**&#x20;

## API Key

Enter a name for the key and click **Create API Key**.&#x20;

## OAuth 2.0 Client

Configure the parameters.&#x20;

- **Client Name**: Enter a name for the client.&#x20;
- **Select RBAC Groups**: Select which access groups will have access to the client. See [Users](<../../../../../Product Features/XqJMqoBNJ5CIdYOv2Rm0V/69owi-gibW6IKrpYCA027/I7w5FhlC0MxvoW6M25rxM.mdx>) to learn more.&#x20;
- **Expiration Date** (Optional): Configure an expiration date for the client.&#x20;
- **Access Token Lifespan in minutes**: Configure how long the access token will be valid.&#x20;

Once the client is created, you must copy the **Client ID** and **Client Secret**. You won't be able to view them again.&#x20;
