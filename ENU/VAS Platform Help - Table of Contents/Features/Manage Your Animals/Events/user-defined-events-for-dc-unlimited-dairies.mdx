If your \{\{masterVAS-Products.DCUnlimited}}\{\{masterVAS-Terminology.d-airy}} has configured user-defined events in \{\{DC-UI.DC-ProdNameShort}}, \{\{masterVAS-Products.Pulse-only}} includes them as available events in the mobile app.

User-defined have an event code (\{\{EC}}) number of 19 or higher. The event numbers and names are defined within \{\{DC-UI.DC-ProdNameShort}}, and \{\{masterVAS-Products.Pulse-only}} mobile app lists them in numerical order on the \{\{Mobile-UI.EventsTab}} screen.

\{\{masterVAS-Products.Pulse-only}} supports user-defined events both with and without protocols. See [Events with Protocols](events-with-protocols.mdx) to learn more about defining \{\{protocol}} information when entering an event.

All available events are listed in event code (EC) order. Be sure to scroll down to locate your user-defined events. If you expect to see an event in the mobile app or \{\{masterVAS-Products.Pulse-only}} and are not able to locate it, then \{\{DC-UI.DC-ProductName}} may not be configured to send it.
