  ### Learn more about GENEX's herd monitoring solutions!

  GENEX offers two \{\{masterVAS-Terminology.h-erd}} monitoring solutions: an on-premise solution, \{\{Partners.HerdMonitor}}, and a cloud solution, \{\{Partners.GENEX}}. While both products offer the same accurate, reliable \{\{masterVAS-Terminology.h-erd}} insights, \{\{Partners.GENEX}} has the processing power and scalability of cloud computing. To learn more about the cloud solution, see [GENEX HerdMonitor — cloud](genex-herdmonitor-cloud-info.mdx).
</ExpandableHeading>

[GENEX HerdMonitor — on-premise](https://genex.coop/herdmonitor/) animal sensors monitor reproduction, rumination, eating, and movements to provide deeper insights on heat detection, overall health, and animals requiring urgent attention. Its all-in-one \{\{masterVAS-Terminology.h-erd}} monitoring solution is an advanced, easy-to-use system available through the \{\{masterVAS-Products.OnPremise-lower}} platform, Velos. \{\{masterVAS-Products.Pulse-only}} displays live sensor data in a \{\{Buttons-UI.Dashboard}} widget and individual animal \{\{masterVAS-Terminology.cowcard}}s on the \{\{Partners.PartnersUIoption}} tab, providing an up-to-date view of animals requiring health checks or reproduction-related protocols.
