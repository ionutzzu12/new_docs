# Copy an Interface Alias

You can copy an alias to use in DeviceHub for Linux serial interfaces that are connected to your network.

To copy a serial interface alias, click the **Copy&#x20;**&#x69;con next to a serial interface containing an alias.&#x20;

# Remove an Interface Alias

You can remove an alias for Linux serial interfaces that are connected to your network.

To remove a serial interface alias, click the **Action menu** for the serial interface that contains an alias and select **Remove Alias**.&#x20;
