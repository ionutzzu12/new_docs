\{\{Buttons-UI.ReportListBuilder}} allows you to quickly build a report in an easy-to-use, drag and drop interface. Builder currently supports List, Count, and Sum reports, as well as \{\{Worklists.ToDoUI-Web}} Lists, \{\{Worklists.VetListUI-Web}}, \{\{Worklists.HospitalListUI-Web}}, and \{\{Worklists.EventListUI-Web}}.

You can find this feature under the :Link[DairyComp/MyDC > Reports/Worklists menu]{label="DairyComp/MyDC > Reports/Worklists menu" overridedLabel="DairyComp/MyDC > Reports/Worklists menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp-Reports" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp-Reports" href="../../../../../../../VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}. Select **Build Report/Worklist** from the menu or \{\{Commands.click}} the button on the top right of any of the main \{\{Buttons-UI.DC-Reports}} pages to access the builder.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/RdAwVaaSaBW8uQ8MAamTA_report-worklist-builder-intro.png" size="57" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/worklists/report-worklist-builder-intro.png" githubPath="Content/Resources/Images/Platform/dairycomp/worklists/report-worklist-builder-intro.png" position="flex-start" showCaption="false"></Image>

You will also use \{\{Buttons-UI.ReportListBuilder}} when editing or updating your [Dairy Reports/Worklists](ManageReportsWorklists.htm).

<hint type="info">
  **NOTE:&#x20;**&#x49;f you build a \{\{Worklists.worklistCommonUse}} report type, you can convert it to an actionable list after it is generated. See [Create Worklist from Report](<../../../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Reports/Worklists/Create New Report or Worklist/create-worklist-from-report.mdx>) to learn more!
</hint>
