The *Utilities&#x20;*&#x70;ane of the *DeviceHub&#x20;*&#x6D;odule allows you to test connectivity to a URL (host name or IP address) and accessibility of a port for a device you want to connect to your edge device for both TCP and UDP. This functionality helps you troubleshoot your network.

# Access Utilities UI

You can access *Utilities&#x20;*&#x66;rom the *DeviceHub*.&#x20;

**To access Utilities:**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **DeviceHub&#x20;**>**&#x20;Utilities**.
   The *Utilities&#xA0;*&#x70;ane appears.
