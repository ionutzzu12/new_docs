The **Applications&#x20;**>**&#x20;Disk Usage** pane shows the current statistics for the applications: the total number, the number of active containers, the size of containers, images, and volumes, as well as the consumed disk space. This is useful for diagnostic purposes.&#x20;

From the *Disk Usage* pane, you can click the **Refresh&#x20;**&#x69;con to refresh the disk usage information.&#x20;
