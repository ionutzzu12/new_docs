To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below for Toolbox v4.6.

**To configure your device:**&#x20;

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Siemens 808D Advanced M
   - **Communication I/F**: Ethernet.&#x20;
2. Press **Set password**.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/scpCTBUZfZsnGhHKsJWoz_siemens-sinumerik-ethernet-password.png" size="80" width="732" height="460" position="flex-start" caption="Set password option" showCaption="true" indent="2"></Image>

3. Enter **SUNRISE&#x20;**&#x66;or authorization to configure the device.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/gd5TgwLQr9BPnOCTCK5zx_siemens-sinumerik-ethernet-authorize.png" size="80" width="712" height="424" position="flex-start" caption="Enter password" showCaption="true" indent="2"></Image>

4. Press **Sys Data**.&#x20;
5. Press **Serv displ.&#x20;**

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Ca1d5HwKYKdCsKpH3PWmO_siemens-sinumerik-ethernet-servdispl.png" size="80" width="695" height="665" position="flex-start" caption="Serv. displ. option" showCaption="true" indent="2"></Image>

6. Press **Service Network**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/GQq7bNeKjbnDd-jJAKfyR_siemens-sinumerik-ethernet-servicenetwork.png" size="80" width="697" height="447" position="flex-start" caption="Service network option" showCaption="true" indent="2"></Image>

7. Enter the network parameters.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/q8xsMzBpJ4I_UCjVL9Abh_siemens-sinumerik-ethernet-parameters.png" size="80" width="649" height="508" position="flex-start" caption="Network configuration screen" showCaption="true" indent="2"></Image>
