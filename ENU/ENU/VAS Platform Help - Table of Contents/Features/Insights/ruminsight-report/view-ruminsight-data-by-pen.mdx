  ### Required Permissions

  To use this feature, your \{\{General.ProductName}} user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

Click the **By&#x20;**\{\{masterVAS-Terminology.P-en-}} tab on the RumINSIGHT report to view detailed fatty acid test results for the most current and previous \{\{masterVAS-Terminology.t-est-days}}, broken down by individual \{\{masterVAS-Terminology.p-ens}}. Use the image and descriptions below to learn how to use the information on this page.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/l--kZAoj7oAJXt5ig5aYM_penfatgraph.png" size="75" isUploading="false" initialPath="../../../Resources/Images/Platform/FattyAcidGroups/PenFatGraph.png" githubPath="Content/Resources/Images/Platform/FattyAcidGroups/PenFatGraph.png" position="flex-start" showCaption="false"></Image>

1. By \{\{masterVAS-Terminology.P-en-}} tab: Select this tab to view the fatty acid test data by \{\{masterVAS-Terminology.p-en}}.
2. \{\{masterVAS-Terminology.T-estday-}} selector: Click the applicable button to view fatty acid data for the **Current** or **Previous**\{\{masterVAS-Terminology.t-estday}}. The data in the graph and chart changes accordingly.
3. RumINSIGHT % graph: This graph provides a visual representation of fatty acid test results for the \{\{masterVAS-Terminology.t-est-day}} selected (i.e., current or previous), broken down by \{\{masterVAS-Terminology.p-en}} number. \{\{masterVAS-Terminology.P-en-}} numbers appear along the bottom of the graph. The three types of fatty acid are represented by a different color in the graphing bar for each \{\{masterVAS-Terminology.p-en}}, according to the color legend beneath the graph.
4. Fatty Acid % graph details: Hover your cursor over a specific bar in the graph to view the exact percentage amount for each type of fatty acid.
5. \{\{masterVAS-Terminology.P-en-}} chart: This chart provides a detailed breakdown of fatty acid and related values for each \{\{masterVAS-Terminology.p-en}}. Data includes the number of cows per \{\{masterVAS-Terminology.p-en}}, the amount of milk produced, etc. Data values of concern (i.e., warnings) are shown in red.
