The \{\{masterVAS-Terminology.SRS2long}} is a portable hand-held device that allows you to quickly read, record, and transfer \{\{EID}} tag information for the animals in your \{\{masterVAS-Terminology.h-erd}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/GZ19Ku8rsKQC7Chs_B_pv_srs2-stick-reader.png" size="43" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-stick-reader.png" githubPath="Content/ContentShare/TruTest/srs2-stick-reader.png" position="flex-start" showCaption="false"></Image>

<hint type="warning">
  **IMPORTANT:&#x20;**\{\{masterVAS-Terminology.SRS2long}}s are currently only supported in the Android version of the mobile app. Always charge your scanner to 100% before configuring settings. Expand the sections below to learn more::::ExpandableHeading

  ### Required Setup

  Before you connect the \{\{masterVAS-Terminology.SRS2long}}, download the latest firmware from the [Tru-Test website](https://www.livestock.tru-test.com/en-us/resources/4328#downloads).

  <ExpandableHeading>
    ### Navigation Tips

    The display is not a touchscreen. You can navigate to options using the **Arrow** buttons.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/YoYHDtxSW5H2kCJl91lIP_srs2-arrow-buttons.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-arrow-buttons.png" githubPath="Content/ContentShare/TruTest/srs2-arrow-buttons.png" position="flex-start" showCaption="false"></Image>

    To select an option, press the **Scan** button.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Woe6IMfopw-SymHgPjrqJ_srs2-scan-button.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-scan-button.png" githubPath="Content/ContentShare/TruTest/srs2-scan-button.png" position="flex-start" showCaption="false"></Image>

    <hint type="info">
      **NOTE:&#x20;**&#x54;he **Scan** button is also referred to as the \{\{masterVAS-Terminology.R-ead-}} and/or **Select** button in your \{\{masterVAS-Terminology.SRS2long}} user manual.
    </hint>

    To return to a previous menu, press the **Return** button.

    :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/m9bRDwdFmTcQllLQrSByx_srs2-return-button.png" alt caption size="10"}
    \::::
  </ExpandableHeading>
</hint>

Follow the steps below to configure a \{\{masterVAS-Terminology.SRS2long}}:

1. Use the **Arrow** buttons to navigate and press the **Scan** button to select **Settings**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/_jW1uPD5ptv7wGzxtY7YF_srs2-settings.png" alt caption size="19"}
2. Select **General**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/CirtfntI2knztdNodXS-l_srs2-general.png" alt caption size="19"}

<ExpandableHeading indent="2">
  ### Set the date and time.

  1. Select **Date/Time**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/DdmcFCmRfr4BKfyeRxYgr_srs2-datetime.png" alt caption size="19"}
  2. Select the current **Date**, **Month**, and **Year**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/AHQLfsBwP49i3Sc7hUH6__srs2-date.png" alt caption size="19"}
  3. Set the **Format** to **MM/DD/YYYY**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/qcMP9NCKf8D65C0Emoc9K_srs2-format.png" alt caption size="19"}
  4. Set the **Hour** and **Minute** time settings for your time \{\{zone}}.:::hint\{type="info"}
     **NOTE:&#x20;**&#x54;ime settings are in military time.
     \:::
  5. Press the **Return** button to return to the **General** menu.
</ExpandableHeading>

<ExpandableHeading indent="2">
  ### Set the language and country.

  1. Select **Language/Country**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/BmHGW9Sq8UsDHhURpXoML_srs2-language.png" alt caption size="19"}
  2. Set **Language** to **English**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/6PR6YVRnyPwAMlCyhmpTS_srs2-english.png" alt caption size="19"}
  3. Select your **Country**.
  4. Press the **Return** button to return to the **General** menu.
</ExpandableHeading>

5. Set **Auto Off** to **10m**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/I8sl93_aSxYrown5MiWU1_srs2-autooff.png" alt caption size="19"}
6. Press the **Return** button to return to the **General** menu.
7. Set **Vibration** to **On**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/XlH29uTmNMAq-GyJtcu8A_srs2-vibration.png" alt caption size="19"}
8. Press the **Return** button to return to the **General** menu.
9. Set **Buzzer** to **Off**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/yDiN1xjXiybhG5ExJFMzY_srs2-buzzer.png" alt caption size="19"}
10. Press the **Return** button to return to the **General** menu.
11. Press the **Return** button to return to the **Settings** menu.
12. Select **Bluetooth**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/5f0KWLVFcWSIEc-H0bQve_srs2-bluetooth.png" alt caption size="19"}
13. Set **Bluetooth** to **Auto**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tCZvr5DvFaKQSGMqTwATw_srs2-bluetooth2.png" alt caption size="19"}

<ExpandableHeading indent="2">
  ### Set advanced Bluetooth features.

  1. Select **Advanced**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/19NW0LJ5eeeewyvXlrcVr_srs2-advanced.png" alt caption size="19"}
  2. Set **Connect Mode** to **Default**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/QRmZGUfJNa0AL1fYzCAau_srs2-connectmode.png" alt caption size="19"}
  3. Set **Output** to **Default**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/LAmwXOpr_hYeyaBrdUIzY_srs2-output.png" alt caption size="19"}
  4. Press the **Return** button to return to the **Settings** menu.
</ExpandableHeading>

15. Press the **Return** button to return to the **General** menu.
16. Press the **Return** button to return to the **Settings** menu.

<ExpandableHeading indent="2">
  ### Set tag reading features.

  1. Select **Tag Reading**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/LMn9WXLv-7VtbhHvtPn52_srs2-tagreading.png" alt caption size="19"}
  2. Set \{\{masterVAS-Terminology.R-ead-}} \{\{Mode}} to **Continuous**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/S2LfFB8jHkr0glJlhyr9x_srs2-readmode.png" alt caption size="19"}
  3. Set **Tag Format** to **Decimal2**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Ve6IARDKiaOYmKwtq1vgk_srs2-tagformat.png" alt caption size="19"}
  4. Set **Allow Duplicates** to **Yes**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/XBGmPhcEA-bP3bZGUw1KS_srs2-allowduplicates.png" alt caption size="19"}
  5. Press the **Return** button to return to the **General** menu.
</ExpandableHeading>
