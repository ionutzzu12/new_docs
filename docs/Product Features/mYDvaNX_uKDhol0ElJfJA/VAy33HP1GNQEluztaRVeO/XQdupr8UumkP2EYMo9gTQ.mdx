You can create an analytics flow by navigating to **Analytics&#x20;**>**&#x20;Instances**.&#x20;

When creating an analytics flow, you're guided through selecting input, function, and output processors and the flow automatically adds connections to the processors.&#x20;

# Create Analytics Flow

**To create an Analytics Flow:**

1. Navigate to **Analytics&#x20;***>&#x20;***Instances**.&#x20;
   The *Instances&#x20;*&#x70;ane appears.
2. At the top right, choose between the classic and new UI.
3. Use the *Analytics Groups* drop-down list to select the group you want to add the processor to.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Z8BUkeXDCi6t68H6UZRnx_image.png" size="80" width="1515" height="748" position="flex-start" showCaption="false" indent="2"></Image>

4. Open the **Action menu** and select **Add flow**.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/tMbpzQIm7yTUhStMWDHRT_image.png" size="40" width="445" height="322" position="flex-start" showCaption="false" indent="2"></Image>

5. In the **Create Flow&#x20;**&#x64;ialog, enter a name for the flow.
6. Select an input processor and define its parameters. See :Link[Input Processors]{label="Input Processors" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="D3DDEKIxUJwJk2YXKZiX1" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.&#x20;
7. Click **Next**.&#x20;
8. Select a function processor and define its parameters. See :Link[Statistical Functions]{label="Statistical Functions" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="NUC_UWNsgYimSJgXxmG22" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.
9. Click **Next**.&#x20;
10. Select an output processor and define its parameters. See :Link[Output Processors]{label="Output Processors" spaceId="0c3K9AkyXsnPI4j8vUaa5" docId="i5kpwlYUyC0dNDmWcFfdT" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.
11. Click **Next**.&#x20;
12. Click **Create Flow**.
    The flow displays in the Analytics canvas with default connections.&#x20;
