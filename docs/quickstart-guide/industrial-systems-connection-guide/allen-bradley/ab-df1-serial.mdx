To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

<hint type="info">
  **Note**: When configuring the settings for the PLC, the *Error Detect* setting must be set to CRC, otherwise a connection with Litmus Edge won't be successful.&#x20;
</hint>

Refer to the following parameters and reference images when setting up the PLC.

- **Device**: PLC5/40C
- **Communication I/F**ꓽ RS232

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/Vuzyjc9Ga0gsbEKsKmp0J_image.png" size="80" width="1509" height="1740" position="center" showCaption="false"></Image>

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/I1cWq7sLWkZLO5s6KmGmE_image.png)

Complete the following steps to setup and configure the PLC.&#x20;

**To configure the PLC:**



1. Connect the PLC to the computer with a serial port 25-pin D-shell cable.
2. Open **RSLinx Classic&#x20;**&#x73;oftware.&#x20;
3. Click the **Configure Drivers** icon. Then, click **Add New\...** and add the *RS-232 DF1 devices* driver.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/Pv7L50fnFDVDhH32HMKQr_plc5-03.png)
4. Configure the driver with the following parameter values and click **OK**.&#x20;
   **Baud Rate**: 9600
   **Bits Per Char**: 8
   **Stop Bits**: 1
   **Parity**: None
   **Error Detect**: CRC
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/l4ly-qvg5A0gpC4UzNtlQ_plc5-04.png)
5. In the driver settings, click **Auto-Configure** and confirm you see an `Auto Configuration Successful!` message.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/1YiiprT-HsMr9d2xZHc2c_plc5-05.png)
6. Ensure these same settings are configured in the PLC to establish a connection.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/SyrwJ7JGbpPGMHPA0p1YX_plc5-06.png)
7. Open **RSLogix 5** software.&#x20;
8. Click **Comms&#x20;**&#x61;nd select **Who Active Go Online**.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/1aKoajDOGyJLH_9qZp0RQ_plc5-07.png)
9. Select the device that you configured and click **Online**.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/FndapLEP2lzeKLxbfrOdJ_plc5-08.png)
10. In the navigation menu, click **Channel Configuration** and select **Open**.&#x20;
    ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/GP2NNwVk0f6nNt3AhQPJs_plc5-09.png)
11. Confirm the *Serial Port* settings for **Channel 0**, which is the RS232 port. Note down the Baud Rate, Data Bits, Stop Bits, and Parity. The *Error Detect* setting must be set to **CRC**.&#x20;
    ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/2NO4ryrjiopaxydtoFlrw_plc5-10.png)
12. You can use **Channel 3A** to configure the Ethernet port, if applicable.&#x20;
    ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/iTtHacTheCd3F6ocBhMMi_plc5-11.png)
