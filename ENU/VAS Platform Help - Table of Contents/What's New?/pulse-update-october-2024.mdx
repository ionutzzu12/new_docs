NEW FEATURE Add Event in Bulk is now available!

Use the new \{\{Buttons-UI.FeaturesMain}} > \{\{Buttons-UI.DC-Animals}} >\{\{Buttons-UI.EventBulk}} option to quickly add an event to multiple animals and adjust details on the fly! The page is designed for keyboard-only activity to speed up your data entry tasks and limit the number of mouse clicks needed. As you add animals to the event, details from the last animal are copied to the next line to make entry even faster. And, \{\{masterVAS-Products.Pulse-only}} saves any draft entries if you accidentally navigate away from the page to help avoid data loss!

Use \{\{Buttons-UI.FeaturesSecondary}} to consolidate all required animal activities for the current day, including all \{\{masterVAS-Terminology.ovsynch}}\{\{Protocols.Protocol-common}}s, wellness checks, and any other scheduled \{\{Worklists.taskCommonUse}}s. Use [Task Builder](../../Features/DairyComp/Tasks/BuildTaskLists.htm) to generate a custom \{\{Worklists.taskCommonUse}} list, or choose one of your saved lists from the [Task Lists Gallery](../../Features/DairyComp/Tasks/TaskListsGallery.htm).
