You can enable and configure an HTTPs proxy for the Device Management, Authentication, and Licensing components.

Consider setting up an HTTPs proxy when you need to activate or setup authentication for a Litmus Edge device without internet access.&#x20;

# Setup an HTTPs Proxy

Add an HTTP/HTTPs Proxy using the following syntax:

`http://proxy-name.com:port`

- where `proxy-name.com` is the name of the additional Litmus Edge instance and `port` is the port number on which the proxy is listening.

To setup an HTTPs proxy:

1. Navigate to **System&#x20;**> **Network&#x20;**&#x61;nd select the **Proxy&#x20;**&#x74;ab.
2. Click **Add Proxy** in one of the available components tiles, for example Device Management.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/8tLk5qtYNFaasE51nocHa_https-proxy.png" size="80" width="1566" height="673" position="flex-start" showCaption="false" indent="2"></Image>

3. Click **Save**.
