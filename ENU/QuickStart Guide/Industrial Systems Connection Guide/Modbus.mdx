<hint type="info">
  **Note**: By default, the Modbus drivers monitor holding address 0 for connection status. If the Modbus device does not have holding address 0, then it will not show the "connected" status. You will still be able to read data from the device.
</hint>

Here are the major differences for the Modbus drivers:

| **Driver**                                                | **Use**                                                        |
| --------------------------------------------------------- | -------------------------------------------------------------- |
| [Enron Modbus](<Modbus/Gen 2 Drivers/Enron Modbus.mdx>)   | Serial connection or TCP connection to an Enron Modbus device. |
| - [Modbus ASCII](<Modbus/Gen 2 Drivers/Modbus ASCII.mdx>) |                                                                |

- [Modbus ASCII Advanced](<Modbus/Gen 2 Drivers/Modbus ASCII Advanced.mdx>)                                                                                                                    | Serial connection to a device using Modbus ASCII. Use the Advanced driver for a multi-drop network with multiple Station IDs.                                                                               |
  \| \* [Modbus RTU](<Modbus/Gen 2 Drivers/Modbus RTU.mdx>)
- [Modbus RTU Advanced](<Modbus/Gen 2 Drivers/Modbus RTU Advanced.mdx>)
- [Modbus Multi-Station RTU to TCP Converter](<Modbus/Gen 2 Drivers/Modbus Multi-Station RTU to TCP Converter.mdx>)  | Serial connection to a device using Modbus RTU. Use the Advanced driver for a multi-drop network with multiple Station IDs. Use the Multi-Station Driver if you want to reuse tag names across Station IDs. |
  \| - [Modbus RTU over TCP](<Modbus/Gen 2 Drivers/Modbus RTU over TCP.mdx>)                                                                                                                                                                                   | TCP connection to a Modbus server which has one or more Modbus RTU connections.                                                                                                                             |
  \| \* [Modbus TCP](<Modbus/Gen 2 Drivers/Modbus TCP.mdx>)                                                                                                                                                                                                     | TCP connection to a Modbus device.                                                                                                                                                                          |
  \| - [Modbus UDP](<Modbus/Gen 2 Drivers/Modbus UDP.mdx>)                                                                                                                                                                                                     | UDP connection to a Modbus device.                                                                                                                                                                          |
