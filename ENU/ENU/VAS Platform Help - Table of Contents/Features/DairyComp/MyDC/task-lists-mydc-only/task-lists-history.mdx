The \{\{Buttons-UI.FeaturesTertiary}} page lists tasks completed for quick and easy access.

You can find this feature under \{\{Buttons-UI.FeaturesMain}} > \{\{Buttons-UI.FeaturesSecondary}} > \{\{Buttons-UI.FeaturesTertiary}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/jwTAViIbbJMY_t9TBdzGS_task-list-history.png" size="77" isUploading="false" initialPath="../../../../Resources/Images/Platform/tasks/task-list-history.png" githubPath="Content/Resources/Images/Platform/tasks/task-list-history.png" position="flex-start" showCaption="false"></Image>

Follow the guidelines below when working with the history page:

1. Use **Filters** to limit results by Task Type or Users.
2. Click \{\{Buttons-UI.FeaturesTertiary}} to build and add a new task list to the gallery. See [Build a Task List](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Task Lists (MyDC only)/build-a-task-list.mdx>) to learn more about creating task lists.
3. Click the task list name to view details including when the task list was assigned, the assignment criteria, and the task count (or how many animals were assigned the task):
