## Enhancements and Improvements

<table isTableHeaderOn="true" selectedColumns="" selectedRows="" selectedTable="false" columnWidths="661">
  <tr>
    <td>
      Description
    </td>
  </tr>
  <tr>
    <td>
      **DeviceHub:**
      - SQL driver query fields have increased character limits to allow for larger and more complex queries.
      - Various improvements have been made for the ARM 64 builds.
      - MQTT Generic and MQTT Generic over SSL connector have been removed from Litmus Edge with this release and replaced the Gen2 connectors.
    </td>
  </tr>
  <tr>
    <td>
      **Analytics:**
      - Now JSON and Array datatype parameters and variables are exported from template downloads as JSON objects and arrays. This make it easier for you to edit JSON parameters.
    </td>
  </tr>
  <tr>
    <td>
      **Applications:**
      - You can now easily add multiple headers in text format in the **HTTPs Proxy** dialog.&#x20;
    </td>
  </tr>
  <tr>
    <td>
      **Integrations:**
      - Validation has been improved when entering a source path for a cloud storage sync job.
    </td>
  </tr>
  <tr>
    <td>
      **System:**
      - Improved user experience with 2FA setup.
      - Added missing logs in Docker support bundle.
    </td>
  </tr>
</table>
