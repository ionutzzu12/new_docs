  ### Required Permissions

  Only users who are assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions can use this feature. See [Edit Dairy User Permissions](EditUserPermissions.htm) and Manage Dairy Roles for more information.
</ExpandableHeading>

This topic explains how to view and update \{\{masterVAS-Terminology.d-airy}} user roles.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/R7Anc1TQ8M9wD5rZhdf3p_roles-dairy.png" size="77" isUploading="false" initialPath="../../../../Resources/Images/Platform/users-roles/roles-dairy.png" githubPath="Content/Resources/Images/Platform/users-roles/roles-dairy.png" position="flex-start" showCaption="false"></Image>

You can find this feature under the :Link[Dairy menu]{label="Dairy menu" overridedLabel="Dairy menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-Dairy" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-Dairy" href="../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}. The **Roles** tab contains the following information:

- All roles that are currently available for this \{\{masterVAS-Terminology.d-airy}}.
- The number of permissions that are currently assigned to each role.
- The number of users assigned to each role.
- The status of each role (active/inactive).

\{\{masterVAS-Terminology.D-airy-}} roles are assigned to \{\{masterVAS-Terminology.d-airy}} users on the **Users** tab. See [Edit Dairy User Permissions](EditUserPermissions.htm) for detailed instructions. If you need to view company user roles, see :Link[View Company User Roles]{label="View Company User Roles" overridedLabel="View Company User Roles" spaceId docId="ZBxNM9nH4pjlw1ThsUy51" version="v2" docAnchorId="#View_Company_Roles" loadingMethod="dynamic" newTab="false" githubPath="../../Company/CompanyUserMgmt.htm#View_Company_Roles" href="../../Company/CompanyUserMgmt.htm#View_Company_Roles"}.

Follow the steps below to continue:

1. On the **Roles** tab, expand the details of a role to view or update the specific permissions that are assigned to it.To learn more about permissions, see [Default Roles and Permissions](Default-Roles-Permissions.htm) and [Permission Descriptions](PermissionDescriptions.htm).
2. Click the **Edit** button to edit permissions for the roleORClick the **Disable** button to deactivate the role.:::ExpandableHeading

### Can't find this feature? Make sure it's included in your subscription!

<Paragraph indent="1">
  This feature/option is available only for \{\{masterVAS-Products.DCU-users-short}}. \{\{masterVAS-Products.DCP-users-short}} will not see it in \{\{masterVAS-Products.Pulse-only}}.
  Click [here](../../../GetStarted/KnowYourPulse.htm) to learn more about the different versions of \{\{masterVAS-Products.Pulse-only}}.
</Paragraph>

<hint type="info" indent="2">
  **NOTE:&#x20;**&#x42;e aware of \{\{Zone}} "write" permissions as you are editing roles. The zone write :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/RAsWMD-IsbPX_etOV-z1q_dairy-zones-write-permissions.png" alt caption size="10"} icon is displayed for any permission level that requires zone selections. When you edit a permission level with this icon, you must also select which zones can be edited by users who are granted that permission.
  \:::
</hint>

3. \{\{Commands.Click}} the **Save** button at the bottom right side of the page.
