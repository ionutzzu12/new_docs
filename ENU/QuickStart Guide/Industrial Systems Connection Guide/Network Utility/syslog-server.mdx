## Tag Parameters

- **Name**: Select a register name from the drop-down list. The available options depend on the names in the register table.&#x20;
- **Value Type**: Select a data type from the drop-down list. The available options depend on the register name selected.&#x20;
- **Tag Name**: Enter a name for the tag.
- **Description** (Optional): Enter a description for the tag. &#x20;
- **Severity**: Select the severity limit.&#x20;
- **Facility**: Select the facility.&#x20;
- **Wildcard**: Select the wildcard character. &#x20;
- **Hostname**: Configure the hostname (String Match).&#x20;
- **Appname**: Configure the appname (String Match).&#x20;
- **Msg ID**: Configure the MessageID (String Match).&#x20;
- **Message**: Configure the Message (String Match).&#x20;
- **Structured Data**: Configure the Structured Data (Field Match).&#x20;
- **Client Address**: Configure the Client CIDR Address (Address Match).&#x20;
- **Order**: Enter a value for match process ordering.&#x20;
- **Format**: Configure the JSON Publish Format. This is not available if you select the Name *Ignore Match (No Publish)*.&#x20;
