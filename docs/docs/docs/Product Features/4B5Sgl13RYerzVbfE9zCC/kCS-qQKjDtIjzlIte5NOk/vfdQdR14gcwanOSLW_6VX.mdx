You can manage flow groups in the Analytics canvas by navigating to **Analytics&#x20;**>**&#x20;Instances**.&#x20;

Refer to the following management options below in the Analytics canvas.&#x20;

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/vjPB89FiaXzVr9hTzvNn-_image.png" size="80" width="756" height="458" caption="Management options for flow groups" position="center" showCaption="true"></Image>

| **Action**                      | **Details**                                                                                                                 |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| \[1] **Lock/Unlock flow group** | You can lock groups so that other users can't make changes to them. Groups default to a locked state when you save the map. |
| \[2] **Pin group as default**   | You can pin a group so that it becomes the default group on the Analytics canvas.                                           |
| \[3] **Create group**           | See [Create Flow Groups](<../../../../../Product Features/Analytics/Analytics Flow Groups/2C86_sd3cK1sDEmBsC9V0.mdx>).      |
| \[4] **Edit group name**        | Modify the name of the group.                                                                                               |
