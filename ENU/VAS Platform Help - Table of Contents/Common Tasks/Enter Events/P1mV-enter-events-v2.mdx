<ExpandableHeading>
  ### Required Permissions

  Only users who are assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions can use this feature.If your account only has \{\{RolesPermissions.PermissionNames}} permissions, you will **not** be able to use this feature. See [Edit Dairy User Permissions](../Features/Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](<../../Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

\{\{masterVAS-Products.Pulse-only}} allows you to create events, add animals to them, and process your changes—all from the mobile app while you are out on the \{\{masterVAS-Terminology.d-airy}}. Add events to one or more animals, either by searching for them manually or using [RFID Scanners (v2)](../DC-Mobile/Scanners/Scanners.htm). When you finish adding animals to an event, tap

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/v5wfvQAcJhnIgw0iXuFul_process-all-changes-icon.png" size="10" caption="Process All Changes button" alt="Process All Changes button" isUploading="false" initialPath="../../Resources/Images/Platform/dc-mobile-img/events-img/process-all-changes-icon.png" githubPath="Content/Resources/Images/Platform/dc-mobile-img/events-img/process-all-changes-icon.png" position="flex-start" showCaption="true"></Image>

\{\{Mobile-UI.EventProcessChangesv2}} to send your updates back to \{\{masterVAS-Products.Pulse-only}} (\{\{Mobile-UI.ConnectionRequired}}).

Expand the topics below to learn more about entering animal events in the mobile app:

  <ExpandableHeading indent="2">
    ### Click here to learn more about REM field length and quarter/quadrant selection!

    The length of the remark field may be 16 or 8 characters, depending on your \{\{masterVAS-Terminology.d-airy}} type/item table \{\{masterVAS-Terminology.c-onfiguration}}. \{\{masterVAS-Products.MyDC-short}}\{\{masterVAS-Terminology.d-airies}} support 16-character remarks. \{\{masterVAS-Products.DCUnlimited-short}}\{\{masterVAS-Terminology.d-airies}} can support 16-character remarks if they have adequate item table storage space. If a \{\{masterVAS-Products.DCUnlimited-short}}\{\{masterVAS-Terminology.d-airy}} has limited item table storage space, it can only support 8-character remarks. Follow the guidelines below and learn how quarter/quadrant selection works for specific events:

    - \{\{Events.30MAST}} and \{\{Events.32LAME}} events:\* 16-character REM: Choose 1, 2, 3, or all quarters/quadrants for MAST and LAME events.
      - 8-character REM: Choose 1 or all quarters/quadrants for MAST and LAME events.
    - \{\{Events.21CULTURE}} events: Choose either 1 or all quarters to follow best practices for this event type, irrespective of the REM field length.
  </ExpandableHeading>

  <Paragraph indent="1">
    The event date defaults to \{\{today}}, so if you’re adding events for previous days, be sure to adjust the date.

  ## Copy Event Details to Multiple Animals

  To speed up entering an event with similar details to multiple animals, you can copy event details. Follow the steps below to continue:

  1. Enter an event for the first animal. That activates the settings button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/WKLpWGZY4hislOAmGD0ay_gear-green.png" size="10" caption="Settings button" alt="Settings button" isUploading="false" initialPath="../../Resources/Images/icons/gear-green.png" githubPath="Content/Resources/Images/icons/gear-green.png" position="flex-start" showCaption="true" indent="2"></Image>

  <Paragraph indent="1">
    at the top right of the screen.
  </Paragraph>

  2. Tap the settings button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/WKLpWGZY4hislOAmGD0ay_gear-green.png" size="10" caption="Settings button" alt="Settings button" isUploading="false" initialPath="../../Resources/Images/icons/gear-green.png" githubPath="Content/Resources/Images/icons/gear-green.png" position="flex-start" showCaption="true" indent="2"></Image>

  4. If you need to adjust event details for an animal, tap the animal and make necessary edits. Manually-entered data overrides copied details.
  5. When you're done, tap the back arrow and then the

  - When you're done adding all animal events and required details, tap the back arrow and then the

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/v5wfvQAcJhnIgw0iXuFul_process-all-changes-icon.png" size="10" caption="Process All Changes button" alt="Process All Changes button" isUploading="false" initialPath="../../Resources/Images/Platform/dc-mobile-img/events-img/process-all-changes-icon.png" githubPath="Content/Resources/Images/Platform/dc-mobile-img/events-img/process-all-changes-icon.png" position="flex-start" showCaption="true" indent="2"></Image>

  - For \{\{masterVAS-Products.MyDC-short}}\{\{masterVAS-Terminology.d-airies}}, the app sends the events to \{\{masterVAS-Products.Pulse-only}} as soon as your mobile connection is active, and the events are immediately added to the animal \{\{masterVAS-Terminology.cowcard}}s. For \{\{masterVAS-Products.DCUnlimited}}\{\{masterVAS-Terminology.d-airies}}, you may see a slight delay as the latest \{\{care package}} is pulled down to \{\{masterVAS-Products.Pulse-only}} and \{\{masterVAS-Products.DC305-short}}.
