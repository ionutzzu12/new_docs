In the *Registry&#x20;*&#x70;ane of the *Applications&#x20;*&#x6D;odule, you can add, modify, or remove registries. A registry is a public or private repository of Docker images used for storing and delivering applications.

# Access Applications Registry UI

**To access the Applications Registry pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **Applications**.
   The *Dashboard&#x20;*&#x70;ane appears.
3. Click **Registry**.&#x20;
   The *Registry&#x20;*&#x70;ane appears.
