NEW FEATURE Import \{\{masterVAS-Terminology.s-ire}}, maternal pedigree, animal identification, and genomic data from a CSV! The new \{\{General.ProductName-short}}**Update Animal Data** feature offers the ability to import \{\{masterVAS-Terminology.s-ire}} mating, maternal pedigree, animal identification, and genomic data using a CSV file from any provider. This additional flexibility extends the power of \{\{General.ProductName-short}}\{\{masterVAS-Terminology.h-erd}} management capabilities for reporting and tracking purposes and helps you leverage critical animal data from multiple sources.

See [Update Animal Data](<../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Animals/update-animal-data.mdx>) to learn more, or watch the :Link[Update Animal Data tutorial video]{label="Update Animal Data tutorial video" overridedLabel="Update Animal Data tutorial video" spaceId docId="Lc_--B1HNccB_LtHX9Yd2" version="v2" docAnchorId="#video" loadingMethod="dynamic" newTab="false" githubPath="../../Features/DairyComp/Animals/UpdateAnimalData.htm#video" href="../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Animals/update-animal-data.mdx"} to walk through the process!

ENHANCEMENTEmbryo data entry improvements and Link to Donor added! The Embryo Management feature includes multiple improvements: 

- If the donor for the embryo is in the \{\{masterVAS-Terminology.h-erd}}, you have the option to link to the donor animal for quick and convenient access! After the embryo is imported, click the Donor link to view the animal's details in a side panel without having to leave the Embryo Management page.
- When adding embryos in batch, the template file now shows an error if the grade is outside the expected 1-4 range.
- \{\{masterVAS-Products.Pulse-only}} now limits the number of \{\{masterVAS-Terminology.f-resh}} or frozen embryos to 2000 in a single load to avoid potential data entry errors. See [Add Embryos to Inventory or Transferred Embryos](../../Features/DairyComp/Embryo/EmbryoAdd.htm) to learn more about adding embryos!

## Bug Fixes

- Command Line fix for metric units. \{\{masterVAS-Products.Pulse-only}} Command Line now understands commands if your user account is set to metric units.
- Reproduction Insights date fix. The [Pregnancy Risk](../../Features/Insights/Reproduction/Reproduction-PregnancyRisk.htm) page now shows dates in the correct format.
- Integration toggle fix. The Connection Active toggle stays turned on for connected integrations.
- \{\{masterVAS-Terminology.P-en-}} capacity fix. The tip that provides \{\{masterVAS-Terminology.p-en}} capacity percentage details on the \{\{masterVAS-Terminology.P-ens-}} List was fixed and is now easier to read.
