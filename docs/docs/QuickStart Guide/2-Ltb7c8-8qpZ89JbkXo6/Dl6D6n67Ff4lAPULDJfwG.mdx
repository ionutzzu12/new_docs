You have the option of connecting an [MQTT Client](<../../../QuickStart Guide/Industrial Systems Connection Guide/MQTT/pNpchH26RoMtYSAOY_ihz.mdx>) and [Industrial OT Server](<../../../QuickStart Guide/Industrial Systems Connection Guide/MQTT/El-BV893OAOKdaDcBfDSd.mdx>) as DeviceHub devices.

<hint type="info">
  **Note**: You can't connect the [MQTT Client](<../../../QuickStart Guide/Industrial Systems Connection Guide/MQTT/pNpchH26RoMtYSAOY_ihz.mdx>) to the DeviceHub [Industrial OT Server](<../../../QuickStart Guide/Industrial Systems Connection Guide/MQTT/El-BV893OAOKdaDcBfDSd.mdx>) if they are using the same Litmus Edge instance. This includes connecting through the Flows Manager or through an Integration connector. If they are using different Litmus Edge instances, they can connect.
</hint>
