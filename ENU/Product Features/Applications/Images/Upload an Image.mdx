You can upload an image using FTP or from your hard drive.

# Before You Begin

- Make sure you have created a registry for the image you are uploading. See [Add a Registry](<././Registry/Add a Registry.md>) to learn more.&#x20;
- Ensure you have enough disk space (See *Data Partition Usage* section of [Info](https://app.archbee.com/docs/JWQMp_kvm0WsA_Fpjt1v_/YIwyPagdCmgYF_px_-wgA)) before pulling any new Docker images onto a Litmus Edge and its Catalog Apps. Certain Docker images may exceed the disk size of your Litmus Edge instance.

# Upload an Image

**To upload an image from your hard drive:**

1. Navigate to **Applications&#x20;**> **Images**.
   The *Images&#x20;*&#x70;ane appears.
2. Click the **Upload Image (+)&#x20;**&#x69;con.&#x20;
   Th&#x65;*&#x20;Import Image* dialog box appears. &#x20;
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/yiwSnitHqDyUA803nP8ld_uploadimageicon.png "Upload image icon")

Select one of the options below: "Upload an Image from your Hard Drive" or "Upload an Image Using FTP".&#x20;

## Upload an Image from your Hard Drive

**From the****\* Import Image****\* dialog box:&#x20;**

1. Select **Upload**.&#x20;
2. Click **Select Docker Image**.&#x20;
3. Navigate to the .tar image file on your hard drive and click **Open**.&#x20;
4. Click **Import**.&#x20;

## Upload an Image Using FTP

**From the****\* Import Image****\* dialog box:&#x20;**

1. Select **FTP**.&#x20;
2. Click **Select Docker Image**.&#x20;
3. Enter the file path for the .tar image file in the **Image File Path&#xA0;**&#x66;ield.&#x20;
4. Click **Import**.&#x20;
