<hint type="info">
  Le module \{\{FRS\_VARS.Announcements}} n'est pas disponible clé en main (OOTB). Cet espace de travail apparaît uniquement si votre administrateur l'a configuré pour votre rôle.
</hint>

L'espace de travail \{\{FRS\_VARS.Announcements}} vous permet d'afficher les messages et alertes les plus récents, comme les avis de coupure de service, les planifications de maintenance, les notifications de mise à niveau, les menaces virales et autres types de bulletin partagés dans toute votre entreprise.

Vous pouvez afficher les annonces. Seul l'\{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator}} peut les créer ou les modifier.

<ExpandableHeading>
  ## Affichage des annonces

  1. Connectez-vous à \{\{FRS\_VARS.ProductName}}.
  2. Dans le menu de gauche, sélectionnez \{\{FRS\_VARS.Announcements}}.

  La liste \{\{FRS\_VARS.Announcements}} s'affiche.

  3. Cliquez sur une entrée \{\{FRS\_VARS.Announcements}} pour afficher les informations suivantes :
     - Description
     - Urgence
     - Région
     - Créé par
     - Date d'entrée en vigueur
     - Date d'expiration
     - Créé le
</ExpandableHeading>

<ExpandableHeading>
  ## Rechercher une entrée Annonces

  1. Dans l'espace de travail \{\{FRS\_VARS.Announcements}}, accédez au champ **Rechercher**, puis entrez un mot-clé lié à la valeur \{\{FRS\_VARS.Announcements}} voulue.
  2. Appuyez sur **Entrée**.

  L'espace de travail affiche l'entrée \{\{FRS\_VARS.Announcements}} correspondant à votre mot-clé.
</ExpandableHeading>

<ExpandableHeading>
  ## Filtrer et trier les annonces

  1. Dans l'espace de travail \{\{FRS\_VARS.Announcements}}, cliquez sur **Filtrer et trier** dans la liste déroulante.
  2. Choisissez l'un des ordres de tri suivants :
     - **Croissant**
     - **Décroissant**
  3. Choisissez le champ à utiliser pour le tri :
     - **Sujet**
     - **Urgence**
     - **Date/Heure de création**
     - **Description**
     - **Région**
  4. Cliquez sur **Appliquer** pour trier la liste.
  5. Pour supprimer le tri, cliquez sur **Tout réinitialiser**.
</ExpandableHeading>

<ExpandableHeading>
  ## Utiliser un filtre de recherche favori

  <hint type="info">
    Cette fonction est disponible uniquement si votre administrateur l'a activée.
  </hint>

  1. Dans l'espace de travail \{\{FRS\_VARS.Announcements}}, cliquez sur **Tout** dans la liste déroulante.
  2. Choisissez des annonces spécifiques, comme **Annonces urgentes**, **Annonces RH** ou **Annonces publiées**, pour filtrer la liste.

  L'affichage de l'espace de travail s'ajuste à votre sélection.
</ExpandableHeading>
