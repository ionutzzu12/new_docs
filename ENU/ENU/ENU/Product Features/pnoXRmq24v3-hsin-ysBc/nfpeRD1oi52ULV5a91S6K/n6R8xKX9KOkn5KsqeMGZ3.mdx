You can upload Machine Learning models by navigating to **Analytics&#x20;**>**&#x20;Models**.&#x20;

<hint type="info">
  **Note**: The maximum model file size is [52.43MB or 50 MiB](https://en.wikipedia.org/wiki/Byte#Multiple-byte_units).
</hint>

# Before You Begin

Make sure you create an appropriate machine learning model. See :Link[Machine Learning Models]{docId="B-kTNYS3tkFCR9bgjB1O9" label="Machine Learning Models" spaceId version="v2" docAnchorId="#create-a-machine-learning-model-for-upload" overridedLabel="Machine Learning Models" href="../../../../Product Features/pnoXRmq24v3-hsin-ysBc/nfpeRD1oi52ULV5a91S6K.mdx"} for details.&#x20;

# Upload a Model

**To upload a model:**

1. Navigate to **Analytics&#x20;***>&#x20;***Models**.&#x20;
2. Click **Upload Model**.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/t5dgo8ZdXIOuzlqndjWxw_uploadmodel.png "Upload Model icon")
3. Navigate to the zip file of the model and click **Open**.&#x20;
   The selected model appears in the *Models&#x20;*&#x70;ane.&#x20;

To remove the model, open the **Action menu** for the model and select **Remove**.&#x20;
