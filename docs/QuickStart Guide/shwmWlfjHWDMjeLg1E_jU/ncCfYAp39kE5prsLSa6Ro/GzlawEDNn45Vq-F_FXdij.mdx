To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.

**To configure your device:**&#x20;

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Yokogawa FA M3 F3SP25-SN
   - **Communication I/F**: Ethernet. Refer to the following figure.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/pGOpRiViey1zBraKAPnx__yokogawafam3ethernetdevice.png" size="80" width="675" height="744" position="flex-start" showCaption="false" indent="2"></Image>

2. Launch WideField3 v3.02 software and create a project with the device model.&#x20;
3. Click **Tools > Set up Environment**.
4. Configure the network parameters.&#x20;
5. Click **Online > Connect&#x20;**&#x61;nd verify the connection is successful.
