Use the **Task Lists&#x20;**\{\{Buttons-UI.GalleryTab}} to manage all the task lists that you have created for your \{\{masterVAS-Terminology.d-airy}} — you can schedule task lists, preview which animals would be included on a \{\{task list}}, edit your task lists, and more.

You can find this feature under \{\{Buttons-UI.FeaturesMain}} > \{\{Buttons-UI.FeaturesSecondary}} > \{\{Buttons-UI.FeaturesTertiary}}.

Follow the guidelines below when using the gallery:

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/2PfhH4jQVRiYuaZvWU5ez_build-task-list.png" size="76" isUploading="false" initialPath="../../../../Resources/Images/Platform/tasks/build-task-list.png" githubPath="Content/Resources/Images/Platform/tasks/build-task-list.png" position="flex-start" showCaption="false"></Image>

1. Click \{\{Buttons-UI.FeaturesTertiary}} to see all your task lists.
2. Click \{\{Buttons-UI.FeaturesTertiary}} to build and add a new task list to the gallery. See [Build a Task List](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Task Lists (MyDC only)/build-a-task-list.mdx>) to learn more about creating task lists.
3. Use the **Schedule** option to set the Task List to run at a specific time and/or a on a daily or weekly basis.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/J0gciZ_l_StjRWPIZB8t-_schedule-task-list.png" size="44" isUploading="false" initialPath="../../../../Resources/Images/Platform/tasks/schedule-task-list.png" githubPath="Content/Resources/Images/Platform/tasks/schedule-task-list.png" position="flex-start" showCaption="false" indent="2"></Image>

<Paragraph indent="1">
  Click **See Results** on any task list to see the list of animals that would be eligible for the task. Note that this doesn't build the task list, it's just a preview — you must schedule a task list to build the task list.
</Paragraph>

5. Select any task list's Options button

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/CDT7MGgVdxUe43MtlyWvh_options-black-dots.png" size="10" isUploading="false" initialPath="../../../../Resources/Images/icons/options-black-dots.png" githubPath="Content/Resources/Images/icons/options-black-dots.png" position="flex-start" showCaption="false" indent="2"></Image>

<Paragraph indent="1">
  in the upper right hand corner to **Edit**, **Duplicate**, or **Delete** the task list from the gallery.
</Paragraph>
