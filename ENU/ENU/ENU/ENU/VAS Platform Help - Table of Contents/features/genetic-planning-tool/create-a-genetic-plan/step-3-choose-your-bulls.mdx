The third step, Choose Your Bulls, provides you with options to choose mating \{\{masterVAS-Terminology.s-ires}} using multiple filters. You can also use this step to add any existing semen inventory to your genetic plan or set a bull's semen type.

Click the arrow to view a bull's trait data and other details, or click a bull's \{\{ID}} to view it's profile in a side panel.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ORg-1xNPQQj94QITD4A4L_genex-step3-main.png" size="43" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-step3-main.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-step3-main.png" position="flex-start" showCaption="false"></Image>

Expand the topics below to learn more about choosing your bulls, including additional bulls, setting dose limits, setting a bull's semen type, or viewing a genetic summary:

  ## Choose Your Bulls

  Follow the steps below to continue:

  1. Set your filters. You have multiple filter options::::ExpandableHeading

  ### Selection

  <Paragraph indent="1">
    Use this filter to display only the \{\{masterVAS-Terminology.s-ires}} you manually selected from the listing on the right side of the page. Quickly locate \{\{masterVAS-Terminology.s-ires}} using the Search field to find bulls by \{\{NAAB}} code, Name, IntRegNum, or country-specific ID. Then, \{\{Commands.click}} the checkboxes for the bulls you'd like to select. You can expand individual bulls to view their trait values. See :Link[GENEX PTA and Trait Data]{label="GENEX PTA and Trait Data" overridedLabel="GENEX PTA and Trait Data" spaceId docId="6GDG6vfN8_CVeBmQ-ng_r" version="v2" docAnchorId="#traits-list" loadingMethod="dynamic" newTab="false" href="step-3-choose-your-bulls.mdx"} below for a listing of the details that are included for each bull.
  </Paragraph>

    ### Semen Type

    \{\{Commands.Click}} the **Sexed** checkbox to include only \{\{UDS-general.UDSBrand}} sexed semen products.

    <ExpandableHeading>
      ### Product Line

      Filter bulls by \{\{UDS-general.UDSBrand}} product line.

      <ExpandableHeading>
        ### Traits

        Filter by specific traits and trait values. You can add a minimum or maximum value for each trait, or add both to set a range.

        <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/oii7xmulv3ghffPoFnNgh_filter-traits.png" size="33" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/filter-traits.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/filter-traits.png" position="flex-start" showCaption="false"></Image>

        <hint type="info">
          **NOTE:&#x20;**&#x49;f necessary, the trait value can be set to a negative value to provide additional control and flexibility in filtering bulls by specific traits. Add a - (hyphen) before the number to set a negative value.

        Click the delete icon

        <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/xsdSlPbnf44rgj428QR0Y_blue-delete.png" size="10" caption="delete icon" alt="delete icon" isUploading="false" initialPath="../../../Resources/Images/icons/blue-delete.png" githubPath="Content/Resources/Images/icons/blue-delete.png" position="flex-start" showCaption="true"></Image>

        <ExpandableHeading>
          ### Indexes

          Filter by numeric Index values for the bulls. You can add a minimum or maximum value for each trait, or add both to set a range.
          \::::Variable\{key="Commands.Click" type="variable"}**Clear filters** to remove all filters and start over.

      </ExpandableHeading>

  ## Include Additional Bulls

  <hint type="info">
    **NOTE:&#x20;**&#x59;ou can manually add bulls if you \{\{did}} not find them in \{\{UDS-general.UDSBrand}} \{\{UDS-general.UDSProduct}}, as well as edit existing bulls that you have manually added or view existing bulls that others have manually added.   </hint>

  To add bulls to your plan that are not on the \{\{UDS-general.UDSBrand}} marketing list, use the Options button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wIXwfkZ07oPcd_rKpJE8J_options-green-small41x29.png" size="10" caption="Options button" alt="Options button" isUploading="false" initialPath="../../../Resources/Images/icons/options-green-small_41x29.png" githubPath="Content/Resources/Images/icons/options-green-small_41x29.png" position="flex-start" showCaption="true"></Image>

  and select **Include Additional Bulls**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/zlM83KpLbuFqQ9tLIAaBs_step4-options-menu.png" size="39" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/step4-options-menu.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/step4-options-menu.png" position="flex-start" showCaption="false"></Image>

  Use the side panel to find the bull by NAAB code, Name, International Registration Number (IntRegNum), or country-specific ID and \{\{Commands.click}}**Save** to include the bull in your plan.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Z9d4AYgY1InA7_vpz3mN4_step4-add-bulls.png" size="70" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/step4-add-bulls.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/step4-add-bulls.png" position="flex-start" showCaption="false"></Image>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/GMy9sOO0nh4bKBDdcciE3_trashcan-blue-outline-on-white-2.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/trashcan-blue-outline-on-white-2.png" githubPath="Content/Resources/Images/icons/trashcan-blue-outline-on-white-2.png" position="flex-start" showCaption="false"></Image>

  ## Set a Bull's Semen Type

  Click a bull's ID to open its side panel and check the Conventional and/or Sexed checkboxes to set the **Semen Type**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tHSnWIgEx0nN5kwRu2wI__genex-bull-sidepanel.png" size="32" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-bull-sidepanel.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-bull-sidepanel.png" position="flex-start" showCaption="false"></Image>

  <hint type="info">
    **NOTE:&#x20;**&#x54;his setting only applies to the current genetic plan and will reset to the default setting in the marketing list when you create a new plan.
  </hint>
</ExpandableHeading>
