You can add users to Litmus Edge by navigating to **System&#x20;**>**&#x20;Access Control&#x20;**&#x66;rom th&#x65;**&#x20;Users&#x20;**&#x74;ab.&#x20;

<hint type="warning">
  **Important**: 

  - Administrators should email credentials to new users. These credentials are temporary, and users need to change their password when they first log in to Litmus Edge.&#x20;
  - New users need to belong to at least one group to be able to log in to Litmus Edge.&#x20;
</hint>

**To add a user:**

1. Navigate to **System&#x20;***>&#x20;***Access Control**.&#x20;
2. From the *Manage Users* section, click the **Add User** icon.
   The *Add User&#x20;*&#x64;ialog box appears.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/n3aFMz7WznFPk135nTr5D_add-user.png)
3. Enter the details for the user.&#x20;
   - First name
   - Last name
   - Username
   - Password
4. Use the drop-down list for **Select Groups&#x20;**&#x74;o add the users to groups. Add as many groups as required. If no groups are assigned, the user will not be able to log in to Litmus Edge.&#x20;
5. Clic&#x6B;**&#x20;Add User**.
   The user is added to the *Users&#x20;*&#x70;ane.&#x20;
