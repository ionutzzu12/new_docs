The Integration module enables you to configure data connections with third-party cloud service providers, databases, MQTT message brokers, and more.&#x20;

You can:

- Publish data (outbound topics) from Litmus Edge to third-party data connections.
- Subscribe to data (inbound topics) from third-party data connections to Litmus Edge.
- Subscribe to a cloud storage service (AWS S3, Azure Blob Storage, Google Cloud Storage, Databricks Unity Catalog Volume)

# Integration Connectors

Refer to [Integration Connectors](<../docs/Product Features/lBL7doO8dZVjlOPVnrAkz/D8fdLNFu4aGD2K_yaXuH5.mdx>) to review available third-party data connections.&#x20;

You can enable database apps in the Applications Marketplace. Database connections allow table names in mixed cases.

# Access Integration UI

**To access the Integration pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **Integration**.
   The *Integration&#x20;*&#x70;ane appears.
