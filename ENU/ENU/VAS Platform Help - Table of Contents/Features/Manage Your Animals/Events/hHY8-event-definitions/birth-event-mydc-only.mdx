The \{\{Events.3003BIRTH}} event is a \{\{masterVAS-Products.MyDC-short}}-only event that is automatically created with a \{\{Events.1FRESH}} event and added to the calf's \{\{masterVAS-Terminology.cowcard}}. Changes to the \{\{Events.3003BIRTH}} event can only be made from the \{\{masterVAS-Terminology.d-am}}'s \{\{Events.1FRESH}} event.

<ExpandableHeading>
  ## Event Guidelines

  Use the mobile app or the web version of \{\{masterVAS-Products.Pulse-only}} to enter events! Tap \{\{Mobile-UI.EventsFeature}} in the [mobile app](<../../../../../../VAS Platform Help - Table of Contents/mobile-apps.mdx>) to get started, or see :Link[Add Events]{label="Add Events" overridedLabel="Add Events" spaceId docId="sp1fQUQTHg-WLIRXAer4X" version="v2" docAnchorId="#Add" loadingMethod="dynamic" newTab="false" githubPath="Events.htm#Add" href="Events.htm#Add"} to learn more about adding events from web. \{\{masterVAS-Products.Pulse-only}} supports adding events that have occurred within the last year.

  Note the guidelines below when working with the \{\{Events.3003BIRTH}} event: 

  - A \{\{Events.3003BIRTH}} event cannot be manually added to a calf; it is automatically added as part of a \{\{Events.1FRESH}} event for the \{\{masterVAS-Terminology.d-am}}.
  - Details that are prompted for the \{\{Events.3003BIRTH}} event are set in :Link[Items to be Prompted for New Calves]{label="Items to be Prompted for New Calves" overridedLabel="Items to be Prompted for New Calves" spaceId docId="BaymhC1urQ-Ad5ucqIccd" version="v2" docAnchorId="#new-calves-items" loadingMethod="dynamic" newTab="false" githubPath="../DairyComp/Settings/DC-LactConfiguration.htm#new-calves-items" href="../DairyComp/Settings/DC-LactConfiguration.htm#new-calves-items"}.
  - If \{\{Events.1FRESH}} event :Link[Optional Details]{label="Optional Details" overridedLabel="Optional Details" spaceId docId="GxXIleQm2y07Vuz9c38iV" version="v2" docAnchorId="#optional" loadingMethod="dynamic" newTab="false" githubPath="../DairyComp/Settings/DC-EventDefinitions.htm#optional" href="../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Settings/General Settings/event-definitions.mdx"} are set to include weight, \{\{masterVAS-Products.Pulse-only}} will prompt for calf birth weight (BWEIGHT) and \{\{masterVAS-Terminology.d-am}} weight (WEIGHT).
</ExpandableHeading>

<ExpandableHeading>
  ## Process Changes

  When you add events from web or the mobile app, your updates are sent to \{\{masterVAS-Products.Pulse-only}} (\{\{Mobile-UI.ConnectionRequired}}).

  The \{\{Events.3003BIRTH}} event is automatically added to the calf after the \{\{Events.1FRESH}} event is processed for the \{\{masterVAS-Terminology.d-am}}. The \{\{EDAY}} for the \{\{Events.3003BIRTH}} event matches the \{\{Events.1FRESH}} EDAY.
</ExpandableHeading>
