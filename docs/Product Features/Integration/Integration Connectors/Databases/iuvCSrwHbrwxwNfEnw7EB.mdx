## Overview

InfluxDB v2.x TCP integration enables an Edge System to connect with an InfluxDB database.

<hint type="info">
  **Note**: The following data types and formats are not viewable in InfluxDB:

  - string
  - char
  - JSON payloads
</hint>

## Supported Versions

- InfluxDB 2.x

## Supported Communication Directions

- Outbound (Write from Edge to Database) - Yes
- Inbound (Read from Database to Edge) - No
