## Options for Setting Up Litmus Edge

You have the following options for setting up Litmus Edge on your site.&#x20;

Learn more about [Installation and Deployments](<././QuickStart Guide/Installation and Deployments.md>).&#x20;

| **Action**   | **Details**                                                                                                            |
| ------------ | ---------------------------------------------------------------------------------------------------------------------- |
| ### Option 1 | Refer to callout **\[1]** in the diagram. &#xA;\* Litmus Edge in a DMZ is connected to PLCs through a Level 2 switch.  |

- Only the "Head" PLC has a network connection in a designated VLAN.&#x20;
- The Litmus Edge in the DMZ must have access to the same VLAN as the "Head" PLC.&#x20;
- The Litmus Edge in the DMZ can read from registers of the "Head" PLC.&#x20;
- Other PLCs attached to the "Head" PLC must transfer their data to the "Head" PLC to be read.  |
  \| ### Option 2 | Refer to callout **\[2]** in the diagram.&#x20;
  \* Litmus Edge in a DMZ is connected to PLCs through a Level 2 switch.&#x20;
- All PLCs have a network connection in a designated VLAN.&#x20;
- The Litmus Edge in the DMZ must have access to the same VLAN as the PLCs.&#x20;
- The Litmus Edge in the DMZ will be able to connect to all PLCs to read data.                                                                                                                     |
  \| ### Option 3 | Refer to callout **\[3]** in the diagram.&#x20;
  \* Litmus Edge in a DMZ is connected to other data sources through a Level 2 switch.
- All data sources have a network connection in a designated VLAN.&#x20;
- The Litmus Edge in the DMZ must have access to the same VLAN as the data sources.&#x20;
- The Litmus Edge in the DMZ will be able to connect to all data sources to read data.                                                                                |
  \| ### Option 4 | Refer to callout **\[4]** in the diagram.&#x20;
  \* Litmus Edge in a DMZ is connected to a PLC through a panel switch.&#x20;
- The panel switch is connected to a Level 2 switch.
- The PLC is connected to the panel switch.&#x20;
- The Litmus Edge in the DMZ must have access to the same VLAN as the PLC.
- The Litmus Edge in the DMZ will read data from any PLC connected to the panel switch.                                                                         |
  \| ### Option 5 | Refer to callout **\[5]** in the diagram.&#x20;
  \* An RS232 to Ethernet converter connects to a PLC through an RS232/484 physical connection.&#x20;
- Litmus Edge can read from the registers on the connected PLC.&#x20;
- The Litmus Edge in the DMZ will read the data from the PLC through the Ethernet converter.                                                                                                                                                        |
  \| ### Option 6 | Refer to callout **\[6]** in the diagram.&#x20;
  \* Litmus Edge in a DMZ connects to a Remote I/O Module/Sensor Gateway.&#x20;
- The Remote I/O Module/Sensor Gateway is connected to the PLC (or in some cases to the Level 2 switch).&#x20;
- Sensors are connected to the gateway through wireless or proprietary technology.&#x20;
- The Litmus Edge reads sensor data through the PLC or through the gateway if it's connected to the Level 2 switch.                     |

## Cloud and Litmus Edge Manager Setup

Refer to the following options for setting up cloud storage and Litmus Edge Manager.&#x20;

### Litmus Edge Manager

Refer to callout **\[7]** in the diagram.&#x20;

Data collected from Litmus Edge is sent to [Litmus Edge Manager](https://docs.litmus.io/edgemanager) for consolidation.&#x20;

### Cloud Storage&#x20;

Refer to callout **\[8]** in the diagram.&#x20;

Litmus Edge Manager sends data to cloud systems or other enterprise systems. Learn more about [Applications](<././Product Features/Applications.md>) and [Applications Guides](<././How-To Guides/Applications Guides.md>).&#x20;

### Integrations

Refer to callout **\[9]** in the diagram.&#x20;

Litmus Edge can also send data directly to cloud systems/other enterprise systems through integrations. Learn more about [Integration](<././Product Features/Integration.md>) and [Integration Guides](<././How-To Guides/Integration Guides.md>).&#x20;
