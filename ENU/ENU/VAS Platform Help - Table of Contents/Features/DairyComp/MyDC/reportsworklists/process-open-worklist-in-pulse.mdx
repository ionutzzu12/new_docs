<ExpandableHeading>
  ### Required Permissions

  To use this feature, your \{\{General.ProductName}} user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](<../../../../../ENU/VAS Platform Help - Table of Contents/features/manage-dairy-users-and-roles/edit-dairy-user-permissions.mdx>) and [Manage Dairy Roles](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

Use the **Open Worklists** page to monitor and process your \{\{masterVAS-Terminology.d-airy}}'s active \{\{Worklists.worklistCommonUse}}s.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/rYwZHH8I4deDhRUnOpBt9_open-worklists-main.png" size="65" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/worklists/open-worklists-main.png" githubPath="Content/Resources/Images/Platform/dairycomp/worklists/open-worklists-main.png" position="flex-start" showCaption="false"></Image>

Select a list to complete tasks or add events to animals, depending on the \{\{Worklists.worklistCommonUse}} type. See the topics below to learn more:

<hint type="warning">
  **IMPORTANT:&#x20;**\{\{masterVAS-Products.Pulse-only}} processes \{\{Worklists.worklistCommonUse}} updates for animals in the order they are completed, and the last event or status change supersedes previous animal updates. For instance, if an animal is on a \{\{Worklists.VetListCommonUse}} for a pregnancy check and is marked as \{\{Events.6PREG}}, and is later determined to have aborted, \{\{Events.12ABORT}} is the final status and animal details adjust accordingly.
</hint>
