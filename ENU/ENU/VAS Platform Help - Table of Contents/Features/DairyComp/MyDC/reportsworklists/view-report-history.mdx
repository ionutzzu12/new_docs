The **Report History** page lists recently run reports (or reports that will run in the future) for quick and easy access.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/R_Lt3CcRkPi35NogQxOAO_reports-history-intro.png" size="84" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/Reports/reports-history-intro.png" githubPath="Content/Resources/Images/Platform/dairycomp/Reports/reports-history-intro.png" position="flex-start" showCaption="false"></Image>

You can find this feature under the :Link[DairyComp/MyDC > Reports/Worklists menu]{label="DairyComp/MyDC > Reports/Worklists menu" overridedLabel="DairyComp/MyDC > Reports/Worklists menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp-Reports" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp-Reports" href="../../../../navigate-pulse/menu-options.mdx"} on the \{\{Buttons-UI.FeaturesSecondary}} tab.

See the topics below to learn more:

- Click the report timestamp to view the report.
- View the date when the report was run (or will be run if the report was future-dated).:::hint\{type="info"}
  **NOTE:&#x20;**&#x46;uture-dated reports will run automatically on the date displayed in the **List Date** column.
  \:::
- Filter reports by type or user.
- See which user generated the report.
- Identify the report or \{\{Worklists.worklistCommonUse}} type by its icon. The following report types are supported in \{\{masterVAS-Products.Pulse-only}}.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/H0hC0w0Cs_N-RNICx4dDq_icon-count.png" alt caption size="10"} A COUNT report or command calculates the total number of animals that match certain items and conditions. See [Build a COUNT Report](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Reports/Worklists/Create New Report or Worklist/Build Custom Report or Worklist/build-a-count-report.mdx>) to learn more about generating COUNT reports!:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ZF7eZR3vlg2JbtN6FUM4N_icon-list.png" alt caption size="10"}A LIST report or command generates a listing of animals with certain items and conditions. See [Build a LIST Report](BuildListReport.htm) to learn more about generating LIST reports!:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/2v4N1OPylyoYUTUYMPPlA_icon-sum.png" alt caption size="10"} A SUM report generates averages for specific items or animals within certain groups and presents details in a table or graph See [Build a SUM Report](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Reports/Worklists/Create New Report or Worklist/Build Custom Report or Worklist/build-a-sum-report.mdx>) to learn more about generating SUM reports!
