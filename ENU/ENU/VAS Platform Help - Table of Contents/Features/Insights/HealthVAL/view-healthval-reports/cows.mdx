  ### Required Permissions

  To use this feature, your \{\{General.ProductName}} user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.

Use the Cows tab for an \{\{masterVAS-Terminology.o-verview}} of your cow \{\{Insights.InsightTypes}}, so you can quickly note any unexpected changes in case counts. Cow \{\{Insights.HealthTrends}} can be viewed in the distinct groupings below.

By \{\{masterVAS-Terminology.F-resh-cow-}} Month: Monthly health event data for all cows by month \{\{masterVAS-Terminology.f-resh-cow}}.

See the topics below to learn more:

1. Select **Filters** to open a side panel on the left of the page to filter the list of lactation events and/or \{\{DIM}} thresholds for event counts. Once a filter is selected, the current filter applied is displayed next to the **Filters** option. Click the **X** on a specific filter to remove it.
2. Click the **Customize** button to edit your \{\{Insights.HealthSummaries}} summaries for cow and heifer events and/or modify your \{\{Insights.HealthSummaries}} benchmark settings.
3. Use the **Scroll To** shortcut to further analyze individual events in the graphs below. See [Cow Event Analysis](EventAnalysis.htm) to learn more about analyzing specific events.
4. **Reference** checkboxes allow you to control which lines are plotted on the graph. Each reference is color-coded to match a specific bar or line on the graph. Check a box to include a plot line on the graph. Uncheck a box to remove a plot line from the graph.
5. Click **Occurrence** to view the reference data in occurrence percentages and include threshold values on the graph, or click **Count** to view the reference data in total counts.
6. If the graph includes data for the current month, the bar is shaded to indicate that data for the month is incomplete.
7. Hover over a bar on the graph to view additional data.
