Tap the **User Account** button

at the top right of the screen to open the \{\{Mobile-UI.AppMyAccount}} screen to open the **User Settings** menu, and then tap the **User Settings** option.

The following settings are available in the current release:

- **Ask Me the PIN:&#x20;**&#x53;et how frequently the app prompts you to enter your PIN (**Always**, **After 8 hours**, or **Once a day**).
- \{\{Care Package}}**:&#x20;**&#x55;se this switch to automatically download new data from \{\{masterVAS-Products.Pulse-only}} each time you choose a \{\{masterVAS-Terminology.d-airy}} from the \{\{Mobile-UI.MyDairiesList}} list (\{\{Mobile-UI.ConnectionRequired}}). If this switch is toggled off, the mobile app will **not** check for new data when you open a \{\{masterVAS-Terminology.d-airy}} from the \{\{Mobile-UI.MyDairiesList}} list.:::ExpandableHeading\{customAnchorSlug="Care\_Package"}

### What is a "Care Package"?

<Paragraph indent="1">
  \{\{General.CompanyName-short}} uses the term "Care Package" to describe the updates that \{\{masterVAS-Products.Pulse-only}} "packages up" and sends to the mobile app. The care package is used to keep the mobile app updated with the most recent on-premise data received by \{\{masterVAS-Products.Pulse-only}}.
  When this setting is on, the mobile app checks with \{\{masterVAS-Products.Pulse-only}} each time you select a \{\{masterVAS-Terminology.d-airy}} from the \{\{Mobile-UI.MyDairiesList}} list to see if a new care package is available. If so, then the app downloads the updated information from \{\{masterVAS-Products.Pulse-only}}.
  \:::
</Paragraph>
