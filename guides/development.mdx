## Hot reload

The dev server watches your project and updates the browser instantly when you:

- Edit any `.mdx` or `.md` file
- Modify `archbee.json`
- Add, rename, or delete docs
- Change files in `/public/`

Updates happen in under 100ms — no manual refresh needed.

## Project structure

```javascript
my-docs/
  archbee.json          Configuration
  guides/               Main docs space
    index.mdx           Home page
    essentials/          Category folder
      configuration.mdx
  blocks/               Blocks showcase space
    callouts-buttons.mdx
  public/               Static assets
    logo-light.png
    favicon.ico
```

## Validation

Check your config and docs for errors:

```bash
archbee validate
```

Verifies that `archbee.json` is valid and all referenced doc files exist. Returns exit code 1 on errors — use it in CI pipelines.

## Link checking

Scan your docs for broken internal links:

```bash
archbee broken-links
```
