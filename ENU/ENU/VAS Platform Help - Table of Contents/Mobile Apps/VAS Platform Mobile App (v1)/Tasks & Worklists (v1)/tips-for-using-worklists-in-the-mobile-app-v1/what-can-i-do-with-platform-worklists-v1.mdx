## In the Web Version of PULSE

Use the [browser version of PULSE](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/reportsworklists.mdx>) to:

- Generate \{\{Buttons-UI.DC-Reports}} from the \{\{masterVAS-Terminology.D-airy-}}\{\{Buttons-UI.DC-Reports}} tab or Gallery. See [Reports/Worklists](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/reportsworklists.mdx>) to learn more.
- View and monitor the status, details, and percent complete of all open \{\{Buttons-UI.DC-Reports}}. See [View Progress on All Open Worklists](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Reports/Worklists/Process Open Worklist in PULSE/view-progress-on-all-open-worklists.mdx>) to learn more.
- Configure \{\{Worklists.worklistCommonUse}}-specific settings, such as customizing \{\{Worklists.VetListCommonUse}} and \{\{Worklists.HospListCommonUse}} event shortcuts and sounds. See [Sounds](../../Features/DairyComp/Settings/DC-Sounds.htm), [Vetcodes](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Settings/Health Settings/vetcodes.mdx>), and [Tasks & Worklist Settings](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Settings/Health Settings/tasks-and-worklist-settings.mdx>) to learn more.

## In the Mobile App

Use the \{\{Mobile-UI.ListsTab}} feature in the \{\{General.ProductName}} to:

- Process animals in \{\{Worklists.worklistCommonUse}}s and set them to \{\{Worklists.DoneUI}}.
- Make \{\{masterVAS-Terminology.p-en}} corrections if an animal's actual \{\{masterVAS-Terminology.p-en}} does not match the one shown on the \{\{Worklists.worklistCommonUse}}. See [Make Pen Corrections (v1)](make-pen-corrections-v1.mdx) to learn more.
- Undo a to‑be‑processed item on the \{\{Worklists.DoneUI}} list to move it back to the \{\{Worklists.ToDoUI-Mob}} list. You cannot undo an item that has already been processed (i.e., is Complete) and sent back to \{\{masterVAS-Products.Pulse-only}} and \{\{DC-UI.DC-ProductName}}.
- Use the \{\{Worklists.ProcessChanges}} button to send list updates back to the \{\{General.ProductName-short}} and \{\{DC-UI.DC-ProductName}}.

See [View Tasks & Worklists in the Mobile App (v1)](<../../../../../../VAS Platform Help - Table of Contents/Mobile Apps/VAS Platform Mobile App (v1)/Tasks & Worklists (v1)/view-tasks-and-worklists-in-the-mobile-app-v1.mdx>) to learn more!
