The \{\{Events.26WEAN}} event records when a calf no longer receives milk for feedings. Expand the sections below to learn more:

<ExpandableHeading>
  ## Event Guidelines

  Use the mobile app or the web version of \{\{masterVAS-Products.Pulse-only}} to enter events! Tap \{\{Mobile-UI.EventsFeature}} in the [mobile app](../../DC-Mobile/MobAppHome.htm) to get started, or see :Link[Add Events]{label="Add Events" overridedLabel="Add Events" spaceId docId="sp1fQUQTHg-WLIRXAer4X" version="v2" docAnchorId="#Add" loadingMethod="dynamic" newTab="false" githubPath="Events.htm#Add" href="Events.htm#Add"} to learn more about adding events from web. \{\{masterVAS-Products.Pulse-only}} supports adding events that have occurred within the last year.

  Expand the topics below to learn more about \{\{Mobile-UI.EventTypes}} event details:

  <ExpandableHeading>
    ### Height

    The animal's height (in the units set in your [PULSE User Account](../Config/EditUserAcct.htm)). To enable Height for this event, select the event from [Event Definitions](<../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx>) and expand :Link[Optional Details]{label="Optional Details" overridedLabel="Optional Details" spaceId docId="GxXIleQm2y07Vuz9c38iV" version="v2" docAnchorId="#optional" loadingMethod="dynamic" newTab="false" githubPath="../DairyComp/Settings/DC-EventDefinitions.htm#optional" href="../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx"}.
  </ExpandableHeading>

  <ExpandableHeading>
    ### Weight

    The animal's weight (in the units set in your [PULSE User Account](../Config/EditUserAcct.htm)). To enable Weight for this event, select the event from [Event Definitions](<../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx>) and expand :Link[Optional Details]{label="Optional Details" overridedLabel="Optional Details" spaceId docId="GxXIleQm2y07Vuz9c38iV" version="v2" docAnchorId="#optional" loadingMethod="dynamic" newTab="false" githubPath="../DairyComp/Settings/DC-EventDefinitions.htm#optional" href="../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx"}.
  </ExpandableHeading>

  <ExpandableHeading>
    ### BCS value

    Body condition score, which is used to evaluate an animal's energy balance based on being extremely thin (1.00) or having excessive fat deposits (5.00). The value can be a whole number or decimal, and you can include additional notes using the \{\{REM}} field for the event (up to 16 alpha-numeric characters). To enable \{\{Events.19BCS}} value for this event, select the event from [Event Definitions](<../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx>) and expand :Link[Optional Details]{label="Optional Details" overridedLabel="Optional Details" spaceId docId="GxXIleQm2y07Vuz9c38iV" version="v2" docAnchorId="#optional" loadingMethod="dynamic" newTab="false" githubPath="../DairyComp/Settings/DC-EventDefinitions.htm#optional" href="../../../DairyComp/MyDC/Settings/General Settings/event-definitions.mdx"}.
  </ExpandableHeading>
</ExpandableHeading>

<ExpandableHeading>
  ## Process Changes

  When you add events from web or the mobile app, your updates are sent to \{\{masterVAS-Products.Pulse-only}} and then on to \{\{DC-UI.DC-ProductName}} for \{\{masterVAS-Products.DCUnlimited}}\{\{masterVAS-Terminology.d-airies}} (\{\{Mobile-UI.ConnectionRequired}}).

  When the updates are received, \{\{masterVAS-Products.Pulse-only}} records \{\{PlatformContentSnippets.A-An}}\{\{Events.EventName}} event and sets the \{\{EDAY}} to the date the event was completed or diagnosed.
</ExpandableHeading>
