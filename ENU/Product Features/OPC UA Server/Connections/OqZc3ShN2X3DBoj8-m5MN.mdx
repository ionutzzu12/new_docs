The Statistics pane is used to manage certificate trust for OPC UA clients. Before a client can connect, the server must be configured to trust that client's certificate.

From here, you can view, trust, or remove client certificates, refresh the client list, and download the server certificate. For more information, see :Link[Manage OPC UA Authentication]{label="Manage OPC UA Authentication" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="_weZBfTZedfxmRhU-zuqu" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;

If you are using the Docker version of Litmus Edge and having issues connecting to the OPC UA server, see :Link[Issues Connecting to OPC UA Server]{label="Issues Connecting to OPC UA Server" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="hrDQvQUWkmP9pK9B-ZlBG" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;

# Access OPC UA Statistics UI

**To access the OPC UA Statistics pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **OPC&#x20;**>**&#x20;Statistics**.
   The *Statistics&#x20;*&#x70;ane appears.
