The \{\{Buttons-UI.FeaturesSecondary}} page allows you to view and edit \{\{masterVAS-Products.AllProductNames}} settings and \{\{masterVAS-Terminology.c-onfiguration}} options. For \{\{masterVAS-Products.DCU-users-short}}, the changes you make here are synchronized with \{\{masterVAS-Products.OnPremise-lower}}\{\{masterVAS-Products.DC-Platform}}. You can find this feature under the :Link[DairyComp/MyDC Menu]{label="DairyComp/MyDC Menu" overridedLabel="DairyComp/MyDC Menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}.

Follow the guidelines below when working with the \{\{Buttons-UI.FeaturesSecondary}} page:

1. View \{\{Buttons-UI.FeaturesSecondary}} Categories: Click the tabs at the top of the page to view \{\{Buttons-UI.FeaturesSecondary}}, \{\{Buttons-UI.FeaturesSecondary}}, and \{\{Buttons-UI.FeaturesSecondary}}.
2. Search for Settings: Use the **Search** field at the top left to find a specific Settings page.
3. Select an Option: Settings for the option selected are displayed on the right side of the page.
4. Filter the List: Some settings include additional search features at the top of the page.
5. View Rows Per Page: Select an option to change how many records are displayed on the page, and click the navigation options to jump to a specific page.
