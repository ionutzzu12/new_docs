# Enable Logging

When you enable logging for a connector, the log displays connection and publish-based error messages. This can increase CPU usage. Therefore, it is preferable to use logging only for debugging.

<hint type="info">
  **Notes**:

  - The connector must be added and connected before logging can be enabled.
  - With the exception of the default connector, all connectors have logging capability.
  - Clicking the connector tile shows the details of the connector. You can view the logs section from there.
  - You can download and refresh log files from the *Details&#x20;*&#x70;ane.
</hint>

**To enable logging:**

1. From a connector tile within an *Integration&#x20;*&#x70;ane, click the **Edit** icon.
   The *Edit Connector* dialog box appears.
2. Select the **Enable logging** check box, and then click **Update**.
   The *Logging&#x20;*&#x69;con appears on the connector tile indicating that logging is active.

# Access Connector Information

You can access device information and edit options by clicking the connector tile.&#x20;

## Connector Dashboard

After clicking a connector tile, the dashboard for the connector displays.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/Hm2VhFNCRZly5VT8VzYbp_image.png "The connector dashboard")

The dashboard displays the following information for the connector:

- Size on Disk
- Disk Usage
- Persistence Size

For *Disk Usage* and *Persistence Size*, you can use the icons to manipulate the charts and also download the charts in SVG, PNG, or CSV format.&#x20;

## Connector Configuration

The *Configuration&#x20;*&#x74;ab allows you to review and update the fields for the connector.&#x20;

Make updates as needed and click **Update**.&#x20;

# Manage Connector Topics

You can manage connector topics from the *Topics&#x20;*&#x74;ab.&#x20;

To manage a topic, click the **Action menu** for the topic.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/qGNCwTyRuus9Zk-GiMBPx_image.png "The action menu for a connector topic")

- **Edit**: Update the parameters for the topic. You can also click the topic name to update parameters.&#x20;
- **Remove**: Remove the topic from the connector.&#x20;
- **Enable/Disable**: Enable or disable the topic for the connector.&#x20;
