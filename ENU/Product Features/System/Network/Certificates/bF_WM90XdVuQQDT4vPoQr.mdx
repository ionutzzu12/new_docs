A certificate signing request (CSR) is a public certificate that contains specific information and is used to apply for an SSL/TLS certificate. You can copy/download the certificate signing request in Litmus Edge and submit it to a certificate authority for authorization. The certificate authority can then send back a signed identity certificate. You can then install this identity certificate in Litmus Edge. See [Install an Identity Certificate](<../../../../ENU/ENU/Product Features/System/b95PhFWmFc516r6yCWwJ0/Ls0gLfyNaT-TDFOgU6SNX/1g12xMN9PFck23XIWUW6L.mdx>) for details.&#x20;

You can manage certificate signing requests by navigating to **System&#x20;**>**&#x20;Certificates** and referring to the *Identity Certificates* section.&#x20;

Click the **Create Request** icon to open the *Certificate Signing Request*.&#x20;

You can copy or download the certificate signing request and give it to a Certificate Authority when applying for an SSL Certificate. A private key is usually created at the same time that you create the Certificate Signing Request, making a key pair.
