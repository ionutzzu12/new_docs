\{\{masterVAS-Terminology.D-airies-}} using \{\{Buttons-UI.PerformanceIndicators}} for the first time need to set a few things up before \{\{masterVAS-Products.Pulse-only}} will examine \{\{PlatformContentSnippets.PerformanceIndicators-ThresholdDifferential}}s and send alerts to your team.

<hint type="info">
  **NOTE:&#x20;**\{\{masterVAS-Products.Pulse-only}} does not currently support alerts and notifications for all available indicator types. Use the following instructions to set up \{\{Buttons-UI.HealthIndicators}}, but note that the steps may differ for other indicator types.
</hint>

This procedure guides you through the main steps for enabling \{\{PlatformContentSnippets.PerformanceIndicators-AlertsAndThresholds}} so that \{\{masterVAS-Products.Pulse-only}} will alert your \{\{masterVAS-Terminology.d-airy}} team when specific events occur across your \{\{masterVAS-Terminology.h-erd}}. While these instructions use indicators and alerts specifically for \{\{mastitis}} events as examples, you can follow the same steps to set up any other event indicators or alerts to meet your \{\{masterVAS-Terminology.d-airy}}'s needs.
