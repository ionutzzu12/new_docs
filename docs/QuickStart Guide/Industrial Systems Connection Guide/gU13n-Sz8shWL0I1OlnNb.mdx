Refer to the following table of compatible DeviceHub drivers for your PLC series.

<hint type="info">
  **Note**: For CompactLogix and ControlLogix PLCs, we recommend using  :Link[AB CompactLogix Ethernet Advanced]{label="AB CompactLogix Ethernet Advanced" spaceId="JWQMp_kvm0WsA_Fpjt1v_" docId="abVmfZeE82lDphyljJZhJ" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}. It implements two access modes: Symbolic and CompactLogix. Symbolic mode uses the same implementation as [AB CompactLogix Ethernet FreeTag](<Allen-Bradley/Gen 2 Drivers/PBoF2y5NhirJ-XS9CYRDZ.mdx>) and is the recommended driver and access mode for connecting to ControlLogix PLCs.
</hint>
