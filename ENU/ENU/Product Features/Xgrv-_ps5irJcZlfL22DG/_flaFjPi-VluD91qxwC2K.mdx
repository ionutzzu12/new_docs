You can transfer files from Litmus Edge to a cloud storage service.&#x20;

# Set Up Cloud Storage

Before you can start leveraging Cloud Storage, you need to set up the source directory that will be used for configuring sync jobs.&#x20;

One option is to start the Litmus Edge FTP Server. See [System Services](<../../../Product Features/System/Device Management/zX-bB0k3Mdx_rsAGhlgpt.mdx>) to learn more.&#x20;

If you start the Litmus Edge FTP Server, you can use an FTP tool such as [Filezilla](https://filezilla-project.org/) to connect to the server and create a dedicated folder that will be used to copy or move files. For example, you can create the following file path to configure a sync job with AWS S3: `/var/lib/customer/ftp-data/s3-sync`. This file path will be used for the *Source&#x20;*&#x70;arameter in the sync job configuration.&#x20;
