<ExpandableHeading>
  ### Required Permissions

  To access and use \{\{UDS-general.UDSBrand}}\{\{UDS-general.UDSProduct}}, you must first be added by your administrator or manager and assigned to specific \{\{masterVAS-Terminology.d-airies}}. See [GENEX Management](<../../../../../VAS Platform Help - Table of Contents/Features/GENEX Genetic Planning Tool/Get Started with GENEX DairySuite/genex-management.mdx>) to learn more about user access.

  <hint type="warning">
    **IMPORTANT:&#x20;**&#x44;o **NOT** use the Request Access button :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/X8Sxc2XLRUtVwCZCx9WBW_request-access144x30.png" alt caption="Request Access button" size="10"} on the \{\{masterVAS-Terminology.d-airy}}'s profile page to request access to the \{\{UDS-general.CommonProductName}}. That option sends a message to the \{\{masterVAS-Terminology.d-airy}}'s owner and may cause confusion.
  </hint>
</ExpandableHeading>

The \{\{UDS-general.GenericProductName}} is available to \{\{UDS-general.UDSBrand}}\{\{masterVAS-Terminology.c-onsultants}} and managers in the \{\{General.ProductName}}. To request access, contact your \{\{UDS-general.UDSBrand}}\{\{UDS-general.UDSProduct}} administrator or manager to be added. Your admin or manager will also assign you to specific \{\{masterVAS-Terminology.d-airies}}. See [GENEX Management](<../../../../../VAS Platform Help - Table of Contents/Features/GENEX Genetic Planning Tool/Get Started with GENEX DairySuite/genex-management.mdx>) to learn more about user access.

<ExpandableHeading>
  ## Log in to GENEX DairySuite

  \{\{UDS-general.UDSBrand}}\{\{UDS-general.UDSProduct}} is available to genetic \{\{masterVAS-Terminology.c-onsultants}} in the \{\{General.ProductName}} at [https://platform.vas.com/login](https://platform.vas.com/login). After you log in, select the \{\{masterVAS-Terminology.d-airy}} for which you would like to build or view a genetic plan.

<ExpandableHeading>
  ## See Your Dairies

  To see which \{\{masterVAS-Terminology.d-airies}} you can access, log into the \{\{General.ProductName-short}} and click the drop-down in the upper left.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/RoMtCNWkeeljvwyYntMgy_view-dairies.png" size="43" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GettingStarted/view-dairies.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GettingStarted/view-dairies.png" position="flex-start" showCaption="false"></Image>

  <hint type="warning">
    **IMPORTANT:&#x20;**&#x42;efore you can begin using the \{\{UDS-general.CommonProductName}}, the \{\{masterVAS-Terminology.d-airy}} must first activate the integration. Share the following link with the \{\{masterVAS-Terminology.d-airy}} to guide them through the activation process: Genetic Planning Tool Integration Activation.
  </hint>

  Once the integration is active, go to the [Integrations Gallery](../../partners/integrations-gallery.mdx) and click **See Details** on the integration card to view the \{\{masterVAS-Terminology.c-onsultants}} assigned to the \{\{masterVAS-Terminology.d-airy}}.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/iFMXag13mk_HJ_mltNEr-_genex-consultants-int-gallery.png" size="56" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GettingStarted/genex-consultants-int-gallery.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GettingStarted/genex-consultants-int-gallery.png" position="flex-start" showCaption="false"></Image>

  ## Navigate GENEX DairySuite

  You can find the

  \{\{UDS-general.UDSBrand}}

  \{\{UDS-general.UDSProduct}}

  button in the main menu bar after

  [logging into](LoginGenPlanning.htm)

  the

  \{\{General.ProductName}}

  and selecting a

  \{\{masterVAS-Terminology.d-airy}}

  .

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tDxmBnxtKwWwTph_b4UZC_click-genex.png" size="59" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GettingStarted/click-genex.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GettingStarted/click-genex.png" position="flex-start" showCaption="false"></Image>

  - \{\{Commands.Click}}**Genetic Plans** to build a new [Genetic Plan](../BuildGeneticPlan/BuildGeneticPlan-intro.htm) or to view existing plans.
  - \{\{Commands.Click}}\{\{masterVAS-Terminology.O-verview-}} to access the [GENEX DairySuite Genetic Dashboard](<../../../../../VAS Platform Help - Table of Contents/Features/GENEX Genetic Planning Tool/genex-dairysuite-genetic-dashboard.mdx>).

  While in the process of building your plan, use the **Back** button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/E7sEvJgJ_UbAuzC50XVNE_button-back66x30.png" size="10" caption="BACK" alt="BACK" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/button-back_66x30.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/button-back_66x30.png" position="flex-start" showCaption="true"></Image>

  to view or change the options in previously completed steps.

  <hint type="info">
    **NOTE:&#x20;**&#x57;hen adjusting previously set options, you will be asked to confirm your updates, as any changes will impact the plan results.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/2OwCxWYwVVA00H12ndhOJ_save-changes-message.png" alt caption size="27"}
  </hint>
</ExpandableHeading>
