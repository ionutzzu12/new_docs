  ## Data Integrity Report

  Download the Data Integrity report to quickly identify missing or incomplete animal data. You can then consult the \{\{masterVAS-Terminology.d-airy}} to add more information or resolve data errors in \{\{masterVAS-Products.Pulse-only}} or \{\{masterVAS-Products.DC305-short}}. When you improve the integrity of the \{\{masterVAS-Terminology.h-erd}} data, \{\{UDS-general.UDSBrand}}\{\{UDS-general.UDSProduct}}\{\{UDS-general.GENEX-Pulse}} assessments and genetic plans become more accurate and provide more value to the \{\{masterVAS-Terminology.d-airy}}. Inaccurate \{\{masterVAS-Terminology.h-erd}} data could result in matches that don't provide the best possible value or the loss of healthy, valuable animals.

  You can download the report by clicking the button at the upper right corner of the page.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/L-CTqgUAiAjkcVSklJ4oR_datareport-genex.png" size="59" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/Dashboard/datareport-genex.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/Dashboard/datareport-genex.png" position="flex-start" showCaption="false"></Image>

  The report denotes missing animal information with one of the following values: a blank space, **No Match in Proofs**, or **UNKNOWN**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/D3-8XS2TllOr2fHCJgFNK_datareport-missing.png" size="59" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/Dashboard/datareport-missing.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/Dashboard/datareport-missing.png" position="flex-start" showCaption="false"></Image>

  ## Genetic Averages by Company

  See the topics below to review genetic traits by semen supplier.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/h6bgYEGCXMXnjb5eXLzBw_gen-average-by-co-widget-new.png" size="66" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/Dashboard/gen-average-by-co-widget-new.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/Dashboard/gen-average-by-co-widget-new.png" position="flex-start" showCaption="false"></Image>

  1. Select the Breeds you want to view.
  2. Select the Trait you want to analyze.
  3. Select the Companies you want to compare. Based on your settings, the bar graph compares the trait across all suppliers for six standard animals groups.
  4. Hover over a group for trait value details by company.
  5. See when the data was last updated.
  6. Click **Show More Info** to view all trait details by company in a table format.
  7. Hover over trait abbreviations to view their full name, or click the column header to sort the data by that trait.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/XQZQPHFTQ9LWVhm9uInRq_gen-average-by-co-widget.png" alt caption size="63"}

  <hint type="info">
    **NOTE:&#x20;**&#x54;his widget is unique in how it calculates trait values: only bulls' traits are included in the average values. The other widgets on the Dashboard include both parents' averages for genetic trait values.
  </hint>

  ## Future Calving

  Use this widget to view all future, unborn heifers grouped by \{\{due}} date. Open the **Heifer Inventory** tab to view this report.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/bIoC84SJQqC4wI3BzyCtQ_future-calving-inventory.png" size="78" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/Dashboard/future-calving-inventory.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/Dashboard/future-calving-inventory.png" position="flex-start" showCaption="false"></Image>

  1. Use the filters to manage the animals that you see in the graph. You can choose to show cows, heifers, or both.
  2. Enter a value for **Heifers Needed Monthly** to show a threshold line in the graph.
  3. The \{\{UDS-general.CommonProductName}} uses these percentages to calculate how many future calves will be female. We recommend that you use the default values: `90%` for **Female Likelihood for Sexed** and `49%` for **Female Likelihood for Conventional**.
  4. Hover over any data point to see exact values.
  5. Click **Show More Info** to view the future calving data in a table format.
</ExpandableHeading>
