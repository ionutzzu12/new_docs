# Manage Individual Networks

You can use the **Action menu** for a network to view management options.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/rar5rLJvQnFEOjfQOjOYL_image.png "Network action menu options")

- **Remove**: Remove the network from the *Networks&#x20;*&#x70;ane.&#x20;
- **Connect/Disconnect container**: After selecting to connect or disconnect the container from the network, select the appropriate container.&#x20;
