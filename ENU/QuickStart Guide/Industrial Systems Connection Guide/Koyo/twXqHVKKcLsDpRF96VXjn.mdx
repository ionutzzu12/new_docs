To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
2. **Device**: Koyo Direct Logic 06 D0-06DR
3. **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/U8QXrm4b0H0_oI96O_Vmx_koyo-direct-serial.png" size="80" width="478" height="233" caption="Koyo Direct Serial reference" position="center" darkWidth="478" darkHeight="233" showCaption="true" indent="2"></Image>

4. Launch DirectSoft6 v6.1 software and click **Reset PLC**.
5. From the Links dialog box, click **Add**.
6. Click **Next**.&#x20;
7. Select the PLC family, and then click **Next**.
8. Select the protocol, and then click **Next**.
9. Click **Finish**.&#x20;
10. Select the link just created for serial communication.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/BlmGbkIiFbo2iQ2kfqmWX_koyo-direct-serial-link.png" size="80" width="830" height="471" caption="Serial communication configuration" position="center" darkWidth="830" darkHeight="471" showCaption="true" indent="2"></Image>
