To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Koyo Click
   - **Communication I/F**: RS232. Refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/AsXqjUutpTs7KFmrJhszS_koyoclickreference.png" size="80" width="621" height="565" position="flex-start" caption="Koyo Click reference" showCaption="true" indent="2"></Image>

2. Launch CLICK Programming v2.10 Software and click **PLC > Connect**.
3. Set up communication parameters and then click **Connect**.
