You can manage marketplaces by navigating to **Applications&#x20;**>**&#x20;Configure**.&#x20;

Click the **Action menu** for a marketplace to manage it.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/DashYNEpaR5gcLnwqphmT_image.png "The action menu for a marketplace")

- **Edit**: Update the parameters for the marketplace.&#x20;
- **Remove**: Remove the marketplace.&#x20;
