View the \{\{Mobile-UI.TransactionLog}} for a list of transactions you've complete in the app over the last 14 days. Tap on the \{\{Mobile-UI.MoreTab}} tab at the bottom right of the screen, and then tap \{\{Mobile-UI.TransactionLog}} to get started!

The \{\{Mobile-UI.TransactionLog}} records the following user actions:

- Adding an animal on the \{\{Mobile-UI.AnimalsTab}} tab
- Editing items from a CowCard
- Completing a task on the \{\{Mobile-UI.ListsTab}} tab
- Completing an event on the \{\{Mobile-UI.EventsTab}} tab

A color-coded line is displayed on the right side of each transaction available in the \{\{Mobile-UI.TransactionLog}} to indicate the following:

- A blue line indicates that the transaction has not been processed.
- A green line indicates that the transaction has been processed.
- A red line indicates that an error occurred when processing the transaction.
