  ### How do I add this widget to a Company Dashboard?

  This widget is only available on \{\{Buttons-UI.D-ashboardsCompany-}}. To add this widget, check the box for it under the \{\{masterVAS-Terminology.H-erd-}} category in the [Page Customization](../../Dashboards/DairyDashboard/CustomizeDashboard.htm) options.

<ExpandableHeading>
  ### What permissions do I need to view this widget?

  To view this widget on your \{\{Buttons-UI.D-ashboardCompany-}}, your user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](../../Config/UserMgmtRoles/ManageRoles.htm) for more information.
</ExpandableHeading>

The \{\{Widgets.CowStats}} by \{\{masterVAS-Terminology.D-airy-}} widget is useful in quickly identifying animal counts by \{\{masterVAS-Terminology.d-airy}}. This widget is only available on \{\{Buttons-UI.D-ashboardsCompany-}}.The example below defines the information shown in this widget.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/X7CBYh5dg-b5kM3wKMz7L_cow-stats-co-by-dairy-widget.png" size="41" isUploading="false" initialPath="../../../../Resources/Images/Platform/dashboard/company/cow-stats-co-by-dairy-widget.png" githubPath="Content/Resources/Images/Platform/dashboard/company/cow-stats-co-by-dairy-widget.png" position="flex-start" showCaption="false"></Image>

1. **References:&#x20;**&#x53;elect to view data by animal status: Heifers, Dry Cows, Milking Cows.
2. **Bar graph:&#x20;**&#x50;resents animal counts for each company \{\{masterVAS-Terminology.d-airy}} in a colored bar to quickly identify Heifers, Dry Cows, and Milking Cows.
3. **Tooltip:&#x20;**&#x48;over over the graph data to view more detailed information, including animal counts and percentages of total \{\{masterVAS-Terminology.h-erd}}.
4. **Updated:&#x20;**&#x54;imestamp (date and time) of the last data sync from \{\{masterVAS-Products.DC305-simple}}.
5. **Show Data Table:&#x20;**&#x53;elect this option to view detailed animal data below the graph in a table format.
6. **Data Table:&#x20;**&#x44;isplays additional animal details by \{\{masterVAS-Terminology.d-airy}}.
7. **Hide Data Table:&#x20;**&#x53;elect this option to collapse the table.
