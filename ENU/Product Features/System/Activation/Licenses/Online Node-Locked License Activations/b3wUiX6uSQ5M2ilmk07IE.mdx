You can upgrade an active license by navigating to **System&#x20;**>**&#x20;Activation&#x20;**>**&#x20;License**.&#x20;

**To upgrade a license online:**

1. Navigate to **System&#x20;***>&#x20;***Activation**.&#x20;
2. Click the **License&#x20;**&#x74;ab.&#x20;
3. From the *Node-Locked License* section, click the **Upgrade License** icon for the license you need to upgrade. The **Upgrade License** icon only appears for the licenses whose type supports the upgrade operation.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/8fz7em7YHWwjHirWAWPjj_upgrade-license.png)
4. Enter the license activation key into the **Upgrade License Online** field. Make sure you include all the dashes when entering the license activation key.
5. Click **Submit**.
6. In the confirmation box that appears, click **Yes**.
