You can configure the CAN interface and the slcan-ttyACM0 or slcan-ttyACM1 interface, or both, by navigating to **System&#x20;**>**&#x20;Info** from the *CAN Interface* section. You only need to set the configuration once.

# Before You Begin

Before completing these steps, make sure you do the following:&#x20;

- Use the HPE EL300 device with CAN Bus module (Atmel chipset)
- Acquire sufficient SocketCAN / CAN Bus knowledge

# Configure the CAN Interface

**To configure the CAN Interface:**

1. In Litmus Edge, navigate to **System**.
   The *Info&#x20;*&#x70;ane appears.
2. Scroll to the *CAN Interface* section, and click **Configure&#xA0;**&#x66;or the slcan-ttyACM0 or sclan-ttyACM1 interface.
3. Check the **Open&#xA0;**&#x63;heck box.
4. Select 1000 from the CAN Speed kbit/s drop-down list, and then click **Save**.
