The \{\{masterVAS-Terminology.SRS2long}} is a portable hand-held device that allows you to quickly read, record, and transfer \{\{EID}} tag information for the animals in your \{\{masterVAS-Terminology.h-erd}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/GZ19Ku8rsKQC7Chs_B_pv_srs2-stick-reader.png" size="43" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-stick-reader.png" githubPath="Content/ContentShare/TruTest/srs2-stick-reader.png" position="flex-start" showCaption="false"></Image>

<hint type="warning">
  **IMPORTANT:&#x20;**\{\{masterVAS-Terminology.SRS2long}}s are currently only supported in the Android version of the mobile app. Expand the sections below to learn more::::ExpandableHeading

  ### Required Setup

  Before you connect the \{\{masterVAS-Terminology.SRS2long}}, download the latest firmware from the [Tru-Test website](https://www.livestock.tru-test.com/en-us/resources/4328#downloads).

  <ExpandableHeading>
    ### Navigation Tips

    The display is not a touchscreen. You can navigate to options using the **Arrow** buttons.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/YoYHDtxSW5H2kCJl91lIP_srs2-arrow-buttons.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-arrow-buttons.png" githubPath="Content/ContentShare/TruTest/srs2-arrow-buttons.png" position="flex-start" showCaption="false"></Image>

    To select an option, press the **Scan** button.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Woe6IMfopw-SymHgPjrqJ_srs2-scan-button.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/srs2-scan-button.png" githubPath="Content/ContentShare/TruTest/srs2-scan-button.png" position="flex-start" showCaption="false"></Image>

    <hint type="info">
      **NOTE:&#x20;**&#x54;he **Scan** button is also referred to as the \{\{masterVAS-Terminology.R-ead-}} and/or **Select** button in your \{\{masterVAS-Terminology.SRS2long}} user manual.
    </hint>

    To return to a previous menu, press the **Return** button.

    :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/m9bRDwdFmTcQllLQrSByx_srs2-return-button.png" alt caption size="10"}

Follow the steps below to delete a \{\{masterVAS-Terminology.SRS2long}} session:

1. Use the **Arrow** buttons to navigate and press the **Scan** button to select **Sessions**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tKL8McviJQnDazDGUx9_O_srs2-sessions.png" alt caption size="19"}
2. Select a **Session** name.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/TkpNEtmJEtvmqF0PpA6-H_srs2-session.png" alt caption size="19"}
3. Select a session \{\{ID}}.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/H7jgY9ahtPRQNY4icJht1_srs2-session-scanset.png" alt caption size="19"}
4. Select the hamburger icon in the bottom right corner

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/j_3dP75KTKoxjwoInQCJE_icon-hamburger34x25.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/icon-hamburger_34x25.png" githubPath="Content/ContentShare/TruTest/icon-hamburger_34x25.png" position="flex-start" showCaption="false" indent="2"></Image>

<Paragraph indent="1">
  .
</Paragraph>

5. Select **Delete**.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/sBsq9FWVHohZFieojAhcR_srs2-session-scanset-delete.png" alt caption size="19"}
6. Select the check mark icon in the bottom right corner

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ug16WyR5cI4leCOYHzXPV_icon-checkmark27x25.png" size="10" isUploading="false" initialPath="../../../ContentShare/TruTest/icon-checkmark_27x25.png" githubPath="Content/ContentShare/TruTest/icon-checkmark_27x25.png" position="flex-start" showCaption="false" indent="2"></Image>

<Paragraph indent="1">
  to confirm the deletion.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Sqib5fVgtfGtw1STd32xb_srs2-session-scanset-delete-confirm.png" alt caption size="19"}
</Paragraph>

<hint type="info">
  **NOTE:&#x20;**&#x41; session holds up to 10,000 scans. When a session fills up, you'll need to start a new session to continue sending data to the mobile app.
