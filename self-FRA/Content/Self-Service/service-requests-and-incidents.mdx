Une demande de service permet de réclamer un service, comme la réservation d'une salle de conférence, l'ouverture d'un nouveau compte d'e-mail, etc. Un incident est une interruption de service, comme le verrouillage d'un compte, un dysfonctionnement de l'imprimante, etc.

Lorsque vous soumettez un incident ou une demande de service, le système journalise cet élément dans le centre de support de votre entreprise, où il est traité. Vous effectuez le suivi depuis la liste **Mes éléments ouverts** de l'espace de travail **Accueil** ou dans l'espace de travail **Mes éléments**.

<ExpandableHeading>
  ## Pour soumettre une demande de service

  1. Dans le menu Espaces de travail, cliquez sur \{\{FRS\_VARS.Service\_Catalog}}.
  2. Naviguez parmi les catégories dans le volet de navigation de gauche. Lorsque vous trouvez l'élément correct sur lequel la demande doit porter, sélectionnez-le dans le volet principal.
  3. Dans le formulaire affiché, remplissez les champs selon vos besoins. S'il existe plusieurs pages, cliquez sur **Continuer** (en bas à droite) pour progresser tout au long du formulaire. Chaque formulaire est différent, selon le type de demande sélectionné.
  4. Après avoir entré toutes les informations requises, cliquez sur **Réviser et soumettre**.
  5. Vérifiez l'exactitude des informations. Pour apporter des changements, cliquez sur **Modifier/Retour**.
  6. Cliquez sur **Soumettre**.

  <hint type="info">
    Une fois la demande soumise, vous ne pouvez plus revenir en arrière, ni modifier ou supprimer le formulaire.
  </hint>

  Effectuez le suivi de la demande depuis la liste **Mes éléments ouverts** de l'espace de travail **Accueil** ou dans l'espace de travail **Mes éléments**.

  Vous pouvez recevoir par e-mail des mises à jour du centre de services (Service Desk) ou de support (Help Desk), vous demandant des informations supplémentaires sur la demande. Si cela se produit, répondez dans le délai précisé dans cet e-mail ; sinon, la demande se ferme automatiquement sans que le problème soit résolu.
</ExpandableHeading>

Si vous avez besoin de mettre à jour une demande de service en cours en lui ajoutant une note ou une pièce jointe (comme une image ou un document), vous pouvez le faire. Pour en savoir plus, reportez-vous à « [Mes éléments](my-items.mdx) ».

<ExpandableHeading>
  ## Pour signaler un incident

  1. Dans le menu de l'espace de travail, cliquez sur **Catalogue de services**.
  2. Dans le volet de navigation de gauche, cliquez sur **Signaler un nouvel incident**. Dans le volet principal, sélectionnez l'option qui correspond à votre problème. Si le problème ne correspond à aucune des options disponibles, cliquez sur **Nouvel incident**.
  3. Dans le formulaire affiché, remplissez les champs selon vos besoins. Avant de le faire, parcourez le volet **Solution possible**, à droite, pour voir si vous pouvez résoudre le problème à l'aide de l'une des solutions proposées :
     - Si un article associé semble correspondre à votre problème, cliquez sur son titre pour l'ouvrir.
     - Si la solution fonctionne, cliquez sur **Résoudre** (en bas à droite du volet) pour fermer l'incident. Si elle ne fonctionne pas, cliquez sur **Précédent** (en bas à gauche du volet) pour chercher d'autres solutions.
  4. Si aucune solution ne fonctionne, remplissez les champs selon vos besoins.
     - Dans le champ **Description**, fournissez des détails pour décrire précisément votre problème.
     - Dans le champ **Urgence**, sélectionnez le niveau d'urgence que présente ce problème.
  5. Cliquez sur **Enregistrer l'incident**.

  Une fois enregistré, le problème devient un incident, avec un ID dont vous pouvez effectuer le suivi dans la liste **Mes éléments ouverts** de l'espace de travail **Accueil** ou dans l'espace de travail **Mes éléments**.

  Vous pouvez recevoir par e-mail des mises à jour du centre de services (Service Desk) ou de support (Help Desk), vous demandant des informations supplémentaires sur l'incident. Si cela se produit, répondez dans le délai précisé dans cet e-mail ; sinon, l'incident se ferme automatiquement sans que le problème soit résolu.
</ExpandableHeading>

<hint type="info">
  Pour trouver des correspondances partielles, vous pouvez utiliser **\*** et **%** comme caractères génériques dans la recherche.
</hint>

Vous ne pouvez pas modifier un incident une fois qu'il a été journalisé, mais vous pouvez le mettre à jour en lui ajoutant une note ou une pièce jointe (comme une image, un document ou autre). Vous pouvez également fermer un incident, quel que soit son état actuel. Pour en savoir plus, reportez-vous à « [Mes éléments](my-items.mdx) ».
