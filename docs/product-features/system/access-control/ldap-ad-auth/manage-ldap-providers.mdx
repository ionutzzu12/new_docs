You can manage LDAP Providers by navigating to **System&#x20;**> **Access Control** >**&#x20;LDAP/AD Auth**.&#x20;

To manage providers, click the **Action menu** for the provider and select a management option.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/bNI2b8w-NAZSScBb4IQp2_image.png "Management options for LDAP providers")

- **Configure**: Update the parameters for the provider.&#x20;
- **Map Groups**: You can map user roles to configured LDAP groups. Before you select this option, make sure the LDAP server is connected to the network and users and groups are defined on the LDAP server. The available roles to select are *Not selected*, *administrator*, *developer*, or *observer*.&#x20;
- **Remove**: You can remove the provider from the *Authentication Providers* list.&#x20;
- **Test**: See [Test a Provider](lldap-ad-auth/test-a-provider.mdx) to learn more.&#x20;
