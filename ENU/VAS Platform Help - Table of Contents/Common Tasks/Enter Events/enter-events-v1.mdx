\{\{masterVAS-Products.Pulse-only}} allows you to create events, add animals to them, and process your changes—all from the mobile app while you are out on the \{\{masterVAS-Terminology.d-airy}}. Add events to one or more animals, either by searching for them manually or using [a scanner](../MobileApp/Scanners/RFIDScanners.htm). When you finish adding animals to an event, tap \{\{Mobile-UI.EventProcessChangesv1}} to send your updates back to \{\{masterVAS-Products.Pulse-only}}and then on to \{\{DC-UI.DC-ProdNameShort}} (\{\{Mobile-UI.ConnectionRequired}}).

Before working with \{\{masterVAS-Products.Pulse-only}}\{\{Mobile-UI.EventsTab}}, review [Tips for Entering Events](../Features/Events/TipsEventEntry.htm). This topic explains how to create and work with events, add animals to them, and process your changes. It also provides helpful hints for working with events. The current \{\{masterVAS-Products.Pulse-only}} release supports the events that are listed in the following topic: [Event Definitions](../Features/Events/EventDefs.htm).
