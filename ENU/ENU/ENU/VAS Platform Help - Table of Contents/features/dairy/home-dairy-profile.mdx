The \{\{Buttons-UI.DairyProfile}} page is the \{\{masterVAS-Terminology.h-omepage}} for the \{\{masterVAS-Terminology.d-airy}} you are currently viewing. It is displayed by default when you select a \{\{masterVAS-Terminology.d-airy}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/otOMYf5_PdGD-UY-ZB8Fe_dairy-profile.png" size="76" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairy-profile/dairy-profile.png" githubPath="Content/Resources/Images/Platform/dairy-profile/dairy-profile.png" position="flex-start" showCaption="false"></Image>

You can find this feature under the :Link[Dairy menu]{label="Dairy menu" overridedLabel="Dairy menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-Dairy" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-Dairy" href="../../../../VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}.

As shown in the example above, the \{\{Buttons-UI.DairyProfile}} page provides \{\{masterVAS-Terminology.d-airy}} information, as well as several helpful shortcuts:

- \{\{masterVAS-Terminology.D-airy-}} Information: The middle left side of the page displays the \{\{masterVAS-Terminology.d-airy}}'s contact information, as well as a description of the \{\{masterVAS-Terminology.d-airy}}'s services. Users with \{\{RolesPermissions.Profile\_Management\_title}} permissions can edit this information. See [Settings (Dairy)](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/settings-dairy.mdx>) to learn more.
- Shortcuts to Common Tasks: The right side of the \{\{Buttons-UI.DairyProfile}} page provides shortcut buttons for the following:\* [Find an Animal](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/yrLo-find-an-animal.mdx>)
  - [Settings (Dairy)](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/settings-dairy.mdx>)
- Quick Actions: The \{\{Buttons-UI.DairyProfile}} page includes convenient links to common \{\{masterVAS-Terminology.h-erd}} management tasks. The Quick Actions available are based on your permission level for that \{\{masterVAS-Terminology.d-airy}}, and may differ from the example shown.\* [Add Animals](<../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Animals/add-animals.mdx>)
  - [Add Embryos](<../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/embryo-management/add-embryos-to-inventory-or-transferred-embryos.mdx>)
  - [Report/Worklist Builder](../../Features/DairyComp/MyDC/reportsworklists/create-new-report-or-worklist/build-custom-report-or-worklist.mdx)
  - [Download Recent Cowfile (DC Unlimited only)](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/Home (Dairy Profile)/download-recent-cowfile-dc-unlimited-only.mdx>)
  - [Milk Recording](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/milk-recording.mdx>)
  - [Export Herd Data (MyDC in Brazil only)](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/Home (Dairy Profile)/export-herd-data-mydc-in-brazil-only.mdx>)
- Announcements: The \{\{Buttons-UI.DairyProfile}} page displays a summary of the current \{\{masterVAS-Products.Pulse-only}} Announcements. Click the links in the preview to view the full announcements.
