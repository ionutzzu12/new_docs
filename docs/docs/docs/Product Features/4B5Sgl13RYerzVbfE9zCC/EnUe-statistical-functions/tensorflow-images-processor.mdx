# TensorFlow Images Overview

- Processor for feeding images to an already created TensorFlow model.

# TensorFlow Images Parameters

<table isTableHeaderOn="true" selectedColumns="" selectedRows="" selectedTable="false" columnWidths="182">
  <tr>
    <td>
      **Parameters**
    </td>
    <td>
      **Details**
    </td>
  </tr>
  <tr>
    <td>
      Model Path
    </td>
    <td>
      This parameter specifies the file path for the pre-trained TensorFlow model that will be used for processing the images.
    </td>
  </tr>
  <tr>
    <td>
      Tags
    </td>
    <td>
      This parameter allows you to specify tags for loading the model. If there are multiple tags, you can separate them using commas.
    </td>
  </tr>
  <tr>
    <td>
      Input Operation
    </td>
    <td>
      This parameter refers to the input tensor that was used when creating the TensorFlow model. Usually the dictionary of values you feed to the model.
    </td>
  </tr>
  <tr>
    <td>
      Output Operation&#x20;
    </td>
    <td>
      This parameter refers to the output tensor used during the creation of the TensorFlow model. It usually represents the output of the model.
    </td>
  </tr>
  <tr>
    <td>
      Gray
    </td>
    <td>
      This parameter specifies whether the images should be processed in grayscale or color.
    </td>
  </tr>
  <tr>
    <td>
      Image Format
    </td>
    <td>
      This parameter likely defines the format of the input images, such as JPEG, PNG, etc.
    </td>
  </tr>
  <tr>
    <td>
      Time Shift Ms
    </td>
    <td>
      This parameter indicates by how many milliseconds the current timestamp should be shifted.
    </td>
  </tr>
</table>
