The \{\{masterVAS-Products.VASPlatformMobile}} mobile app \{\{Mobile-UI.MobileAppVersion1}} provides the most recent animal data while you are out with the \{\{masterVAS-Terminology.h-erd}} or away from the \{\{masterVAS-Terminology.d-airy}}.

**Please be aware that the mobile app displays only the most recent events, tests, and lactations, and only for animals that are alive and on the&#x20;**\{\{masterVAS-Terminology.d-airy}}. For full access to historical animal activity, tests, etc., use the [web version of PULSE](https://platform.vas.com/login).

<ExpandableHeading>
  ## Online and Offline Mode

  When the mobile app has an active WiFi or data connection, you can refresh your data. If you go outside of data range, the app will continue to work in offline \{\{mode}}. Once you are back in data range, you can refresh your data and view the latest updates.

  <hint type="info">
    **NOTE:&#x20;**&#x57;hile the mobile app allows you to continue working in offline mode, some features in the mobile app require you to be online to see data.
  </hint>

  See the topics below to learn more:

<ExpandableHeading>
  ## Set Up Multiple User Accounts on a Shared Device

  \{\{masterVAS-Terminology.D-airy-}} owners can easily keep a tablet onsite for \{\{masterVAS-Terminology.d-airy}} workers to use during their shifts. Make sure to set up Platform accounts for all workers who will need to use the tablet (see [Invite a User to Your Dairy](<../../../Features/Manage Dairy Users and Roles/invite-a-user-to-your-dairy.mdx>)), and then allow them each to log in to \{\{masterVAS-Products.Pulse-only}} app on the tablet. Once each user has logged in and set up their PIN, they can choose their user account from the app start‑up screen and enter their PIN to use the app.
</ExpandableHeading>
