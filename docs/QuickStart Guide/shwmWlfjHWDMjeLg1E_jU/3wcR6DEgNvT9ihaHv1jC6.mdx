You have the option of connecting an :Link[MQTT Client]{docId="vwrlrDdwjGnlBOqK-lrjX" label="MQTT Client" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} and [Industrial OT Server](<../Industrial Systems Connection Guide/MQTT/ACrhHvGLG_GpHpOvQQvnL.mdx>) as DeviceHub devices.

<hint type="info">
  **Note**: You can't connect the :Link[MQTT Client]{docId="vwrlrDdwjGnlBOqK-lrjX" label="MQTT Client" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to the DeviceHub [Industrial OT Server](<../Industrial Systems Connection Guide/MQTT/ACrhHvGLG_GpHpOvQQvnL.mdx>) if they are using the same Litmus Edge instance. This includes connecting through the Flows Manager or through an Integration connector. If they are using different Litmus Edge instances, they can connect.
</hint>
