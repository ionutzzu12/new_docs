After you add roles, you can manage them by navigating to **System&#x20;**> **Access Control&#x20;**&#x61;nd referring to the *Manage Roles* section.&#x20;

# Search for an Existing Role

You can use the search bar from the *Roles&#x20;*&#x74;ab to search for existing roles.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/gheDcMY9xTLHGyj60qcE6_image.png "The search bar for existing roles")

# Manage Existing Roles

You can use the **Action menu&#x20;**&#x66;or a role to manage it.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/9FbK3GHUDL6AQYherBqLB_image.png)

- **Edit**: Update the name and user permissions for the role. See [Role Permissions](<Role Permissions.mdx>) for more information.&#x20;
- **Delete**: Remove the role from the *Roles&#x20;*&#x74;ab.&#x20;
