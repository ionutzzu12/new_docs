To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

The setup shown here is for the Rockwell Automation Allen-Bradley DF1 (Gen1.3) PLC. Refer to the following parameters.&#x20;

**Communication Mode:**

DF1 Full Duplex protocol 19200, None, 8, 1 (default)

Error Check: CRC

**To configure your device:&#x20;**

1. Record the following information and refer to it as needed.&#x20;
   **Device**: AB DF1 SLC 5/04
   **Communication I/F**: RS232. Refer to the following figure.


<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Zx0ExTUn_RHbbI5EVM1bj_abdf1reference.png" size="80" width="523" height="795" position="flex-start" caption="AB DF1 reference" showCaption="true" indent="2"></Image>

2. Launch RSLinx Classic and then click **Configure Drivers**.
3. Add an RS-232 DF1 devices driver and then click **Configure**.
4. Set up communication parameters, click **Auto-Configure** to test. Then click **OK**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/NOjKDxWnJPl-iHG06B1f2_ab-df1-auto.png" size="100" width="766" height="357" position="flex-start" caption="Communication parameters" showCaption="true" indent="2"></Image>

5. Launch RSLogix 500 V8.10.00 and then navigate to **Comms > System Comms**.
