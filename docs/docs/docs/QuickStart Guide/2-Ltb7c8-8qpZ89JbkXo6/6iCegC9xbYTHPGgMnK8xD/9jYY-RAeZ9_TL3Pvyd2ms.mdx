To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: Parker ACR9000 Ethernet
   - **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/I4FQLheUqfAlk5NRWBtKH_parkeracr9000ethernet.png" size="80" width="963" height="410" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch the V6.4.2.1014 software and create a project according to device model.
3. Click the device in the Navigation pane, set up the device IP, and then click **Connect**.
