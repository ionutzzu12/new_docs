You can configure an LTE Modem either in Litmus Edge or in the Terminal User Interface (TUI).&#x20;

# Configure an LTE Modem in Litmus Edge

**To configure an LTE modem in Litmus Edge:**

1. After completing the basic set up for your LTE modem, go to **System&#x20;**>**&#x20;Network** on Litmus Edge.
2. At the top of the page, you can see **Host** and **Gateway** information. The Gateway IP address comes from the modem software.
3. Scroll down to the **Modem Configuration** section of the page to view the assigned APN for your modem.
4. Click the **Edit icon** to request a new APN from your network provider.
5. Insert an LTE SIM card into Litmus Edge devices to connect them to the LTE network.

<hint type="info">
  **Note**: For more information about your LTE modem, go to **System&#x20;**>**&#x20;Info** and scroll to the *Modem Interfaces* section.
</hint>

# Connect an LTE Modem with the Terminal Interface (TUI)

**To use the TUI to connect Litmus Edge to an LTE network:**

1. Perform the initial setup of your modem and enter Litmus Edge gateway information. Insert an LTE SIM card into the device to connect it to the LTE network.
2. Open the **TUI** to see information about your LTE connection below eth0 and eth1 information.
3. Press the **F2** key to enter setup. Make sure to choose an appropriate keyboard layout if this is your first time using the TUI. The **us** layout is a standard English keyboard layout. 
4. Use the **Arrow keys** to navigate the TUI. Select **Configure LTE modems** and press **Enter**. Select **Next** with the arrow keys and press **Enter** once again to open the next menu.
5. Select your LTE modem with th&#x65;**&#x20;Arrow keys** and the **Enter&#x20;**&#x6B;ey.
6. Type the APN that you would like to request from your network provider. Press **Enter**.
