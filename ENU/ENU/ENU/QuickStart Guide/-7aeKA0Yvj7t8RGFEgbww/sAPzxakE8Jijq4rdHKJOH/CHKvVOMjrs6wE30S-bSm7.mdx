To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Sick Flexi FX3-CPU1
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/aL0EikBMT3iu7lXU2pY_4_sickflexiserialdevice.png" size="80" width="623" height="690" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch the Flexi Soft Designer v2.01.0 software and click **Com Settings**.&#x20;
3. Edit the connection profile.&#x20;
4. Check **COM auto-detection**.&#x20;
5. Test the connection.&#x20;
6. Click **Connect**. &#x20;
