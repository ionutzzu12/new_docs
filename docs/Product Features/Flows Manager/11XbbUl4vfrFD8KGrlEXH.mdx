# Upload Modules

You can upload a pre-existing module or custom modules from an individual flow.&#x20;

Module files contain a collection of nodes that can be uploaded simultaneously to the Flow canvas.&#x20;

<hint type="warning">
  **Important**:&#x20;

  - Make sure the module file you upload contains all dependencies.&#x20;
  - If your custom node modules are version specific, make sure they work with `nodejs 10.16.3.`
  - If custom node modules have any binding or require C/C++ compilation, the upload may not work.
</hint>

**To upload modules:&#x20;**

1. In your project directory, create a folder named `node_modules`.&#x20;
2. Add the nodes files you want included in the `node_modules` folder.&#x20;
3. Compress the `node_modules` folder into a ZIP file.
   To create the ZIP file, you can right-click the folder in *Windows File Explorer*, and then select **Send to&#x20;**> **Compressed (zipped) folder**. &#x20;
4. In Litmus Edge, navigate to **Flows Manager**.&#x20;
5. Click the **Upload Modules** icon for the flows manager you want to update.&#x20;
6. Select the ZIP file of the folder you compressed and select **Open**.&#x20;
7. After the upload, stop and start the flows manager to implement the changes.&#x20;

If you don't see the expected new nodes after uploading and then stopping/starting, check the flow log to troubleshoot the issue.&#x20;
