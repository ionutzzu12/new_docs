You can manage nodes by navigating to **OPC&#x20;**>**&#x20;Hierarchy**. Refer to the image and callouts for management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/yDbxaUh2BerFCAXbkyIja_image.png "Management options for nodes")

| **Action**                    | **Details**                                                                                                                                                                                                                                                                                            |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| \[1] **Show filter**          | Search for nodes.                                                                                                                                                                                                                                                                                      |
| \[2] **Remove selected node** | After confirming the removal of the node from the *Hierarchy&#x20;*&#x70;ane, click the **Save&#x20;**&#x69;con. After saving the update, you must restart the OPC UA server. See [Manage the OPC UA Server](<././Management/Manage the OPC UA Server.md>) to learn more.                              |
| \[3] **Reset hierarchy**      | Resetting the hierarchy removes all the nodes in the hierarchy. After confirming the reset of the hierarchy, click the **Save&#x20;**&#x69;con. After saving the update, you must Reset the OPC UA server. See [Manage the OPC UA Server](<././Management/Manage the OPC UA Server.md>) to learn more. |
| \[4] **Export as JSON**       | Open the **Action menu** to export the hierarchy exports all of the nodes in the hierarchy to a JSON file and download it to your hard drive. A hierarchy must exist in the *Hierarchy&#x20;*&#x70;ane to view this option.                                                                            |
| \[5] **Import JSON**          | Open the **Action menu** to import all the nodes from a JSON file. The imported nodes merge with the existing hierarchy rather than replace it. After saving the update, you must Reset the OPC UA server. See [Manage the OPC UA Server](<././Management/Manage the OPC UA Server.md>) to learn more. |

<hint type="warning">
  **Caution**: All child nodes and their configurations are deleted when you remove a selected node.
</hint>
