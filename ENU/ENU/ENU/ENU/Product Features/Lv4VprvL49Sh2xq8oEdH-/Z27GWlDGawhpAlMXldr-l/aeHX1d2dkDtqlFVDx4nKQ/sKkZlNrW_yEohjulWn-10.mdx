After you add users, you can manage them by navigating to **System&#x20;**> **Access Control&#x20;**&#x61;nd referring to the *Manage Users* section. &#x20;

# Search for an Existing User

You can use the search bar from the *Users&#x20;*&#x74;ab to search for existing users.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/dSZaMNFgq3On1W9ja3Xih_image.png "The search bar for existing users")

# Manage Existing Users

You can use the **Action menu&#x20;**&#x66;or a user to manage it.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/VuZTp_smnjXHr4_JK4BzI_image.png "The action menu for a user")

- **Edit**: Update the following details for the user.&#x20;
  - First and Last name
  - Username
  - Password
  - Groups user belongs to
- **Remove**: Remove the user from the *Users&#x20;*&#x74;ab.&#x20;
- **Disable/Enable**: Disable or enable the user.&#x20;
