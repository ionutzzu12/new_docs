Multiple users can work on the same lists at the same time. As each users processes changes, they will be synchronized across other *online* mobile devices to ensure that all mobile users have the most up‑to‑date list details.

<hint type="warning">
  **IMPORTANT:&#x20;**&#x49;f your device does not have an active data connection, it cannot send your list updates, nor can it receive list updates from other users.
</hint>
