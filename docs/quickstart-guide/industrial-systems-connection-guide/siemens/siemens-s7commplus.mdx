To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, use PLC software on your computer to configure your device for Ethernet access.

1. Open your TIA Portal project.&#x20;
2. In the project tree, right-click your PLC and select **Properties&#x20;**&#x66;rom the popup menu. The *Properties&#x20;*&#x77;indow opens.
3. To obtain the device’s IP address (if you do not already have it):
   - From the left pane, select **PROFINET** **interface > Ethernet addresses**.
   - In the right pane, go t&#x6F;**&#x20;Internet protocol version 4 (IPv4)**, find the IP address of the PLC, and copy it aside.
4. From the left pane, select **PROFINET interface** > **Advanced options** > **Port \[X1 P1]** > **Port interconnection.**
5. In the right pane, enter `Any partner` in the **Partner port** field.
6. From the left pane, select **Protection & Security > Access control.**
7. In the right pane, under **Access control configuration**, ensure that **Disable access control** is selected.
8. In the project tree, right-click the data block you want to read and select **Properties&#x20;**&#x66;rom the popup menu.
   The *Properties&#x20;*&#x77;indow opens.
9. From the left pane, select **Attributes**.
10. In the right pane, disable the **Only store in load memory** check box.
11. Click **OK.**
12. Repeat steps 8-11 for all data blocks you want to read.
13. Click th&#x65;**&#x20;Compile&#x20;**&#x69;con on the toolbar.
14. When the project finishes compiling, clic&#x6B;**&#x20;Download to Device**.
