You can test an LDAP provider from the *Authentication Providers* list or the *Configuration Update Provider* dialog box by navigating to **System&#x20;**>**&#x20;Access Control&#x20;**>**&#x20;LDAP/AD Auth**.

Tests include:

- Connection status
- Group status, count, and details
- Users status, count, and details

<hint type="warning">
  **Important**: To receive meaningful test results, the LDAP server must be connected to the network.
</hint>

# Test an LDAP Provider

**To test an LDAP provider:**

1. Navigate to **System&#x20;**> **Access Control**.
2. Click the **LDAP / AD Auth** tab.
   The *Authentication Providers* pane appears.
3. Click the **Action menu** next to a provider and select **Test**. The *Provider Test Result&#x20;*&#x64;ialog box appears detailing the test results.&#x20;

# View Groups and Users Details

You can view information on Groups and Users from the *Provider Test Result* dialog box. To view the information, click the **Info&#x20;**&#x69;con in the *Groups&#x20;*&#x6F;&#x72;*&#x20;Users&#x20;*&#x73;ection. The details for the users or groups configured on the LDAP server drop down.&#x20;

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/EpmFzdRlpnE-UkpUjPfJS_image.png" size="80" width="809" height="560" caption="The Provider Test Result dialog box" position="center" showCaption="true"></Image>

Details for Groups include (for each listed group):

- DN
- Group ID
- Group Name
- Users DN, Group ID and Group Name

Details for Users include (for each listed user):

- DN
- Group ID
- First Name
- Last Name

# Test an LDAP Provider Configuration

**To test an LDAP provider configuration:**

1. From the *Authentication Providers* pane, click the **Actions&#xA0;**&#x6E;ext to a provider and select **Configure**.
2. Click **Test**.&#x20;
   The *Update Provider&#x20;*&#x64;ialog box changes to include test results for the selected provider.&#x20;
