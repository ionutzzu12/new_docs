OPC UA server hierarchies involve different types of nodes that map to OPC UA clients:

- **Folder** to represent an edge device, which might be responsible for an industrial plant
- **Device** to represent devices on the industrial plant floor (such as a boiler or lathe PLC)
- **Tag** to represent specific device registers (such as temperature or pressure sensors)

From the Hierarchy pane, you can add, import, modify and delete nodes. You can also reset, export a JSON, and import a JSON for the hierarchy.

# Access OPC UA Hierarchy UI

**To access the OPC UA Hierarchy pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **OPC**.
   The *Hierarchy&#x20;*&#x70;ane appears.
