Review the following information for setting up and configuring the OSI Agent driver.&#x20;

<hint type="warning">
  **Important**:&#x20;

  - The DH Historian Agent can only have one OSI PI agent and one OSI PI server connection with one username/password configured for the OSI PI server.
  - The username/password in the DeviceHub configuration is to access the DH Historian Agent, not the OSI PI server.
  - If you would like to manage access to tags/data being sent through the OSI PI agent and server, you can set up multiple Litmus Edge connections through the NATS connector. See [Set Up Edge Cascading with the NATS Connector](<../../../How-To Guides/Integration Guides/Q4h1xmUjh59N-bcorfkda.mdx>) to learn more. &#x20;
</hint>
