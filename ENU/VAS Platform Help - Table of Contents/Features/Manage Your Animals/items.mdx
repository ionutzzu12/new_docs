\{\{masterVAS-Products.Pulse-only}}\{\{masterVAS-Terminology.cowcard}}s display all of your items on one page. You can expand and collapse each item card and :Link[edit the titles and items contained within each card]{label="edit the titles and items contained within each card" overridedLabel="edit the titles and items contained within each card" spaceId docId="UQaqnJVy7pv-R1Jl4p0AK" version="v2" docAnchorId="#Edit" loadingMethod="dynamic" newTab="false" href="items.mdx"}. You can also :Link[hide item cards]{label="hide item cards" overridedLabel="hide item cards" spaceId docId="UQaqnJVy7pv-R1Jl4p0AK" version="v2" docAnchorId="#Hide" loadingMethod="dynamic" newTab="false" href="items.mdx"} from view altogether or only on the mobile app. If you're unsure what an abbreviation means, hover your cursor over it to display a pop‑up tooltip.

<ExpandableHeading>
  ## Add or Edit a Card

  The instructions for adding a new item card and editing one are very similar. Follow the steps below to get started.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tMdTO3HhYRQ5X0nMTTYx2_new-item-card.png" size="50" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/new-item-card.png" githubPath="Content/Resources/Images/Platform/cowcards/new-item-card.png" position="flex-start" showCaption="false"></Image>

  1. To add or edit a new item card, follow the steps below:\* To create a new item card: Click the **Add New Card** button at the top right of the **Items** page.
     - To edit an item card: Click the Options button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/CDT7MGgVdxUe43MtlyWvh_options-black-dots.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/options-black-dots.png" githubPath="Content/Resources/Images/icons/options-black-dots.png" position="flex-start" showCaption="false" indent="3"></Image>

  <Paragraph indent="2">
    and select **Edit Card** from the drop-down list.
  </Paragraph>

  2. Enter a name in the **Card Name** field. The **Card Preview** pane on the right side of the page updates to show the new card name.
  3. Check or clear the \{\{MorphHelp.ShowInMobileApp}} box. If the box is checked, the web and mobile app versions of \{\{masterVAS-Products.Pulse-only}} display this card. If the box is cleared, the card is visible in the web version of \{\{masterVAS-Products.Pulse-only}} but hidden in the mobile app.
  4. Click the **Color** link below the Card Name field to select a background color and text color for the item card title bar.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/S52hNUvF3A5xUMtIs9h4j_edit-item-card-colors.png" alt caption size="34"}
  5. \{\{Commands.Click}} on items in the **Add Items** list to add them to the card. Each item you add is displayed in the **Card Preview** pane.
  6. To rearrange the items in the card, drag and drop them in the **Card Preview** pane.
  7. To delete an item from the card, click the **X** on that item in the **Card Preview** pane.
  8. Click the **Save** button to add the new card to the Items page or save changes to an existing card.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/OwIc7pxHGsPwMLgK-D0vI_new-item-card-added-new.png" alt caption size="85"}
</ExpandableHeading>

<ExpandableHeading>
  ## Edit Item Values

  Users with [Herd Write permission](../Config/UserMgmtRoles/EditUserPermissions.htm) can change the value of certain items within \{\{masterVAS-Terminology.cowcard}}s. To do so, click in the item to open a side panel in which to make your changes. When you're done editing, click **Save.**

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/dvZ9jX9V2Ju2_DxtDUY87_itemediting.png" size="43" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/ItemEditing.png" githubPath="Content/Resources/Images/Platform/cowcards/ItemEditing.png" position="flex-start" showCaption="false"></Image>

  Most stored item values can be edited. Calculated item values (such as \{\{Age}}), as well as certain restricted items, cannot be edited.
</ExpandableHeading>

<ExpandableHeading>
  ## View Item Details and History

  Click any item to view its details and history in a side panel. The **All Lactation Values** table shows the last value entered for that item in each lactation, as well as the start and end dates for each lactation.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/UpuywkoO2ICReMiaOu8Hw_cc-itemhistory.png" size="86" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/cc-itemhistory.png" githubPath="Content/Resources/Images/Platform/cowcards/cc-itemhistory.png" position="flex-start" showCaption="false"></Image>
</ExpandableHeading>

<ExpandableHeading>
  ## Hide a Card from View

  Though you can't delete item cards from \{\{masterVAS-Products.Pulse-only}}, you can hide them in both the web and mobile versions:

  - To hide an item card in both the web and mobile version: \{\{Commands.Click}} the Options button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/CDT7MGgVdxUe43MtlyWvh_options-black-dots.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/options-black-dots.png" githubPath="Content/Resources/Images/icons/options-black-dots.png" position="flex-start" showCaption="false" indent="2"></Image>

  <Paragraph indent="1">
    at the top right of the card and select **Hide Card**.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/2O2DtYrtLk8rzkltwaLL3_hide-card-new.png" size="86" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/hide-card-new.png" githubPath="Content/Resources/Images/Platform/cowcards/hide-card-new.png" position="flex-start" showCaption="false"></Image>
  </Paragraph>

  - To show an item card again, click the **Restore Hidden Card** button at the top right of the **Items** page and select the cards you want to restore to view.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/hhpzf1ZDU3u6jkxh7_Sct_restore-hidden-card.png" size="44" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/restore-hidden-card.png" githubPath="Content/Resources/Images/Platform/cowcards/restore-hidden-card.png" position="flex-start" showCaption="false" indent="2"></Image>

  <Paragraph indent="1">
    To hide a card only in the mobile app: \{\{Commands.Click}} the Options button
  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/CDT7MGgVdxUe43MtlyWvh_options-black-dots.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/options-black-dots.png" githubPath="Content/Resources/Images/icons/options-black-dots.png" position="flex-start" showCaption="false" indent="2"></Image>

  <Paragraph indent="1">
    and select **Edit Card**. Locate the \{\{MorphHelp.ShowInMobileApp}} box at the top left of the **Edit Items Card** page and clear it (see example under :Link[Add or Edit a Card]{label="Add or Edit a Card" overridedLabel="Add or Edit a Card" spaceId docId="UQaqnJVy7pv-R1Jl4p0AK" version="v2" docAnchorId="#Create" loadingMethod="dynamic" newTab="false" href="items.mdx"} above). With the check cleared from the box, the card is visible in the web version of \{\{masterVAS-Products.Pulse-only}} but hidden in the mobile app.
  </Paragraph>

  <hint type="info">
    **NOTE:&#x20;**&#x54;he color of the More Details button changes depending on the header color of the item card.
  </hint>
</ExpandableHeading>
