To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Simatic S5-95U
   - **Communication I/F**: Ethernet Softing S5-PCLINK Ethernet gateway. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/4kwr5AJ6CWdtTfZVsJ3rB_siemenss5ethernetdevice.png" size="80" width="589" height="778" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch S5-LAN Manager Step5/ST v7.23, click **Search&#xA0;**&#x69;n the local network to browse the gateway hardware, and then click **Settings&#xA0;**&#x74;o set up the network parameters.
3. Launch PLC-VCom and click **Configure&#xA0;**&#x74;o configure the S5-PCLINK to have a virtual port along with its IP. Make sure the status is connected.
4. Launch Step 5, and then click **File > Project > Set**.
5. From the PLC page, double-click **Offline**.
6. Connect to the device.&#x20;
