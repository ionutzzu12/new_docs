You can download an image from a registry.

# Before You Begin

Make sure you have created a registry for the image you are downloading. See [Add a Registry](../registry/add-a-registry.mdx) to learn more.&#x20;

# Download an Image from a Registry

**To download an image from a registry:**

1. Navigate to **Applications&#x20;**> **Images**.
2. Click **Pull Image** at top right. &#x20;

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/zFMiIUdj3N5_RnJ3_39kj_image.png" size="80" width="1915" height="927" position="flex-start" showCaption="false" indent="2"></Image>

3. From the **Pull Image** dialog box, select an option from the **Registry&#x20;**&#x64;rop-down list.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Oh_4cOff4ZHv4l29fcRt9_image.png" size="80" width="1185" height="714" position="flex-start" showCaption="false" indent="2"></Image>

4. Enter a name for the image and click **Pull**.
