Follow the steps below to set up your \{\{masterVAS-Products.PCC-long}} scanner so you can scan or tap genomic sample data using \{\{masterVAS-Products.PCC-short}}, then deliver the data to \{\{DC-UI.DC-ProdNameShort}}:

<ExpandableHeading>
  ## Create Commands to Use PCC

  Before you can start using the scanner, you must first create the following commands in \{\{masterVAS-Products.DC-Platform}}:

  1. In \{\{DC-UI.DC-ProdNameShort}}, enter \{\{ALTER}}**\3** in the command line to open the **Commands Definitions** list.
  2. Click **Add \[Ins]** on the left side of the window to add the following commands:

  | Abbreviation | Content                                          | Title                                                         |
  | ------------ | ------------------------------------------------ | ------------------------------------------------------------- |
  | ZGSAMP       | ENTER\Z GSDAT=\{\{SCDAT}} GSAMP=?                | \{\{masterVAS-Products.PCC-short}} genomic sample batch entry |
  | ZTOTSU       | SERVER \{\{ID}} GSAMP FOR \{\{LACT}}=0 GSAMP="-" | PCC list of animals to sample                                 |
</ExpandableHeading>

<ExpandableHeading>
  ## Set Up Your PCC Scanner

  1. Download the \{\{masterVAS-Terminology.h-erd}} database from \{\{DC-UI.DC-ProdNameShort}}.
  2. Pair the Bluetooth \{\{masterVAS-Terminology.radiofrequencyID-short}} scanner to \{\{masterVAS-Products.PCC-short}}.
  3. Pair the built-in QR scanner to \{\{masterVAS-Products.PCC-short}}.
  4. Navigate to Batch Command Entry in the \{\{masterVAS-Products.PCC-short}} app using your handheld mobile device.
  5. Select the batch command for entering samples: **ZGSAMP**.
  6. In the **For Cows in List** drop-down list, select **ZTOTSU**.
</ExpandableHeading>

<ExpandableHeading>
  ## Scan Genomic Samples

  1. On your \{\{masterVAS-Products.PCC-short}} handheld device, enter the date.
  2. Scan the first animal's \{\{masterVAS-Terminology.radiofrequencyID-short}} tag to identify the animal.
  3. \{\{masterVAS-Products.PCC-short}} will ask you to enter the sample ID. Manually enter the sample ID or scan the sample's QR code using the 2D scanner.
  4. After you record all the samples, post all transactions in the batch command.
  5. On the home screen of the \{\{masterVAS-Products.PCC-short}} app, select **Send Changes** to update \{\{DC-UI.DC-ProdNameShort}} with the new sample assignments.

  After the changes are posted in \{\{DC-UI.DC-ProdNameShort}}, \{\{masterVAS-Products.Pulse-only}} will automatically synchronize with \{\{DC-UI.DC-ProdNameShort}} and the sample data will be available in \{\{masterVAS-Products.Pulse-only}}.
</ExpandableHeading>

After you scan the genomic samples, you can submit them to \{\{Partners.Zoetis}} in \{\{masterVAS-Products.Pulse-only}}. See [Submit Zoetis Genomics Samples](zoetis-genomics-submit-samples.htm) to get started.
