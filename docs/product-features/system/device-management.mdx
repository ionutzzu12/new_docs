From the *Device Management* pane, you can do the following:

- Reboot and factory reset the Litmus Edge device. See :Link[Manage Edge Devices]{label="Manage Edge Devices" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="7vR-_2YDxS0i6t3TB3rNb" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;
- Upgrade the version of Litmus Edge. See [Software Updates](../../quickstart-guide/installation-and-deployments/software-updates.mdx).&#x20;
- Manage the disk space policy.&#x20;
- Manage FTP and SSH services. See :Link[System Services]{label="System Services" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="2IIwsVZ4SFgrCy63Wqcm0" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;
- Configure and initiate system backups and restore operations. To prepare for possible hardware failures, Litmus Edge provides a disaster recovery plan to support device replacement. See [Backup/Restore](device-management/backup-restore.mdx).&#x20;
- Create detailed, granular backup templates and exchange these templates between edge devices as JSON files. See [Backup/Restore](device-management/backup-restore.mdx).

<hint type="info">
  **Note:** Litmus Edge supports using Trusted Platform Module (TPM) components to encrypt certain Litmus Edge Manager credentials using hardware security. After configuring TPM on your device, LEM credentials become encrypted by Litmus Edge. If you require assistance with enabling TPM, contact Litmus support.
</hint>

Access Device Management UI

From the Litmus Edge navigation panel, navigate to **System&#x20;**>**&#x20;Device Management**.
The *Device Management&#x20;*&#x70;ane appears.&#x20;
