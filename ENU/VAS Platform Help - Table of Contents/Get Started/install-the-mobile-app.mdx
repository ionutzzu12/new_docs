The mobile app offers many of the same features as the web version and can be used on \{\{MorphHelp.AndroidOS-TM}} and Apple \{\{MorphHelp.AppleOS-TM}} devices. Review the [Minimum Requirements](https://vas.com/technology-specifications/) on the \{\{masterVAS.CompanyName-short}} website before installing the mobile app on \{\{MorphHelp.AppleOS-TM}} and \{\{MorphHelp.AndroidOS-TM}} devices. Visit Google Play or the \{\{Mobile-UI.AppStore}} on your device to download and install the app.
