\{\{Buttons-UI.DC-WorklistsONLY}} are based on reports you create in \{\{masterVAS-Products.Pulse-only}} and then convert into an actionable list. Follow the steps below to continue:

1. Build a \{\{Worklists.worklistCommonUse}} report using [Dairy Reports/Worklists](ManageReportsWorklists.htm) or [Report/Worklist Builder](ReportWorklistBuilder.htm).
2. View the report and click the **Create Open Worklist** button on the [ReportWorklistResults.htm](see-reportsworklists-results.mdx)[Reports/Worklists Results](see-reportsworklists-results.mdx) page.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/D4Oqlp87hmF4wx6yvN5_E_report-results-worklist.png" alt caption size="56"}:::hint\{type="info"}
   **NOTE:&#x20;**&#x54;he option to create a \{\{Worklists.worklistCommonUse}} is only available on reports run \{\{today}} to ensure the list captures your latest \{\{masterVAS-Terminology.h-erd}} data. If you do not see the **Create Open Worklist** button, rerun the report using today's date.
   \:::
3. \{\{masterVAS-Products.Pulse-only}} converts the \{\{Worklists.worklistCommonUse}} report to a \{\{Worklists.worklistCommonUse}} and opens the list. If you navigate away from the report, you can access it again from your [Open Worklists](ProcessOpenWorklistsWeb.htm) or [Worklist History](../view-worklists-history.mdx) pages.Once the \{\{Worklists.worklistCommonUse}} is available, you can begin checking or treating animals on the list and marking tasks as complete from the web or using mobile app. See [Open Worklists](ProcessOpenWorklistsWeb.htm) and [Complete Tasks (v1)](<../../../../../../Mobile Apps/VAS Platform Mobile App (v1)/Tasks & Worklists (v1)/complete-tasks-v1.mdx>) or [Complete Tasks (v2)](<../../../../../../Mobile Apps/DC on Mobile (v2)/complete-tasks-v2.mdx>) to learn more!:::hint\{type="info"}
   **NOTE:&#x20;**&#x46;or \{\{masterVAS-Products.DCU-users-short}}, \{\{Worklists.worklistCommonUse}} updates in \{\{masterVAS-Products.Pulse-only}} remain synchronized with \{\{masterVAS-Products.DC305-short}}.
   \:::

  ## Understanding Event Lists

  \{\{Worklists.EventListUI-Web}} are a type of \{\{Worklists.worklistCommonUse}} that allow you to add events to animals on the list and then mark the animal as \{\{Worklists.DoneUI}}. An \{\{Worklists.EventListCommonUse}} is generated from [Report/Worklist Builder](ReportWorklistBuilder.htm) using the **Event** option when building a report.
