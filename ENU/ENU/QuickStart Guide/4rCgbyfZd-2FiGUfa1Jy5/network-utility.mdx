- Use :Link[Ping (Gen1)]{label="Ping (Gen1)" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="gCJ5TAtPH1UX6sFk1E9Kp" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to to ping an IP address.
