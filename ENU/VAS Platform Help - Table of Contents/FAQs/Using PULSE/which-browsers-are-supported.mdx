When you log in to \{\{masterVAS-Products.Pulse-only}}, it checks to ensure that you are using a supported browser. For security reasons, the supported browsers are [Google Chrome](https://www.google.com/chrome/), [Safari (Mac)](https://support.apple.com/downloads/safari), [Mozilla Firefox](https://www.mozilla.org/en-US/firefox/), and [Microsoft Edge](https://www.microsoft.com/en-us/windows/microsoft-edge). If you attempt to log in to \{\{masterVAS-Products.Pulse-only}}, with an unsupported browser, the system will display a message that lets you know the browser is not supported, and it will provide a list of supported browsers so that you can easily download the one of your choice.
