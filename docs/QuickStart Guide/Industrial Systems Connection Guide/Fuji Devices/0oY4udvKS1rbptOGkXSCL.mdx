To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it as needed.&#x20;
   - **Device**: Fuji NBOU14
   - **Communication I/F**: RS485 2W. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/i3JUAda-DNpYt8sTfvQu__fuji-nb-serial01.png" size="80" width="642" height="828" position="flex-start" caption="Fuji NB Serial reference" showCaption="true" indent="2"></Image>

2. Launch Flex PC Programmer v2.1.0.35 software.
3. Click **Options > Communication**.
4. Set up COM port and communication parameters.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/kIjq8zrdzXflX2kgEb4WS_fuji-nb-serial-parameters.png" size="80" width="699" height="361" position="flex-start" caption="COM port and communication parameters" showCaption="true" indent="2"></Image>

5. Click the **Online&#xA0;**&#x69;con.
