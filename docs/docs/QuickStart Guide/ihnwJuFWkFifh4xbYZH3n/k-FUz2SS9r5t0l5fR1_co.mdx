Human Interface Devices (HID) are USB-connected peripheral devices that can be polled for data. Examples include scanners used for barcodes and QR codes. To connect these devices to Litmus Edge, you can use the [HID Device (Gen1)](<../../../QuickStart Guide/Industrial Systems Connection Guide/HID Devices/Jl7QCjdrJYHUGutRVicW1.mdx>) serial DeviceHub driver.&#x20;
