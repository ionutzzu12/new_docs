You access and submit \{\{FRS\_VARS\_Self.Request\_Offering}} through external offerings by clicking a URL on an external website. You view and request services, track your request using user and offering IDs in the URL, and choose how the request displays—either in a new window, inside \{\{Primary.Product}}, or within the **Service Catalog**.

To use the external offerings created by your administrator, follow these steps:

1. Log in to \{\{Primary.Product}}.
2. From the **Service Catalog&#x20;**&#x77;orkspace, use the search bar to search for the external offering record.

<Paragraph indent="1">
  The external offering records are displayed.
</Paragraph>

3. Click an external offering card.

<Paragraph indent="1">
  A dialog will prompt you to continue or stay on the page.
</Paragraph>

4. If you select **Continue**, the URL opens in a new tab with your user ID and offering ID included.
