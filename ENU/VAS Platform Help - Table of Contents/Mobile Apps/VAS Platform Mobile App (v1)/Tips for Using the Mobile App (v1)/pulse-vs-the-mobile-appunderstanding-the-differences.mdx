As mentioned in other topics in the [Get Started](GettingStarted.htm) section, \{\{masterVAS-Products.Pulse-only}} has two main components for viewing and managing \{\{masterVAS-Terminology.d-airy}} data—the **web version of&#x20;**\{\{masterVAS-Products.Pulse-only}} and the mobile app. While the web version and the mobile app share the same data, what you can do with each differs. Expand the sections below to learn more:

<ExpandableHeading>
  ## Use PULSE to monitor your data

  The web version of \{\{masterVAS-Products.Pulse-only}} is available through a browser at [https://platform.vas.com](https://platform.vas.com) from any desktop computer, laptop, or mobile device that has an **active** WiFi or data connection (e.g., a tablet or mobile phone).

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/xOXo-jfRPoLdzEiWA4YNi_platform-device-square-web.png" size="27" isUploading="false" initialPath="../../Resources/Images/Platform/getting-started/platform-device-square-web.png" githubPath="Content/Resources/Images/Platform/getting-started/platform-device-square-web.png" position="flex-start" showCaption="false"></Image>

  The primary purpose of the web version of \{\{masterVAS-Products.Pulse-only}} is to **monitor, manage, and analyze the status of your&#x20;**\{\{masterVAS-Terminology.d-airy}}. As such, it is typically used by \{\{masterVAS-Terminology.d-airy}} owners and mangers, \{\{masterVAS-Terminology.h-erd}} and feed managers, and \{\{masterVAS-Terminology.d-airy}}\{\{masterVAS-Terminology.c-onsultants}}.

  \{\{masterVAS-Products.Pulse-only}} brings together your most important feed, \{\{masterVAS-Terminology.h-erd}}, parlor, and industry partner data into a single web-based platform, offering the following capabilities from any web browser:

  - Easy access to your data in a single, web-based format
  - Access to data such as animal records, lactation information, and milk production
  - Dashboards so you can graph animal data to provide important insights
  - [Command line](<../../../../ENU/VAS Platform Help - Table of Contents/features/command-line.mdx>) functionality to generate many of the same reports you do in \{\{DC-UI.DC-ProductName}} (for \{\{masterVAS-Products.DCUnlimited-short}}\{\{masterVAS-Terminology.d-airies}})
  - \{\{Worklist}} and event status data
</ExpandableHeading>

<ExpandableHeading>
  ## Use the mobile app to perform work

  The mobile app is primarily used for work on the \{\{masterVAS-Terminology.d-airy}}. While the mobile app allows you to look up most of the same [CowCard](../Features/CowCards/WorkingWithCowCards.htm) information that's in the web version of \{\{masterVAS-Products.Pulse-only}}, you can also use the mobile app to scan animals, enter animal events (BRED, MOVE, etc.), process worklists, and enter embryo transfer events.

  Unlike the web version of \{\{masterVAS-Products.Pulse-only}}, the mobile app can be used both online **or offline** and has the ability to work with a paired Bluetooth device to make scanning and working with animals even easier.

  With the functionality defined above, the mobile app is typically used while workers are outside the office or cow‑side.
</ExpandableHeading>
