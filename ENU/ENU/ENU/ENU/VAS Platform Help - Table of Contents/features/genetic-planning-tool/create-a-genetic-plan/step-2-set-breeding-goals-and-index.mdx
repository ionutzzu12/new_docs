Set Breeding Goals & Index is the second step of the planning process and allows you to generate a custom index to establish priorities and goals for the next generation of animals in the \{\{masterVAS-Terminology.h-erd}}. Breeding goals in \{\{UDS-general.UDSBrand}}\{\{UDS-general.UDSProduct}} are based on industry-standard indexes, which assign weights to three major categories: Production, Health, and Conformation.

<ExpandableHeading>
  ### See a complete list of Production, Health, and Conformation traits here!

  - Production Traits\* Milk (Lbs)
    - Fat (Lbs)
    - Protein (Lbs)
  - Health Traits\* Productive Life (Months)
    - Daughter Pregnancy Rate
    - Somatic Cells
    - \{\{masterVAS-Terminology.S-ire-}} Calving \{\{Ease}}
    - Daughter Calving Ease
    - \{\{masterVAS-Terminology.S-ire-}} Stillbirth
    - Daughter Stillbirth
    - Cows \{\{Insights.ConceptionRateTab}}
    - Heifers \{\{Insights.ConceptionRateTab}}
    - Livability
  - Conformation Traits\* Udder Composite (UDC)- Fore Udder Attach
    - Rear Udder Height
    - Rear Udder Width
    - Udder Cleft
    - Udder Depth
    - Fore Teat Placement
    - Rear Teat Placement
    - Teat Length

    ### Preset Index Options

    \{\{UDS-general.GENEX-Pulse}} offers four preset indexes: 

    - NM$ (Net Merit Dollars): 40% Production, 34% Health, 26% Conformation
    - CM$ (Cheese Merit Dollars): 41% Production, 34% Health, 25% Conformation
    - FM$ (Fluid Merit Dollars): 43% Production, 32% Health, 25% Conformation
    - ICC (Ideal Commercial Cow Index): \* Holsteins: 53% Production, 34% Sustainability, 13% Fertility. This is a custom index. :Link[Contact your administrator]{label="Contact your administrator" overridedLabel="Contact your administrator" spaceId docId="Kk0UeFPEsM0CshK-aRILO" version="v2" docAnchorId="#Contact" loadingMethod="dynamic" newTab="false" githubPath="../GettingStarted/UserMgmt.htm#Contact" href="../../../../../../../VAS Platform Help - Table of Contents/Features/GENEX Genetic Planning Tool/Get Started with GENEX DairySuite/genex-management.mdx"} if you need to adjust the ICC trait weights.
      - Jerseys/XD: 46% Production, 36% Sustainability, 18% Fertility. This is a custom index that cannot be adjusted by trait.

    <hint type="info">
      **NOTE:&#x20;**\{\{UDS-general.UDSBrand}}'s indexes are reevaluated and adjusted based on current economic trends and industry standards. As industry indexes are updated, \{\{UDS-general.UDSBrand}} \{\{UDS-general.UDSProduct}}'s presets will likely be adjusted as well.
    </hint>

    If applicable, make additional adjustments to the major categories, Production, Health, and Conformation, making sure the weights add up to 100%.

    View the Current \{\{masterVAS-Terminology.H-erd-}} average and Progeny projections for each trait.:::ExpandableHeading

    ### How is this value calculated?

    - Current \{\{masterVAS-Terminology.H-erd-}}: Average value for this trait for all current animals in the \{\{masterVAS-Terminology.h-erd}}, or:(sum of all animal values for this trait)/(total number of animals in \{\{masterVAS-Terminology.h-erd}}). The source for this information is \{\{masterVAS-Products.Pulse-only}}.
    - Progeny projection (greatest potential improvement): The highest possible improved value for the next generation of the \{\{masterVAS-Terminology.h-erd}} for this trait. This calculation is based on matings with optimal bulls for each specific trait.
      \:::
  </WorkflowBlockItem>

  <WorkflowBlockItem>
    Hover over the graph for deviation from breed average data details.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/rJO4O19NE9p95C4t9w5Gc_step3-hover.png" alt caption size="17"}:::ExpandableHeading

    ### How is this data calculated?

    Each trait has an average value based on the \{\{masterVAS-Terminology.h-erd}} breed. The potential improvement (or deviation) to each trait value is calculated and shown for each of the bull tiers: Silver, Gold, Platinum.
    \:::
