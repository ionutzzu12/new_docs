The *Connections&#xA0;*&#x70;ane allows you to manage clients that connect to the OPC UA server using certificate-based authentication. See [Manage OPC UA Authentication](<../../../Product Features/OPC UA Server/Management/dXPq-8fpo0U_HE-gK800u.mdx>) for more information.&#x20;

For a certificate-authenticated OPC UA client to work, the OPC UA server must accept (trust) that client's requests.

You can add and refresh an OPC UA client connection, view, refresh and remove client certificates, and manually download the OPC UA server certificate.

If you are using the Docker version of Litmus Edge and having issues connecting to the OPC UA server, refer to [Issues Connecting to OPC UA Server](<../../../Troubleshooting/OPC UA Troubleshooting/kBk45E-_KBkAeJ8bCIxl4.mdx>).&#x20;

# Access OPC UA Connections UI

**To access the OPC UA Connections pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **OPC&#x20;**>**&#x20;Connections**.
   The *Connections&#x20;*&#x70;ane appears.&#x20;
