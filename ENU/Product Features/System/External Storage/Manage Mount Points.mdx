<hint type="info">
  **Note**: The status of mounted storage is monitored. If the external storage is down, then the storage is automatically unmounted. The storage is automatically mounted when the storage reconnects.&#x20;
</hint>

You can manage mount points by navigating to **System&#x20;**>**&#x20;External Storage**.&#x20;

Click the **Action menu** for a mount point to view management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/KHOFd5LyuIcOcFm1dUgb4_image.png "The action menu for a mount point")
