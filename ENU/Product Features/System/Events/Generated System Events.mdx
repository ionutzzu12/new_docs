Events are displayed when you complete a specific action for a component.&#x20;

<hint type="info">
  **Note**: Some actions performed are local.&#x20;
</hint>

The event can be a success or a failure. Review below what each event may display and the possible events for each component.&#x20;

| --------------------- | ----------------------------------------------- | ------------------- |

- Failed to delete user and failure reason
- Role created
- Failed to create role and failure reason
- Role deleted
- Failed to delete role and failure reason

- Failed to assign role to the user and failure reason

- User logged in
- Login lockdown initiated                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
