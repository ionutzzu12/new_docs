To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
2. **Device**: Keyence KV-5000 Series
3. **Communication I/F**: Ethernet. Refer to the following figures.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/ZJS7Az-QzXe41kRUHpEJ7_keyence-nano-ethernet.png" size="80" width="373" height="420" caption="Keyence Nano Ethernet reference" position="center" darkWidth="373" darkHeight="420" showCaption="true" indent="2"></Image>

4. Launch the KV-Studio Software, and then click **Tools > Ethernet IP Settings**.
5. Set up the IP address.&#x20;

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/8yrnI9S-IHaPDD1phYgLg_keyence-nano-ethernet-ip-address.png" size="80" width="568" height="444" caption="IP address setup" position="center" darkWidth="568" darkHeight="444" showCaption="true" indent="2"></Image>
