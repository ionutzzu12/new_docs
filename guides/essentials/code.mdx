## Basic Code Block

Use triple backticks with a language tag:

````markdown
```javascript
const greeting = 'Hello, world!'
console.log(greeting)
```
````

Renders as:

```javascript
const greeting = 'Hello, world!'
console.log(greeting)
```

## Supported Languages

Common languages: `javascript`, `typescript`, `python`, `bash`, `json`, `yaml`, `html`, `css`, `jsx`, `tsx`, `sql`, `go`, `rust`, `ruby`, `php`, `java`, `c`, `cpp`, `swift`, `kotlin`, `markdown`.

## Shell Commands

```bash
# Install
npm install -g @archbee/cli

# Create a project
archbee new my-docs

# Start dev server
cd my-docs && archbee dev

# Validate
archbee validate
```
