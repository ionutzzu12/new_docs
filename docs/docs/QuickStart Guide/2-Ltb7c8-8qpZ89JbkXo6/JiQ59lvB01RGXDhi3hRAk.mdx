The :Link[Profinet Acyclic]{docId="YeAxA_BhNznpMvAukBrO6" label="Profinet Acyclic" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId overridedLabel="Profinet Acyclic"} driver supports read-only communication with devices that use PROFINET Acyclic communication. The driver also supports two optional :Link[application profiles]{href="https://www.profinet.com/profinet-explained/profinet-application-profiles" newTab="true" hasDisabledNofollow="false"}: `PROFIdrive` and `PROFIenergy`.

If you have a :Link[Turck TBEN-L5-8IOL]{href="https://www.turck.us/en/product/0000001600019e750008003a" newTab="true"}, use the :Link[Profinet Acyclic Turck]{label="Profinet Acyclic Turck" spaceId="JWQMp_kvm0WsA_Fpjt1v_" docId="YruiD_Uc6fybWo0p9D0FO" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} driver. The Turck TBEN I/O Master uses Profinet Acyclic communications, but does not use the Slot and Subslot device parameters.
