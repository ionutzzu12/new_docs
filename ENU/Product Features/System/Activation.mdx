From the *Activation* pane, you can do the following:

- Activate the Litmus Edge device with Litmus Edge Manager. See [Activate an Edge Device](#) to learn more.&#x20;
- From the *License&#x20;*&#x74;ab, you can:
  - Configure a license server. See [License Server Installation](<./././QuickStart Guide/Installation and Deployments/License Server Installation.md>) to learn more.&#x20;
  - Manage node-locked licenses for the Litmus Edge device.&#x20;
  - View the current Litmus Edge version and License ID.
  - View the currently installed modules in the Litmus Edge device.&#x20;

Refer to the following to learn more about licenses:

- [Licenses](Activation/Licenses.mdx)
- [Site and Node-Locked License Overview for Litmus Edge](<./././Resources/Site and Node-Locked License Overview for Litmus Edge.md>)

Activating your Litmus Edge device does the following:

- Creates a connection to communicate with Litmus Edge Manager.&#x20;
- Creates a default SSL connector in Integration.
- Adds a Custom CA certificate for Litmus Edge Manager.
- Configures a Litmus Edge so you can have remote access to the Litmus Edge device from Litmus Edge Manager.

# Access Activation UI

From the Litmus Edge navigation panel, navigate to **System&#x20;**>**&#x20;Activation**.
The *Cloud Activation* pane appears.&#x20;
