The *DB Users* pane is used:

- For role-based access control to the `tsdata` database
- To create new users
- To define `READ`, `WRITE`, and `ALL` access to the database

<hint type="warning">
  **Important**: InfluxDB uses **admin**, **user**, and **observer&#x20;**&#x61;s default users. Therefore, you cannot use these user names when creating a new user.
</hint>

# Access DataHub DB Users UI

You can access the *DB Users&#x20;*&#x55;I from the *DataHub*.&#x20;

**To access DB Users:**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **DataHub&#x20;**>**&#x20;DB Users**.
   The *DB Users* pane appears.
