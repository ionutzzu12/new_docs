La page Bienvenue dans Enterprise Self Service est un hub qui regroupe tout votre contenu Self Service, organisé par département. De là, vous pouvez naviguer dans le portail Self Service Mobile en sélectionnant le département auquel vous souhaitez accéder, comme Services IT, Ressources humaines, Services généraux ou Opérations de sécurité.

Par exemple, si vous choisissez **Ressources humaines** dans la page Bienvenue dans Enterprise, vous êtes redirigé vers le portail Self Service Mobile, mais le contenu visible (**Catalogues de services**, **Annonces**, **FAQ**, **Mes éléments** et **Articles de connaissances**) est propre aux ressources humaines. De même, si vous sélectionnez **Opérations de sécurité**, le portail affiche le contenu SecOps.

Cette fonction est disponible uniquement si l'administrateur de l'entreprise l'a activée à votre attention. Contactez l'administrateur de votre entreprise pour toute question ou inquiétude.

Lors de la conception de la page Bienvenue dans Enterprise, suivez les recommandations ci-dessous concernant le format d'image/de logo, la taille et la résolution :

| Élément              | Format     | Limite de taille | Résolution                      |
| -------------------- | ---------- | ---------------- | ------------------------------- |
| Logo                 | .svg       | 1 Mo             | -                               |
| Image d'arrière-plan | .jpg, .png | 2 Mo             | 1 920 (largeur) x 550 (hauteur) |
| Logo de carte        | .svg       | 1 Mo             | -                               |
