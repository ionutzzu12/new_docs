To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

Refer to the following parameters.&#x20;

**Online simulator**: YES

**Extend address mode**: NO

**Broadcast command**: NO

**To configure your device:&#x20;**

1. Note down the following information and refer to it when needed.&#x20;

- **Device**: Mitsubishi Q03UDCPU and QJ7124N
- **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/zYTT-nh8UyvV0haFGquzl_mitsubishi-q00-01h-serial-reference.png" size="80" width="590" height="735" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

1. Launch GX Works2 v1.560 software, and then click **Online > Read from PLC**.
2. From the PLC Series Selection dialog box, select **QCPU (Q Mode)** from the **PLC Series&#xA0;**&#x64;rop-down list, and then click **OK**.
3. Set up communication parameters, and then click **Connection Test**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/RjFvb33CZafgUnNahw-ts_mitsubishi-q00-01h-serial-test.png" size="80" width="1001" height="608" position="flex-start" caption="Connection Test option" showCaption="true" indent="2"></Image>

4. Right-click **Intelligent Function Module > New Module**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Lr691etkMvjOkCIrWXQo1_mitsubishi-q00-01h-serial-new-module.png" size="80" width="796" height="441" position="flex-start" caption="New Module option" showCaption="true" indent="2"></Image>

5. Set up module parameters as shown below.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/rbnPuT6U6VPzNk2n0bu9x_mitsubishi-q00-01h-serial-parameters.png" size="80" width="1120" height="579" position="flex-start" caption="Module parameters" showCaption="true" indent="2"></Image>

6. Click **Switch Setting&#xA0;**&#x61;nd set up communication parameters as shown below.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/OAD9hEvHnDkbUBDkTWHmt_mitsubishi-q00-01h-serial-switch.png" size="80" width="758" height="425" position="flex-start" caption="Switch Setting option" showCaption="true" indent="2"></Image>
