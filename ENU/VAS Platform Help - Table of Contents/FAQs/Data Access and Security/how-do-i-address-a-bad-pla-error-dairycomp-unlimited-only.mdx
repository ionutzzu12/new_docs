If you encounter a bad \{\{masterVAS-Terminology.PLA}} file error message, [contact us](https://dc-help.vas.com/GetStarted/ContactUs.htm) for support.

This error message is related to \{\{masterVAS-Products.Pulse-only}}. A bad \{\{masterVAS-Terminology.PLA}} file prevents \{\{masterVAS-Products.Pulse-only}} from properly updating with your \{\{masterVAS-Products.DC305-short}} data. This message often displays as a separate message box after restarting \{\{masterVAS-Products.DC305-short}}. In some cases, you might have seen and closed this message previously.

If you receive a bad \{\{masterVAS-Terminology.PLA}} file error message, or if your \{\{masterVAS-Products.Pulse-only}} isn't refreshing with \{\{masterVAS-Products.DC305-short}}, [contact us](https://platform-docs.vas.com/en-us/Content/Platform/GetStarted/ContactUs.htm) for support. This issue requires specialized assistance, and attempting to fix it yourself can lead to further complications.

  **NOTE:&#x20;**&#x54;hese instructions apply only to \{\{General.DCUnlimited}}.
