On the *Configure&#x20;*&#x70;ane of the *Applications&#x20;*&#x6D;odule, you manage marketplaces. Initially, the pane includes only the default marketplace catalog. You can add, modify, and remove multiple marketplaces.

You can deploy a Private Marketplace catalog so you can leverage existing custom Docker applications. You can include custom Docker applications that are under development and third-party Docker applications in the Private Marketplace.

You can add multiple Private Marketplaces, and each Private Marketplace can install multiple Docker applications. The number of Docker applications that can be added is limited only by the Litmus Edge’s hardware size.

# Access Applications Configure UI

**To access the Applications Configure pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **Applications**.
   The *Dashboard&#x20;*&#x70;ane appears.
3. Click **Configure**.&#x20;
   The *Configure* pane appears.
