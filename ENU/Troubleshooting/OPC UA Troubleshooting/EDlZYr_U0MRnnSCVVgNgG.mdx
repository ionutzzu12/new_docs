If you are using the Docker version of Litmus Edge and have issues connecting to the OPC UA server, do the following:

1. When adding the OPC UA server from the client, use advanced mode instead of discovery.&#x20;
2. Manually enter the endpoint URL with the respective port of the Docker.&#x20;
3. In Litmus Edge, navigate to **OPC&#x20;**>**&#x20;Management**, click **Configure**, and configure the IP address to 0.0.0.0. &#x20;
