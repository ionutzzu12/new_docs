### WeighComp Release v2.7.11 - October 2025

<ExpandableHeading>
  ### Improvements

  - Physical Harvest ticket improvements: We made several updates to improve the visibility and usability of physical Harvest tickets. Tickets now show the full dairy name, even if the name exceeds 24 characters. We also added a header and a footer to extend the length of the ticket for better readability.
</ExpandableHeading>

<ExpandableHeading>
  ### Bug Fixes

  - Data recovery update: If your internet connection drops while updating harvest loads, you no longer have to manually enter information from printed receipts. \{\{masterVAS.CompanyName-short}} can now recover the data from log files. Simply contact [fwsupport@vas.com](mailto\:fwsupport@vas.com) to retrieve your data.
  - Several more bugs were squashed in this update. For more information, please contact the \{\{masterVAS.CompanyName-short}} Feed Support Team at [fwsupport@vas.com](mailto\:fwsupport@vas.com).
</ExpandableHeading>

    ## WeighComp Release v2.7.9 - August 2024

    <ExpandableHeading>
      ### Improvements

      - Add a truck directly from the Harvest screen: When you set up a harvest, you can now easily add a new truck directly from the **Select truck** screen. Simply tap **+ Add new truck**, then enter and save the truck details.
      - Quickly search for trucks or truck companies: You can now quickly add or remove trucks from a harvest by using the new search feature for a truck or trucking company. Instead of scrolling through a long list of trucks, just enter the truck name or company, then tap your selection.
      - \{\{masterVAS-Products.WC-Platform}} apps are now compatible with newer Android OS devices: We updated the \{\{masterVAS-Products.WC-Platform}} app and the Driver app to be compatible with devices that run on newer versions of the \{\{masterVAS-Terminology.AndroidOS}} operating system. If you have a newer Android device, you can now download the latest versions of the \{\{masterVAS-Products.WC-Platform}} app and the Driver app without error messages.
    </ExpandableHeading>

    <ExpandableHeading>
      ### Bug Fixes

      - Resolved an issue where an older version of \{\{masterVAS-Products.WC-Platform}} displayed old transactions if you selected the **Forgot the name you added?** button.
      - Several more bugs were squashed in this update. For more information, please contact the \{\{masterVAS.CompanyName-short}} Feed Support Team at [fwsupport@vas.com](mailto\:fwsupport@vas.com).

    ## WeighComp Release v2.7.8 - January 2024

    <ExpandableHeading>
      ### New Features

      \{\{masterVAS-Products.WC-Platform}} now logs activity to improve troubleshooting: We now track your spot load transaction data to make sure your data is accurate and up-to-date on the app and in \{\{masterVAS-Products.Pulse-only}}. The app captures all transaction data, like where the data is being stored and how the data is transferred between the app and \{\{masterVAS-Products.Pulse-only}}. The app sends daily logs to \{\{masterVAS.CompanyName-short}}, which we can use to better resolve issues like duplicate tickets or lost data.
    </ExpandableHeading>

    <ExpandableHeading>
      ### Improvements

      - Improved communication between \{\{masterVAS-Products.Pulse-only}} and the \{\{MorphHelp.WeighComp}} app: We improved communication between \{\{masterVAS-Products.Pulse-only}} and the \{\{MorphHelp.WeighComp}} app to ensure that the data is always accurate and up-to-date.
      - Improved the Harvest dashboard: We made various improvements and fixed multiple issues in the Harvest dashboard, including the following updates:\* Moved the Print Harvest button to a more accessible place: When you open a harvest summary, the **Print Harvest** button is now positioned at the bottom right part of the screen.
        - Added the ability to assign a trucking company: You can now add a trucking company when you add a new truck or edit a truck in the \{\{masterVAS-Products.WC-Platform}} app.
    </ExpandableHeading>
