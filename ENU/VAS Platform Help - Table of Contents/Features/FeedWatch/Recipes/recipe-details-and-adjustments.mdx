  ## Recipe Details

  To view this page, \{\{Commands.click}} the **Details** link at the top right of any \{\{recipe}} card. The example below illustrates the information on this page.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tm4_K74uvNAjGIxuFY9Py_recipe-details-illustrated.png" size="68" isUploading="false" initialPath="../../../Resources/Images/Platform/feedwatchplatform/recipe-details-illustrated.png" githubPath="Content/Resources/Images/Platform/feedwatchplatform/recipe-details-illustrated.png" position="flex-start" showCaption="false"></Image>

  1. Details Summary: The top of the page displays a summary of the most relevant recipe details.:::ExpandableHeading

  2. Alternate Recipes: If a recipe has alternate options (i.e., other recipes that can replace it), they display at the far right of the Details Summary, according to the following parameters:\* If the recipe has only one alternate, the name of that recipe displays in this section.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/lI80BmDExUBjcPbeiInVg_altrecipesoff.png" alt caption size="18"}
     - To select the alternate recipe, click the switch to active it. When you do so, the Alternate Recipes field turns green.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/uZQZX1InaNQWfqVjn562I_altrecipeson.png" alt caption size="10"}
     - If the recipe has multiple alternates, a **SEE OPTIONS** link displays. Click that link to display a window containing the alternate recipes, then choose the recipe you want. Additionally, you can view recipe details.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/LTw5RMqze35Y9nSEv4fWv_alt-recipes-dialog.png" alt caption size="33"}
  3. Recipe Composition: The Recipe Composition bar provides a quick glance at recipe composition by color‑coded ingredient type. The legend above the composition bar shows the percentage of each ingredient type that is included in the recipe. The recipe composition is based on dry matter quantities, so it will not include ingredients such as water that are 0% dry matter.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/qpOI9-Hu0vgKZ0RFi3Fbi_recipe-details-composition.png" alt caption size="38"}
  4. Recipe Directions: This section lists all ingredients in the recipe, as well as the order in which they should be mixed. The locations, quantities, and percentages are also listed for each ingredient. If a numbered row displays a gray bar with the clock symbol, it indicates a time lapse. See :Link[Time Lapse]{label="Time Lapse" overridedLabel="Time Lapse" spaceId docId="j3JUncSmAEj170hFY5ckK" version="v2" docAnchorId="#Time_Lapse" loadingMethod="dynamic" newTab="false" href="recipe-details-and-adjustments.mdx"} below for details.
  5. Ingredient Type: The ingredient type is indicated by the color‑coded bar at the beginning of the row. The predefined types are Forage, Protein, Concentrate, Additive, Mineral, and Premix. Any type that does not belong to one of these categories is displayed with the color‑coded label for Other ingredient types (for example, Grain, Hi-Fiber, Fat, etc.).
  6. Link to Ingredient Details: \{\{Commands.Click}} on any ingredient name to view its details. See [Ingredients](../ingredients.mdx) for help reviewing ingredient details.
  7. Time Lapse: If a recipe requires a pause during the mixing sequence, the pause (i.e., time lapse) will be displayed as a numbered item in the Recipe Directions. A time lapse in the mixing process can be used to allow the ingredients to be mixed for an allotted time. A time lapse may be desired after a specific ingredient is added to the \{\{mixer}} (for example, to mix coarse hay for an additional amount of time before the feeder is prompted to add the next ingredient to the mixer).To easily distinguish a time lapse from an ingredient, time lapses are displayed as a gray bar and clock icon on their own numbered row in the recipe. The length of the pause is shown in **m**inutes and **s**econds (for example, 4**m**:30**s**).:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wdE3TgXMXAarmUObhUmfA_recipe-details-time-lapse.png" alt caption size="43"}
  8. Premix Ingredient: Premix ingredients are displayed on an expandable row. \{\{Commands.Click}} anywhere on the row to expand it and view all ingredients that compose the premix. For more information on premixes, see :Link[Premix Recipes]{label="Premix Recipes" overridedLabel="Premix Recipes" spaceId docId="mFmG6tjKhh5JEpRfxMuwv" version="v2" docAnchorId="#Premix" loadingMethod="dynamic" newTab="false" githubPath="FeedWatchPlatform-RecipesList.htm#Premix" href="FeedWatchPlatform-RecipesList.htm#Premix"}.:::ExpandableHeading

  ## Recipe Order Adjustments

  Users with permissions to edit this page can make adjustments to ingredient mixing order by selecting **Reorder Sequence**. Use the arrows on the left to move ingredients up or down, or drag and drop ingredients to adjust their position in the sequence. Once the mixing order is correct, select **Save** to capture your changes.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/O4eWKtApY5WGhPfM7heVb_recipe-details-edit-reorder.png" size="55" isUploading="false" initialPath="../../../Resources/Images/Platform/feedwatchplatform/recipe-details-edit-reorder.png" githubPath="Content/Resources/Images/Platform/feedwatchplatform/recipe-details-edit-reorder.png" position="flex-start" showCaption="false"></Image>

  All sequence changes made during the current window session remain highlighted until you select **Save**.
