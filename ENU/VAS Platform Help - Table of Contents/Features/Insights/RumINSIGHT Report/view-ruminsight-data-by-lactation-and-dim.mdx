Click the **By&#x20;**\{\{Lact}}**/**\{\{DIM}} tab on the RumINSIGHT report to view detailed fatty acid test results organized by Days in Milk (DIM) and lactation groups. Use the image and descriptions below to learn how to use the information on this page.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/xUwinTrU0PkXvzN4nxwgC_lactfatgraph.png" size="75" isUploading="false" initialPath="../../../Resources/Images/Platform/FattyAcidGroups/LactFatGraph.png" githubPath="Content/Resources/Images/Platform/FattyAcidGroups/LactFatGraph.png" position="flex-start" showCaption="false"></Image>

1. By Lact/DIM tab: Select this tab to view the fatty acid test data organized by Days in Milk (DIM) and lacation groups.
2. Lactation Stage chart: This chart provides detailed fatty acid data and related values grouped by DIM *for all lactation groups*. Data values of concern (i.e. warnings) are shown in red.
3. Lactation group selector: Click the applicable button to choose which lacatation group's data displays in the Lactation Breakdown chart.
4. Lactation Breakdown chart: This chart provides detailed fatty acid data and related values broken down by lactation group. Data values of concern are shown in red.
