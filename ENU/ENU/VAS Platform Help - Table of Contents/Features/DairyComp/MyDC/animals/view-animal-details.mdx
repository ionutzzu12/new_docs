The \{\{masterVAS-Products.Pulse-only}}[Animals](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/animals.mdx>) list allows you to view each animal's details. You can also add events, view event details, or edit the details for an animal from here.

You can find this feature under the :Link[DairyComp/MyDC]{label="DairyComp/MyDC" overridedLabel="DairyComp/MyDC" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../navigate-pulse/menu-options.mdx"} > :Link[Animals]{label="Animals" overridedLabel="Animals" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp-Testdays" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp-Testdays" href="../../../../navigate-pulse/menu-options.mdx"} menu.

\{\{Commands.Click}} the animal's \{\{ID}} on the \{\{Buttons-UI.DC-Animals}} list to view its details in a side panel.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/96TkB06SuwATCw4CSqp52_herdsidepanel.png" size="57" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/HerdSidePanel.png" githubPath="Content/Resources/Images/Platform/dairycomp/HerdSidePanel.png" position="flex-start" showCaption="false"></Image>

In the side panel, you can use the left and right arrow keys on your keyboard to step through side panels for the next/previous \{\{Side-Panels.PanelContent}} in the list. To close the side panel, click the **X** button in the top-right corner or press the **Esc** key.

Expand the sections below to learn more:

<ExpandableHeading>
  ## Add Events

  1. Click the animal's ID to view its details in a side panel. \{\{Commands.Click}} the **Add events** button.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/TnkrnGXp0crs13a8GSLiX_dc-animals-addevent.png" size="30" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/dc-animals-addevent.png" githubPath="Content/Resources/Images/Platform/dairycomp/dc-animals-addevent.png" position="flex-start" showCaption="false" indent="2"></Image>

  1. From the animal's \{\{masterVAS-Terminology.cowcard}}, click the **Add New Event** button.
  2. In the side panel, select an event from the **New Event** drop-down list.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/YSse6M1U7VSPZXrU2Y0xk_addeventsidepanel.png" alt caption size="47"}:::ExpandableHeading

  ### Click here to learn more about working with events!

  <Paragraph listStyleType="disc" indent="2">
    Events are listed in order by Event Code (EC).
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Scroll down the list to view all events.
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    The \{\{masterVAS-Products.DC-Platform}} event command is displayed below each event.
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Events that allow custom remarks display "PROMPT" in the event command. See :Link[event details]{label="event details" overridedLabel="event details" spaceId docId="XGUpytk7Zwvz0GGKLpkd6" version="v2" docAnchorId="#event-details" loadingMethod="dynamic" newTab="false" href="view-animal-details.mdx"} for more information.
    \:::
  </Paragraph>

  3. Enter the details for the event in the applicable fields.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/A5HYQ5pcqd1muVkFWYwHf_addeventdetails.png" alt caption size="43"}The details required in this step depend upon the type of event you're adding. The example above adds a \{\{Events.4HEAT}} event. For more information about the details required for each type of event, see [Event Entry (v1)](../../../MobileApp/MobEventsIntro.htm) and [Event Definitions](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/Events/hHY8-event-definitions.mdx>).:::ExpandableHeading\{customAnchorSlug="event-details"}

  ### Click here to learn more about event details!

  <Paragraph listStyleType="disc" indent="2">
    All required fields are marked with the **\*** (asterisk) symbol.
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    If the event requires a remark, you can select it from the remark pick list. If the event allows a custom remark, you can type it into the \{\{REM}} field instead of using the pick list. The custom field allows you to leave the remark blank as well. See :Link[Edit Event Definitions]{label="Edit Event Definitions" overridedLabel="Edit Event Definitions" spaceId docId="GxXIleQm2y07Vuz9c38iV" version="v2" docAnchorId="#Edit" loadingMethod="dynamic" newTab="false" githubPath="../Settings/DC-EventDefinitions.htm#Edit" href="../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Settings/General Settings/event-definitions.mdx"} for more information.When you enter a custom remark in the **REM** field, the pick list opens and an **+Add** option is available. Click the **+Add** option to select your custom remark and add it to the **REM** field.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/74aj2vJ_9MlZM0ZC-3lcy_addeventsidepanelcustomrem.png" alt caption size="34"}
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    To add additional events to the animal, click **Add another event**.
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    To remove an event, click the delete icon
  </Paragraph>

  4. Define \{\{Protocols.Protocol-common}}s for the event (if applicable).Certain events allow you to select a treatment \{\{Protocols.Protocol-common}}, such as drugs to be administered or treatments to be applied to the animal's hooves or udders. If you are entering an event that includes \{\{Protocols.Protocol-common}}s, you will see the \{\{Protocols.Protocols}} field as an option for the event.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ZgRU-SsHNGfaKHV3ZTuRu_protocols1.png" alt caption size="24"}::::ExpandableHeading

  ### How do you add protocols to an event?

  <Paragraph indent="1">
    The treatment \{\{Protocols.Protocol-common}}s for each eligible event are pre-configured so that they are available when you add the event to an animal. See [Protocols](../Settings/DC-Settings-Protocols.htm) to learn more.
  </Paragraph>

  <Paragraph listStyleType="decimal" indent="2">
    Click the \{\{Protocols.Protocols}} field to view the available \{\{Protocols.Protocol-common}}s for the event.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/23FNiDAVwONRhCe4MWLgx_protocols2.png" alt caption size="61"}For each \{\{Protocols.Protocol-common}}, the system displays the corresponding Milk and Meat withholdings. Additionally, the system auto-fills the **Remark** field and the animal's \{\{masterVAS-Terminology.p-en}} (**Go To&#x20;**\{\{masterVAS-Terminology.P-en-}}) based on the selected \{\{Protocols.Protocol-common}}. **You may select only one&#x20;**\{\{Protocols.Protocol-common}}**&#x20;per event.**:::hint\{type="info"}
    **NOTE:&#x20;**&#x49;f the event includes a default remark or pick list, the \{\{Protocols.Protocol-common}}'s REM overrides it.
    \:::
  </Paragraph>

  <Paragraph listStyleType="decimal" listStart="2" indent="2">
    If you select a \{\{Protocols.Protocol-common}} that requires you to choose which hoof or quarter of the udder is affected, the system opens a page where you can select the quarter(s).:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/PEmlUNrQGsma3Zj7yYaDO_protocols4.png" alt caption size="34"}The labels you see may be different from those listed below, but the typical ones are:\* **ALL** - Selects all quarters/quadrants
  </Paragraph>

  <Paragraph listStyleType="disc" indent="3">
    **LF** - **L**eft **F**ront hoof or teat
  </Paragraph>

  <Paragraph listStyleType="disc" indent="3">
    **RF** - **R**ight **F**ront hoof or teat
  </Paragraph>

  <Paragraph listStyleType="disc" indent="3">
    **LH** - **L**eft **H**ind hoof or teat
  </Paragraph>

  <Paragraph listStyleType="disc" indent="3">
    **RH** - **R**ight **H**ind hoof or teat:::hint\{type="info"}
    **NOTE:&#x20;**&#x54;he labels for quarters/quadrants can be customized in \{\{masterVAS-Products.Pulse-only}} [Animal Configuration](../Settings/DC-AnimalConfiguration.htm) settings or in \{\{DC-UI.DC-ProdNameShort}} for \{\{masterVAS-Products.DCUnlimited-short}} \{\{masterVAS-Terminology.d-airies}}.
    \:::See [Events with Protocols](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/Events/events-with-protocols.mdx>) to learn more.
  </Paragraph>

  <Paragraph listStyleType="decimal" listStart="3" indent="2">
    When you are done selecting the affected quarters, click **Continue** to return to the Add Event window. From here, you can choose additional events for the animal, and then click **Save**.:::hint\{type="warning"}
    **IMPORTANT:&#x20;**&#x54;he mobile app section of this document includes additional information about using \{\{Protocols.Protocol-common}}s with events. See [Events with Protocols](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/Events/events-with-protocols.mdx>) to learn more.
    \:::
    \::::
  </Paragraph>

  5. Click **Add another event** to add additional events to the animal (optional).
  6. Once you have entered the events and details, click **Save**.

  ## Update Events

  1. \{\{Commands.Click}} the animal's ID to view its details in a side panel. \{\{Commands.Click}} the **Go to Events Page** link.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/zd7gSGN92hSFfK5rqukgv_dc-animals-deleteevent.png" size="28" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/dc-animals-deleteevent.png" githubPath="Content/Resources/Images/Platform/dairycomp/dc-animals-deleteevent.png" position="flex-start" showCaption="false" indent="2"></Image>

  <WorkflowBlock>
    <WorkflowBlockItem>
      Click on the desired event to open the event's side panel.
    </WorkflowBlockItem>

    <WorkflowBlockItem>
      <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/uF8PpTMSl6FZI89Qrbzbb_updateevent.png" size="83" isUploading="false" initialPath="../../../../Resources/Images/Platform/cowcards/UpdateEvent.png" githubPath="Content/Resources/Images/Platform/cowcards/UpdateEvent.png" position="flex-start" showCaption="false"></Image>

    </WorkflowBlockItem>

    <WorkflowBlockItem>
      - You can only edit events from the current lactation.
      - Some event updates may reverse past event actions. For example, if you delete a MOVE event where an animal was moved to a new \{\{masterVAS-Terminology.p-en}}, the system will move the animal back to its previous \{\{masterVAS-Terminology.p-en}}.
        Click **Save** to apply your changes.
    </WorkflowBlockItem>
  </WorkflowBlock>

  ## Edit Animal Details

  <ExpandableHeading>
    ### Required Permissions

    Only users who are assigned a role with \{\{RolesPermissions.PermissionNames}} permissions can edit data or customize settings.

    See [Edit Dairy User Permissions](../../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
  </ExpandableHeading>

  - **Required Info:** Cow Number and \{\{masterVAS-Terminology.P-en-}} are not editable, but they can be changed by adding animal events. Add an \{\{XID}} event to change Cow Number or a MOVE event to change the animal's \{\{masterVAS-Terminology.p-en}}. Required fields are marked with an asterisk (\*).

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/of7bhcVE8n1tHh2tf4jug_dc-animals-requiredinfo.png" size="23" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/dc-animals-requiredinfo.png" githubPath="Content/Resources/Images/Platform/dairycomp/dc-animals-requiredinfo.png" position="flex-start" showCaption="false" indent="2"></Image>

  <Paragraph indent="1">
