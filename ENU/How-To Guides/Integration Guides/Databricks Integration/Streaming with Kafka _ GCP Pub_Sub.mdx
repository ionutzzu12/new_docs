You can connect Litmus Edge to Databricks using Kafka and GCP Pub/Sub connector for real-time data streaming and analysis.

# Before You Begin

Make sure you have the following:

- Access to Databricks workspace.
- API Keys and Secrets for Kafka and GCP Pub/Sub.

There are two ways to stream data with Databricks:&#x20;

1. [Kafka Integration](<Streaming with Kafka _ GCP Pub_Sub/Kafka Integration.mdx>)&#x20;
2. [GCP Pub/Sub Integration](<Streaming with Kafka _ GCP Pub_Sub/GCP Pub_Sub Integration.mdx>)&#x20;
