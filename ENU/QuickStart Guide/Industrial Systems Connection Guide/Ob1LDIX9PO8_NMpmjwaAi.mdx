Refer to the following table of compatible DeviceHub drivers for your PLC series.

<hint type="info">
  **Note**: For CompactLogix, ControlLogix, and FlexLogix PLCs, we recommend using [AB Logix Ethernet (3.16.4)](<Allen-Bradley/Gen 2 Drivers/ab-logix-ethernet.mdx>). It implements two access modes: Symbolic and Logical. Symbolic mode uses string-based tag requests and Logical mode uses ID-based tag requests.
</hint>
