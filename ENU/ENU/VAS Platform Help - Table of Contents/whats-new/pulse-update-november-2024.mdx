## Bug Fixes

- Bulk event entry fixes. We've squashed several bugs associated with the **Add Event in Bulk** option, including the following:\* Event dates and calendar selection were adjusted and now include the appropriate checks and messaging.
  - Remarks now display correctly in the event side panel.
  - \{\{BNAME}} search is now working correctly.
  - \{\{Events.12ABORT}} events now show correct error messaging and apply the correct check for \{\{ADDAT}} and the Abort Days Ago field.
  - Animal data now displays correctly when an animal is selected in the list.
  - A negative animal \{\{ID}} is no longer accepted.
  - Success and failure messages and animal counts display accurately on the Summary page.

`MyDC ONLY`\{\{Events.3002COLOSTRUM}} event fix. The volume of \{\{masterVAS-Terminology.c-olostrum}} is no longer required for the \{\{masterVAS-Products.MyDC-short}}\{\{Events.3002COLOSTRUM}} event.

- `MyDC ONLY`\{\{Events.1FRESH}} event issues. When adding \{\{Events.1FRESH}} events, \{\{masterVAS-Products.MyDC-short}}\{\{masterVAS-Terminology.d-airies}} will no longer encounter issues with multiple lactations and unexpected errors.
- `MyDC ONLY`\{\{Task list}} error. An unexpected error when creating \{\{Buttons-UI.FeaturesSecondary}} was addressed for \{\{masterVAS-Products.MyDC-short}}\{\{masterVAS-Terminology.d-airies}}.
