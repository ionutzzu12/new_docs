The \{\{Buttons-UI.FeaturesTertiary}} tab provides information on embryos that were transferred into recipient animals. This data is updated after embryos from inventory are used for \{\{Events.5BRED}} events and then processed from mobile or web.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/XndzfJSR_w9rxgEuc6g8V_embryo-transferred.png" size="95" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/embryo/embryo-transferred.png" githubPath="Content/Resources/Images/Platform/dairycomp/embryo/embryo-transferred.png" position="flex-start" showCaption="false"></Image>

<ExpandableHeading>
  ## View Transfer Details

  Filter transferred embryos: Use the filters to limit which embryos are shown. For instance, you may want to only view embryos transferred on a specific date.

  View a summary: A summary quickly shows the total number of transferred \{\{masterVAS-Terminology.f-resh}} and frozen embryos, as well as those in an undetermined status. The summary adjusts as you apply filters to your view.

  View transferred embryo details:  Click the Embryo \{\{ID}} to view a side panel with additional details. Scroll to the right if necessary to view additional information for your transfers.

  <ExpandableHeading>
    ### Embryo Details

    - \{\{masterVAS-Terminology.S-ire-}}: The bull whose semen was used to fertilize the donor's egg to produce the embryo. If this column shows **Pool**, it means that an \{\{oocyte}} was fertilized by \{\{IVF}} from a pool containing the semen of multiple sires.
    - Donor: TThe female who provided the embryo or oocyte (sometimes referred to as the donor dam).
    - Embryo Breed: The embryo breed. Select from the standard list of \{\{NAAB}} breeds.
    - Fertilization Date: Date the donor egg was fertilized using the \{\{masterVAS-Terminology.s-ire}} semen.
    - Recovery Type: The recovery type used:\* **Flush** is used for embryos
      - **IVF** for in vitro fertilization (\{\{oocytes}})
      - **Other** for neither of the above
    - Tank: The liquid nitrogen storage tank containing the cane with the embryo's straw.
    - Incubator: The warming unit containing the embryo's straw or mating.
    - Cane: The container holding the embryo's straw.
    - Mating: A description of the embryo mating details, such as \{\{masterVAS-Terminology.s-ire}}, donor, and fertilization date. May contain multiple embryos.
    - Straw number: Number of the straw holding the embryo. Canes contain multiple straws and each straw holds a single embryo.
    - Stage: The stage at which the embryo was harvested:\* **1** Unfertilized
      - **2** Simple Celled
      - **3** Early Morula
      - **4** Morula
      - **5** Early Blastocyst
      - **6** Blastocyst
      - **7** Expanded Blastocyst
      - **8** Hatched Blastocyst
      - **9** Expanded and Hatched Blastocyst
    - Grade: The quality of the embryo:\* **1** Good
      - **2** Fair
      - **3** Poor
      - **4** Dead (or Degenerating)
    - Lab ID: Identifier for the lab from which the embryo was purchased.
    - Lab Embryo ID: Identifier for the embryo from the lab. This value differs from the \{\{masterVAS-Products.Pulse-only}} Embryo ID.
    - Embryo Owner: Name or company that owns the embryo.
    - Frozen Date: Date the embryo was frozen.
    - Freeze Method: The method used to freeze the embryo:  \* **Direct Transfer (DT)** is the process of using high levels of cryoprotectant to quickly cool an embryo to freezing. A common cryoprotectant used for this process is ethylene glycol.
      - **Slow** is the process of using low levels of cryoprotectant and a programmable freezing machine to slowly lower the temperature of the embryo to freezing.
    - Recipient: The animal that received the embryo transfer. Including the Recipient and Transfer details automatically adds a \{\{Events.3000TRANSFER}} event to the recipient animal. See :Link[Import Transferred Embryos]{label="Import Transferred Embryos" overridedLabel="Import Transferred Embryos" spaceId docId="QMJysXEoHcmgu2GzWwO0-" version="v2" docAnchorId="#Transferred" loadingMethod="dynamic" newTab="false" href="transferred-embryos.mdx"} for more information.
    - Transferred \{\{Tech}}: The embryo technician who completed the transfer.
    - Transferred Date: Date the embryo was transferred to the recipient.
    - Breeding Code: The code associated with the breeding activity. Check your \{\{masterVAS-Terminology.d-airy}}'s [Breeding Codes](<../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Settings/Reproduction Settings/breeding-codes.mdx>) for more details on each code.
  </ExpandableHeading>

      ## Filter Columns

      Some columns that contain numeric values can be filtered to show only certain data. For example, to filter the **Lactation** column so that only animals with a lactation value of 3 display, click the filter option, and then use the fields to create your filter.

      ## Reorder Columns

      You can reorder the way columns display by clicking in the header of the column you want to move and dragging it to its new location.

    <ExpandableHeading>
      ## Change which Columns to Display

<ExpandableHeading>
  ## Delete Transferred Embryos

  If necessary, you can delete one or more \{\{Events.3000TRANSFER}} events from recipient animals from the Transferred tab. Check the embryos and select the **delete** option at the top right in the blue bar. Then choose whether to return the embryos to inventory or discard them.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/-28OctEbllkv7to0CFl-s_embryo-transferred-delete.png" size="77" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/embryo/embryo-transferred-delete.png" githubPath="Content/Resources/Images/Platform/dairycomp/embryo/embryo-transferred-delete.png" position="flex-start" showCaption="false"></Image>

  Removing the [TRANSFER Event](../../Events/TransferEvent.htm) also removes the associated [BRED Event](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/Events/Event Definitions/bred-event.mdx>) from the recipient animal.

  <hint type="info">
    **TIP:&#x20;**&#x55;se the **Return to Inventory** option if you accidentally selected the wrong embryo for a \{\{Events.3000TRANSFER}} event.
  </hint>
</ExpandableHeading>
