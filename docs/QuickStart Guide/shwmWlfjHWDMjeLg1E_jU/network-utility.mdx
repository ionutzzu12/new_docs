You can configure the following network utilities.&#x20;

- Use [Ping (Gen1)](<../Industrial Systems Connection Guide/Network Utility/NJdKhRKvFReepcdqjy51A.mdx>) to to ping an IP address.
- Use [SNMP Manager](<../Industrial Systems Connection Guide/Network Utility/snmp-manager.mdx>) to poll an SNMP agent or start an SNMP trap receiver.&#x20;
- Use [Syslog Server](<../Industrial Systems Connection Guide/Network Utility/syslog-server.mdx>) to receive Syslog messages from a Syslog client.
