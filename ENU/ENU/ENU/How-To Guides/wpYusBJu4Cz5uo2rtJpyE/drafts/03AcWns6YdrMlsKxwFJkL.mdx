You can use flows to request data from a serial device.

This procedure shows you how to request data from a serial device and output the requested data to the debug console.

If you scroll to the bottom of the steps, you can copy the JSON for the nodes described in the steps and import them into Flows.

**To request and output data from a serial device:**

1. From the navigation panel, select **Flows Manager**.
2. Click the **Expand** icon next to a selected Flow.
3. Drag the **Inject**, **Function&#xA0;**&#x61;nd **Serial Out&#xA0;**&#x6E;odes to the canvas and wire them together.
4. Double-click on the **Inject&#x20;**&#x6E;ode and change the payload to `string`. Write the string that requests the data that you want from the device. In this example the string is `F` for the temperature in Fahrenheit. Click **Done**.
5. Double-click on the function node and paste the following code into the node:
   `msg.payload = msg.payload+"\r"; // Adds a carriage return to the string so the device can read it.   `
   `return msg;`
   Click **Done**.
6. Double-click on the **serial out** node and select the serial port of your device. Click **Done**.
7. Add a **serial in node** and a debug node connected together in a separate section of your flow.
8. Double-click on the **serial in** node and select the serial port for your device. Click **Done**.
9. Ensure that your flow looks like the picture below and click **Save**.
10. Click the tab to the left of the inject node to have the Flows console display the data that you are requesting from the device.
