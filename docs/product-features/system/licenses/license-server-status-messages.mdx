The following table shows all the possible Status Messages when your Litmus Edge instance connects to a license server.

<table columnWidths="145,465" isTableHeaderOn="true" selectedColumns="" selectedRows="" selectedTable="false">
  <tr>
    <td align="left">
      **Status Message**
    </td>
    <td align="left">
      **Details**
    </td>
  </tr>
  <tr>
    <td align="left">
      **PINGING**
    </td>
    <td align="left">
      The Litmus Edge instance is trying to query license server status.
    </td>
  </tr>
  <tr>
    <td align="left">
      **UP**
    </td>
    <td align="left">
      The Litmus Edge instance has successfully established a communication with the license server.
    </td>
  </tr>
  <tr>
    <td align="left">
      **NO HEART BEAT**
    </td>
    <td align="left">
      The Litmus Edge instance has lost its connection with the license server and is attempting a reconnection. This state can only occur after **UP**.
    </td>
  </tr>
  <tr>
    <td align="left">
      **DOWN**
    </td>
    <td align="left">
      The Litmus Edge instance was not able to successfully establish a connection with the license server or has ceased in attempting a reconnection with the License Server. This state will occur after prolonged duration of **PINGING** or **NO HEART BEAT**.
    </td>
  </tr>
</table>
