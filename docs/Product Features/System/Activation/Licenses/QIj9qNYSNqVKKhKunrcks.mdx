You can set a license server for your Litmus Edge by navigating to **System&#x20;**>**&#x20;Activation&#x20;**>**&#x20;License**.&#x20;

Once you set the license server, you have access to the site license.&#x20;

# Before You Begin

Obtain the License Server Configuration key from Litmus.&#x20;

# Set License Server

**To set a license server:**

1. Navigate to **System&#x20;***>&#x20;***Activation**.&#x20;
2. Click the **License&#x20;**&#x74;ab.&#x20;
3. From the *License Server Settings* section, click **Set Server**.
   The *Set License Server Configuration* dialog box appears.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/uVyz5sJnqPxODa3Bfa0Ey_set-server-button.png)
4. In the **Configuration&#x20;**&#x66;ield, paste or enter the License Server Configuration key obtained from Litmus.
5. Click **Save**. &#x20;
   The license server is set for your Litmus Edge.&#x20;
