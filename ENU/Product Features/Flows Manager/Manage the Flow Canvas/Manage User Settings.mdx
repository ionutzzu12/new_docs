You can select from multiple options when modifying user settings. This enables you to customize the flow views.

You can customize settings for:

- **View**: Options for the Grid, Nodes, and showing tips in the sidebar palette. You can select a language from here.
- **Keyboard**: Options to assign or modify keyboard shortcuts for actions performed in the Actions list. The scope for Actions can be assigned to "global" or "workspace".
- **Palette**: Options to enable or disable nodes and install additional node modules.

# Manage Flow Canvas User Settings

You can manage the Flows user settings from the Flow Definition window.&#x20;

**To manage Flows user settings:**

1. Navigate to **Flows Manager**.
   The *Overview&#x20;*&#x70;ane appears.
2. Click the **Go To Flow Definition** icon next to a selected flow.
   The flow canvas opens in a new browser tab.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/DeJ3y4SUZncVlRm4KsmP7_gotoflowdefinitionicon.png "Go To Flow Definition icon")
3. Open the flow canvas menu and select **Settings**.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/BOIqzPp3QoEJfGUF1d2K0_settingsmenuitem.png" size="100" caption="Settings menu item" isUploading="false" showCaption="true" indent="2"></Image>

The *User Settings* dialog box appears and the *View&#x20;*&#x73;ettings appear by default.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/3Ey-hbbkZYMvj7Y2MGkCg_image.png" size="80" width="700" height="588" caption="Flow Manager User settings dialog box" position="center" showCaption="true"></Image>

You can manage the following settings.&#x20;

## View Settings

You can manage the following View settings:

- **Language**: Update the language in the Flows view.&#x20;
- **View**: Select if the zoom level and scroll position should restore on load.&#x20;
- **Grid**: Select options to show grid and snap to grid. Also select the grid size.&#x20;
- **Nodes**: Select to show node status and label of new added nodes.&#x20;
- **Other**: Select to show tips and guided tours for new versions.&#x20;

## Palette Settings

You can customize settings for the Flows palette from a Flows View.

This allows you to enable or disable nodes and install additional node modules.

**To customize palette settings:**

1. From the *User Settings* dialog box, click the **Palette** tab.
2. In the *filter nodes* search bar, enter `node-` or `loop` in the **Search** bar.
   A list of uninstalled modules appears.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/3jqCc3OqR7btVphGp7SFL_palettenodessettings.png" size="80" width="702" height="793" caption="Palette user settings section" position="center" indent="2" showCaption="true"></Image>

3. Click **Install** next to a selected module from the list.
   The *Confirmation&#x20;*&#x64;ialog box appears.
4. Clic&#x6B;**&#x20;Open node information** to make sure there are no dependencies that cannot be automatically resolved.
5. Click **Install**.
   The module installs in Flows.

## Keyboard Settings

You can customize the keyboard settings from a Flows View.

This includes options to assign or modify keyboard shortcuts for actions performed in the Actions list. The scope for Actions can be assigned to global or workspace.

**To update a keyboard shortcut:**

1. Click an unassigned action from the list.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Dwl71TIPzMWIhQUNXwoSa_keyboardusersetting.png" size="80" width="702" height="382" caption="Update keyboard setting section" position="center" indent="2" showCaption="true"></Image>

2. &#x20;Enter the command you want to use for the action in the **shortcut** field. For example, `ctrl-alt-f`.
3. Select **global** or **workspace** from the **scope** drop-down list.
4. Click the **Check** icon to accept the changes.
