You can configure the network interfaces by navigating to **System&#x20;**>**&#x20;Network** > **Interfaces**.&#x20;

The Wi-Fi option in the navigation panel only appears if your edge device supports Wi-Fi.

From the *Wi-Fi&#xA0;*, you can select a network for your edge device to allow a connection. You can also disconnect any device that appears on the Wi-Fi pane.

Litmus Edge Wi-Fi uses WPA for authentication to ensure a secure wireless network.

You must configure the wlan0 network interface in the *Interfaces&#xA0;*&#x74;ab on the Network pane before it appears in the Navigation panel.

# Manage Network Devices

You can manage network devices by navigating to **System** > **Network&#x20;**>**&#x20;Wi-Fi**.&#x20;

Litmus Edge Wi-Fi uses WPA for authentication to ensure a secure wireless network.

<hint type="info">
  **Note**: The *Wi-Fi&#x20;*&#x6F;ption only appears in the Navigation panel if the applicable device supports Wi-Fi.
</hint>

## Before You Begin

Make sure you have configured the wlan0 network interface. See :Link[Configure Network Interfaces]{docId="SUfaaMmemOR0_zmzckssE" label="Configure Network Interfaces" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId overridedLabel="Configure Network Interfaces"} for more information.

## Connect a Device

**To connect a device:&#x20;**

1. Navigate to **System&#x20;**> **Network**.
2. Click the **Wi-Fi** tab. &#x20;
3. Click **Networks&#x20;**&#x6F;n the the network device tile, select a network from the drop-down list, and then click **Join**. You can enter the name of the network (SSID) into the **Enter SSID&#xA0;**&#x66;ield to quickly find a network.
   The *Wireless Password* dialog box appears.&#x20;
4. Enter a password for the wireless network in the **Password&#xA0;**&#x66;ield, and then click **Connect**.

The state on the device tile displays CONNECTED.

## Disconnect a Device

To disconnect a device, navigate to th&#x65;*&#x20;Wi-Fi&#x20;*&#x70;ane and click **Disconnect**.&#x20;
