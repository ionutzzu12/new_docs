When you assign certain events to an animal (for example, a BRED event), you're given the option to assign a particular technician to the event. See the topics below to get started with change the technician assigned to these events directly from the animal's \{\{masterVAS-Terminology.cowcard}}:

1. On the animal's \{\{masterVAS-Terminology.cowcard}}, go to the \{\{masterVAS-Terminology.O-verview-}} tab or the Events tab.
2. Click on the desired event to open its side panel.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ET27Nsyg0L2nef75Z9H6t_eventtech.png" size="83" isUploading="false" initialPath="../../Resources/Images/Platform/cowcards/EventTech.png" githubPath="Content/Resources/Images/Platform/cowcards/EventTech.png" position="flex-start" showCaption="false" indent="2"></Image>

<Paragraph indent="1">
  Click \{\{Tech}} and select a different technician from the drop-down menu.:::hint\{type="info"}
  **NOTE:&#x20;**&#x59;ou can edit other event data besides the technician, but you can only edit events from the current lactation. See :Link[Update Events]{label="Update Events" overridedLabel="Update Events" spaceId docId="sp1fQUQTHg-WLIRXAer4X" version="v2" docAnchorId="#Update" loadingMethod="dynamic" newTab="false" githubPath="../Features/Events/Events.htm#Update" href="../Features/Events/Events.htm#Update"} to learn more.
  \:::
</Paragraph>

4. Click **Save** to apply your changes.
