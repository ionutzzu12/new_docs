The *External Storage* pane allows you to manage external storage with Litmus Edge.&#x20;

You can mount and manage network storage. This enables you to share the same storage location with all marketplace applications.&#x20;

# Access External Storage UI

From the Litmus Edge navigation panel, navigate to **System&#x20;**>**&#x20;External Storage**.
