<ExpandableHeading>
  ### Required Permissions

  Only users who are assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions can use this feature. See [Edit Dairy User Permissions](../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](../Config/UserMgmtRoles/ManageRoles.htm) for more information.
</ExpandableHeading>

With dashboards being such a key component in how you view and control your data in \{\{masterVAS-Products.Pulse-only}}, knowing how to add and edit those dashboards will help you optimize them for your specific operation. See the topics below to get started!

<ExpandableHeading>
  ## Rearrange Dashboards

  To quickly rearrange the order of your dashboard tabs at the top of the page, simply drag and drop them left or right.

</ExpandableHeading>

<ExpandableHeading>
  ## Add a New Dashboard

  Follow these steps to create a new dashboard:

  1. In the top-right corner of the page, click the Options button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wIXwfkZ07oPcd_rKpJE8J_options-green-small41x29.png" size="10" caption="Options button" alt="Options button" isUploading="false" initialPath="../../../Resources/Images/icons/options-green-small_41x29.png" githubPath="Content/Resources/Images/icons/options-green-small_41x29.png" position="flex-start" showCaption="true" indent="2"></Image>

  <Paragraph indent="1">
    .
  </Paragraph>

  2. Select **Create New Dashboard** to open the customization page.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Yg3x8gT177ITeaphBxGzl_multiple-dashboards.png" alt caption size="36"}
  3. In the **New Dashboard** field, type a name up to 20 characters long.
  4. Click **Save** to create the new dashboard.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/NUuNosiYfbmUKPQPYCcQI_co-customize-new.png" alt caption size="41"}
  5. You can now begin adding widgets and customizing their order.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/on4kzQ0yaNhkUJGlkCxBt_co-customize.png" alt caption size="57"}See [Customize Dashboards](DairyDashboard/CustomizeDashboard.htm) for more information.
</ExpandableHeading>

<ExpandableHeading>
  ## Edit an Existing Dashboard

  Follow these steps to rename, delete, or edit a dashboard:

  1. Navigate to the dashboard you want to change.
  2. In the top-right corner of the page, click the Options button

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wIXwfkZ07oPcd_rKpJE8J_options-green-small41x29.png" size="10" caption="Options button" alt="Options button" isUploading="false" initialPath="../../../Resources/Images/icons/options-green-small_41x29.png" githubPath="Content/Resources/Images/icons/options-green-small_41x29.png" position="flex-start" showCaption="true" indent="2"></Image>

  <Paragraph indent="1">
    .
  </Paragraph>

  3. Select **Customize Current Dashboard** to open the customization page.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/ByIigUH8Yd-CJsiZ92yM__multiple-dashboards3.png" alt caption size="57"}
  4. From the customization page, you can then:\* Rename the dashboard by typing a new name in the field at the top left.
     - Delete the dashboard by clicking the read Delete button at the top right.
  5. You can also add, remove, and rearrange the widgets.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/on4kzQ0yaNhkUJGlkCxBt_co-customize.png" alt caption size="57"}See [Customize Dashboards](DairyDashboard/CustomizeDashboard.htm) for more information.

  <hint type="info">
    **NOTE:&#x20;**&#x49;f you do not see the Options button :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wIXwfkZ07oPcd_rKpJE8J_options-green-small41x29.png" alt="Options button" caption="Options button" size="10"}, check your roles and permissions. This button is only visible for users who are assigned to a Role with **Dashboard Management** permissions. See [Edit Dairy User Permissions](../Config/UserMgmtRoles/EditUserPermissions.htm) for more information.
  </hint>
</ExpandableHeading>
