The **Conversion** processor helps you convert commonly used units in a factory setting.

# Conversion Overview

- Ready conversions of commonly used Units.
- You have the following base conversion options:
  - timestamp -> local time&#x20;
  - timestamp -> UTC&#x20;
  - Celsius -> Fahrenheit&#x20;
  - Fahrenheit -> Celsius&#x20;
  - degree -> radian&#x20;
  - radian -> degree&#x20;
  - degree -> gon&#x20;
  - gon -> degree&#x20;
  - radian -> gon&#x20;
  - gon -> radian
