To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:&#x20;**

1\. Note down the following information and refer to it when needed.&#x20;

- **Device**: Cutler-Hammer ELC-PV28NNDR
- **Communication I/F**: RS232, refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/S0CKD5lAmibbvuwri6spB_image.png" size="80" width="401" height="366" position="flex-start" caption="ELC Serial (Gen1) reference [1]" showCaption="true"></Image>

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/4jK6p-X6ePRP-LbNWKxY2_image.png" size="80" width="360" height="456" position="flex-start" caption="ELC Serial (Gen1) reference [2]" showCaption="true"></Image>

2\. Launch ELCSoft v2.07 software and navigate to **Options > Communication Setting**.

3\. Click **Communication > Transfer Setup**.

4\. Read from PLC to PC.
