You can manage FTP and System Services by navigating to **System&#x20;**> **Device Management&#x20;**>**&#x20;System Services**.&#x20;

To learn more about interacting with the FTP directory, see :Link[Litmus Edge File Browser]{label="Litmus Edge File Browser" spaceId="SHlZRB8sbLUIhUOl3qzzX" docId="NaypFhoKi6NNR5e0zuwsE" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;

<hint type="info">
  **Note**: The base directory for the FTP service on Litmus Edge is: */var/lib/customer/ftp-data****.&#x20;***
</hint>

# Manage FTP Service

From the *FTP Service&#x20;*&#x73;ection, you can manage the FTP Service.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/pS2RkzXv7tjy2ltXKFz9e_image.png "FTP Service management options")

## Start and Stop FTP Service

You can start and stop the FTP Service manually by clicking the **Start&#x20;**&#x6F;r **Stop&#x20;**&#x62;utton. The FTP Service is stopped and disabled by default.

<hint type="info">
  **Note**: If you reboot Litmus Edge, the FTP Server will not start unless **Start service automatically** is enabled and at least one FTP user is created.&#x20;
</hint>

## Start Service Automatically

You can click the toggle for **Start service automatically** to enable or disable the FTP service automatically starting when you log in.&#x20;

# Manage System Services

From the *System Services&#x20;*&#x70;ane, you can start and stop the SSH service. If you stop the service, no one can log in remotely.&#x20;
