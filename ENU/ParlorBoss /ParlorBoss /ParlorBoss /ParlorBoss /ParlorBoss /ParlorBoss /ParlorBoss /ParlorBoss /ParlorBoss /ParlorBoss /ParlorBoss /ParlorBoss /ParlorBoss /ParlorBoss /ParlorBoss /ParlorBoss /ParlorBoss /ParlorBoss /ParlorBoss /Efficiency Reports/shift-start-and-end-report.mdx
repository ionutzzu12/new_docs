<ExpandableHeading indent="2">
  ## View Report Details

  If your \{\{masterVAS-Terminology.d-airy}} has multiple \{\{FeedComp.FeedCompFeatures-Lower}}, \{\{Commands.Click}} the down arrow

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/GBvcd0YBt0W4PxXfaIEHe_expand27x25.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/expand_27x25.png" githubPath="Content/Resources/Images/icons/expand_27x25.png" position="flex-start" showCaption="false"></Image>

  to expand a row and view details for a specific \{\{Partners.parlor}}. Follow the guidelines below when \{\{Commands.UserActions}}\{\{FeedComp.FeedCompFeatures-Lower}}\{\{Controls.ControlTypes}}:

  - Average Time Spent: An average time for all milking shifts for the selected date displays in a **GRAY** bar at the top of the \{\{General.UI-gui}}.
  - \# Shift: Milking shift number. Click the column header to sort the data in descending or ascending order by shift.
  - Start Time: Time the shift started.
  - End Time: Time the shift ended.
  - Time Spent: Total time spent on the milking shift.
  - \{\{masterVAS-Terminology.P-en-}} Number: \{\{masterVAS-Terminology.P-ens-}} milked during the shift. Hover over the + value to see any additional \{\{masterVAS-Terminology.p-ens}}.
  - Number of Pens: Total number of \{\{masterVAS-Terminology.p-ens}} milked during the shift.
  - Number of Cows: Total number of cows milked during the shift.
</ExpandableHeading>
