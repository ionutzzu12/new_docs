Refer to the image and callouts below for management options for marketplace applications.

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/ML94b6ld_ZNPj6kKafdgW_image.png)

| **Action**                    | **Details**                                                                                                                                                         |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| \[1] **Filter apps**          | Use the **Catalog** and **Category** lists to filter the catalogs that display on the page.                                                                         |
| \[2] **Search**               | Enter a search term to display specific marketplace catalogs. A marketplace catalog must be added and deployed to the Litmus Edge instance to be visible.           |
| \[3] **Last sync time**       | View the last time that the catalog was synced.                                                                                                                     |
| \[4] **Synchronize catalogs** | Use **Sync Catalog** to downloads the latest configurations for Docker applications in the selected marketplace catalog or **Sync All** to update all applications. |
