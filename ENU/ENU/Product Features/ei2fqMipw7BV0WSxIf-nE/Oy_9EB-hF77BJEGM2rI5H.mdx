Litmus Edge OPC UA supports many different options for managing security policies and user authentication.

The OPC UA *Management&#xA0;*&#x70;ane includes the following sections:

- **OPC UA Management**: Allows you to start, reset, stop, and configure the server. The OPC UA server is color-coded. A started server displays a green OK. A stopped server displays a red Stopped. Configure allows you to set the host, port, and allow nodes under root.&#x20;
  - If you are using the Docker version of Litmus Edge and having issues connecting to the OPC UA server, refer to [Issues Connecting to OPC UA Server](<../../../Troubleshooting/OPC UA Troubleshooting/kBk45E-_KBkAeJ8bCIxl4.mdx>).&#x20;
- **Policy Management**: Allows you to enable different encryption algorithms and security modes for the OPC UA server.&#x20;
- **Authentication Management**: Allows you to enable multiple identity types and choose security policies.&#x20;
- **User Management**: Allows you to view current OPC UA users.&#x20;

# Access OPC UA Management UI

**To access the OPC UA Management pane:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, navigate to **OPC&#x20;**>**&#x20;Management**.
   The *Management&#x20;*&#x70;ane appears.&#x20;
