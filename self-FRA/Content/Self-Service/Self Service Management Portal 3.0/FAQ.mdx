<hint type="info">
  Le module **FAQ** n'est pas disponible clé en main (OOTB). Cet espace de travail apparaît uniquement si votre administrateur l'a configuré pour votre rôle.
</hint>

L'espace de travail **FAQ** vous aide à trouver la réponse aux questions fréquemment posées. Vous pouvez utiliser cet espace de travail pour résoudre les problèmes courants ou les contourner, avant de signaler un incident.

<ExpandableHeading>
  ## Parcourir toutes les FAQ publiées

  1. Connectez-vous à \{\{FRS\_VARS.ProductName}}.
  2. Dans le menu de gauche, sélectionnez **FAQ**.

  La liste de toutes les FAQ publiées apparaît.

  3. Cliquez sur l'icône de développement, en regard de n'importe quelle question, pour afficher son contenu complet.
  4. Cliquez sur une valeur dans l'angle inférieur droit pour afficher sur la page le jeu de FAQ correspondant.
</ExpandableHeading>

<ExpandableHeading>
  ## Rechercher une question FAQ spécifique

  1. Dans l'espace de travail **FAQ**, accédez à la zone **Rechercher un article**.
  2. Entrez un mot-clé lié à votre problème, puis appuyez sur **Entrée**.

  Le système affiche les FAQ correspondantes.
</ExpandableHeading>

<ExpandableHeading>
  ## Filtrer les FAQ par catégorie

  1. Dans l'espace de travail **FAQ**, cliquez sur la liste déroulante **Filtrer par**.
  2. Sélectionnez la catégorie à afficher, parmi les options suivantes :
     - Toutes les catégories
     - Matériel
     - Logiciel
     - Téléphonie
</ExpandableHeading>
