The **Applications&#x20;**>**&#x20;System Info** pane shows a JSON file that contains information about your Docker applications and Docker configurations.

From the *System Info* pane, you can click the **Show filter** icon to search for text in the JSON file. You can also click the **Refresh&#x20;**&#x69;con to refresh the system information.&#x20;
