Puede utilizar el bot de service manager de \{\{FRS\_VARS.Company}} para realizar múltiples acciones, como crear una incidencia, comprobar el estado de la incidencia y chatear con un analista.

1. Abra el Bot de Service Manager \{\{FRS\_VARS.Company}} y haga ping a un mensaje.

<hint type="info">
  Si el bot no está disponible para usted, contacte con el admistrador de su empresa para que lo habilite.
</hint>

2. Seleccione alguna de las siguientes opciones.
   - **Crear un ticket**: crea un incidente.
   - **Comprobar el estado de mi ticket**: se muestra el estado de todos los incidentes abiertos.
   - **Chatear con el servicio de asistencia**: se pondrá en contacto con un analista que le ayudará con su problema o consulta.
     - Si un analista no contesta su chat, recibirá un mensaje que dice "Por favor espere, mientras le conectamos con otro analista". Puede esperar a que le conecten.
     - Si no hay ningún analista conectado, aparecerá el mensaje: "Todos los analistas están desconectados". Puede seguir esperando hasta que un analista esté en línea o salir. El tiempo máximo de espera para conectarle con un Analista es de 5 minutos, después de los cuales aparece el siguiente mensaje: "Lo sentimos, no podemos conectarle en este momento." Haga clic en **Sí** para continuar utilizando el bot o **No** para salir.
   - **Salir de la conversación de chat**: abandona el chat.
   - **Ayuda**: muestra un diálogo que describe las opciones anteriores.

<hint type="info">
  - Si envía un mensaje de texto en lugar de seleccionar una de las opciones dadas, aparecerá el siguiente mensaje: "Selección no válida. Seleccione una de las opciones disponibles."
  - Siempre que el bot muestre opciones, asegúrese de seleccionar una de las opciones dadas. De lo contrario, aparecerá un mensaje que le pedirá que seleccione solo una de las opciones mostradas.
</hint>
