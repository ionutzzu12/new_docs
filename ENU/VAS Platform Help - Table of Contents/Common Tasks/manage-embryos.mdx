The \{\{Events.3000TRANSFER}} event is a \{\{masterVAS-Products.Pulse-only}}-only event added to recipient animals for embryo transfers. \{\{masterVAS-Products.Pulse-only}} allows you to manage your embryo inventory and enter \{\{Events.3000TRANSFER}} events in web or the mobile app.Adding embryos is simple, and defining embryo details in advance saves you precious time during embryo transfer tasks while out on the \{\{masterVAS-Terminology.d-airy}} with your embryo technician. You can find this feature under the :Link[DairyComp/MyDC Menu]{label="DairyComp/MyDC Menu" overridedLabel="DairyComp/MyDC Menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../NavPlatform/Menu.htm#Menu-DairyComp" href="../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}.

After your embryos are collected and prepared for transfer, you can add them to inventory and then select them while adding embryo \{\{Events.3000TRANSFER}} events. Or, use the batch import process to import embryos to inventory or add completed transfers. Then track transferred embryos and your results from \{\{masterVAS-Products.Pulse-only}}.

After \{\{masterVAS-Products.Pulse-only}} processes the Embryo \{\{Events.3000TRANSFER}} event, it forwards the event details on to \{\{masterVAS-Products.DC305-short}} for \{\{masterVAS-Products.DCUnlimited}}\{\{masterVAS-Terminology.d-airies}}.

The \{\{Buttons-UI.FeaturesSecondary}} feature in the web version of \{\{masterVAS-Products.Pulse-only}} allows you to add and edit embryos, as well as view and analyze completed transfers and their results. If you need help creating a \{\{Events.3000TRANSFER}} event or for detailed information on what happens when you process the event, see the [TRANSFER Event](../Features/Events/TransferEvent.htm) topic.
