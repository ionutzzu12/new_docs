You can remove the following licenses from the **System&#x20;**>**&#x20;Activation&#x20;**>**&#x20;License&#x20;**&#x70;ane.&#x20;

- Add-on node-locked licenses
- Core node-locked licenses if no active add-on licenses are associated with them

**To remove a license:**

1. Navigate to **System&#x20;**\*> \***Activation**.
2. Click the **License&#x20;**&#x74;ab.&#x20;
3. From the *Node-Locked License* section, click the **Remove License** icon for the license you want to remove. The **Remove License** icon appears grayed out for those licenses that cannot be removed. To remove a core node-locked license, you need to first remove all add-on licenses associated with it.
   The *Confirm License Removal* dialog box appears.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Ue8HdfqkFGK4R64thYlye_remove-license.png)
4. Click **Yes, remove license**.
