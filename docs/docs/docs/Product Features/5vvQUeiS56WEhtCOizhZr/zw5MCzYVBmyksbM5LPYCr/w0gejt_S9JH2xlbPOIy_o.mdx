You can manage the Timeseries DB by navigating to **DataHub&#x20;**>**&#x20;DB Management**.&#x20;

DataHub's DB Management module offers the following functionalities:

- Drop Tables
- Delete a Database

# View Tables

You can view database tables from the *DB Management* pane by clicking the **tsdata&#x20;**&#x64;atabase name.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/kKpxmh_1m9pcL4o89RjVr_image.png" size="80" width="1910" height="255" position="center" showCaption="false"></Image>

The DataHub tables are the data from the devices you added in DeviceHub if you enabled data store.



After clicking *tsdata*, the tables drop down and are selectable.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/hYAgQk7291IE-nyXJbkFZ_image.png" size="80" width="1910" height="426" caption="Individual database tables" position="center" showCaption="true"></Image>



# Drop Tables

You can drop a table from the timeseries database in the *DB Management* pane.

<hint type="info">
  **Note**: By dropping a table, you delete the data stored for devices you had added in DeviceHub with data store enabled.
</hint>

**To drop a table from the database:**

1. From the *DB management* pane, click the database name: **tsdata**.
   The selected database's tables dropdown.
2. Select a table's check box and click the **Remove Selected Data&#x20;**&#x69;con.
   The *Confirm Deletion* dialog box appears.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/3Lb1j4ySA55UblahyIy6Y_j6z6a-mgdbprmop2jc5fqimage.png" size="80" width="1911" height="425" caption="The Remove selected table icon" position="center" showCaption="true"></Image>
3. Click **Yes**.
   The selected table's data is removed from the database.

# Delete a Database

You can delete the time series database from the *DB Management* pane.

**To delete the database:**

1. From the *DB management* pane, click the **Remove Database&#x20;**&#x69;con by the database name.
   The *Confirm Deletion* dialog box appears.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/busOOGimx8Hp_y-yThaQ9_scj9cpc4nmdqh5vnupvp5image.png" size="80" width="1911" height="425" caption="The Remove Database icon" position="center" showCaption="true"></Image>
2. Click **Yes**.&#x20;
   The database is removed from the *DB Management* pane.&#x20;
