- Manage LDAP/AD authentication providers. See [LDAP/AD Auth](<../../ENU/Product Features/Lv4VprvL49Sh2xq8oEdH-/Z27GWlDGawhpAlMXldr-l/re6d45HDCLbzy6NU6SxfY.mdx>).&#x20;
- Manage the password policy for Litmus Edge. See [Manage the Password Policy](<../../ENU/Product Features/Lv4VprvL49Sh2xq8oEdH-/Z27GWlDGawhpAlMXldr-l/0jw6cNDwOiEGsdVdT6932.mdx>).
