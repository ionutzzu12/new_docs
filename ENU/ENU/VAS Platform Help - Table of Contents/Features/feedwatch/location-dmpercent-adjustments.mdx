  ### Is the Location DM% Adjustments list long? Sort it!

  <hint type="info">
    **NOTE:&#x20;**&#x46;or \{\{masterVAS-Terminology.d-airies}} with a long \{\{Buttons-UI.FC-DMPercentAdj}} list, you can click the column titles to sort by **Ingredient Name**, **Location**, or **Type**.
  </hint>
</ExpandableHeading>

1. **Ingredient Name:&#x20;**&#x49;ngredient name as defined in \{\{masterVAS-Products.FeedWatchOnPremise}}.\* Click the column title to sort the list by Ingredient.
   - Click on the inventory name to view ingredient details. Note that this action takes you into the :Link[ingredient  Details]{label="ingredient  Details" overridedLabel="ingredient  Details" spaceId docId="2qPNJ_MJBUJ4uJc0qopDZ" version="v2" docAnchorId="#ingred-details" loadingMethod="dynamic" newTab="false" githubPath="FeedWatchPlatform-IngredientsList.htm#ingred-details" href="../../../../VAS Platform Help - Table of Contents/Features/FeedWatch/ingredients.mdx"} page under \{\{masterVAS-Products.Pulse-only}} Ingredients list (the breadcrumbs links at the top of the page will help you find your current location). To return to the \{\{Buttons-UI.FeaturesSecondary}} page, simply select it again from the \{\{Buttons-UI.FeaturesMain}} main menu.
2. **Type:&#x20;**&#x54;he ingredient type is shown in a color‑coded label. The predefined types are Forage, Protein, Concentrate, Additive, Mineral, and Premix. Any type that does not belong to one of these categories is displayed with the color‑coded label for Other ingredient types (for example, Grain, Hi-Fiber, Fat, etc.). Click the column title to sort the list by Type.
3. **Location:&#x20;**&#x4C;ocation of the ingredient. Click the column title to sort the list by Location.
4. **Last Change:&#x20;**&#x54;he last time the DM% was adjusted for the ingredient.
5. **Responsible:&#x20;**&#x54;he person who last adjusted the DM%.
6. **DM%:&#x20;**&#x54;he current dry matter percentage of the ingredient.
