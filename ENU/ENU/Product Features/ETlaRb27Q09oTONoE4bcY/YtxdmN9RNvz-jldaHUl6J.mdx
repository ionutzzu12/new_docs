Analytics flows are used to collect data, process that data, and then export that data to your individual configurations.&#x20;

Flows are made up of processors and require at least one input processor, one function processor, and one output processor. See [Input Processors](input-processors.mdx) and [Output Processors](output-processors.mdx) for details.&#x20;

To create an entire flow at once, see [Create an Analytics Flow](<../../ENU/Product Features/ETlaRb27Q09oTONoE4bcY/YtxdmN9RNvz-jldaHUl6J/jSA9xZtjDdpJOB7IZpWAH.mdx>). To add an individual processor to the Analytics canvas, see [Add Processors and Processor Connections](<../../ENU/Product Features/ETlaRb27Q09oTONoE4bcY/YtxdmN9RNvz-jldaHUl6J/k_P8mfL-X_bZhHDbE4Tnx.mdx>).&#x20;
