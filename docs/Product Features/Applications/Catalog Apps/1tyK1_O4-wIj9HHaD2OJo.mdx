You can manage catalog apps by navigating to **Applications&#x20;**>**&#x20;Catalog Apps**.&#x20;

Refer to the following actions you can take from the Catalog Apps Web UI:

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/45dUewBos7aW2eTi3Gaef_image.png)

<table isTableHeaderOn="true" selectedColumns="" selectedRows="" selectedTable="false" columnWidths="303,303">

    </td>

      **Details**
    </td>
  </tr>
  <tr>
    <td align="left">
      \[1] **Filter**
    </td>
    <td align="left">
      Use the search bar to filter the applications by name.&#x20;
    </td>
  </tr>
  <tr>
    <td align="left">
      \[2] **Search**
    </td>
    <td align="left">
      Use the search bar to filter the applications by name.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[3] **Start or stop application**
    </td>
    <td align="left">
      This action allows you to initiate or halt application functionality, enabling you to manage its operation based on current needs.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[4] **Refresh application**
    </td>
    <td align="left">
      Use this option to update the application state, ensuring that any changes or new configurations are applied and reflected in real-time.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[5] **Remove application**
    </td>
    <td align="left">
      Before removing an application, you must stop a running container. Please note that removing an application removes all the application data and configuration files for the application.&#x20;
    </td>
  </tr>
  <tr>
    <td align="left">
      \[6] **Display Application Information**
    </td>
    <td align="left">
      When you click an application tile (tile view) or application name (list view), information displays about the application. You can click the Container drop-down list to view the containers that are running for the application.
      You can view the IP address, port number, memory usage, and the up state of the application container. The up state indicates how long the application container has been running.
      You can view the logs for the application. You can download the logs filtered by container tags for diagnostic purposes, refresh it, and expand the log file view. You can select the amount of time the log shows.

    </td>
