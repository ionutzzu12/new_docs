<ExpandableHeading>
  ### Required Permissions

  To use this feature, your \{\{General.ProductName}} user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../manage-dairy-users-and-roles/edit-dairy-user-permissions.mdx) and [Manage Dairy Roles](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

The Change Log provides an easy to use summary and historical list of animal activity and data changes made within the \{\{masterVAS-Terminology.d-airy}}. The top of the Change Log shows a breakdown of animal activity and events for the past 7 days, while beneath that is a list of all major data changes that have occurred for the \{\{masterVAS-Terminology.d-airy}}. You can find this feature under the :Link[Dairy menu]{label="Dairy menu" overridedLabel="Dairy menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-Dairy" loadingMethod="dynamic" newTab="false" githubPath="../../NavPlatform/Menu.htm#Menu-Dairy" href="../../../../VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"}.

<ExpandableHeading>
  ## View Summary of Changes over the Last 7 Days

  The top of the Change Log shows a breakdown of animal activity and events for the past 7 days.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/zifTz4RbAp1nE1PYxxHMB_last7days.jpg" size="67" isUploading="false" initialPath="../../../Resources/Images/Platform/Last7Days.jpg" githubPath="Content/Resources/Images/Platform/Last7Days.jpg" position="flex-start" showCaption="false"></Image>

  - The counter for each event is color coordinated to the event's color for the \{\{masterVAS-Terminology.d-airy}}.
  - The number of animals that were added, sold, and died in the last 7 days are shown.
  - Event information covers the six major event types: \{\{masterVAS-Terminology.F-resh-cow-}}, Dry, \{\{Preg}}, Bred, Heat, and Move

  <hint type="info">
    **NOTE:&#x20;**&#x53;ee [Event Entry (v1)](../../MobileApp/MobEventsIntro.htm) for more information about working with events.
  </hint>
</ExpandableHeading>

  ## View All Changes

  The Change Log displays a list of all the major changes to data that have occurred for the \{\{masterVAS-Terminology.d-airy}}. The list is presented in reverse-chronological order (i.e., the top change in the list is the most recent).

<ExpandableHeading>
  ## View Change Details

  Click on a change in the list to view a side panel containing the full details about the change.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/MOMC5ZCqGOtvJ-JXwgkQd_changelogsidepanel.png" size="52" isUploading="false" initialPath="../../../Resources/Images/Platform/ChangeLogSidePanel.png" githubPath="Content/Resources/Images/Platform/ChangeLogSidePanel.png" position="flex-start" showCaption="false"></Image>

  Use the following guidelines for the data in the side panel:

  - The groupings of information displayed in the side panel vary according to the type of change.
  - If the change has an active subject associated with it, that subject will be linked from the side panel.
  - For Event Entry type changes, details about the event are shown in the side panel.
  - If the change has a \{\{masterVAS-Products.DC305-simple}} command associated with it, the side panel includes the command. You can copy the command to the clipboard by clicking the copy icon

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/4KaDVpgfpCEeaG2_G7C58_copyicon.png" size="10" isUploading="false" initialPath="../../../Resources/Images/Platform/CopyIcon.png" githubPath="Content/Resources/Images/Platform/CopyIcon.png" position="flex-start" showCaption="false" indent="2"></Image>
