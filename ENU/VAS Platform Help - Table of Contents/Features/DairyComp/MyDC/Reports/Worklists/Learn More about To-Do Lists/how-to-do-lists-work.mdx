The primary purpose of \{\{Worklists.ToDoCommonUse}}s in \{\{masterVAS-Products.Pulse-only}} is to track **one** action that needs to be completed for all animals on the list (send to dry, move to a different \{\{masterVAS-Terminology.p-en}}, etc.). **All animals on the list should require that same&#x20;*****single*****&#x20;action.**

\{\{masterVAS-Products.Pulse-only}} also supports \{\{Worklists.OnDone}} and \{\{Worklists.PostProcess}} routines. Before scheduling follow‑up routines, however, you should review the instructions in :Link[Create To-Do Lists]{label="Create To-Do Lists" overridedLabel="Create To-Do Lists" spaceId docId="bifG42_6IpH_xJCb9_RsU" version="v2" docAnchorId="#PlanLists" loadingMethod="dynamic" newTab="false" githubPath="CreateToDoLists.htm#PlanLists" href="CreateToDoLists.htm#PlanLists"} to make sure you understand how \{\{masterVAS-Products.Pulse-only}} mobile app will handle them.
