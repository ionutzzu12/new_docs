To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Mitsubishi A2NCPU
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/SqMt6Ya6_dDmMX2dzVeZX_ls-xbm-serial-reference.png" size="80" width="715" height="282" position="flex-start" caption="Mitsubishi A2N Serial reference" showCaption="true" indent="2"></Image>

2. Launch GX developer v8.103 software and connect the device with a serial cable.
3. Click **Online > Read from PLC**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/MLA3ZFZ2hfcagelDvbR-D_mitsubishi-a2n-serial-read.png" size="80" width="616" height="433" position="flex-start" caption="Read from PLC option" showCaption="true" indent="2"></Image>

4. Select the PLC series.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/eZnK_RaZ7LsAWZIm8BepA_mitsubishi-a2n-serial-series.png" size="60" width="302" height="106" position="flex-start" caption="PLC series drop-down list " showCaption="true" indent="2"></Image>

5. Set up communications and then click **Communication Test**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/LQkmOe9yRKuJjTp0F2Ykc_mitsubishi-a2n-serial-communication.png" size="80" width="750" height="480" position="flex-start" caption="Communication Test option " showCaption="true" indent="2"></Image>
