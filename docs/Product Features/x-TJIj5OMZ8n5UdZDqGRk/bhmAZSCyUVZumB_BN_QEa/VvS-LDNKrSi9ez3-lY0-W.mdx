You can mount and manage network storage. This enables you to share the same storage location with all marketplace applications.&#x20;

You can use CIFS or FTP to mount external storage.

To add a CIFS mount point, you must do one of the following before creating the CIFS mount point in Litmus Edge:&#x20;

- [Set up a CIFS Mount Point on Linux](<../../System/External Storage/Network Storage/YjzVRF-Fui6WIcEHRuvzI.mdx>)
- [Set up a CIFS Mount Point on Windows 10](<../../System/External Storage/Network Storage/oaLLYsFdVK7NaD68kAXRl.mdx>)
