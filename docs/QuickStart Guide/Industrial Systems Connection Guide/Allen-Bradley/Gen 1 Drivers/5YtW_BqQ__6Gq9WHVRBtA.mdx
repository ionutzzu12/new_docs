To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it as needed.&#x20;
   **Device**: AB ControlLogix FreeTag 1756-L55
   **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/SSp4v8C87SjqfCgvoTLwD_ab-controllogix-freetag.png" size="100" width="864" height="279" position="flex-start" caption="AB ControlLogix FreeTag" showCaption="true" indent="2"></Image>

2. Launch RSLinx Classic, and then click **Configure Drivers**.
3. Add a new RS-232-DF1 driver.
4. Set up the communication port and set up parameters as follows, and then click **Auto-Configure**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/tqIEohWhk6dy_BhOWLt9g_ab-controllogix-freetag-auto-configure482x338.png" size="80" width="482" height="338" position="flex-start" caption="Configure parameters" showCaption="true" indent="2"></Image>

5. Right-click the Ethernet communication module and select **Module Configuration&#xA0;**&#x74;o set up its network parameters.
