  ### Learn more about Nedap's herd monitoring solutions!

  \{\{Partners.Nedap-parent}} offers two \{\{masterVAS-Terminology.h-erd}} monitoring solutions: an on-premise solution, [Nedap — on-premise](https://www.nedap.com/cowcontrol), and a cloud solution, [Nedap — cloud](https://www.nedap.com/now). While both products offer the same accurate, reliable \{\{masterVAS-Terminology.h-erd}} insights, \{\{Partners.NedapNow-product}} has the processing power and scalability of cloud computing. To learn more about the cloud solution, see [Nedap — cloud](nedap-now-info.mdx).
</ExpandableHeading>

[Nedap — on-premise](https://www.nedap.com/cowcontrol) animal sensors monitor reproduction, rumination, eating, and movements to provide deeper insights on heat detection, overall health, and animals requiring urgent attention. It's all-in-one \{\{masterVAS-Terminology.h-erd}} monitoring solution is an advanced, easy-to-use system available through the \{\{masterVAS-Products.OnPremise-lower}} platform, Velos. \{\{masterVAS-Products.Pulse-only}} displays live sensor data in a \{\{Buttons-UI.Dashboard}} widget and individual animal \{\{masterVAS-Terminology.cowcard}}s on the \{\{Partners.PartnersUIoption}} tab, providing an up-to-date view of animals requiring health checks or reproduction-related protocols.
