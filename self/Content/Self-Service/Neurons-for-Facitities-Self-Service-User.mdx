**Minimum Version:** 2022.4

\{\{Primary.Company}} \{\{FRS\_VARS\_Self.Neurons\_for\_Facilities}} 2022.4 provides Self-Service Users with enhanced facilities functionality for their existing \{\{Primary.Product}} system.

When logged on to \{\{Primary.Company}} \{\{Primary.Product}} with \{\{FRS\_VARS\_Self.Neurons\_for\_Facilities}} 2022.4, a Self-Service User can:

- Reserve available conference rooms, facilities spaces, and cubicles.
- Select multiple facilities assets that are assigned to them and that they wish to take with them when moving to another facility.
- Facilities Asset Request feature is improved for better user experience and usability.

All of the above mentioned enhancements are available in the Service Catalog Facility Services section.

For general instructions on making a service request, see [Service Requests and Incidents](service-requests-and-incidents.mdx).

For information on all request features you can select in the service catalog, see [Default Service Catalog Offerings](<https://help.ivanti.com/ht/help/en_US/ISM/2026/admin-user/Content/ServiceDesk/ServiceCatalog/Default Catalog Offerings.htm>)..

The request features described below are new or improved for \{\{Primary.Company}} \{\{FRS\_VARS\_Self.Neurons\_for\_Facilities}} 2022.4.

## Employee Move Request

Request to move from one facility to another, including any assets that are assigned to the requester.

**To create an Employee Move request:**

1. In the **Home** workspace of the Self-Service portal, select **More Services**.
2. Select the **Facility Services** category, and then select **Employee Move Request**.
3. See [Employee Move Request](<https://help.ivanti.com/ht/help/en_US/ISM/2026/admin-user/Content/ServiceDesk/ServiceCatalog/Default Catalog Offerings.htm#h3_Employee_Move_Request>) in Default Service Catalog Offerings for details on completing the request.
4. Click **Submit** when all fields are completed.

## Facility Reservation

Make a reservation for a conference room, cubicle or facilities space, including catering and required equipment.

**To make a reservation:**

1. In the **Home** workspace of the Self-Service portal, select **More Services**.
2. Select the **Facility Services** category, and then select **Facility Reservation**.
3. See [Facility Reservation Request](<https://help.ivanti.com/ht/help/en_US/ISM/2026/admin-user/Content/ServiceDesk/ServiceCatalog/Default Catalog Offerings.htm#h3_Facility_Reservation_Request>) in Default Service Catalog Offerings for details on completing the request.
4. Click **Submit** when all fields are completed.

## Facilities Asset Request

Request a new facilities asset from the Product Catalog.

**To request a facilities asset:**

1. In the **Home** workspace of the Self-Service portal, select **More Services**.
2. Select the **Facility Services** category, and then select **Facilities Asset Request**.
3. See [Facilities Asset Request](<https://help.ivanti.com/ht/help/en_US/ISM/2026/admin-user/Content/ServiceDesk/ServiceCatalog/Default Catalog Offerings.htm?#h3_Facility_Asset_Request>) in Default Service Catalog Offerings for details on completing the request.
4. Click **Submit** when all fields are completed.
