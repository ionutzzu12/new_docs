Vous consultez et soumettez une offre de demande (\{\{FRS\_VARS.Request\_Offering}}) via des offres externes, en cliquant sur une URL sur un site Web externe. Vous affichez et demandez des services, suivez votre demande à l'aide d'ID d'utilisateur et d'offre dans l'URL, et choisissez comment la demande doit s'afficher, à savoir dans une nouvelle fenêtre, dans \{\{FRS\_VARS.ProductName}} ou dans le **catalogue de services**.

Pour utiliser les offres externes créées par votre administrateur, procédez comme suit :

1. Connectez-vous à \{\{FRS\_VARS.ProductName}}.
2. Dans l'espace de travail **Catalogue de services**, utilisez la barre de recherche pour trouver l'enregistrement d'offre externe voulu.

<Paragraph indent="1">
  Les enregistrements d'offre externe s'affichent.
</Paragraph>

3. Cliquez sur une carte d'offre externe.

<Paragraph indent="1">
  Une boîte de dialogue vous demande si vous souhaitez continuer ou rester sur cette page.
</Paragraph>

4. Si vous choisissez **Continuer**, l'URL s'ouvre dans un nouvel onglet avec votre ID d'utilisateur et l'ID d'offre.
