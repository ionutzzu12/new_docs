To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.

<hint type="info">
  **Note**: Make sure Litmus Edge IP address is configured in TWINCAT.
</hint>

**To configure your device:**

1\. Record the following information and refer to it as needed.&#x20;
&#x20;    **Device**: Beckhoff CX1100-002
&#x20;    **Communication I/F**: Ethernet. Refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/rnp7bXed_-apBuJ-_kua6_image.png" size="80" width="654" height="500" position="flex-start" caption="Beckhoff ADS/EMS Ethernet reference" showCaption="true"></Image>

2\. Launch TwinCAT V2.11 Build 2302 software.

<hint type="info">
  **Note**: The Beckhoff PLC is also a PC with its own OS and an IP address can be set up like a PC.
</hint>



3\. Click the TwinCAT icon in the Notification area, launch the system, and then click the system status indicator switch from Config Node (blue) to Running (green).

4\. Launch the PLC.

5\. Set up the target.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/uUfSL4AFniSWnT0tfQwMD_beckhott-ads-ams-ethernet-target.png" size="80" width="786" height="487" position="flex-start" caption="Ethernet target [1]" showCaption="true"></Image>

![](https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/NhJWkfPTOnGte7-9NsZLW_beckhoff-ads-ams-ethernet-target3.png "Ethernet target \[3]")

![](https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/77Tcz7vaxcKGZ3WGlR_89_beckhoff-ads-ams-ethernet-target4.png "Ethernet target \[4]")

![](https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/gxweNBAv05Wc4zLBLauc9_beckhoff-ads-ams-ethernet-target5.png "Ethernet target \[5]")

6\. Select the target.&#x20;

![](https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/fiM2YkaXCR0knFiWSiTrX_image.png "Target select \[1]")

8\. Click **Online > Login**, and then download the program and run the PLC.
