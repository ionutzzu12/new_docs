The charts below list the default permissions that are allowed for each \{\{masterVAS-Products.Pulse-only}} role. See [Permission Descriptions](permission-descriptions.mdx) for a list of the user actions that each permission controls.

<hint type="info">
  **NOTE:&#x20;**&#x42;e aware of \{\{Zone}} "write" permissions as you are editing roles. The zone write :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/RAsWMD-IsbPX_etOV-z1q_dairy-zones-write-permissions.png" alt caption size="10"} icon is displayed for any permission level that requires zone selections. When you edit a permission level with this icon, you must also select which zones can be edited by users who are granted that permission.
</hint>
