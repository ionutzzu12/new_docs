Manage your Digital Twins models from **Digital Twins** > **Models**.

<hint type="info">
  **Note**: Digital Twins requires a Scale or Growth Litmus license.
</hint>

## Change model view

Click **Show models grid** or **Show models table** to switch between the grid and table views.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/v7921AaBj4o4XbqcJzAL6_image.png" size="80" width="922" height="495" position="flex-start" caption="Change model view icons" darkWidth="922" darkHeight="495" showCaption="true"></Image>

Use the search bar to filter by model name.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/S8kyvZIqFXImm4x_gU9M6_image.png" size="80" width="922" height="495" position="flex-start" caption="Model search bar" darkWidth="922" darkHeight="495" showCaption="true"></Image>

## Model tile actions

From the grid view, each model tile provides the following actions.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/74SjVVg_uCUa04MBZSAzA_image.png" size="80" width="1148" height="487" position="flex-start" caption="Model tile actions" darkWidth="1148" darkHeight="487" showCaption="true"></Image>

<table isTableHeaderOn="true" columnWidths="127,488" selectedColumns="" selectedRows="" selectedTable="false">
  <tr>
    <td align="left">
      Action
    </td>
    <td align="left">
      Details
    </td>
  </tr>
  <tr>
    <td align="left">
      \[1] **Export**
    </td>
    <td align="left">
      Downloads a `<model name>.json` file containing the model's static and dynamic attribute definitions. Use this file to import the model into another Litmus Edge instance.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[2] **Clone**
    </td>
    <td align="left">
      Creates a copy of the model. Enter a new name and description, then click **Update**.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[3] **Edit**
    </td>
    <td align="left">
      Update the model name, description, type, or UNS connector, then click **Update**.
    </td>
  </tr>
  <tr>
    <td align="left">
      \[4] **Delete**
    </td>
    <td align="left">
      Removes the model. Deleting a model does not delete its instances, but those instances will no longer have a valid model reference.
    </td>
  </tr>
</table>
