You can manage images by navigating to **Applications&#x20;**>**&#x20;Images**.&#x20;

Refer to the image and callouts below for management options.&#x20;

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/Mby-I5yEx-BUeqNrP8Hq5_image.png" size="80" width="1913" height="420" caption="Image management options" position="center" showCaption="true"></Image>

- **Copy image**: You can copy the image to your clipboard.
- **More info**: View details on the image. Details include the following:
  - ID
  - Size
  - Date and time created
  - Build
  - Tags
  - Dockerfile Details
  - Layers
- **Delete image**: Remove the image from the *Images&#x20;*&#x70;ane. You cannot remove a Docker image that is currently being used to run the respective Docker container.  Docker images not being used as Docker containers should be removed to reduce disk usage.&#x20;
