You can add a new group and associate *Roles&#x20;*&#x74;o it by navigating to **System&#x20;**>**&#x20;Access Control&#x20;**&#x66;rom th&#x65;**&#x20;Users&#x20;**&#x74;ab.

<hint type="warning">
  **Important**: New users need to belong to at least one group to be able to log in to Litmus Edge.&#x20;
</hint>

**To add a group**:

1. Navigate to **System&#x20;**\*> \***Access Control**.&#x20;
2. From the *Manage Groups* section, click the **Add Group** icon.
   The *Add Group* dialog box appears.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/ro1jNUTadRL3C8zC_y1NL_add-group.png)
3. Enter a name for the group.&#x20;
4. Click **Add Roles**. All existing *Roles&#x20;*&#x64;isplay. Select a role's check box to associate the role with the new group. Add as many roles as required.&#x20;
5. Click **Save**.
   The group is added to the *Group&#x20;*&#x74;ab.
