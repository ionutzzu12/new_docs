<ExpandableHeading>
  ### Required Setup

  For the \{\{Partners.PartnersUIoption}} tab to display \{\{Partners.Forster}} data, your \{\{masterVAS-Terminology.d-airy}} must first activate the integration. See [Activate Förster-Technik Integration](<../../../../../../../../VAS Platform Help - Table of Contents/Features/Partners/Calf Management/Förster-Technik/activate-forster-technik-integration.mdx>) to learn more.
</ExpandableHeading>

The \{\{Widgets.ForsterVisits}} widget includes details on each individual feeding visit for the previous day and the current day. View the example below to for more information on the data shown in the widget.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/Q8miQKkIbMkjCEo21XpMI_milk-cons-widget.png" size="50" isUploading="false" initialPath="../../Resources/Images/Partners/forster/milk-cons-widget.png" githubPath="Content/Resources/Images/Partners/forster/milk-cons-widget.png" position="flex-start" showCaption="false"></Image>

1. Units: Choose the units in which to display data\:Units: Choose the units in which to display data:\* Metric: Display data in Kilograms and Liters.
   - Imperial: Display data in Pounds and Gallons.:::hint\{type="info"}
     **NOTE:&#x20;**&#x54;his setting is independent of the Units selected in your [Account Settings](<../../../../../../../../VAS Platform Help - Table of Contents/Features/User Menu/edit-your-user-account.mdx>) and applies to all users in your \{\{masterVAS-Terminology.d-airy}}.
     \:::
2. **Yesterday's Consumption:** Shows the total milk consumed and how it compares to the target (or entitlement) for the day.
3. **Current Entitlement:&#xA0;**&#xA0;Calculates the difference between the milk consumed in \{\{today}}'s visit(s) and, if lower than the entitled amount, the available milk to reach the entitlement.In the example shown, the animal consumed 0.58 gallons of the entitled 0.66 gallon amount, which makes the current entitlement: 0.66 - 0.58 = 0.08 gal.
4. **Show More Info:&#x20;**\{\{Commands.Click}} this option to view detailed information for each visit.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/A3R8QI10bGPHIlH6F0pSt_milk-cons-widget-show-more.png" alt caption size="47"}
