Review the following information for setting up and configuring the Siemens Logo Ethernet (Gen1.3) driver.&#x20;

# Litmus Edge Compatible Series&#x20;

| Compatibility parameter      | Compatible items                                                                                        |
| ---------------------------- | ------------------------------------------------------------------------------------------------------- |
| **Driver type**              | Ethernet                                                                                                |
| **Validated devices/series** | [Siemens LOGO!](https://new.siemens.com/global/en/products/automation/systems/industrial/plc/logo.html) |

To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: LOGO BM 24CE
   - **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/EsaqC7hQh5_I2caKzX_wO_siemens-logo-ethernet-gen1.png" size="80" width="371" height="390" position="flex-start" caption="Reference image of device" indent="2" showCaption="true"></Image>

2. Set up the IP address under ADMIN mode. The default password to switch from Operator mode is LOGO.
3. Select **Network > IP address**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/owz8tFDqQ2U46uXCMjnw3_siemens-logo-ethernet-gen1-devices.png" size="80" width="308" height="649" position="flex-start" caption="Reference image of device" indent="2" showCaption="true"></Image>

4. Enter the IP address for your device. For example: 192.168.447.110.&#x20;
5. Log in to the web server to configure the device. The default password is LOGO.
6. Launch the Logo Soft Comfort v1.08.02 software and select **Tools > Transfer > Logo > PC**.
7. Enter the IP address in the **Target IP Address&#xA0;**&#x66;ield, and then click **OK**.
8. Select **Tools > Ethernet connection**.
9. From the *Configure Ethernet Connection* dialog box, click **Ethernet Connectors**.
10. From the *Local Properties (Server)* section, select **Accept all connection results**.
11. From the *Remote Properties (Client)* section, enter the local TSAP in the **TSAP&#xA0;**&#x66;ield. The local TSAP default is 20.00. For this configuration, the Remote TSAP is set to 21.00.
12. Click **OK**.&#x20;

## Tag Parameters

- **Name**: Select a register name from the drop-down list. The available options depend on the names in the register table.&#x20;
- **Value Type**: Select a data type from the drop-down list. The available options depend on the register name selected.&#x20;
- **Polling Interval**: Enter a value in seconds. This determines how often the tag should poll the register for data.
- **Tag Name**: Enter a name for the tag.
- **Description** (Optional): Enter a description for the tag. &#x20;
- **Address**: Enter a tag address. The value must in the decimal format, within the allowed min/max range. Omit the leading zeros. The device might use the decimal (D), hexadecimal (H), or octal (O) address format. See the Register Table at the bottom of the dialog box. For Bit tags, append the bit address without the bit separator. The lowercase letters in the address format indicate how many digits you should enter in the Bit address.
- **Tag Formula**: Enter a formula for the tag to process the generated data. Two variables are permitted: `value` (current tag value) and `timestamp` (current tag UNIX time in milliseconds). The following math functions are available:
  - sin
  - cos
  - sqrt
  - tan
  - power: power(`x`) performs the operation `10^x`.
  - log: log(`x`) is the natural logarithm (the logarithm is in base `e`).&#x20;
  - exp: exp(`x`) performs the operation `e^x`.
- **Only Publish on Change of Value**: Select the checkbox to only publish NATS messages when the *value&#x20;*&#x70;arameter changes to a new value. *Change of Value* only applies to boolean, numeric (such as int or float), and simple string data types. It does not apply to complex types, such as JSON or array. Poll-once topics will not be affected by *Change of Value* settings. These topics will still only see a single message.&#x20;
- **Meta Data**: Metadata summarizes basic information about data. This feature allows you to define key-value pair data for the device output payload later on. Metadata can then be used to find, use, and reuse particular instances of data.&#x20;

<hint type="info">
  **Note**: If you use special characters in meta data key names, the special characters are replaced with underscore characters in the payload. This can cause two key names to be combined into one. For example, configuring the key names a\*\*b and a&\&b will cause only one key name to be created (a\_\_b).&#x20;
</hint>
