ENHANCEMENTNew users are automatically prompted to enter their preferred account settings. When a user creates a \{\{masterVAS-Products.Pulse-only}} account, the system automatically prompts them to set their preferred measurement units (imperial or metric), date/time format, and language. This is a usability improvement, as new users won't need to navigate to [Account Settings](../../Features/Config/EditUserAcct.htm) manually.

## Bug Fixes

- Wrong field label for \{\{Events.12ABORT}} event. The \{\{REM}} label now shows as expected on the \{\{Events.12ABORT}} event entry page.
- BNAME and REG now accept alpha characters. \{\{masterVAS-Products.Pulse-only}} now accepts alpha characters for an animal's \{\{BNAME}} or \{\{REG}} number (for country code).
- EXPCALF fix. The EXPCALF item now displays correctly in reports and worklists.
- `MyDC ONLY`SSF export fix. If applicable, multiple events for the same animal are correctly exported. Previously, \{\{masterVAS-Products.Pulse-only}} limited the export to a single recent event per animal.
