  ### Learn more about the FNAME and BNAME in Search Results

  For \{\{masterVAS-Terminology.d-airies}}that identify animals by source farm or barn name, the search results will display the respective FNAME (source farm name) or BNAME (barn name) for each match. The system displays both the FNAME and BNAME up to their first 7 characters.

  :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/jC-6hH71L919cU-_-_qgW_search-fname-results.png" alt caption size="16"}:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/1lS3GkeGxYQadZHn3b16d_search-bname-results.png" alt caption size="14"}

  Likewise, the \{\{masterVAS-Terminology.cowcard}} header identifies the source farm name/ID and barn name.

  :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/PuMzCBapGdLcLDZs3mX1t_fname-header.png" alt caption size="25"}:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/EzWITMIWSNqWRKzyMbsNr_bname-header.png" alt caption size="27"}

  For \{\{masterVAS-Terminology.d-airies}} that use source farms, \{\{masterVAS-Products.Pulse-only}} displays the source farm ID as it is configured in \{\{masterVAS-Products.DC305-simple}}. This feature allows you to use a dot when you want to search for animals by source farm. For example, if the source farm ID is 11, you can start typing `11.` or `.11`, depending on whether \{\{masterVAS-Products.DC305-simple}} is configured to display the source farm extension before or after the animal ID. As shown in the example below, this \{\{masterVAS-Terminology.d-airy}} configures its source farm ID at the end of the animal ID.

    ### Tell me more about the Source Farm Name and ID Number

    In both the search results and the \{\{masterVAS-Terminology.cowcard}} header, animals with source farms are displayed with a source farm extension on their animal IDs. As illustrated here, the source farm number is displayed after a dot at the end of the ID and indicates the farm from which the animal came.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/b48K6M7___aHYsaehT2vP_fname-id-number.png" size="31" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/fname-id-number.png" githubPath="Content/Resources/Images/Platform/cowcards/fname-id-number.png" position="flex-start" showCaption="false"></Image>

    In the example below, the combined \{\{DC-UI.DC-ProdNameShort}}FNAME and animal ID is **1109123**. It can be interpreted as:

    - The Source Farm ID number 11 is at the beginning, followed by an extra zero (i.e., 110) as part of the \{\{masterVAS-Products.DC-Platform}} FNAME ID coding.
    - The animal ID 9123 is at the end.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/q6F8NOA-igCD6r5q-Tm4Z_fname-illustrated.png" size="13" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/fname-illustrated.png" githubPath="Content/Resources/Images/Platform/cowcards/fname-illustrated.png" position="flex-start" showCaption="false"></Image>

    ### Tell me more about the Barn Name

    The Barn Name (BNAME) is a distinct name assigned to an animal, separate from the animal ID. The BNAME is displayed in \{\{masterVAS-Products.Pulse-only}} as a 7‑character-max letter/number combo, such as an assigned name, the color of the ear tag, or any other distinct name that helps identify each specific animal. A BNAME could look like this: *Daisy*, *Sammie*, or *Janie* when they are only names, or combos such as *GRE1234*, *BL1234*, *OR909*, *R909*, *Daisy123*, *123Daisy*, etc.

    When searching for an animal at a \{\{masterVAS-Terminology.d-airy}} with BNAMEs, you can search by numbers and letters. The search then returns all animal ID numbers and BNAMEs that contain those numbers and letters. For example, if you type `123`, the results could show an animal ID 12344 as well as BNAMEs such as 123Daisy or Daisy123.
  </ExpandableHeading>
