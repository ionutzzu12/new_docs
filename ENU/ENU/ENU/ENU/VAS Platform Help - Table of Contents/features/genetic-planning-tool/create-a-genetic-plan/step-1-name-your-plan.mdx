Name Your Plan is the first step of the planning process and allows you to add a plan Name and Description. Follow the steps below to continue:

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/RgyU_doHspTLsmZY5hdEH_genex-step1-new.png" size="46" isUploading="false" initialPath="../../../Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-step1-new.png" githubPath="Content/Resources/Images/PlatformAddons/GenPlanning/GenPlan/Genex-step1-new.png" position="flex-start" showCaption="false"></Image>

1. Add a descriptive name. For example, you may choose to include the animals the plan applies to and the date: *Heifers - Q4 2024*. Descriptive names will make sorting through plans easier as more plans are added.
2. Select the animals to include in the plan. Select **All animals** to include all animals in the plan or **Custom** and follow the steps below to only include animals from a specific \{\{pen}} or PSTRG::::ExpandableHeading

### What is a PSTRG?

<Paragraph indent="1">
  A PSTRG is another way to group animals from a \{\{masterVAS-Terminology.h-erd}} together, similar to a \{\{masterVAS-Terminology.p-en}}. However, a PSTRG can include multiple \{\{masterVAS-Terminology.p-ens}}. To view a list of existing PSTRGs, enter `SUM BY pen PSTRG`[in the command line](<../../../../../../VAS Platform Help - Table of Contents/features/command-line.mdx>).
  \:::- PSTRG users: From the PSTRG drop-down list, select the PSTRG. From the PEN drop-down list, select **All** to include all \{\{masterVAS-Terminology.p-ens}} from the PSTRG or **Custom** and enter specific \{\{masterVAS-Terminology.p-en}} or a pen range.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  All other users: From the PSTRG drop-down list, select **PSTRG=-** (hyphen)or **PSTRG=0**. In the PEN drop-down list, select **Custom** and enter a specific \{\{masterVAS-Terminology.p-en}} or a \{\{masterVAS-Terminology.p-en}} range.
</Paragraph>

3. Select the breed(s) to include in the plan from the drop-down list. This selection will filter only animals of the selected breed(s) for the new genetic plan. Unselected animals will not be mated. Once set, the breeds cannot be changed after you click **Continue**. All genetic data is breed-specific, so if you select the wrong breed in this step, you will need to create a new plan.
4. Select the \{\{masterVAS-Terminology.s-ire}} breed you want to mate your animals to.
5. Select the breed you want to use for index and parent average calculations, projections, and missing data if needed.
6. Add a note to capture the primary goals for the plan. For example: *Heifers plan for Q4*.
7. Click **Continue** and proceed to [Step 2: Set Breeding Goals & Index](step-2-set-breeding-goals-and-index.mdx).

<hint type="warning">
  **IMPORTANT:&#x20;**&#x4E;ote that if you edit the choices in this step, it may reset selections in later steps. Review all of your selections before activating your plan.
</hint>
