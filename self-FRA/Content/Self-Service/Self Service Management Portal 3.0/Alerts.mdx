<hint type="info">
  Le module **Alertes** n'est pas disponible clé en main (OOTB). Cet espace de travail apparaît uniquement si votre administrateur l'a configuré pour votre rôle.
</hint>

L'espace de travail **Alertes** permet d'afficher des avis importants, notamment :

- Erreurs du système ou d'une application (comme les problèmes de performances ou les arrêts d'application)
- Avertissements
- Messages Informations

Les alertes vous aident à identifier et à résoudre proactivement les problèmes, avant qu'ils n'affectent les utilisateurs.

<ExpandableHeading>
  ## Accès à l'espace de travail Alerte

  1. Connectez-vous à \{\{FRS\_VARS.ProductName}}.
  2. Dans le menu de gauche, sélectionnez **Alertes**.

  La liste des alertes s'affiche, avec des détails clés comme la catégorie, l'état et la description de chaque alerte.

  3. Cliquez sur une valeur au-dessous pour afficher le jeu d'alertes correspondant sur cette page.
</ExpandableHeading>

<ExpandableHeading>
  Afficher et gérer les alertes

  Afficher les détails de l'alerte
  --------------------------------

  1. Cliquez sur l'alerte à afficher.

  Les détails suivants sont affichés :

  <Paragraph listStyleType="disc" indent="2">
    Catégorie
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    État
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Description détaillée
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Gravité
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Créé par
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Dernière modification par
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Date de création
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    Date de dernière modification
  </Paragraph>

  **Trier les alertes**

  1. Cliquez sur la liste déroulante **Filtrer et trier**.
  2. Choisissez l'un des ordres de tri suivants :
     - **Croissant**
     - **Décroissant**
  3. Choisissez le champ à utiliser pour le tri :
     - **Catégorie**
     - **État**
     - **Description**
  4. Cliquez sur **Appliquer** pour trier la liste.
  5. Pour supprimer le tri, cliquez sur **Tout réinitialiser**.

  **Rechercher des alertes**

  1. Dans le champ **Rechercher**, entrez un mot-clé.
  2. Appuyez sur **Entrée**.

  La liste se met à jour et les alertes correspondantes s'affichent.
</ExpandableHeading>

<ExpandableHeading>
  ## Personnaliser la vue Alerte dans l'application mobile Self Service

  1. Dans les espaces de travail **Alertes**, cliquez sur **Tout** dans la liste déroulante.
  2. Cliquez sur l'icône **Vue Vignette** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/_pj5WN6eiB60jVUU7jDLR_tile-view-ssm-3022x19.png" alt caption} pour afficher les alertes sous forme de vignettes (ou briques).

  Les vignettes optimisent l'espace et rendent les données plus lisibles.

  3. Cliquez sur **la vue Liste** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/umzVTDiJZNYG1HpyKvjyv_list-record-view-ssm-3018x18.png" alt caption} pour afficher les enregistrements en vue Liste d'enregistrements.

  La vue Liste d'enregistrements affiche les enregistrements dans des lignes.
</ExpandableHeading>
