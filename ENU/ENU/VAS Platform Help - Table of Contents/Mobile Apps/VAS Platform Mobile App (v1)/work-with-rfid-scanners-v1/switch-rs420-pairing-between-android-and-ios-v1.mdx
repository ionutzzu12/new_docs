If your \{\{masterVAS-Terminology.d-airy}} uses only \{\{MorphHelp.AppleOS-TM}} devices, or a variety of \{\{MorphHelp.AndroidOS-TM}} and \{\{MorphHelp.AppleOS-TM}} equipment, you must update the wand \{\{masterVAS-Terminology.c-onfiguration}} to pair it with \{\{MorphHelp.AppleOS-TM}} devices. You will then also need to update the \{\{masterVAS-Terminology.c-onfiguration}} again if you want to switch back to pairing it with \{\{MorphHelp.AndroidOS-TM}} devices. Follow the instructions below to pair the \{\{MorphHelp.RS420short}} with an \{\{MorphHelp.AppleOS-TM}} device and then (if necessary) switch it back to an \{\{MorphHelp.AndroidOS-TM}} device.

<ExpandableHeading>
  ### Pair RS420 with an iOS Device

  To reconfigure the \{\{MorphHelp.RS420short}} scanner to be used with \{\{MorphHelp.AppleOS-TM}} devices, you'll first need to see if Menu Access is enabled. If it is not, use the AllFlex \{\{EID}} Tag Manager client to change the wand Menu Access setting. Then, proceed to setting the scanner to be discoverable for connection.

  Test for Menu Access on the \{\{MorphHelp.RS420short}} & Change if Needed

  1.   5. \{\{Commands.Click}}**Tools > Reader Config > Menu Access** and change the setting to **Enabled**.
  6. \{\{Commands.Click}}**Apply Settings**.
  7. Disconnect the scanner from the computer.

  Manually Connect the \{\{MorphHelp.RS420short}} to an \{\{MorphHelp.AppleOS-TM}} Device

  1. Press and hold the top (black) button on the scanner for 3 seconds to access the menu.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/jZEzTKENtmqfE158wSclv_allfelx-rs420scanner.png" alt caption size="22"}
  2. Use the bottom (green) button to arrow down to **Bluetooth settings**::inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/DSst1GGfcmXpUsJfevMtg_allflex-scanner-menu.png" alt caption size="10"}Press the top button to select the option.
  3. From the Bluetooth settings, choose **iPhone discoverable**::inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/fgQw50lftaoppRoH-a8Gp_allflex-scanner-bluetooth-menu.png" alt caption size="10"}
  4. The scanner discovers the \{\{MorphHelp.AppleOS-TM}} device, and displays its name and the option to connect. Select **Yes**.
  5. From the \{\{MorphHelp.AppleOS-TM}} device, view the Bluetooth settings to see available devices. The scanner displays as \{\{MorphHelp.RS420short}}\_\<serial number>. The scanner's serial number is located on the back of the scanner on the Allflex decal.
  6. Tap the correct scanner. If prompted for a password, use **1234** or **0000**.
  7. The scanner displays a message to confirm the connection. Verify the connection on the \{\{MorphHelp.AppleOS-TM}} device.
  8. Once the scanner and \{\{MorphHelp.AppleOS-TM}} device are connected, the scanner displays an M next to the Bluetooth symbol

<ExpandableHeading>
  ### Pair RS420 with an Android Device

  If you have reconfigured a scanner for use with an \{\{MorphHelp.AppleOS-TM}} device and want to return it to \{\{MorphHelp.AndroidOS-TM}} setup, you'll need to reconnect it to your \{\{MorphHelp.AndroidOS-TM}} device. See [Use a Scanner with Android Devices (v1)](<../../../../../VAS Platform Help - Table of Contents/Mobile Apps/VAS Platform Mobile App (v1)/Work with RFID Scanners (v1)/use-a-scanner-with-android-devices-v1.mdx>) for details.
</ExpandableHeading>

To avoid having to reconfigure the \{\{MorphHelp.RS420short}} when switching between device types, consider maintaining dedicated scanners for \{\{MorphHelp.AndroidOS-TM}} and \{\{MorphHelp.AppleOS-TM}} devices. Or, you may choose to purchase an additional, lower-cost scanner for use with the less common device type.
