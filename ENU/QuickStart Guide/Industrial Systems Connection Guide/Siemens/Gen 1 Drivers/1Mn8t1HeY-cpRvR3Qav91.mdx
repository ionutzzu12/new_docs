To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**&#x20;

1. Record the following information and refer to it when needed.&#x20;
   - **Device**: Siemens CPU 222
   - **Communication I/F**: RS485. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/d964M7ieFCw7hCxKiKHQT_siemens-s7-200-serialdevice.png" size="80" width="725" height="878" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch Step7-Micro/WIN software and click **Set PG/PC Interface**.&#x20;
3. Select **PC/PPI Cable**, and then click **Property**.
4. From th&#x65;*&#x20;Local Communication* dialog box, select **USB**.
5. Click **Communication**, and then click **Double-Click to Refresh**.
   The device is detected.&#x20;
