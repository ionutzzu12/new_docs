To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Record the following information and refer to it as needed.&#x20;
   - **Device**: Mitsubishi Q02CPU
   - **Communication I/F**: RS232. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/lPSCxGR3FS0adX0y1bQTM_mitsubishi-q02-02h-serial-reference.png" size="80" width="440" height="613" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

2. Launch GX Works2 v1.560J software, click **Create a Project**, and then click **Connection 1&#xA0;**&#x69;n the navigation panel.
3. Set up communication parameters, and then click **Connection Test**.
