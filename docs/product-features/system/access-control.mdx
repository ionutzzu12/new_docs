The *Access Control&#x20;*&#x70;ane allows you to do the following:

- Add and manage permissions for Litmus Edge users. See [Users](access-control/users.mdx).
- Create and manage API tokens and accounts. See [Tokens](access-control/tokens.mdx).&#x20;
- Manage LDAP/AD authentication providers. See :Link[LDAP/AD Auth]{label="LDAP/AD Auth" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="Cb8fI66OSul3TnlKrlKp8" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.&#x20;
- Manage the password policy for Litmus Edge. See [Manage the Password Policy](access-control/manage-password-policy.mdx).
