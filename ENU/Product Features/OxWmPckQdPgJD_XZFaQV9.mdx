The Dashboard gives you a real-time view of your Litmus Edge instance. The Dashboard displays system statistics, connected device and tag counts, message broker activity, and the health of running modules all in one place. This information is on the operational network and is completely offline.

Once you connect devices to Litmus Edge, you can use the Dashboard to monitor the overall health and activity of your system.

# Dashboard Charts

The Dashboard includes preset charts that display system performance data over time. Hover your cursor horizontally over the charts to view data for additional dates.

## CPU Utilization

Displays the usage of processing resources over time for the IUD Gateway, industrial PCs, all virtual machines, and the user.

# Module Status

At the bottom of the dashboard, status indicators show the health of the modules included in your plan. Hover over each indicator to see the module version. Each indicator displays a colored circle reflecting the online status of the selected module.

# Considerations

- The Dashboard operates on the operational network and is completely offline.
- Charts display data for the last 5 minutes by default.
- The Dashboard becomes meaningful once you have connected devices and tags in DeviceHub.
