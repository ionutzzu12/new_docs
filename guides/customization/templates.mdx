Archbee includes 4 built-in templates. Set the `template` field in `archbee.json`:

```json
"template": "stripe"
```

## Stripey

The default template. Clean, modern layout with a sidebar and two-column content area.

**Best for:** Technical documentation, API references, developer guides.

```json
"template": "stripe"
```

## Classic

Bold sidebar with colored navigation and prominent category headers.

**Best for:** Product documentation with many categories and sections.

```json
"template": "colorNav"
```

## Windy

Minimalist design inspired by Tailwind CSS documentation. Clean typography with subtle styling.

**Best for:** Open-source projects, design systems, minimalist docs.

```json
"template": "windy"
```

## Booklet

Book-style layout that renders all documents on a single scrollable page.

**Best for:** Single-page guides, tutorials, onboarding docs, articles.

```json
"template": "booklet"
```

## Switching Templates

Change the `template` field in `archbee.json` and save. The dev server updates instantly — no restart needed.

All templates support:

- Dark mode
- Custom colors
- Custom fonts
- Navigation icons
- External links
- Multiple spaces
