  This feature/option is available only for \{\{masterVAS-Products.DCP-users-short}} in \{\{Countries.002Brazil}}.

  Click [here](../../../GetStarted/KnowYourPulse.htm) to learn more about the different versions of \{\{masterVAS-Products.Pulse-only}}.

Use the **Export&#x20;**\{\{masterVAS-Terminology.H-erd-}}**&#x20;Data** feature on your \{\{masterVAS-Terminology.d-airy}}'s \{\{masterVAS-Terminology.h-omepage}} to easily create and export detailed \{\{masterVAS-Terminology.h-erd}} data reports.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/8x_bmDk76PthAEovW9NUB_dp-export-herd-data.png" size="61" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairy-profile/dp-export-herd-data.png" githubPath="Content/Resources/Images/Platform/dairy-profile/dp-export-herd-data.png" position="flex-start" showCaption="false"></Image>

Follow the steps below to export a \{\{masterVAS-Terminology.h-erd}} data report:

1. Click **Export** at the bottom left side of the page to open the Export \{\{masterVAS-Terminology.H-erd-}} Data side panel.
2. Select the report you want to download: select **Breeding Data Export** to download a report that lists all animals in the selected breeding event date range, or select **Replacements Data Export** to download a report that lists all live and archived animals that have a birth date in the selected breeding event date range.
3. Select the start date and end date for your report.
4. Click **Export** at the bottom right side of the side panel to create and download the report.
