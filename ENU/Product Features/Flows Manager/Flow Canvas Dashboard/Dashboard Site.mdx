The *Site&#x20;*&#x74;ab allows you to make selections affecting your dashboard, format the date and configure the size, spacing and padding for widgets and groups.

The **Title** field allows you to set the title of the dashboard.

The following options are available for further configuration of the dashboard website:

| **Option** | **Available configurations** |
| ---------- | ---------------------------- |
| Title Bar  | - Show the title bar         |

- Hide the title bar                                                                         |
  \| Side Menu  | \* Click to show side menu
- Always show side menu
- Always show icons only                                        |
  \| Tabs       | - No swipe between tabs
- Allow swipe between tabs
- Allow swipe (+mouse) between tabs
- Swipe to open/close menu |
  \| Theme      | \* Node-RED theme everywhere
- Use Angular theme in ui\_template
- Angular theme everywhere                        |

The date format can be set up by entering what you would like to see on the Dashboard.

<hint type="info">
  **Example**: DD/MM/YYYY
</hint>

You can control the size and spacing of widgets as well as the padding and spacing of groups on the Dashboard.
