This section details the requirements for the Immunization notifications for Targeted Messaging file format. This format allows pharmacies to notify the required providers when Immunizations are administered to patients. This is typically used to notify primary care providers in alignment with state regulations.

<hint type="info">
  **Note:&#x20;**&#x55;se of Targeted Messaging does not replace any state mandated reporting measures, such as reporting to state immunization registries.
</hint>

## Customer-Submitted Immunization File Information

This section details the file format which customers will submit for Immunization notification files.

### Record Information

This section specifies and describes the different record fields involved in the Immunization notification file, which Surescripts will validate and use to populate the unique messages that will be sent to the targeted provider.

Records that do not meet the base validations are not processed further and are included in the Targeted Messaging Delivery Report File with an applicable error code.

  **Note:&#x20;**&#x56;alidation does not test data types, valid value rules or formats. All elements are treated as plain text during validation.
