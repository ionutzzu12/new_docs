\{\{Commands.Click}} the \{\{masterVAS-Terminology.P-en-}}**&#x20;Changes** tab on the \{\{masterVAS-Terminology.cowcard}} for a detailed view of an animal's \{\{masterVAS-Terminology.p-en}} changes, organized by lactation cycle.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/4x0W6BNXhTlxPnXQI7rOg_pen-changes.png" size="45" isUploading="false" initialPath="../../../Resources/Images/Platform/cowcards/pen-changes.png" githubPath="Content/Resources/Images/Platform/cowcards/pen-changes.png" position="flex-start" showCaption="false"></Image>

See the topics below to learn more:

1. Details for the current lactation cycle. The section for the current cycle is expanded by default.
2. **Date:** The day when an event triggered a \{\{masterVAS-Terminology.p-en}} change for this animal.
3. **Moved to&#x20;**\{\{masterVAS-Terminology.P-en-}}**:** The animal's new \{\{masterVAS-Terminology.p-en}}.
4. A visual bar reference for the amount of time the animal was in each \{\{masterVAS-Terminology.p-en}} (blue) over the course of the lactation (gray). The blue portion of the bar helps you see how long the animal was in each \{\{masterVAS-Terminology.p-en}} during the lactation.
5. **Days in&#x20;**\{\{masterVAS-Terminology.P-en-}}**:** The number of days the animal spent in the \{\{masterVAS-Terminology.p-en}}.
6. **Premise&#x20;**\{\{ID}}**:&#x20;**\{\{masterVAS-Terminology.P-en-}} identifier related to each \{\{masterVAS-Products.Pulse-only}}\{\{Zone}} of the \{\{masterVAS-Terminology.d-airy}}.
7. **Source:&#x20;**&#x54;he event that triggered the \{\{masterVAS-Terminology.p-en}} change.
8. \{\{DIM}}**:&#xA0;**&#x54;he number of days the animal was in milk until the source triggered the \{\{masterVAS-Terminology.p-en}} change.
9. **Responsible:&#x20;**&#x54;he \{\{masterVAS-Terminology.d-airy}} worker responsible for the event that triggered the \{\{masterVAS-Terminology.p-en}} change.
10. You can expand any other lactation section to see previous \{\{masterVAS-Terminology.p-en}} change details. Multiple sections can be opened at the same time.
