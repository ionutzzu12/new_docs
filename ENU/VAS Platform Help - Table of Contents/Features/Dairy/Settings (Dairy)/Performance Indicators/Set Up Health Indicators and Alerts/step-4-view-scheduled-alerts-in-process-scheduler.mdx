Follow the steps below to continue:

1. Begin by selecting \{\{Buttons-UI.FeaturesMain}} > \{\{Protocols.ProcessScheduler}} from the \{\{masterVAS-Products.Pulse-only}} menu.
2. Click on a date in the \{\{Protocols.ProcessScheduler}} calendar to open a side panel that displays the date's scheduled processes::inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/wYN99aK9lD0hyGm0699La_dailyview-mastitis-alert.png" alt caption size="58"}
3. Click a process to view its details.This example illustrates details for the \{\{Mastitis}}**&#x20;Events Alert Group** process that we created as an example in [Step 3: Add Alert Groups](Step3AlertGroups.htm).:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/jW07x9OytLT0VfuhlKt5C_dailyview-mastitis-alert-click-intro.png" alt caption size="59"}:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/gDgYujyErxmrmCnVAYLLY_mastitis-alert-details.png" alt caption size="42"}The details note which indicators \{\{masterVAS-Products.Pulse-only}} is tracking for this alert group, which users will be notified if your \{\{masterVAS-Terminology.h-erd}} exceeds the thresholds, and the time of day that notifications will be sent.
4. Click **Edit Alert Group** to edit the details if necessary.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/9zcP2sw13KRfAZ4mUdAKI_mastitis-alert-click-edit.png" alt caption size="37"}::::ExpandableHeading
