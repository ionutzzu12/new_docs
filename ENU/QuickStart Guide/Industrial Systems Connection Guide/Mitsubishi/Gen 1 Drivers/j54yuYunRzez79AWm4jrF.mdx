To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed. ****
   **Device**: Mitsubishi A3A/A3N/A1SH/A2SH****
   **Communication I/F**: RS232 with a serial cable. Refer to the following figure.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/uobvr4WMIsUjrpnAVF7bY_mitsubishi-a2sh-serial.png" size="100" width="758" height="422" position="center" showCaption="false" indent="2"></Image>

2. Launch GX developer v8.103 software and connect the device with a serial cable.
3. Click **Online > Read from PLC**.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/MLA3ZFZ2hfcagelDvbR-D_mitsubishi-a2n-serial-read.png" size="80" width="616" height="433" caption="Read from PLC option" position="center" showCaption="true" indent="2"></Image>

4. Select the PLC series.&#x20;

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/eZnK_RaZ7LsAWZIm8BepA_mitsubishi-a2n-serial-series.png" size="60" width="302" height="106" caption="PLC series drop-down list " position="center" showCaption="true" indent="2"></Image>

5. Set up communications, and then click **Communication Test**.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/LQkmOe9yRKuJjTp0F2Ykc_mitsubishi-a2n-serial-communication.png" size="80" width="750" height="480" caption="Communication Test option " position="center" showCaption="true" indent="2"></Image>
