You can create an Access account by navigating t&#x6F;**&#x20;System&#x20;**>**&#x20;Access Control** >**&#x20;Tokens**.&#x20;

**To create an Access account:**

1. Navigate to **System&#x20;***>&#x20;***Access Control**.
2. Click the **Tokens&#x20;**&#x74;ab.&#x20;
3. From the *Access Accounts* section, click the **Add&#x20;**&#x69;con.&#x20;
   The *Create Access Account* dialog box appears.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/jQO6QNSeMUUG7B0GyaThc_create-api-account.png" size="100" width="1198" height="842" position="center" showCaption="false" indent="2"></Image>

4. Enter the details for the account.&#x20;
   - **Account Name**: Name of account
   - **Enable Account**: Account is enabled by default. &#x20;
5. Click **Create Account**.
