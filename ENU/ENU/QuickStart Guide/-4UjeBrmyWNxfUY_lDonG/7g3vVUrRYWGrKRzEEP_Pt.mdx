You have the option of connecting an [MQTT Client](<../../ENU/QuickStart Guide/-4UjeBrmyWNxfUY_lDonG/7g3vVUrRYWGrKRzEEP_Pt/ZoC9i90sRBxfEkj-XleEM.mdx>) and [Industrial OT Server](<../../ENU/QuickStart Guide/-4UjeBrmyWNxfUY_lDonG/7g3vVUrRYWGrKRzEEP_Pt/9Bd6OFOLfv9qKKNDFGJt7.mdx>) as DeviceHub devices.

<hint type="info">
  **Note**: You can't connect the [MQTT Client](<../../ENU/QuickStart Guide/-4UjeBrmyWNxfUY_lDonG/7g3vVUrRYWGrKRzEEP_Pt/ZoC9i90sRBxfEkj-XleEM.mdx>) to the DeviceHub [Industrial OT Server](<../../ENU/QuickStart Guide/-4UjeBrmyWNxfUY_lDonG/7g3vVUrRYWGrKRzEEP_Pt/9Bd6OFOLfv9qKKNDFGJt7.mdx>) if they are using the same Litmus Edge instance. This includes connecting through the Flows Manager or through an Integration connector. If they are using different Litmus Edge instances, they can connect.
</hint>
