Litmus offers a renewable 2-hour trial that gives you access to all functionality in Litmus Edge. The trial is suitable for developers who want to get started quickly with exploring, developing, and testing in Litmus without needing to set up a production environment. After the 2-hour time period expires, you can simply renew to continue using Litmus Edge with your data preserved.

To get started with the 2-Hour Trial:

- Decide on whether to install Litmus Edge on a VM, ISO, or a container&#x20;
- Get an install image from&#x20;  :Link[Litmus Central Portal]{href="https://portal.litmus.io/" newTab="true" hasDisabledNofollow="false"} and install Litmus Edge
- Access the Litmus Edge in your browser and activate your trial
- Renew the trial when it expires

## Download a VM, ISO, or a Container

You can install Litmus Edge on a VM, with an ISO, or on a container such as Docker.&#x20;

To download a Litmus Edge image:

Get an ISO or Docker image from :Link[Product Downloads]{href="https://portal.litmus.io/downloads" newTab="true" hasDisabledNofollow="false"}. If you downloaded:

- an ISO image, see :Link[ISO Installation on Hypervisors]{label="ISO Installation on Hypervisors" spaceId="wrAXt1hpevKwz3x_oRuPB" docId="DWzONQkDiDn4SnSz2FPSL" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.
- a Docker image, see :Link[Docker Installation]{label="Docker Installation" spaceId="wrAXt1hpevKwz3x_oRuPB" docId="xH3VHlG7P12KTlzHEzOnD" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.
- a Raspberry Pi image, see [Raspberry Pi Installation](raspberry-pi-installation.mdx).

## Activate the 2-Hour Trial

Activate your 2-Hour trial to begin working in Litmus Edge.

<hint type="info">
  **Note:** When you login to Litmus Edge Manager for the first time, provide the default credentials `admin `and `litmusedge`. Change the initial password after being prompted to do so.
</hint>

To activate the trail:

1. After logging in to Litmus Edge, select **2-Hour Trial**.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/pfvd4Yu2vif7uq3t8gr12_two-hr-trial-01.png" size="80" width="1456" height="792" position="flex-start" showCaption="false" indent="2"></Image>

2. Click **Activate**.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/yGWAOdHt178WwPk_9ySbB_image.png" size="80" width="1452" height="792" position="flex-start" showCaption="false" indent="2"></Image>

3. Litmus Edge authenticates to Central Portal to get product and solution updates.
4. Accept the End User License Agreement (EULA).

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/nx0SGyvHZgofcJMY_9fMe_image.png" size="80" width="1515" height="907" position="flex-start" showCaption="false" indent="2"></Image>

Optionally, download Digital Factory Demo to populate Litmus Edge with sample data. For more information on installing the demo, see :Link[Digital Factory Demo]{label="Digital Factory Demo" spaceId="SHlZRB8sbLUIhUOl3qzzX" docId="dpIZrXUi0RtDkBv5eDwWp" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}.

<hint type="info">
  **Note:** After two hours, click **Reset** and complete the reCAPTCHA to continue your trial. Your data is fully preserved and you can pick up exactly where you left off.
  Some services may take a few minutes to return online.
</hint>
