The [Profinet Acyclic](profinet/profinet-acyclic.mdx) driver supports read-only communication with devices that use PROFINET Acyclic communication. The driver also supports two optional :Link[application profiles]{href="https://www.profinet.com/profinet-explained/profinet-application-profiles" newTab="true" hasDisabledNofollow="false"}: `PROFIdrive` and `PROFIenergy`.

If you have a :Link[Turck TBEN-L5-8IOL]{href="https://www.turck.us/en/product/0000001600019e750008003a" newTab="true"}, use the [Profinet Acyclic Turck](profinet/profinet-acyclic-turck.mdx) driver. The Turck TBEN I/O Master uses Profinet Acyclic communications, but does not use the Slot and Subslot device parameters.
