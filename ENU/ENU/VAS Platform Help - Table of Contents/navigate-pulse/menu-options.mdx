  ## Command Line

  <ExpandableHeading>
    ### Is this relevant to me? Make sure it's included in your subscription!

    The contents below require access to the \{\{masterVAS-Products.OnPremise-lower}} version of \{\{masterVAS-Products.DC305-long}} and are only available to \{\{masterVAS-Products.DCU-users-short}}.

    Click [here](../GetStarted/KnowYourPulse.htm) to learn more about the different versions of \{\{masterVAS-Products.Pulse-only}}.
  </ExpandableHeading>

  The Command Line brings the power of \{\{masterVAS-Products.DC305-simple}} into \{\{masterVAS-Products.Pulse-only}}, allowing you to generate many of the same reports you're able to view in \{\{masterVAS-Products.DC305-simple}}. \{\{masterVAS-Products.Pulse-only}} Command Line currently supports the [LIST (SHOW)](../Features/PlatCmdLine/ListCmd.htm), [SUM](<../../../VAS Platform Help - Table of Contents/Features/Command Line/sum-command.mdx>), [COUNT,](../Features/PlatCmdLine/CountCmd.htm) and [ENTER](../Features/PlatCmdLine/EnterCmd.htm) commands. You can find this feature on the main menu bar, or by selecting **F9** from your keyboard.

  See [Command Line menu](../Features/PlatCmdLine/CmdLineIntro.htm) to learn more!
</ExpandableHeading>
