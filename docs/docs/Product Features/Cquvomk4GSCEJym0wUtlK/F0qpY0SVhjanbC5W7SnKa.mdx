Analytics flows are used to collect data, process that data, and then export that data to your individual configurations.&#x20;

Flows are made up of processors and require at least one input processor, one function processor, and one output processor. See [Input Processors](<../../../Product Features/Analytics/input-processors.mdx>) and [Output Processors](output-processors.mdx) for details.&#x20;

To create an entire flow at once, see [Create an Analytics Flow](<../../../Product Features/Analytics/Analytics Flows and Processors/HB_-yW2A2ghEBv7WsbRMY.mdx>). To add an individual processor to the Analytics canvas, see [Add Processors and Processor Connections](<../../../Product Features/Analytics/Analytics Flows and Processors/GExwUFwj-jxEsanA9dyh-.mdx>).&#x20;
