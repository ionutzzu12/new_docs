  ## Command Line

  **Preferred Keyword Language:** Use the drop-down to select the language used for Command Line keywords (e.g., LIST, SHOW).
</ExpandableHeading>

<ExpandableHeading>
  ## Parlor Settings

  **Milk Testing Provider:** Use the drop-down to select the provider your \{\{masterVAS-Terminology.d-airy}} uses for milk testing.

  **Milking Frequency:** Enter the general number of times per day that your \{\{masterVAS-Terminology.d-airy}}'s animals are milked. You must enter a digit between 0-4.
</ExpandableHeading>
