Review the following information for setting up and configuring the AB MicroLogix (Gen1) driver.&#x20;

To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, use PLC software on your computer to configure the device for Ethernet access.&#x20;

Refer to the following parameters.&#x20;

**Communication mode**: Port Setting: 10/100 Mbps Full Duplex/Half Duplex

# Step 2: Set up the PLC Device in Litmus Edge DeviceHub

Configure the following parameters when you [Connect a Device]() with this driver. Update default values to the specific setup of your device.&#x20;

| Parameter           | Value                                    |
| ------------------- | ---------------------------------------- |
| **Type**            | Allen-Bradley                            |
| **Driver**          | AB MicroLogix (Gen1)                     |
| **Network Address** | The device address you defined in Step 1 |
| **Port**            | 102                                      |
| **Slot Number**     | 0                                        |
