You can quickly check the connection status between the \{\{masterVAS-Products.WC-Platform}} app and \{\{masterVAS-Products.Pulse-only}}. Click on the Integration Status icon (located at the top of \{\{masterVAS-Products.Pulse-only}}) to open the [Integration Status](<../../../../../VAS Platform Help - Table of Contents/Features/Dairy/integration-status.mdx>) page where you can view the \{\{masterVAS-Products.WC-Platform}} status card.

### Red Integration Status Icon (Sync Status Error Condition)

This icon is located at the top of the page, toward the right side of the screen. If the status icon is red, it indicates that data is not currently synchronized for the default platform zone.

\{\{Commands.Click}}the **Integration Status** icon

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/PnuJWlqeFywrnHnUVdZwT_sync-error.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/sync-error.png" githubPath="Content/Resources/Images/icons/sync-error.png" position="flex-start" showCaption="false"></Image>

to view the sync error date, time, and location. When the error has been corrected, the system displays the date and time that synchronization was restored. Click the small gray arrow buttons to view sync status for other zones.

If your data fails to sync within 24 hours, the \{\{masterVAS-Products.Pulse-only}} displays an **Integration Status Alert** each time you access the \{\{masterVAS-Terminology.d-airy}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/NcbU4ieDd8ilNjqkKsNhq_int-status-alert-wc.png" size="73" isUploading="false" initialPath="../../../Resources/Images/Platform/features-icons-examples/int-status-alert-wc.png" githubPath="Content/Resources/Images/Platform/features-icons-examples/int-status-alert-wc.png" position="flex-start" showCaption="false"></Image>

This is a critical error that requires **immediate attention** to ensure that all animal information, \{\{worklist}} details, and event data continue to sync correctly with the \{\{masterVAS-Products.Pulse-only}} and mobile app. Proceed to Troubleshooting to address any errors.

Synchronization errors are defined as either :Link[Communication Status Errors]{label="Communication Status Errors" overridedLabel="Communication Status Errors" spaceId docId="iTWEl5OGWb8F8UVsmpXEi" version="v2" docAnchorId="#Communic" loadingMethod="dynamic" newTab="false" href="weighcomp-integration-status.mdx"} or :Link[Last Data Sync Errors]{label="Last Data Sync Errors" overridedLabel="Last Data Sync Errors" spaceId docId="iTWEl5OGWb8F8UVsmpXEi" version="v2" docAnchorId="#Last" loadingMethod="dynamic" newTab="false" href="weighcomp-integration-status.mdx"}. See below for more information.

### Green Communication Status Icon

This icon can be found inside the Integration Status window, which can be opened by clicking the **Integration Status** icon at the top of the page. A green **Communication Status** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/NgdFYyHAdbUlYbtbxpHF4_heartbeat-healthy32x29.png" alt caption size="10"} indicates that the connection is healthy.

The system also displays the date/time of the last successful communication check (or check on the pulse "beat") between the \{\{masterVAS-Products.Pulse-only}} and the \{\{masterVAS-Products.WC-Platform}} app. The \{\{masterVAS-Products.Pulse-only}} is scheduled to check the communication status every 10 minutes.

<ExpandableHeading>
  ## Communication Status Errors

  A sync error in **Communication Status**

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/MV5g4-gr9bJj_GAcPYHE7_heartbeat-not-healthy22x25.png" size="10" isUploading="false" initialPath="../../../Resources/Images/icons/heartbeat-not-healthy_22x25.png" githubPath="Content/Resources/Images/icons/heartbeat-not-healthy_22x25.png" position="flex-start" showCaption="false"></Image>

  means that the connection (or "heartbeat") between \{\{masterVAS-Products.Pulse-only}} and the \{\{masterVAS-Products.WC-Platform}} app has broken down.

  The heartbeat essentially checks the pulse of the communication channel between the \{\{masterVAS-Products.Pulse-only}} and the \{\{masterVAS-Products.WC-Platform}} app. If there is a disconnect along that communication channel, the \{\{masterVAS-Products.Pulse-only}} is not able to receive data from the \{\{masterVAS-Products.WC-Platform}} app.
</ExpandableHeading>

<ExpandableHeading>
  ## Last Data Sync Errors

  A **Last Data Sync** error means that despite a healthy heartbeat between the \{\{masterVAS-Products.Pulse-only}} and the \{\{masterVAS-Products.WC-Platform}} app, the \{\{masterVAS-Products.Pulse-only}} is unable to import or process the data. The error message displays the date/time when data was last synchronized successfully, along with icons that indicate whether the data sync is healthy

  or has encountered errors
