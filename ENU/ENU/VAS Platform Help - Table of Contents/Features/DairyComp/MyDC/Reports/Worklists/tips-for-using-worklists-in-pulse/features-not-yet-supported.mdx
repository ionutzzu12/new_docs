\{\{Buttons-UI.PlatformWorklists}} do not yet support the following features:

- Using voice commands to look up an animal on a list
- Searching for an animal that is not on the list
- Parlor-based \{\{Worklists.HospListCommonUse}}s
- Configurable sounds for \{\{Worklists.HospListCommonUse}}s
