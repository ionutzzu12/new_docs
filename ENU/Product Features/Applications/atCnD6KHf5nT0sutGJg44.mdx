Use the *Marketplace&#x20;*&#x70;ane of the *Applications&#x20;*&#x6D;odule to deploy applications from your public and private Marketplace Catalogs. Also view Docker applications from specific Marketplace Catalogs and synchronize these Docker applications with the server where they reside.

<hint type="info">
  The Marketplace feature is only available for Virtual Machine or the ISO version of Litmus Edge. Marketplace is not available for the Docker version of Litmus Edge.
</hint>

Each Litmus Edge instance includes a default public Marketplace Catalog, which provides a selection of launchable Docker applications, including:

- Elasticsearch
- KDB
- Minio
- MongoDB
- Microsoft SQL for Linux
- MySQL
- PipelineDB
- PostgreSQL & Adminer

Each application displays the publisher and the category applied to the application.

<hint type="info">
  **Note:** The specific applications available may vary in your environment.
</hint>

# Access Applications Marketplace UI

**To access the Applications Marketplace:&#x20;**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **Applications**.
   The *Dashboard&#x20;*&#x70;ane appears.
3. Click **Marketplace**.&#x20;
   The *Marketplace* pane appears.
