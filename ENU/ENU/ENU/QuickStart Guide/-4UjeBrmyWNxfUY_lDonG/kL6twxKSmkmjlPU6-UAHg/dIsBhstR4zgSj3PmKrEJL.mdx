# Set up and Configure the EUROMAP 63 Driver

**To set up and configure th****e&#x20;****EUROMAP 63&#x20;****driver:**

1. Create a shared folder. See [Create a Shared Folder in Windows 10](<../../../../../Product Features/System/External Storage/Network Storage/Set up a CIFS Mount Point on Windows 10/0XImZn0oIa8oHH9A6FAUS.mdx>) to learn more.&#x20;
2. In Litmus Edge, follow the steps to [Mount External Storage](<../../../Product Features/Lv4VprvL49Sh2xq8oEdH-/-eJWLAK8nFnkQRDqbsVJc/HryaEQTXolf1kd7iapXDv.mdx>) and configure the following parameters.&#x20;
   - **Mount Point Name**: Enter the name of the shared folder you created previously.&#x20;
   - **Share**: Enter the share name in the following format: `//[IP address of the PC running the simulator]/[shared folder name]`. For example: **//192.168.53.34/Euromap63folder**.&#x20;
   - **Read Only**: Clear the checkbox.&#x20;
   - **Username**: Enter the username you configured.&#x20;
   - **Password**: Enter the password you configured.&#x20;
3. In Litmus Edge, follow the steps to [Connect a Device](<../../../Product Features/qDV3AZpxurItY0hScn5FH/36ZpLd_nEeqQvUAOKFxvf/sCrWEhYJ8UZnuYSbQ-0se.mdx>) and configure the following parameters. Configure other parameters as needed.&#x20;
   - **Device Type**: Select **Euromap63**.
   - **Driver Name**: Select **EUROMAP 63**.
   - **Path**: Enter the path of the shared folder in the following format: `/data/[shared folder name]`—for example: **/data/Euromap63folder**.&#x20;
4. Follow the steps to :Link[Browse Tags]{label="Browse Tags" spaceId="JWQMp_kvm0WsA_Fpjt1v_" docId="akkXnVv6EdVzAZ40SHbWM" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} and add tags to the DeviceHub device.&#x20;

## Free Tags

This driver doesn't have a list of available register names in the DeviceHub *Add Tag* form. To add tags, you must enter the case-sensitive register name exactly as it is defined in the PLC.

A **Connected** status for the device won't display until a tag with a correct register name is created.

## Available Value Types

- `int`
- `float`
- `int64`
- `float64`
- `string`
