The [University of Wisconsin-Madison Dairy Brain](https://dairybrain.wisc.edu/) is a diverse set of analysis tools that leverage integration of data from multiple farm systems in order to provide insights to help optimize \{\{masterVAS-Terminology.d-airy}} management practices. \{\{Partners.DairyBrain-short}}'s one-directional integration with \{\{masterVAS-Products.Pulse-only}} allows \{\{General.CompanyName-short}} animal and event data to be shared with the University of Wisconsin-Madison's \{\{Partners.DairyBrain-short}} suite of tools for data collection and analysis.
