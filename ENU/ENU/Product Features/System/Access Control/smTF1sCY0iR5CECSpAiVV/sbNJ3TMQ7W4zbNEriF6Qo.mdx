After you add groups, you can manage them by navigating to **System&#x20;**> **Access Control&#x20;**&#x61;nd referring to th&#x65;*&#x20;Manage Groups* section. &#x20;

# Search for an Existing Group

You can use the search bar from the *Groups&#x20;*&#x74;ab to search for existing groups.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/2zYSIdNeOU7xgt6X6Muzj_image.png)

# Manage Existing Groups

You can use the **Action menu&#x20;**&#x66;or a group to manage it.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/B-AzKwLZGj-7ATEBtv5zz_image.png)

- **Edit**: Update the name and roles for the group.&#x20;
- **Delete**: Remove the group from the *Groups&#x20;*&#x74;ab.&#x20;
- **Edit Group Users**: Review the current list of members and click **Assign People** to add or remove users to the group.&#x20;
