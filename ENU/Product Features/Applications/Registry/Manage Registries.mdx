You can manage registries by navigating to **Applications&#x20;**>**&#x20;Registry**.

Click the **Action menu** for individual registries to manage them.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/6N2nOppsRO7cORSLQPhHR_image.png "Management options for registries")

- **Edit**
- **Remove**

<hint type="warning">
  **Important**: If you edit an AWS\_ECR container registry, you must re-enter the AWS Region, AWS Key ID, and AWS Secret Key before saving the updates.
</hint>
