To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device**

1. Record the following information and refer to it as needed.&#x20;
   - **Device**: AB FlexLogix L24ER-QB1B
   - **Communication I/F**: Ethernet and USB. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/GbbzCKrgmVLWpRH5UjYkv_ab-flexlogix.png" size="80" width="613" height="341" position="flex-start" caption="AB FlexLogix reference" showCaption="true" indent="2"></Image>

2. Connect the device to a PC using a serial port.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/mJdLZWqx7p9MbnipjxQWQ_ab-flexlogix-serial.png" size="80" width="407" height="197" position="flex-start" caption="Serial port" showCaption="true" indent="2"></Image>

3. Launch RSLinx classic and click **Configure Drivers**.
4. Select **RS232 DF1 Device driver** and click **Add New**.
5. Set up the *Comm&#x20;*&#x61;nd click **Auto-Configure**. Make sure the device can be detected successfully. The new device appears.&#x20;
6. Right-click the **02 1788-ENBT/A module > Module Configuration**.
7. Set up Ethernet parameters in *Port Configuration* page.
8. Create another Ethernet driver. Select and add the **EthernetIP** Driver.
9. Select the adapter with the same subnet as the device. The Rockwell device appears including the target.&#x20;
10. Launch RSLogix5000 v16. Create a project with the correct device model and version and then navigate to **Communication > Who Active**.
11. Select the target.&#x20;
