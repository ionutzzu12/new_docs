Devices is where you connect and manage PLCs, sensors, and other industrial equipment.&#x20;Once a device is connected, Litmus Edge can collect data from it using the configured driver and make that data&#x20;available across the platform.

# Access Devices UI

You can access **Devices** from DeviceHub.&#x20;

**To access Devices:**

1. Log in to Litmus Edge.
2. From the Navigation panel, select **DeviceHub**.
   The *Devices* pane appears. Devices appear as tiles.
