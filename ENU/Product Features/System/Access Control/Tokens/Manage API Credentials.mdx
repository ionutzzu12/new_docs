</hint>

You can manage tokens by navigating to **System&#x20;**>**&#x20;Access Control&#x20;**> **Tokens**.&#x20;

## API Keys

Click the **Action menu** to view management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/fr-qpvvgwuABAeKS9UjcQ_image.png)

- **Edit**: Update the name for the token.&#x20;
- **View Details**: View details and copy the API token value.&#x20;
- **Validate**: A message appears showing validation success.&#x20;
- **Remove**: Delete the token.&#x20;

## OAuth Clients

Click the **Action menu** to view management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Vqni2N5vInN7hvZjH8U80_image.png)

- **Edit**: Update the name of the client.&#x20;
- **View Details**: View details for the client.&#x20;
- **Remove**: Delete the client.
