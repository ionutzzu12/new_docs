To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<hint type="info">
  **Note**: Only one connection is allowed per port defined in the PLC. See the steps below to view and edit the open ports on the PLC.
</hint>

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;

- **Device**: Mitsubishi FX5U Ethernet Binary FX5U-32M
- **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/lTXYbKV5HwJLobE_-4nhQ_mitsubishi-fx5u-ethernet-binary-reference.png" size="80" width="955" height="588" position="flex-start" caption="Reference image of device" showCaption="true" indent="2"></Image>

1. Launch GXWork3 V1.035M software, and then click **Online > Read from PLC**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/Sw3n3iOUDE9NhWXoF5MjS_mitsubishi-fx5u-gx-online-read482x338.png" size="80" width="482" height="338" position="flex-start" caption="Read from PLC option" showCaption="true" indent="2"></Image>

2. Select the series.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/VCGUFwGmsLbIgU9TSSMr9_mitsubishi-fx5u-ethernet-binary-series-select.png" size="70" width="323" height="111" position="flex-start" caption="Series Selection dialog box" showCaption="true" indent="2"></Image>

3. Select the connection destination.&#x20;

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/6SBNb9RA0DbV-q7dU55w7_mitsubishi-fx5u-ethernet-binary-destination.png" size="80" width="820" height="429" position="flex-start" caption="Select Connection Destination dialog box" showCaption="true" indent="2"></Image>

4. Double-click the **PLC module** icon, and click **Find** to browse the device on the network.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/MrFx2s2JWTYdvd5BZOph-_mitsubishi-fx5u-ethernet-binary-browse.png" size="80" width="1121" height="550" position="flex-start" caption="Network browse step" showCaption="true" indent="2"></Image>

5. From the Navigation panel, click **Ethernet Port&#xA0;**&#x74;o set up communication parameters. Use **Binary&#x20;**&#x66;or this device.&#x20;
6. Double-click **External Device Configuration**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/z_GMbNgF_qKC9kS9laMs2_mitsubishi-fx5u-ethernet-binary-communication.png" size="80" width="1048" height="510" position="flex-start" caption="External Device Configuration option" showCaption="true" indent="2"></Image>

7. Set up the connections. In this case, port 4996 through 4999 are able to connect.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/SzbfndO0OvOkjsWGUqMu2_mitsubishi-fx5u-ethernet-binary-ports.png" size="80" width="975" height="185" position="flex-start" caption="Connection setup screen" showCaption="true" indent="2"></Image>
