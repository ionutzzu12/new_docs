The \{\{Buttons-UI.DC-Animals}} page displays the animals in your \{\{masterVAS-Terminology.h-erd}} by \{\{ID}} number, along with numerous other columns of information about each animal.

You can find this feature under the :Link[DairyComp/MyDC]{label="DairyComp/MyDC" overridedLabel="DairyComp/MyDC" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../navigate-pulse/menu-options.mdx"} > :Link[Animals]{label="Animals" overridedLabel="Animals" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp-Testdays" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp-Testdays" href="../../../../navigate-pulse/menu-options.mdx"} menu.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/rrcgj8T703pchAIDZaYO__animalslist-main.png" size="81" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/AnimalsList-main.png" githubPath="Content/Resources/Images/Platform/dairycomp/AnimalsList-main.png" position="flex-start" showCaption="false"></Image>

Follow the guidelines below when viewing your animals and their details: 

- Click**Filters** to search for specific animals or to limit the animals shown by \{\{masterVAS-Terminology.p-en}}, lactation, or reproductive code (\{\{RC}}).
- To view a summary of an animal, click on its ID to open a side panel without leaving the page.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/96TkB06SuwATCw4CSqp52_herdsidepanel.png" alt caption size="57"}

<hint type="info">
  **NOTE:&#x20;**&#x52;emember that you can also search for a specific animal at any time using the **Search for an Animal&#x20;**&#x66;ield at the top of the screen. See [Find an Animal](<../../../../../../VAS Platform Help - Table of Contents/Features/Manage Your Animals/yrLo-find-an-animal.mdx>) for more information.
</hint>

In the side panel, you can use the left and right arrow keys on your keyboard to step through side panels for the next/previous \{\{Side-Panels.PanelContent}} in the list. To close the side panel, click the **X** button in the top-right corner or press the **Esc** key.

Expand the topic below to learn more about customizing the \{\{Buttons-UI.DC-Animals}} page:

<ExpandableHeading>
  ## Customize the View

  Interactive column headers allow you to arrange the data into a way that's most useful to you. Depending on the features available for each column, you can add new columns of data, rearrange and reorder them, and filter some columns. To access these features, click on the column menu in the column you want to change.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/qnuufb85hPEOhW9C5fPsS_columnmenuopen.png" size="42" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/ColumnMenuOpen.png" githubPath="Content/Resources/Images/Platform/dairycomp/ColumnMenuOpen.png" position="flex-start" showCaption="false"></Image>

  The top menu can contain three different options—one for resizing the column, one for filtering the data displayed in the column, and one for controlling which columns are displayed. Each menu contains various ways for manipulating what's shown on the page.

  Expand the sections below to learn more:

  <ExpandableHeading>
    ## Filter Columns

    Some columns that contain numeric values can be filtered to show only certain data. For example, to filter the **Lactation** column so that only animals with a lactation value of 3 display, click the filter option, and then use the fields to create your filter.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/qj1RjbFtFHfdQMp1O0Yxl_herdfilter.png" size="37" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/HerdFilter.png" githubPath="Content/Resources/Images/Platform/dairycomp/HerdFilter.png" position="flex-start" showCaption="false"></Image>
  </ExpandableHeading>

  <ExpandableHeading>
    ## Reorder Columns

    You can reorder the way columns display by clicking in the header of the column you want to move and dragging it to its new location.
  </ExpandableHeading>

  <ExpandableHeading>
    ## Change which Columns to Display

    To add or remove columns from the page, click the columns option and then check or uncheck the boxes to select which columns to display.

    <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/DQd5aSkjYd-ETqW95S84r_herdcolumns.png" size="35" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/HerdColumns.png" githubPath="Content/Resources/Images/Platform/dairycomp/HerdColumns.png" position="flex-start" showCaption="false"></Image>

    You can also remove columns by clicking on the header of the column you want to remove, holding down the mouse key, and dragging the column up and out of the table.
  </ExpandableHeading>
</ExpandableHeading>
