You can set up the Influx client to execute influxQL queries.

# Before You Begin

Make sure you do the following:

- Add a database user. See [Add Database Users](<./././Product Features/DataHub/DB Users/Add Database Users.md>) for details.&#x20;
- Pull the Influxdb application image into Litmus Edge. See [Upload an Image](<./././Product Features/Applications/Images/Upload an Image.md>) for details.&#x20;
- Launch a container using the Docker Run command. See [Manage Containers](<./././Product Features/Applications/Containers/Manage Containers.md>)  for details.&#x20;

# Step 1: Deploy Influx Container

You will need to deploy the Influx container in Litmus Edge and specify the version you want to deploy.&#x20;

**To deploy the Influx container:&#x20;**

1. In Litmus Edge, navigate to **Applications&#x20;**> **Containers**.&#x20;
   The *Containers&#x20;*&#x70;ane appears.&#x20;
2. Click **Run**.&#x20;
   The *Enter command to run* field appears.&#x20;
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/zT58_FEgdS6lV15Q6BFHC_run-button.png "The Run button")
3. Enter the following command. Replace `[your influxdb version]` with the correct InfluxDB version (for example, influxdb:2.0).&#x20;

```none
docker run -d --name influxdb influxdb:[your influxdb version]
```

The named container appears in the *Containers&#x20;*&#x70;ane and displays a running state. The container runs locally.

# Step 2: Provide Database User Access to Influx Database

To provide the database user your created access to the Influx database, refer to the following resources and review the appropriate InfluxDB version:

- [influx - InfluxDB command line interface (2.0)](https://docs.influxdata.com/influxdb/v2.0/reference/cli/influx/)
- [influx - InfluxDB command line interface (2.7)](https://docs.influxdata.com/influxdb/v2.7/reference/cli/influx/)
- [influxdb - Official Image | Docker Hub](https://hub.docker.com/_/influxdb)

# Step 3: Enter and Execute InfluxQL Queries

After completing step 2, you can enter and execute InfluxQL Queries.

Refer to [InfluxData Documentation](https://docs.influxdata.com/) to learn more.&#x20;
