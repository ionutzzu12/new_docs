If your device supports LTE, a Modem Configuration section appears by navigating to **System&#x20;**>**&#x20;Network** and clicking the **WWAN&#x20;**&#x74;ab. Once you configure the modem, the information appears on the *Info&#x20;*&#x70;ane.

# Before You Begin

Make sure your configuration is using the same network on which your device is located.

# Configure a Modem

**To configure a modem:**

1. Navigate to **System&#x20;***>&#x20;***Network**.&#x20;
2. Click the **WWAN&#x20;**&#x74;ab.&#x20;
3. Confirm that there are configurations available.
4. Click the **Edit&#xA0;**&#x69;con.
   The *Update Modem Config* dialog box appears.&#x20;
5. Enter an access point name in the **APN&#xA0;**&#x66;ield.&#x20;
6. Click **Submit**.&#x20;
   The APN name appears in the *Assigned APN***&#xA0;**&#x61;nd the *Requested APN***&#xA0;**&#x66;ield.&#x20;
