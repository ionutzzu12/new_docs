## Overview

AWS SiteWise integration enables an Edge System to connect with Amazon AWS IoT SiteWise.

## Supported Communication Directions

- Outbound - Publish data from Edge to AWS IoT SiteWise
- Inbound - Subscribe data from AWS IoT SiteWise

## Variable Definitions

| **Variable**             | **Details**                                                                                                                                       |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                     | Enter a name for the connector                                                                                                                    |
| Region                   | Enter the AWS Region. See [AWS IoT SiteWise endpoints and quotas](https://docs.aws.amazon.com/general/latest/gr/iot-sitewise.html) to learn more. |
| Access Key               | Enter the AWS Access Key                                                                                                                          |
| Secret Key Access        | Enter the AWS Secret Access Key                                                                                                                   |
| Subscribe interval       | The time between fetching inbound messages (in seconds)                                                                                           |
| Number of batch messages | Maximum amount of outbound messages sent in one batch                                                                                             |
| Throttling limit         | Maximum throughput counted in messages per second                                                                                                 |
