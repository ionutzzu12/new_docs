Click the **Enabled&#x20;**&#x74;oggle to enable or disable the account.&#x20;

Click the **Action menu** for an account to access management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/qoIPcAZ5_mwl4PyeUT9o2_image.png "Action menu options for REST API accounts")

- **Edit topics**: See the *Add Topics to Accounts&#x20;*&#x73;ection below.&#x20;
- **View API Key**: You can view and copy the API Key.&#x20;
- **Reset Account**: If you reset the account, all settings and modifications are deleted.
- **Delete Account**: If you delete the account, all settings and modifications are deleted.

# Add Topics to Accounts

You can add one or more **Read** or **Write** Topics to an API account.

**To add topics to accounts:**

1. From **System** > **Access Control** > **Tokens**, click the **Actions menu&#x20;**&#x6E;ext to a selected account and select **Edit topics**.
   The account form drops down.
2. Do one or both of the following. From:
   - **Read Topics**, enter a name in the **Topic Name&#xA0;**&#x66;ield, and then click **Add Read Topic**.
   - **Write Topics**, enter a name in the **Topic Name&#xA0;**&#x66;ield, and then click **Add Write Topic**.

<hint type="info" indent="2">
  **Note:** Easily add multiple topics by separating topic names with a comma in the **Topic Name&#xA0;**&#x66;ield.
</hint>

3. Add additional topics as needed.
4. When you are done adding topics, click **Update**.
5. Click **OK**.
