You can manage your networks by navigating to **Applications&#x20;**>**&#x20;Networks**.

# View Networks

Refer to the image and callouts below for viewing options for your networks.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/JLSGFPqkMuG62N7tpEVDG_image.png "Network viewing options")

| **Action**           | **Details**                                                                                  |
| -------------------- | -------------------------------------------------------------------------------------------- |
| \[1] **Name column** | Click the **Name&#x20;**&#x63;olumn to sort network names in ascending and descending order. |
| \[2] **Show filter** | Open the search bar to search by network name.                                               |

# Manage Individual Networks

You can use the **Action menu** for a network to view management options.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/rar5rLJvQnFEOjfQOjOYL_image.png "Network action menu options")

- **Remove**: Remove the network from the *Networks&#x20;*&#x70;ane.&#x20;
- **Connect/Disconnect container**: After selecting to connect or disconnect the container from the network, select the appropriate container.&#x20;
