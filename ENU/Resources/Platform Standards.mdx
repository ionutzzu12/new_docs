The protocols and standards used by Litmus software comprise these categories.

### Southbound (PLC to Litmus Edge)

- ORiN device interface
- DOSA (Distributed-power Open Standards Alliance)
- OPC UA
- MTConnect open standard

### Northbound (device to cloud/database)

- MQTT (IETF)
- REST/HTTP

### Security

- Authentication (follow northbound)
- Transport Layer (SSL, TLS 1.3/1.2)
- Certificate Management (EPID)

### Data

- OPC UA
