Analytics flows are used to collect data, process that data, and then export that data to your individual configurations.&#x20;

Flows are made up of processors and require at least one input processor, one function processor, and one output processor. See :Link[Input Processors]{label="Input Processors" spaceId="7QvCN5MYpwobQXnPvWc7V" docId="2ZCIGXmgNtg1ooXyngDhu" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} and :Link[Output Processors]{label="Output Processors" spaceId="lwfN6vLUNp54T_hfVp5AA" docId="ocV9YHtR8lY_-KGSxYJtI" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} for details.&#x20;

To create an entire flow at once, see :Link[Create an Analytics Flow]{docId="TfwEzikn35IrSnJ_ZzSdO" label="Create an Analytics Flow" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId overridedLabel="Create an Analytics Flow"}. To add an individual processor to the Analytics canvas, see :Link[Add Processors and Processor Connections]{docId="PKUbZoUTUW1Igh3MwOLRs" label="Add Processors and Processor Connections" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId overridedLabel="Add Processors and Processor Connections"}.&#x20;
