Use the **Download Recent&#x20;**\{\{masterVAS-Terminology.C-owfile}} feature on your \{\{masterVAS-Terminology.d-airy}}'s \{\{masterVAS-Terminology.h-omepage}} to download the last available \{\{masterVAS-Terminology.c-owfile}} cloud backup that \{\{masterVAS-Products.OnPremise-lower}}\{\{masterVAS-Products.DC305-simple}} sent to \{\{masterVAS-Products.Pulse-only}}. For more information about \{\{masterVAS-Products.DC305-short}} cloud backups, see [Back DC305 Up to the Cloud](https://dc-help.vas.com/ReferenceGuide/Backups/Backups.htm#BackuptoCloud).

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/w7EemA4g1mL_uxEbbymmY_dp-download-cowfile.png" size="76" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairy-profile/dp-download-cowfile.png" githubPath="Content/Resources/Images/Platform/dairy-profile/dp-download-cowfile.png" position="flex-start" showCaption="false"></Image>

The file downloaded is saved to your downloads folder. The process is simple, quick, and allows you to download the last available \{\{masterVAS-Terminology.c-owfile}} backup wherever and whenever you prefer.
