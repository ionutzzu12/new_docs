You can create flows by navigating to the **Flows Manager**.&#x20;

See :Link[Flows Guides]{label="Flows Guides" spaceId="JWQMp_kvm0WsA_Fpjt1v_" docId="ciAEyVS1R5uCJ8jtkkstS" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more about leveraging Flows functionality.&#x20;

# Step 1: Select a Flows Manager

<hint type="info">
  **Note**: You can create a total of 16 Flows Managers.&#x20;
</hint>

By default, the *flow-1* Flows Manager is created with the status of `RUNNING`. You can control the status of the flow by clicking **Start Flow** and **Stop Flow**. See :Link[Manage a Flows Manager]{label="Manage a Flows Manager" spaceId="JWQMp_kvm0WsA_Fpjt1v_" docId="3IgCsKonJDpOD2tICEmRL" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel} to learn more.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/HBxCojGrjRY5PWDMleOHO_image.png "Status management icon for a Flows Manager")

If you'd like to organize your flows and store them in multiple Flows Managers, you can follow the steps below and create additional Flows Managers. If you are selecting an existing Flows Manager, proceed to Step 2. &#x20;

**To create a Flows Manager:**

1. Navigate to **Flows Manager**.
   The *Overview&#x20;*&#x70;ane appears.&#x20;
2. From the *Overview&#x20;*&#x70;ane, click **Add Flow**.&#x20;
   The *Create Flow* dialog box appears.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/Fa34mv1FnBu-NEPdpe5hc_addflowbutton.png" size="80" width="1910" height="278" position="center" caption="Add Flow button" showCaption="true"></Image>
3. Enter a name.&#x20;
4. Select the memory limit for the flow manager. You have the following options:
   - 256 MB
   - 512 MB
   - 1024 MB
   - 2048 MB
   - 4096 MB
5. Click **Create Flow**.&#x20;
   The Flows Manager is created.&#x20;

# Step 2: Create Individual Flow

After creating the Flows Manager, you can open the flow canvas to create individual flows.&#x20;

**To create an individual flow:**

1. For the flow manager you want to use, click the **Go To Flow Definition** icon.&#x20;
   The flow canvas opens in a new browser tab.&#x20;
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/kXh4sc3s7gKYzewaP09UU_gotoflowdefinitionicon.png "Go To Flow Definition icon")
2. Drag and drop nodes onto the canvas.
3. To connect nodes, hover over a node's gray dot until it changes color, and then drag the connector to another node's gray dot until it changes color.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/-rUTXiKiCqxiWNjozIRDE_connectnodes.png" size="80" width="886" height="456" caption=" A node connection" position="center" showCaption="true"></Image>

# Step 3: Configure Connector Nodes

If you are creating a flow for the first time in a Flows Manager, you will need to configure a connection to the connector nodes.&#x20;

**To configure the Connector Nodes:**

1. If you have a connector node on the flow canvas (such as Datahub Subscribe, Datahub Publish, MySQL, and MongoDB), double-click the node.&#x20;
   The edit node dialog box appears.&#x20;
2. In the first field in the *Properties&#x20;*&#x74;ab, click the **Edit&#x20;**&#x69;con. &#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/tkERuarFt3Eg-YJUMTW9l_editconnection.png" size="60" width="512" height="292" caption="Edit icon for configuring connection" position="center" showCaption="true"></Image>
3. Review the fields for configuring the connection and make any necessary updates.&#x20;
4. Click **Add**.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/yYQN_aDIiTtlXSyHRs_-E_addbutton.png" size="60" width="508" height="242" caption="Add button" position="center" showCaption="true"></Image>
5. In the edit node dialog box, click **Done**.&#x20;
6. Click **Deploy**.&#x20;
