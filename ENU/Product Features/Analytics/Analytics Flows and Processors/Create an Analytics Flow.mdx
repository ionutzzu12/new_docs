You can create an analytics flow by navigating to **Analytics&#x20;**>**&#x20;Instances**.&#x20;

When creating an analytics flow, you're guided through selecting input, function, and output processors and the flow automatically adds connections to the processors.&#x20;

# Create Analytics Flow

**To create an Analytics Flow:**

1. Navigate to **Analytics&#x20;**\*> \***Instances**.&#x20;
   The *Instances&#x20;*&#x70;ane appears.&#x20;
2. Use the *Analytics Groups* drop-down list to select the group you want to add the processor to.&#x20;

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/9OHzVMs7ocuASYixUaeVY_groupsdropdownlist.png" size="80" width="687" height="415" caption="Analytics Groups drop-down list" position="center" indent="2" showCaption="true"></Image>

3. Open the **Action menu** and select **Add flow**.&#x20;
   The *Create Flow&#x20;*&#x77;izard appears.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/lZM_PB1BAeaaq6wRXSOGF_addanalyticsflow.png" size="80" width="1408" height="458" caption="Add Flow option" position="center" indent="2" showCaption="true"></Image>

4. Enter a name for the flow.
5. Select an input processor and define its parameters. See [Input Processors](<././Input Processors.md>) to learn more.&#x20;
6. Click **Next**.&#x20;
7. Select a function processor and define its parameters. See [Statistical Functions](<././Statistical Functions.md>) to learn more.
8. Click **Next**.&#x20;
9. Select an output processor and define its parameters. See [Output Processors](#) to learn more.
10. Click **Next**.&#x20;
11. Click **Create Flow**.
    The flow displays in the Analytics canvas with default connections.&#x20;
