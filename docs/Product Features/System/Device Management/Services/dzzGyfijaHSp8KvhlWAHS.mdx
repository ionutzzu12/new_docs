You can manage FTP Users by navigating to **System&#x20;**>**&#x20;Device Management&#x20;**> **System Services**.&#x20;

You can click the **Enable&#x20;**&#x74;oggle for an FTP user in the *FTP Service* section to enable or disable the user.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/mW4jpnl5zSF2_7ksJfN3S_image.png "The Enable toggle for FTP Users")

You can click the **Action menu** for an FTP user to access management options.&#x20;

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/ac9hV145tO7ieFvSpuo02_image.png "Action menu for FTP users")

- **Reset Password**: After resetting a password, click the **Copy&#x20;**&#x69;con next to the password and save it to a text file to send to the user. Save the password in a safe place. The user cannot use the user account without this password.
- **Remove User**: You can remove a user from the FTP user list.&#x20;
