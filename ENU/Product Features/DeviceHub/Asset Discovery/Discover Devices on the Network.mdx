You can discover devices on the network by navigating t&#x6F;**&#x20;DeviceHub&#x20;**> **Asset Discovery**.&#x20;

# Enable/Disable Asset Discovery

<hint type="warning">
  **Important**: To discover devices on CIP, your firewall must allow incoming UDP packets on 58107. Once you enable discovery for CIP, a firewall rule in Litmus Edge will automatically be created to allow UDP messages on port 58107. When device discovery for CIP is disabled, the firewall rule will be automatically deleted. You can view this rule on the Firewall page, but you can't edit or delete it. See [Manage Firewall Rules](<./././System/Network/Manage Firewall Rules.md>) to learn more.&#x20;
</hint>

You can enable and disable discovery of devices that are connected to your network. Use Asset Discovery to discover which nodes are for edge devices.

To enable asset discovery, you will first need to select which interfaces to search on. Open the menu for Interfaces to select them.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/MCM38hic7x7-TNZxJWnio_image.png)

Then, toggle the Auto-Refresh switch to **ON&#x20;**&#x74;o start looking for devices.&#x20;

![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/A4I3UgZw6sVCOw4sDqbvs_image.png)

Once devices are discovered, you can review the following. &#x20;

## Overview

The overview section displays the following information.&#x20;

- The total number of devices discovered.&#x20;
- The total number of devices per vendor.&#x20;
- The total number of devices per device type.&#x20;

## Device Table

The device table displays the following information for specific devices.&#x20;

- IP address
- Device name
- Protocols
- Time last seen

# Discover Devices on the Network

Depending on the enterprise, Litmus Edge may become lost among the thousands of devices in the private network. You can use Litmus Edge with VGA/HDMI to discover devices.&#x20;

## Litmus Edge with VGA/HDMI

If the system is running normal Litmus Edge, it can be discovered using TUI (Terminal User Interface) access. Connect a monitor to the Litmus Edge IPC and the terminal shows the IP address of the device. See [Access the Terminal User Interface](<././././QuickStart Guide/Access the Litmus Edge Web UI/Access the Terminal User Interface.md>). &#x20;
