You can manage your OPC UA Client Certificates by navigating to **OPC&#x20;**>**&#x20;Statistics**.&#x20;

You can click the **Action menu** for client certificates you've added to manage them.&#x20;

- **Trust**: You can trust the client certificate
- **Remove**: The client certificate is removed from the list.&#x20;

Click **Shows****&#x20;Action menu** for all client certificates to remove all certificates from the list.
