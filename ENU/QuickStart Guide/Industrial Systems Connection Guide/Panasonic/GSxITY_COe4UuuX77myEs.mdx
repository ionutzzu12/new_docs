- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it as needed.&#x20;
2. **Device**: Panasonic FP7 CPS31ES
3. **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/pjbJN6l25JZOYCwAflUpZ_panasonic-fp7-ethernet.png" size="50" width="284" height="478" caption="Reference image of de" position="center" showCaption="true" indent="2"></Image>

4. Connect the device to a PC with a USB cable.
5. Launch FPWIN GR7S v2.24, and click **Online > Communication**.
6. Set up communication parameters.
7. Click the **Online&#x20;**&#x69;con and then go online.
8. Click **Options > FP7 Configuration > Build-in ET-LAN**.
9. Set up the Ethernet communication parameters and then click **OK**.
10. Download the setting into the PLC.
11. Set up the communication settings.
