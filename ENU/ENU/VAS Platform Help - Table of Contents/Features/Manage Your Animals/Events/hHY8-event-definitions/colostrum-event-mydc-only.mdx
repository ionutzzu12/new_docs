The \{\{Events.3002COLOSTRUM}} event is a \{\{masterVAS-Products.MyDC-short}}-only event that captures when newborn calves receive \{\{masterVAS-Terminology.c-olostrum}} or \{\{masterVAS-Terminology.c-olostrumReplacer}}. \{\{masterVAS-Terminology.C-olostrum-}}, the first milk produced by a cow after birth, is high in nutrients and antibodies that provide calves initial protection against disease. Calves need about 2 quarts (2 liters) of \{\{masterVAS-Terminology.c-olostrum}} within 4 hours of birth and 1 gallon (4 liters) within 12 hours.

<ExpandableHeading>
  ## Event Guidelines

  Use the mobile app or the web version of \{\{masterVAS-Products.Pulse-only}} to enter events! Tap \{\{Mobile-UI.EventsFeature}} in the [mobile app](<../../../../../../VAS Platform Help - Table of Contents/mobile-apps.mdx>) to get started, or see :Link[Add Events]{label="Add Events" overridedLabel="Add Events" spaceId docId="sp1fQUQTHg-WLIRXAer4X" version="v2" docAnchorId="#Add" loadingMethod="dynamic" newTab="false" githubPath="Events.htm#Add" href="Events.htm#Add"} to learn more about adding events from web. \{\{masterVAS-Products.Pulse-only}} supports adding events that have occurred within the last year.

  Note the guidelines below when working with the \{\{Events.3002COLOSTRUM}} event: 

  - This event can only be added for Lactation 0 animals.
  - The system warns you if you attempt to add the event to calves older than 2 days.
  - All fields are optional.
  - If \{\{masterVAS-Terminology.c-olostrumReplacer}} is used, the system does not prompt for the source of the \{\{masterVAS-Terminology.c-olostrum}}.

  Expand the topic below to learn more about \{\{Mobile-UI.EventTypes}} event details:

  <ExpandableHeading>
    ### Brix value

    \{\{masterVAS-Terminology.BrixValue}} is based on a system that uses a refractometer to measure colostral antibodies. \{\{masterVAS-Products.Pulse-only}} accepts values between 14 and 40. \{\{masterVAS-Terminology.C-olostrum-}} with a \{\{masterVAS-Terminology.BrixValue}} above 22 is considered optimal. When the \{\{masterVAS-Terminology.BrixValue}} is below 22, \{\{masterVAS-Terminology.d-airies}} may choose to supplement with \{\{masterVAS-Terminology.c-olostrum}}\{\{masterVAS-Terminology.c-olostrumReplacer}}.
  </ExpandableHeading>
