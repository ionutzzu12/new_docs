Each workspace in the Self Service portal has different capabilities and is used to perform different functions. To find out more about each one and the tasks you can complete as a Self Service User, select a link below:

- View, update, or close one of your submitted service requests and incidents. For details, see [My Items](my-items.mdx).
- Submit a new service request or report a new incident. For details, see [Service Requests and Incidents](service-requests-and-incidents.mdx).
- Browse knowledge articles that can help you resolve a problem. For details, see [Knowledge Center](knowledge-center.mdx).
- View announcements, FAQs, and alerts that are posted to the portal. For details, see [Announcements, FAQs, and Alerts](announcements-FAQs-and-alerts.mdx).
- Use the Social Board to send and receive messages from other users. Learn how to search for, filter, and delete messages, as well as set up groups of users to share messages with. For details, see [Social Board messages](social-board-messages.mdx).
- Chat with others using the instant messaging Chat Zone. For details, see [Chat Zone](chat-zone.mdx).
