When you :Link[Add a Tag]{docId="XgWOkQbTPevII7OR82LL0" label="Add a Tag" spaceId="JWQMp_kvm0WsA_Fpjt1v_" version="v2" docAnchorId loadingMethod="dynamic" overridedLabel}, you have the option of adding a formula to process the generated data.&#x20;

You can use the following variables:

- `value`: The current tag value.&#x20;
- `timestamp`: The time the tag value was generated represented in UNIX time in milliseconds.&#x20;

You can use the following functions.
