Links are used for navigation purposes in the dashboard. You can use links to navigate between tabs in your dashboard or to navigate to a specific website.

For example, you can add a link to an outside website that displays more information about node flows such as [Node-RED](https://flows.nodered.org/).

# Navigate to the Dashboard Layout tab

You can add, modify, and remove dashboard links from the Dashboard Layout tab.

**To navigate to the Dashboard Layout tab:**

1. From the Navigation panel, selec&#x74;**&#xA0;Flows Manager.**
2. Click the Expand icon next to a selected Flow.
3. Click the Menu icon, select **View&#x20;**> **Dashboard**, and then drag the **Sidebar&#xA0;**&#x75;p from the bottom of the canvas.

![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/D_hIaJnsuxHnBXRyMBpAC_image.png "The Dashboard Layout tab")

# Add a Dashboard Link

You can add a dashboard tab from the Dashboard Layout tab.

**To add a dashboard tab:**

1. Navigate to the Dashboard Layout tab for the Flow you wish to work in. From the *Tabs & Links* box, click **+ Link**.
   The link appears in the Tabs & Links box.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/F23MQQmYLjtsYlFM-5uev_bgdzxtutnpzmjx0m0rz2023-09-0810-44-15.png "Flows canvas")
2. Click **Deploy**.

# Modify a Dashboard Link

You can add the link URL in the dashboard, select the navigation, and select a specific Flow or all Flows for which to apply the changes from the Dashboard Layout tab.

**To modify a dashboard link:**

1. Navigate to the Dashboard Layout tab for the Flow you wish to work in.
2. Hover over a link in the Tabs & Links box and click **Edit**.
   The *Edit link node* dialog box appears.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/L3XqaYL-068Kk9LLQazIL_2023-09-0811-30-44.png "Edit button (hover to view)")
3. From the *Edit link node* dialog box, do the following:
   - Enter a URL for the link in the **Link&#xA0;**&#x66;ield.
   - Select an option from the **Open in&#xA0;**&#x73;ection.
   - Select an option from the Flows drop-down list.
   - Click **Update**.
4. Click **Deploy**.

# Remove a Dashboard Link

You can remove a link in the dashboard from the Dashboard Layout tab.

**To remove a dashboard link:**

1. Navigate to the Dashboard Layout tab for the Flow you wish to work in.
2. Hover over a link and click **Edit**.
   Th&#x65;*&#x20;Edit link node* dialog box appears.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/aDxCg78Ck3_ozGu-3qeIt_2023-09-0811-38-45.png" size="80" width="764" height="614" position="center" showCaption="false"></Image>
3. Click **Delete**.
4. Click **Deploy**.
