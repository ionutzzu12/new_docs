The *Access Control&#x20;*&#x70;ane allows you to do the following:

- Add and manage permissions for Litmus Edge users. See [Users](<Access Control/Users.mdx>).
- Create and manage API tokens and accounts. See [Tokens](<Access Control/Tokens.mdx>).&#x20;
- Manage LDAP/AD authentication providers. See [LDAP/AD Auth](<Access Control/LDAP_AD Auth.mdx>).&#x20;
- Manage the password policy for Litmus Edge. See [Manage the Password Policy](<Access Control/Manage the Password Policy.mdx>).
