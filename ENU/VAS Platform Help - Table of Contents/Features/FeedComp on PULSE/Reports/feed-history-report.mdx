  ## Search and Filter the Feed History Report

  Options to search and filter the \{\{FeedComp.FeedCompReportNames}}\{\{Controls.ControlTypes}} are available at the top of the \{\{General.UI-gui}}. Follow the guidelines below when \{\{Commands.UserActions}} the \{\{FeedComp.FeedCompReportNames}}\{\{Controls.ControlTypes}}:

  - \{\{Commands.Click}}**Filters** at the top left of the page to limit the data that is displayed. The **Filters** side panel opens.
  - Use the **From** and **To** filters to only view data from that range.
  - \{\{Commands.UserActions}}a \{\{FeedComp.FeedCompFeatures-Upper}} to limit the \{\{Controls.ControlTypes}} to a specific \{\{FeedComp.FeedCompFeatures-Lower}}. You can type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}} by name.
  - \{\{Commands.UserActions}}\{\{FeedComp.FeedCompFeatures-Upper}} to limit the \{\{Controls.ControlTypes}} to specific \{\{FeedComp.FeedCompFeatures-Lower}}. You can type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}} by name.
  - \{\{Commands.UserActions}} a \{\{FeedComp.FeedCompFeatures-Upper}} to limit the \{\{Controls.ControlTypes}} to a specific \{\{FeedComp.FeedCompFeatures-Lower}}. You can type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}} by name.
  - \{\{Commands.UserActions}}a \{\{FeedComp.FeedCompFeatures-Upper}} to limit the \{\{Controls.ControlTypes}} to a specific \{\{FeedComp.FeedCompFeatures-Lower}}. You can type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}} by name or number.
  - \{\{Commands.UserActions}} a \*\*\{\{FeedComp.FeedCompFeatures-Upper}} \*\***Type** to limit the \{\{Controls.ControlTypes}} to a specific type of \{\{FeedComp.FeedCompFeatures-Lower}}. Multiple \{\{FeedComp.FeedCompFeatures-Lower}} can be \{\{Commands.UserActions}}. You can also type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}}\{\{Controls.ControlTypes}}by name.
  - Select a **Status** to limit the report to a specific feeding status. Multiple \{\{FeedComp.FeedCompFeatures-Lower}} can be \{\{Commands.UserActions}}. You can also type text directly in the field to search for \{\{FeedComp.FeedCompFeatures-Lower}}\{\{Controls.ControlTypes}}by name.
  - \{\{Commands.UserActions}} the **Clear Filter\*\*\*\*s** option to clear all of the filters currently applied.
</ExpandableHeading>
