The multiple **Input&#x20;**&#x70;rocessors help you calculate the statistical properties of the data from the PLC machines.

# Inputs Overview

Use one of the following processors for your inputs:

- **Inputs Average&#xA0;**&#x67;ives you the average of the inputs values in the Flow.
- **Inputs Maximum&#xA0;**&#x67;ives you the maximum value between the inputs in the Flow.
- **Inputs Minimum&#xA0;**&#x67;ives you the minimum value between the inputs in the Flow.
- **Inputs Sum&#xA0;**&#x61;dds all the inputs values in the Flow.

The following information describes these processors:

- For **Field Name**, you can enter the field name for which you want to apply the function. The field name must be available in all the inputs. For example, **value&#xA0;**&#x6F;r **downtime**.
- For **Processing Type**, you can select from **float**, **signed&#xA0;**(signed integer) or **unsigned&#xA0;**(unsigned integer).
- Connect the inputs to the function using the **values&#xA0;**&#x63;onnection. The values connection acts as a time synchronizer.
- Connect the function to the output using the **events&#xA0;**&#x63;onnection.
