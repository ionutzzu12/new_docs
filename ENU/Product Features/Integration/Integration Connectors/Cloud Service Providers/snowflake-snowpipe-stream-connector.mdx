  **Note:** This feature is available starting with Litmus Edge 4.1.x.
</hint>

The Snowflake Snowpipe Stream connector allows you to connect Litmus Edge with Snowflake. For more information, see the [Snowflake Snowpipe Stream Integration Guide](<../../../../How-To Guides/Integration Guides/snowflake-snowpipe-stream-integration-guide.mdx>).

## Supported Communication

- Outbound (Write from Edge to Snowflake): Yes
- Inbound (Read from Snowflake to Edge): No
