To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<hint type="info">
  **Note**: Only one connection is allowed per port defined in the PLC. See the steps below to view and edit the open ports on the PLC.
</hint>

If the device is configured for Ethernet access, find its network parameters. Otherwise, follow the steps below.&#x20;

**To configure your device:**

1. Note down the following information and refer to it as needed. ****
   **Device**: Mitsubishi Q02CPU+Q71E71-100****
   **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/eRudfJaBjVu9Tagi4HBbC_mitsubishi-qj71e71-binary-reference.png" size="80" width="580" height="490" caption="Reference image of device" position="center" showCaption="true" indent="2"></Image>

2. Launch GX Works2 v1.560J software, click **Create a project**, and then click **Connection1&#xA0;**&#x69;n the navigation panel.
3. Set up communication parameters, and then click **Connect**.
4. Set up the network parameters. The *Station Number* cannot be duplicated.
5. Click **Operation Setting&#xA0;**&#x74;o set up IP for modules.
6. Click **Online > Write to PLC**, and then reboot the device to activate the settings.
7. Click **Connection1&#xA0;**&#x69;n the navigation panel.
