Standard items are available in \{\{DC-UI.DC-ProductName}} to capture specific animal data. Items are either stored or calculated values used by applications that connect to \{\{DC-UI.DC-ProdNameShort}} to generate reports or display animal details. Standard item types can be either default (generally used by all \{\{masterVAS-Terminology.d-airies}}) or pre-defined (for optional use in \{\{masterVAS-Products.Pulse-only}}).

<hint type="info">
  **NOTE:&#x20;**

  Items are customizable in

  \{\{masterVAS-Products.DC305-simple}}

  , and your

  \{\{masterVAS-Terminology.d-airy}}

  may use additional items not shown here. See the following for item number ranges and their associated types:

  - 1-89: default standard items
  - 90-1023: custom items, configured in the \{\{masterVAS-Terminology.c-owfile}}
  - 1024-1999: on-the-fly items, for advanced \{\{masterVAS-Products.DC305-simple}} command line use
  - 2000+: pre-defined standard items, for use in \{\{masterVAS-Products.Pulse-only}} only
</hint>
