Follow these steps to add a new **AI Model** connection.

# Add an AI Model Connection

1. Click **Add Connection**.
   The *Add Connection&#x20;*&#x63;onfiguration dialog box appears.

<Image src="https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/kXEBZwTDFpsves716bNoq_image.png" size="80" width="1854" height="652" position="flex-start" indent="2" showCaption="false"></Image>

2. Enter Connection Details:
   - **Name:** Provide a user-friendly name for the connection.
   - **Provider:** Select the desired provider from the available options.&#x20;
     - OpenAI API
     - Cloudflare AI Gateway
     - Grok API
     - Ollama API
     - Google Gemini API
     - Nvidia API
     - Anthropic API
   - **Configuration Fields:** Enter the **URL** endpoint, **API** key, and **Model ID&#x20;**&#x73;pecific to the provider.
3. Verify Connection:
   - Use the **Verify** button to perform a connection test via the List Models API call.
   - Ensure that the connection is successfully validated before proceeding.
4. Once verified, save the configuration to add it to your list of connections.
