The settings within the Reproduction Constraints page are used to manage the limits and warnings the system uses when reproductive events are entered. The settings on this page are broken down into specific breeding and \{\{masterVAS-Terminology.f-reshening}} categories to make them easier to understand and apply.

You can find this feature under the :Link[DairyComp/MyDC menu]{label="DairyComp/MyDC menu" overridedLabel="DairyComp/MyDC menu" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} > :Link[Settings]{label="Settings" overridedLabel="Settings" spaceId docId="WFLX7SPMpZfy7zoMyrvRh" version="v2" docAnchorId="#Menu-DairyComp" loadingMethod="dynamic" newTab="false" githubPath="../../../NavPlatform/Menu.htm#Menu-DairyComp" href="../../../../../../ENU/VAS Platform Help - Table of Contents/navigate-pulse/menu-options.mdx"} option on the \{\{Buttons-UI.FeaturesSecondary}} tab.

<ExpandableHeading>
  ## General Breeding

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/QEHIJMvw14W-Malh2DcTC_generalbreeding.png" size="40" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/GeneralBreeding.png" githubPath="Content/Resources/Images/Platform/dairycomp/GeneralBreeding.png" position="flex-start" showCaption="false"></Image>

  **Voluntary Wait Period:** In this field, enter the number of days that should pass before another reproductive event is entered for an animal. Entering a reproduction event for an animal prior to the amount of days specified here will result in a warning message before the event can proceed.

  \{\{masterVAS-Terminology.S-ire-}}**&#x20;Mating Strategy:** Use the buttons to choose how the per-animal \{\{masterVAS-Terminology.s-ire}} preference items are used on your \{\{masterVAS-Terminology.d-airy}}. Choose either *Prefer mating with the specified&#x20;*\{\{masterVAS-Terminology.s-ires}} or *Never mate with the specified&#x20;*\{\{masterVAS-Terminology.s-ires}}.

  **Minimum Days Since Last Heat for Pregnancy Entry:** In this field, enter the minimum number of days since the last heat event before a pregnancy can be entered. This value must be between 1-100 days.
</ExpandableHeading>

<ExpandableHeading>
  ## Cow Freshening/Calving

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/zTJ5DLNkYTADbCvdYnaMi_cowfreshcalving.png" size="45" isUploading="false" initialPath="../../../../Resources/Images/Platform/dairycomp/CowFreshCalving.png" githubPath="Content/Resources/Images/Platform/dairycomp/CowFreshCalving.png" position="flex-start" showCaption="false"></Image>

  **Warn When Not Dry:** Use this switch to display a warning when attempting to enter a \{\{Events.1FRESH}} event for a cow that is not dry.

  **Minimum Days in Milk:** In this field, enter the minimum number of days since the last \{\{masterVAS-Terminology.f-reshening}}/calving before the next \{\{Events.1FRESH}} event can occur. This value must be between 100-400 days.

  **Minimum Days Dry:** In this field, enter the minimum number of dry days expected before the next \{\{Events.1FRESH}} event can occur. This value must be between 0-560 days.

  **Minimum Days Carried Calf:** In this field, enter the minimum expected duration of pregnancy before the next \{\{Events.1FRESH}} event can occur. This value must be between 140-410 days.

  **Maximum Days Carried Calf:** In this field, enter the maximum expected duration of pregnancy before the next \{\{Events.1FRESH}} event can occur. This value must be between 150-410 days.

  **Minimum Days in Milk for an Open Cow:** In this field, enter the minimum number of days since the last \{\{masterVAS-Terminology.f-reshening}}/calving before entering a \{\{Events.1FRESH}} event for an Open animal. This value must be between 1-400 days.
</ExpandableHeading>
