You can manage database users by navigating to **DataHub&#x20;**>**&#x20;DB Users**.&#x20;

# Search for a Database User

You can search for a database user in the DB Users pane.

**To search for a database user:**

1. Navigate to **DataHub**\* > \***DB Users**.&#x20;
2. Click the **Show filter** icon.
   The search bar appears.&#x20;
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/PLkidQoKFeVbJed3fmbet_sfu0egevkx1qwhmmvikcgimage.png "Show filter icon")
3. Enter the search terms in the search bar.&#x20;
   The selections narrow as you type.

# Manage Database Users

You can manage individual database users.

**To manage database users:**

1. Navigate to **DataHub**\* > \***DB Users**.
2. Click the **Action menu&#x20;**&#x66;or the user.
   ![](https://api.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/x_s56VqzCKJlymqysuhsd_o5ndiznqybnwt0tigfurimage.png "Action menu for DataHub users")
3. Select an option for the user.
   - **Reset password**
   - **Delete User**: The user and all their privileges will be removed.&#x20;
   - **Add Privilege**: If the user is already assigned with a privilege, adding a new privilege will not combine privileges. The newly added privilege will replace the previous one.&#x20;

# Remove a Database User Privilege

You can remove an existing database privilege by clicking **X&#x20;**&#x6E;ext to the privilege you want to remove.&#x20;
