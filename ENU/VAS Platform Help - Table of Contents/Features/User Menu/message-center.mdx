The \{\{General.ProductName}}includes a Message Centerto serve as a communication hub and provide an easily-accessible area to view important messages from the \{\{General.ProductName}}.

To view all messages, selec&#x74;**&#x20;Message Center** from the [User Menu](../Config/UserMenu.htm) in the top right corner of the page.

See the example and the explanations below for more information.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/WDyygvz9Ri-4VC1t3vvO2_message-center-numbered.png" size="58" isUploading="false" initialPath="../../../Resources/Images/Platform/operations/message-center-numbered.png" githubPath="Content/Resources/Images/Platform/operations/message-center-numbered.png" position="flex-start" showCaption="false"></Image>

1. View: Use the **View** drop‑down option to filter the list and display only certain types of messages.
2. Unread messages: Unread messages display with a blue background.
3. Message link: Click the link to view the full message.
4. See more details: \{\{Commands.Click}} this option to view additional information related to the message in a new browser tab.
5. Mark as read checkbox: Select the checkbox for any unread messages to use the MARK AS READ option.
6. MARK AS READ: Select this option to mark any checked messages as read.
7. \{\{masterVAS-Terminology.R-ead-}} messages: Previously viewed messages display with a white background.
