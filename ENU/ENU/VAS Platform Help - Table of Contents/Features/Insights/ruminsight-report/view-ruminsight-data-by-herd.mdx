<ExpandableHeading>
  ### Required Permissions

  To use this feature, your \{\{General.ProductName}} user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](<../../../../ENU/VAS Platform Help - Table of Contents/features/manage-dairy-users-and-roles/edit-dairy-user-permissions.mdx>) and [Manage Dairy Roles](<../../../../../VAS Platform Help - Table of Contents/Features/Manage Dairy Users and Roles/manage-dairy-roles.mdx>) for more information.
</ExpandableHeading>

Click the **By&#x20;**\{\{masterVAS-Terminology.H-erd-}} tab on the RumINSIGHT report to view testing data for the entire \{\{masterVAS-Terminology.h-erd}}. The top of the page contains a summary of the data by \{\{masterVAS-Terminology.t-est-day}}. Beneath that are graphs showing the production trend of the three main fatty acid types (de novo, mixed, and preformed), as well as graphs regarding \{\{masterVAS-Terminology.h-erd}} fat and protein.

Expand the sections below to learn more:

<ExpandableHeading>
  ## Herd Summary Analysis Table

  The \{\{masterVAS-Terminology.H-erd-}}**&#x20;Summary Analysis** table shows a detailed breakdown of fatty acid test results arranged by \{\{masterVAS-Terminology.t-est-day}}.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/LbmPZbWAZdSs2Ir27nYlU_herdsummarytable.png" size="97" isUploading="false" initialPath="../../../Resources/Images/Platform/FattyAcidGroups/HerdSummaryTable.png" githubPath="Content/Resources/Images/Platform/FattyAcidGroups/HerdSummaryTable.png" position="flex-start" showCaption="false"></Image>

  Follow these guidelines when viewing the data in this table:

  - The \{\{masterVAS-Terminology.T-estday-}} column can be sorted in either ascending or descending order. (See [Testdays](<../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/testdays.mdx>) to learn more about \{\{masterVAS-Terminology.t-est-day}} data.)
  - The table displays summary data for all three types of fatty acid—de novo, mixed, and preformed.
  - Data values that are of concern (i.e., warnings) display in the table in red.
</ExpandableHeading>

<ExpandableHeading>
  ## Trend Graphs

  The trend graphs show the progress of production for each of the three types of fatty acids (de novo, mixed, and preformed) over a range of \{\{masterVAS-Terminology.t-est-days}}. An example of a **De Novo Trends** graph is shown below.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/-2o4-ZOoGnEOkNnwn3Vy8_denovograph.png" size="50" isUploading="false" initialPath="../../../Resources/Images/Platform/FattyAcidGroups/DeNovoGraph.png" githubPath="Content/Resources/Images/Platform/FattyAcidGroups/DeNovoGraph.png" position="flex-start" showCaption="false"></Image>

  Move your cursor over the \{\{masterVAS-Terminology.t-est-days}} in the graph to display precise information for each of the following:

  - \{\{Standard Deviation}} (displayed in dotted red)
  - \{\{masterVAS-Terminology.H-erd-}} Mean (displayed in dotted light blue)
  - \{\{masterVAS-Terminology.T-estday-}}\{\{masterVAS-Terminology.H-erd-}} Value (displayed in solid gold)
  - Benchmark (displayed in dotted green)

  <hint type="info">
    **NOTE:&#x20;**&#x42;enchmark values shown in the trend graphs are calculated using various values collected for the applicable month and are influenced by factors such as breed type and milking frequency. They provide a value against which the other values in the graph can be estimated and compared.
  </hint>

  In addition to the three fatty acid graphs, there is also a trend graph for \{\{masterVAS-Terminology.H-erd-}}**&#x20;Fat and Protein**. This graph works in the same manner as the other trend graphs and displays the following data:

  - \{\{masterVAS-Terminology.H-erd-}} Mean Fat % (displayed in dotted gold)
  - \{\{masterVAS-Terminology.H-erd-}}\{\{masterVAS-Terminology.T-estday-}} Fat (displayed in solid gold)
  - \{\{masterVAS-Terminology.H-erd-}} Mean Protein % (displayed in dotted light blue)
  - \{\{masterVAS-Terminology.H-erd-}}\{\{masterVAS-Terminology.T-estday-}} Protein % (displayed in solid light blue)
  - Benchmark Fat % (displayed in solid green)
  - Benchmark Protein % (displayed in solid dark blue)
</ExpandableHeading>

<ExpandableHeading>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/0fFya1WTV7lrO2UiKgJzC_herdfatgraph.png" size="42" isUploading="false" initialPath="../../../Resources/Images/Platform/FattyAcidGroups/HerdFatGraph.png" githubPath="Content/Resources/Images/Platform/FattyAcidGroups/HerdFatGraph.png" position="flex-start" showCaption="false"></Image>

</ExpandableHeading>
