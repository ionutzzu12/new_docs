The **Reproduction Codes** page allows you to view the descriptions for the reproductive status values assigned to animals in your \{\{masterVAS-Terminology.d-airy}}.\{\{masterVAS-Products.DCU-users-short}} can also edit the descriptions for the codes if they have the required permissions.   ## Edit Reproduction Code Descriptions

  <hint type="info">
    **NOTE:&#x20;**&#x4F;nly the descriptions of reproduction codes can be changed; the actual code numbers themselves cannot be altered. Additionally, reproduction codes cannot be deleted, nor can new reproduction codes be added.
  </hint>

  1. Click on the applicable reproduction code in the list to edit its **Description** field in the side panel.:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/tKmBjo6A_rArGu92BeQi4_dc-repro-codes-side-panel.png" alt caption size="63"}The description can be up to 30 characters long.
  2. Click **Save**.A message is displayed at the bottom of the page indicating that your changes have been saved.
</ExpandableHeading>
