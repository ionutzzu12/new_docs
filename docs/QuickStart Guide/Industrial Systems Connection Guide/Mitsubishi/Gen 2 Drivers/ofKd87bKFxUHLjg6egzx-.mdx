To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<hint type="info">
  **Note**: Only one connection is allowed per port defined in the PLC. See the steps below to view and edit the open ports on the PLC.
</hint>

If the device is configured for Ethernet access, find its network parameters. Otherwise, use PLC software on your computer to configure the device for Ethernet access.&#x20;

1. Open the PLC project in GXWorks 3.
2. From the Navigation panel, click **Ethernet Port&#xA0;**&#x74;o view the communication parameters. Note down the **IP Address**, **Subnet Mask**, **Default Gateway**, and **Communication Data Code**.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/z_GMbNgF_qKC9kS9laMs2_mitsubishi-fx5u-ethernet-binary-communication.png" size="80" width="1048" height="510" position="center" caption="Ethernet Port" showCaption="true"></Image>
3. Some CPUs have an **Enable/Disable Online Change&#x20;**&#x73;etting. To allow the driver to write tag values to the PLC, this setting needs to be set to **Enable All (SLMP)**.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/s2tgMv9KxlDyAahTP3jr5_image.png" size="80" width="1099" height="549" position="center" showCaption="false"></Image>
4. Double-click **External Device Configuration**.
5. View the available connections. If necessary, add more TCP connections to the SLMP Connection Module.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/PTKshBgq4vrjxR431qWrL_screenshot-2024-06-18-101802.png" size="80" width="950" height="559" position="center" showCaption="false"></Image>

<hint type="info">
  **Note:** If you modify any CPU or module parameters, you must power cycle the PLC after downloading the changes.
</hint>
