To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

# Step 1: Set up and Configure the PLC Device

<ContentSnippet docRefId="ZPTCmzo7MNjZIP6UvGpD8">

</ContentSnippet>

**To configure your device:**

1. Note down the following information and refer to it when needed.&#x20;
   - **Device**: Keyence KV-40DT
   - **Communication I/F**: RS232. Refer to the following figures.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/9nNvf2tfJmMHSfdZbJsXT_keyenceserial01.png" size="80" width="465" height="481" position="flex-start" caption="Keyence Serial reference" darkWidth="465" darkHeight="481" showCaption="true" indent="2"></Image>

2. Launch KV-Studio software.&#x20;
3. Select HMI and PLC Devices.
4. Set up the PLC.&#x20;
