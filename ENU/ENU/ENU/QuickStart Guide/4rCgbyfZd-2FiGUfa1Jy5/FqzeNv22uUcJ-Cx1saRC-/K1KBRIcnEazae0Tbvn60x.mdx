To set up and configure this device in Litmus Edge, you will need to do the following:

- Step 1: Set up and Configure the PLC Device
- Step 2: Set up the PLC Device in Litmus Edge DeviceHub
- Step 3: Configure the List of Registers

<hint type="info">
  **Note:** Incorrect tag configurations, unsupported tags, or tags missing from the PLC cause the device to cycle repeatedly between Connected and Disconnected statuses. A stable connection that all configured tags are valid, supported by the driver, and correctly mapped to the PLC.
</hint>

## Configure Device

Refer to the following configurations if the device doesn't connect right away.&#x20;

**To configure the device:**

1. Note down the following information and refer to it when needed. ****
   **Device**: Fanuc 0i-F****
   **Communication I/F**: Ethernet. Refer to the following figure.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/fwoBsmtOAexVXak7alS2T_01fanuc-cnc-ethernet.png" size="80" width="471" height="545" position="flex-start" caption="Fanuc CNC Ethernet reference" showCaption="true" indent="2"></Image>

2. Press **Soft key** five times to set up the IP Address.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/fwvMVtQf330-iyRTm01Vq_02fanuc-cnc-ethernet-ip482x338.png" size="80" width="482" height="338" position="flex-start" caption="IP Address set up screen" showCaption="true" indent="2"></Image>

3. Press the Soft key **Embed Port&#xA0;**&#x74;o set up the network parameters, and then press the Soft key **OPRT**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/LPmF1PLNNS_a0oZ7-ByJM_03fanuc-cnc-ethernet-ip482x338.png" size="80" width="535" height="386" position="flex-start" caption="Set up network parameters screen" showCaption="true" indent="2"></Image>

4. Press the Soft key **Restart&#xA0;**&#x74;o activate the settings.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/TBGmJBCz6DUz3H9rvKBj2_04fanuc-cnc-ethernet-settings.png" size="80" width="624" height="372" position="flex-start" caption="Activate settings screen [1]" showCaption="true" indent="2"></Image>

5. Press the Soft key **Execute&#xA0;**&#x74;o activate the settings.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/OdVxfjpyDOFcc9_GS-9-R_05fanuc-cnc-ethernet-activate.png" size="80" width="624" height="392" position="flex-start" caption="Activate settings screen [2]" showCaption="true" indent="2"></Image>

6. Press the Soft key two times to reach this page, and then press the Soft key **FOCAS2**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/yKdznXawbByL6y0g2gJxp_06fanuc-cnc-ethernet-focas2.png" size="80" width="606" height="406" position="flex-start" caption="FOCAS2 button reference" showCaption="true" indent="2"></Image>

7. Set up the port number, and then press the Soft key **OPRT**.

<Image src="https://app.archbee.com/api/optimize/SSUUxKZUk9bFTEPNn_6Zo/ccr3yWUc7OHaoNw4yGRI__07fanuc-cnc-ethernet-port.png" size="80" width="695" height="421" position="flex-start" caption="Port setup screen" showCaption="true" indent="2"></Image>
