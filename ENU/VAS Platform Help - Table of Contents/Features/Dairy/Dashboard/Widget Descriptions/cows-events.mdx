<ExpandableHeading>
  ### What permissions do I need to view this widget?

  To view this widget on a Dashboard or \{\{masterVAS-Terminology.O-verview-}} Page, your user account must be assigned to a Role with \{\{RolesPermissions.PermissionNames}} permissions. See [Edit Dairy User Permissions](../../Config/UserMgmtRoles/EditUserPermissions.htm) and [Manage Dairy Roles](../../Config/UserMgmtRoles/ManageRoles.htm) for more information.
</ExpandableHeading>

The \{\{Widgets.Events}} widget displays all the events that have occurred for cows for the last 7 days. This widget reflects data only for cows; there is another widget that shows [this data for heifers](21-dayPregRisk-Heifer.htm).

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/QjalyvC_Eyr95bGir3p2_/JSYKWl3q6KqYMxcTfU_FT_events-cows-widget.png" size="63" isUploading="false" initialPath="../../../../Resources/Images/Platform/dashboard/events-cows-widget.png" githubPath="Content/Resources/Images/Platform/dashboard/events-cows-widget.png" position="flex-start" showCaption="false"></Image>

Follow these guidelines when reviewing the data in this widget:

- The left side of the chart displays each type of event that has occurred over the last 7 days.
- The days of the week appear across the top of the chart. Beneath each day is a breakdown of events for the day.
- Totals for each type of event, as well as the total for all events, display across the bottom of the chart.
