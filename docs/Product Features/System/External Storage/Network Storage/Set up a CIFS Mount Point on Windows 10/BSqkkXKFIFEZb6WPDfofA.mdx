Configuring the network path manually makes it easy to have a shared folder be stored anywhere on Windows 10. If you want your shared folder on the Desktop, for example, it is best to set a custom network path so it is easier to remember.

<hint type="info">
  **Note**: When accessing the folder from Litmus Edge, or elsewhere on the network, make sure that you use the full network path of the shared folder. You can find this information in folder properties and configure a custom network path with *Advanced Sharing*.
</hint>

**To view and configure a shared folder network path in Windows 10:**

1. Create a folder that will be used for sharing on the network. In this example, the folder is *MyFolder*.&#x20;
2. Right-click the folder, then click **Properties**. Here you can see the full network path to use if you want to access the shared folder through the network. Make sure to share the folder with the **Share** button, if you have not already done so.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/9sGhY8dOYoDKuj0sxxFCH_myfolderproperties1.png" size="50" width="386" height="475" caption="Folder properties dialog box" position="center" showCaption="true"></Image>
3. Click **Advanced Sharing** to set a custom network path.
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/9UYIDFgViwYhzo7xp938g_advancedsharing.png" size="50" width="353" height="363" caption="Advanced Sharing dialog box" position="center" showCaption="true"></Image>
4. **Share this folder** if it is not already checked.
5. Click **Apply**, then click **OK**.&#x20;
   You can see the updated network path in the folder properties. It should be much shorter and more usable.
