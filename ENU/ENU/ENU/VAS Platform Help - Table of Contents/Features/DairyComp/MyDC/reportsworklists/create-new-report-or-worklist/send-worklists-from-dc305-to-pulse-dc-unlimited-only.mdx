Based on how \{\{DC-UI.DC-ProdNameShort}} creates each type of list, the actions you must take to send them to the \{\{General.ProductName-short}} varies.

- Send \{\{Worklists.HospitalListUI-Web}} to \{\{General.ProductName-short}}Use \{\{masterVAS-Products.Pulse-only}} or \{\{DC-UI.DC-ProdNameShort}} to create your \{\{Worklists.HospListCommonUse}}s with a special type of LIST command. For \{\{DC-UI.DC-ProdNameShort}} lists, adding the \HK switch to the commands sends them to \{\{masterVAS-Products.Pulse-only}}. Most \{\{masterVAS-Terminology.d-airies}} schedule \{\{Worklists.HospListCommonUse}}s to run daily to ensure treatment protocols are properly followed, as drugs are usually administered on a daily basis.You can then process your lists in the mobile app and [monitor them in the web version of PULSE](<../../../../../../../../VAS Platform Help - Table of Contents/Mobile Apps/VAS Platform Mobile App (v1)/Tasks & Worklists (v1)/view-animal-status-in-the-mobile-app-v1.mdx>). See the topics below to learn more:\* [Learn More about Hospital Lists](HospLists.htm)
  - [Learn More about Vet Lists](<../../../../../../../../VAS Platform Help - Table of Contents/Features/DairyComp/MyDC/Reports/Worklists/learn-more-about-vet-lists.mdx>)
  - [Complete Tasks (v1)](<../../../../../../../../VAS Platform Help - Table of Contents/Mobile Apps/VAS Platform Mobile App (v1)/Tasks & Worklists (v1)/complete-tasks-v1.mdx>)
  - [Complete Tasks (v2)](<../../../../../../../../VAS Platform Help - Table of Contents/Mobile Apps/DC on Mobile (v2)/complete-tasks-v2.mdx>)
- Send \{\{Worklists.ToDoUI-Web}} Lists to \{\{General.ProductName-short}}: \{\{Worklists.ToDoUI-Web}} lists are very flexible and can be created from \{\{masterVAS-Products.Pulse-only}} or \{\{DC-UI.DC-ProdNameShort}}. For \{\{DC-UI.DC-ProdNameShort}} lists, adding the \K switch to the commands sends them to \{\{masterVAS-Products.Pulse-only}}. See [Learn More about To-Do Lists](ToDoLists.htm) for more information.
- Send \{\{Worklists.VetListUI-Web}} to \{\{General.ProductName-short}}: Use \{\{masterVAS-Products.Pulse-only}} or \{\{DC-UI.DC-ProdNameShort}} to create your \{\{Worklists.VetListCommonUse}}s with a special VETLIST command. For \{\{DC-UI.DC-ProdNameShort}} lists, adding the \K switch to the commands sends them to \{\{masterVAS-Products.Pulse-only}}. The flexibility of the system allows you to schedule the \{\{Worklists.VetListCommonUse}}s to generate on days the veterinarian is scheduled to visit, or to generate and send them on an as-needed basis.
