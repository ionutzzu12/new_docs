You can add a new role customized with user permissions by navigating to **System&#x20;**>**&#x20;Access Control&#x20;**&#x66;rom the **Users&#x20;**&#x74;ab.

Learn more about [Role Permissions](../Users/pbvGMbHUfDXL8ThqJL8Pg.mdx).&#x20;

**To add a role**:

1. Navigate to **System&#x20;***>&#x20;***Access Control**.&#x20;
2. From the *Manage Roles* section, click the **Add Permissions (+)** icon.
   The *Add Role Details* dialog box appears.
   ![](https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/ScgL6HSJ4A5YkqrK4qB_k_add-role.png)
3. Enter a name for the role.
4. Under *Permissions*, select the check boxes for each permission the role will receive. Based on your selection, additional permissions may be automatically added to ensure proper functionality. See [Role Permissions](../Users/pbvGMbHUfDXL8ThqJL8Pg.mdx) for more information.&#x20;
   <Image src="https://archbee-image-uploads.s3.amazonaws.com/SSUUxKZUk9bFTEPNn_6Zo/O3YdW38kS6eJXHUqWL_uD_addroledetailsdialogbox.png" size="80" width="1171" height="772" caption="Add Role Details dialog box" position="center" showCaption="true"></Image>
5. When done, click **Add Role Details**.
   The role is added to the *Roles&#x20;*&#x74;ab.
